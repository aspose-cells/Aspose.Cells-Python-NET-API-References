##  ControlMatchEntryType перечисление
Показывает, как ListBox или ComboBox выполняет поиск в списке по мере ввода данных пользователем.



Тип ControlMatchEntryType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| FIRST_LETTER | Элемент управления ищет следующую запись, начинающуюся с введенного символа.<br/> Повторное нажатие одной и той же буквы циклически перебирает все записи, начинающиеся с этой буквы.|
| COMPLETE | При вводе каждого символа элемент управления ищет запись, соответствующую всем введенным символам.|
| NONE | При вводе символов поиск в списке производиться не будет.|
