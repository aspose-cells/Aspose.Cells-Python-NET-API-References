##  ControlMatchEntryType перечисление
Представляет, как ListBox или ComboBox выполняет поиск в своем списке по мере ввода пользователем.



Тип ControlMatchEntryType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| FIRST_LETTER | Система управления ищет следующую запись, которая начинается с введенного символа.<br/> Повторный ввод одной и той же буквы приводит к циклическому просмотру всех записей, начинающихся с этой буквы.|
| COMPLETE | По мере ввода каждого символа элемент управления ищет запись, соответствующую всем введенным символам.|
| NONE | Список не будет искаться при вводе символов.|
