##  ControlPicturePositionType перечисление
Представляет расположение изображения элемента управления относительно его подписи.



Тип ControlPicturePositionType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| LEFT_TOP | Картинка находится слева от подписи.<br/> Подпись выровнена по верхнему краю изображения.|
| LEFT_CENTER | Картинка находится слева от подписи.<br/> Подпись выравнивается по центру относительно изображения.|
| LEFT_BOTTOM | Картинка находится слева от подписи.<br/>Подпись выровнена по нижнему краю изображения.|
| RIGHT_TOP | Картинка находится справа от подписи.<br/> Подпись выровнена по верхнему краю изображения.|
| RIGHT_CENTER | Картинка находится справа от подписи.<br/> Подпись выравнивается по центру относительно изображения.|
| RIGHT_BOTTOM | Картинка находится справа от подписи.<br/>Подпись выровнена по нижнему краю изображения.|
| ABOVE_LEFT | Картинка отображается над подписью.<br/> Подпись выравнивается по левому краю рисунка.|
| ABOVE_CENTER | Картинка отображается над подписью.<br/> Подпись располагается по центру под изображением.|
| ABOVE_RIGHT | Картинка отображается над подписью.<br/> Подпись выравнивается по правому краю изображения.|
| BELOW_LEFT | Картинка находится под подписью.<br/> Подпись выравнивается по левому краю рисунка.|
| BELOW_CENTER | Картинка находится под подписью.<br/> Подпись располагается по центру над изображением.|
| BELOW_RIGHT | Картинка находится под подписью.<br/> Подпись выравнивается по правому краю изображения.|
| CENTER | Картинка появляется в центре элемента управления.<br/> Подпись располагается по центру изображения по горизонтали и вертикали.|
