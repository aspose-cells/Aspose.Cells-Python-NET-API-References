##  ControlPicturePositionType перечисление
Представляет расположение изображения элемента управления относительно его заголовка.



Тип ControlPicturePositionType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| LEFT_TOP | Изображение появляется слева от подписи.<br/> Подпись выравнивается по верхнему краю изображения.|
| LEFT_CENTER | Изображение появляется слева от подписи.<br/> Надпись располагается по центру относительно изображения.|
| LEFT_BOTTOM | Изображение появляется слева от подписи.<br/> Подпись выравнивается по нижнему краю изображения.|
| RIGHT_TOP | Изображение появляется справа от подписи.<br/> Подпись выравнивается по верхнему краю изображения.|
| RIGHT_CENTER | Изображение появляется справа от подписи.<br/> Надпись располагается по центру относительно изображения.|
| RIGHT_BOTTOM | Изображение появляется справа от подписи.<br/> Подпись выравнивается по нижнему краю изображения.|
| ABOVE_LEFT | Изображение появляется над подписью.<br/> Подпись выравнивается по левому краю изображения.|
| ABOVE_CENTER | Изображение появляется над подписью.<br/> Подпись по центру под картинкой.|
| ABOVE_RIGHT | Изображение появляется над подписью.<br/> Подпись выравнивается по правому краю изображения.|
| BELOW_LEFT | Изображение появляется под заголовком.<br/> Подпись выравнивается по левому краю изображения.|
| BELOW_CENTER | Изображение появляется под заголовком.<br/>Подпись располагается по центру над картинкой.|
| BELOW_RIGHT | Изображение появляется под заголовком.<br/> Подпись выравнивается по правому краю изображения.|
| CENTER | Изображение появляется в центре элемента управления.<br/> Подпись располагается по центру по горизонтали и вертикали в верхней части изображения.|
