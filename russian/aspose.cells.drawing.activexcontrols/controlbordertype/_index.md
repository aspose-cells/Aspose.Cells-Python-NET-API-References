##  ControlBorderType перечисление
Представляет тип границы элемента управления ActiveX.



Тип ControlBorderType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Без границ.|
| SINGLE | Единая линия.|
