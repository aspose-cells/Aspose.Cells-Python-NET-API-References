##  ControlBorderType перечисление
Представляет тип границы элемента управления ActiveX.



Тип ControlBorderType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Границы нет.|
| SINGLE | Одна линия.|
