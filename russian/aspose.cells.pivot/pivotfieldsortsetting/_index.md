##  PivotFieldSortSetting класс
Представляет собой настройку сортировки полей сводки.



Тип PivotFieldSortSetting предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [sort_type](/cells/python-net/ru/aspose.cells.pivot/pivotfieldsortsetting/sort_type) | Представляет собой [`SortOrder`](/cells/python-net/ru/aspose.cells/sortorder).|
| [is_sort_by_labels](/cells/python-net/ru/aspose.cells.pivot/pivotfieldsortsetting/is_sort_by_labels) | Указывает, сортируется ли само поле или поле данных.|
| [field_index](/cells/python-net/ru/aspose.cells.pivot/pivotfieldsortsetting/field_index) | Представляет индекс поля, по которому выполняется сортировка.<br/> -1 означает сортировку PivotField по меткам, другие означают сортировку по полю данных.|
| [line_type_sorted_by](/cells/python-net/ru/aspose.cells.pivot/pivotfieldsortsetting/line_type_sorted_by) | Тип опорной линии, по которому производится сортировка.|
| [is_simple_sort](/cells/python-net/ru/aspose.cells.pivot/pivotfieldsortsetting/is_simple_sort) | Указывает, будет ли применяться простая операция сортировки данных.|
| [cell](/cells/python-net/ru/aspose.cells.pivot/pivotfieldsortsetting/cell) | Сортирует по значениям в строке или столбце.|
