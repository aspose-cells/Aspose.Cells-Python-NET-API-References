##  PivotFieldSubtotalType перечисление
Краткое описание для PivotFieldSubtotalType.



Тип PivotFieldSubtotalType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Представляет собой тип промежуточного итога None.|
| AUTOMATIC | Представляет тип автоматического промежуточного итога.|
| SUM | Представляет тип промежуточного итога суммы.|
| COUNT | Представляет собой тип промежуточного итога «Количество».|
| AVERAGE | Представляет собой тип среднего промежуточного итога.|
| MAX | Представляет тип максимального промежуточного итога.|
| MIN | Представляет собой тип минимального промежуточного итога.|
| PRODUCT | Представляет собой тип промежуточного итога продукта.|
| COUNT_NUMS | Представляет собой тип промежуточного итога Count Nums.|
| STDEV | Представляет тип промежуточного итога Stdev.|
| STDEVP | Представляет тип промежуточного итога Stdevp.|
| VAR | Представляет собой тип промежуточного итога Var.|
| VARP | Представляет собой тип промежуточного итога Varp.|
