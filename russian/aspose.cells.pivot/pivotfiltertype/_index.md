##  PivotFilterType перечисление
Представляет тип фильтра сводной таблицы.



Тип PivotFilterType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| CAPTION_BEGINS_WITH | Указывает фильтр «начинается с» для заголовков полей.|
| CAPTION_BETWEEN | Указывает фильтр «между» для заголовков полей.|
| CAPTION_CONTAINS |Указывает фильтр «содержит» для заголовков полей.|
| CAPTION_ENDS_WITH | Указывает фильтр «заканчивается на» для заголовков полей.|
| CAPTION_EQUAL | Указывает фильтр «равно» для заголовков полей.|
| CAPTION_GREATER_THAN | Указывает фильтр «больше чем» для заголовков полей.|
| CAPTION_GREATER_THAN_OR_EQUAL | Указывает фильтр «больше или равно» для заголовков полей.|
| CAPTION_LESS_THAN | Указывает фильтр «меньше чем» для заголовков полей.|
| CAPTION_LESS_THAN_OR_EQUAL | Указывает фильтр «меньше или равно» для заголовков полей.|
| CAPTION_NOT_BEGINS_WITH | Указывает фильтр «не начинается с» для заголовков полей.|
