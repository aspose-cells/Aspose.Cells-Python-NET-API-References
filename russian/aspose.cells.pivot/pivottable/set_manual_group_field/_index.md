##  set_manual_group_field {#int-float-float-list-float}
Устанавливает группу полей вручную с помощью сводной таблицы.



```python
def set_manual_group_field(self, base_field_index, start_val, end_val, group_by_list, interval_num):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| base_field_index | int | Индекс поля строки или столбца в базовых полях|
| start_val | float | Указывает начальное значение для числовой группировки.|
| end_val | float | Указывает конечное значение для числовой группировки.|
| group_by_list | list | Указывает список типов группировки. Задается типом PivotTableGroupType.|
| interval_num | float | Указывает группу номеров интервалов посредством числовой группировки.|

используйте метод PivotField.GroupBy().
