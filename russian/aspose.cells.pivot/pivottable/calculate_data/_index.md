##  calculate_data() {#}
Вычисляет данные сводной таблицы в ячейки.



```python
def calculate_data(self):
    ...
```


###  Примечания

Cell. Значение в диапазоне поворота не может вернуть правильный результат, если метод не был вызван.
Этот метод вычисляет данные с помощью внутреннего сводного кеша, а не исходного источника данных.
Поэтому, если источник данных изменен, сначала вызовите метод RefreshData().
