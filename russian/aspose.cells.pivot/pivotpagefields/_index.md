##  PivotPageFields класс
 Представляет элементы поля сводной страницы.
если источником данных сводной таблицы являются диапазоны консолидации.
Он может содержать не более 4 полей.



Тип PivotPageFields предоставляет следующие члены:

###  Конструкторы
| Конструктор| Описание|
| :- | :- |
| [PivotPageFields()](/cells/python-net/ru/aspose.cells.pivot/pivotpagefields/__init__/#) | Представляет элементы поля сводной страницы.|


###  Характеристики
| Свойство| Описание|
| :- | :- |
| [page_field_count](/cells/python-net/ru/aspose.cells.pivot/pivotpagefields/page_field_count) | Получает количество полей страницы.|


###  Методы
| Метод| Описание|
| :- | :- |
| [add_page_field(page_items)](/cells/python-net/ru/aspose.cells.pivot/pivotpagefields/add_page_field/#list) | Добавляет поле страницы.|
| [add_identify(range_index, page_item_index)](/cells/python-net/ru/aspose.cells.pivot/pivotpagefields/add_identify/#int-list) | Устанавливает, какую метку элемента в каждом поле страницы использовать для идентификации диапазона данных.<br/> Значение pageItemIndex.Length должно быть равно PageFieldCount, поэтому сначала добавьте поле страницы.|
