##  PivotItemPositionType перечисление
Представляет тип позиции базового элемента сводной таблицы в базовом поле при использовании вычисления ShowDataAs.



Тип PivotItemPositionType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| PREVIOUS | Представляет предыдущий элемент сводной таблицы в PivotField.|
| NEXT | Представляет следующий элемент поворота в PivotField.|
| CUSTOM | Показывает значения в другом формате на основе индекса элемента сводной таблицы в PivotField.|
