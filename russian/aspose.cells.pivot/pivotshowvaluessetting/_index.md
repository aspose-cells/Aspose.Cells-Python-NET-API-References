##  PivotShowValuesSetting класс
Представляет настройки отображения значений при использовании расчета ShowDataAs.



Тип PivotShowValuesSetting предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [calculation_type](/cells/python-net/ru/aspose.cells.pivot/pivotshowvaluessetting/calculation_type) | Показывает, как отображать значения поля данных в сводном отчете.|
| [base_field_index](/cells/python-net/ru/aspose.cells.pivot/pivotshowvaluessetting/base_field_index) | Представляет базовое поле для расчета ShowDataAs, когда используется расчет ShowDataAs.|
| [base_item_position_type](/cells/python-net/ru/aspose.cells.pivot/pivotshowvaluessetting/base_item_position_type) | Представляет тип базового элемента сводки в базовом поле при использовании вычисления ShowDataAs.<br/> Действительно только для полей данных.<br/>Поскольку PivotItemPosition.Custom предназначен только для чтения, если вам нужно установить PivotItemPosition.Custom,<br/> пожалуйста, установите атрибут PivotField.BaseItemIndex.|
| [base_item_index](/cells/python-net/ru/aspose.cells.pivot/pivotshowvaluessetting/base_item_index) | Представляет пользовательский индекс элемента сводки в базовом поле при использовании вычисления ShowDataAs.<br/> Действительно только для полей данных.|
