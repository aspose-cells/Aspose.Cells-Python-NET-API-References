##  PivotTableFormatCollection класс
Представляет коллекцию форматов, применяемых к сводной таблице.



Тип PivotTableFormatCollection предоставляет следующие элементы:

| Свойство| Описание|
| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells.pivot/pivottableformatcollection/capacity) | Возвращает или задает количество элементов, которые может содержать список массива.|
