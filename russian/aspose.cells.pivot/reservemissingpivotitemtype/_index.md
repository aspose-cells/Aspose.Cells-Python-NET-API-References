##  ReserveMissingPivotItemType перечисление
Показывает, как сохранить недостающие опорные элементы.



Тип ReserveMissingPivotItemType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| DEFAULT | Удаляет старые отсутствующие элементы сводной таблицы и резервирует видимые элементы, которых нет в текущем источнике данных, как отсутствующие элементы.|
| ALL | Резервирует все недостающие предметы.|
| NONE | Удаляет все отсутствующие опорные элементы.|
