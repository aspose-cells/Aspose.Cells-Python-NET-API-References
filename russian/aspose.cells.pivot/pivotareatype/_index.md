##  PivotAreaType перечисление
Указывает тип правила, используемого для описания области или аспекта сводной таблицы.



Тип PivotAreaType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Нет области вращения.|
| NORMAL | Представляет заголовок или элемент.|
| DATA | Представляет что-то в области данных.|
| ALL | Представляет всю сводную таблицу.|
| ORIGIN | Представляет пустые ячейки в левом верхнем углу сводной таблицы (вверху справа для листов с письмом справа налево).|
| BUTTON | Представляет кнопку поля.|
| TOP_RIGHT | Представляет пустые ячейки в правом верхнем углу сводной таблицы (в верхнем левом углу для листов с письмом справа налево).|
