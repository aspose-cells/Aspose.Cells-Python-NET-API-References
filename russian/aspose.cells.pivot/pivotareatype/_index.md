##  PivotAreaType перечисление
Указывает тип правила, используемого для описания области или аспекта сводной таблицы.



Тип PivotAreaType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Нет поворотной зоны.|
| NORMAL | Представляет заголовок или элемент.|
| DATA | Представляет что-либо в области данных.|
| ALL | Представляет всю сводную таблицу.|
| ORIGIN | Представляет пустые ячейки в левом верхнем углу сводной таблицы (в правом верхнем углу для листов RTL).|
| BUTTON |Представляет собой кнопку поля.|
| TOP_RIGHT | Представляет собой пустые ячейки в правом верхнем углу сводной таблицы (в левом верхнем углу для листов RTL).|
