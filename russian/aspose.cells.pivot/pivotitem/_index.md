##  PivotItem класс
Представляет элемент в отчете PivotField.



Тип PivotItem предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [is_hidden](/cells/python-net/ru/aspose.cells.pivot/pivotitem/is_hidden) | Получает и устанавливает, скрыт ли сводный элемент.|
| [is_hide_detail](/cells/python-net/ru/aspose.cells.pivot/pivotitem/is_hide_detail) | Получает и задает, скрывает ли элемент сводки детали.|
| [is_missing](/cells/python-net/ru/aspose.cells.pivot/pivotitem/is_missing) | Указывает, имеет ли элемент пропущенное значение.|
| [value](/cells/python-net/ru/aspose.cells.pivot/pivotitem/value) | Получает значение элемента сводной таблицы|
| [name](/cells/python-net/ru/aspose.cells.pivot/pivotitem/name) | Получает имя элемента сводной таблицы.|
| [index](/cells/python-net/ru/aspose.cells.pivot/pivotitem/index) | Получает индекс элемента сводной таблицы в поле сводной таблицы.|


###  Методы
| Метод| Описание|
| :- | :- |
| [move](/cells/python-net/ru/aspose.cells.pivot/pivotitem/move/#int-bool) | Перемещает элемент вверх или вниз|
| [get_string_value](/cells/python-net/ru/aspose.cells.pivot/pivotitem/get_string_value/#) | Получает строковое значение элемента сводной таблицы.<br/> Если значение равно нулю, оно вернет ""|
| [get_double_value](/cells/python-net/ru/aspose.cells.pivot/pivotitem/get_double_value/#) | Получает двойное значение элемента сводной таблицы.<br/> Если значение равно нулю или не является числом, оно вернет 0|
| [get_date_time_value](/cells/python-net/ru/aspose.cells.pivot/pivotitem/get_date_time_value/#) |Получает значение даты и времени сводного элемента.<br/> Если значение равно нулю, оно вернет DateTime.MinValue.|
