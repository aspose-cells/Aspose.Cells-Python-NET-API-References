##  PivotItem класс
Представляет элемент в отчете PivotField.



Тип PivotItem предоставляет следующие члены:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [is_hidden](/cells/python-net/ru/aspose.cells.pivot/pivotitem/is_hidden) | Получает и задает, скрыт ли элемент сводки.|
| [is_hide_detail](/cells/python-net/ru/aspose.cells.pivot/pivotitem/is_hide_detail) | Получает и задает, скрывает ли элемент сводки детали.|
| [value](/cells/python-net/ru/aspose.cells.pivot/pivotitem/value) | Получает значение элемента сводки|
| [name](/cells/python-net/ru/aspose.cells.pivot/pivotitem/name) | Получает имя элемента сводки.|
| [index](/cells/python-net/ru/aspose.cells.pivot/pivotitem/index) | Получает индекс элемента сводки в поле сводки|


###  Методы
| Метод| Описание|
| :- | :- |
| [move(count, is_same_parent)](/cells/python-net/ru/aspose.cells.pivot/pivotitem/move/#int-bool) | Перемещает элемент вверх или вниз|
| [get_string_value()](/cells/python-net/ru/aspose.cells.pivot/pivotitem/get_string_value/#) | Получает строковое значение элемента сводки<br/> Если значение равно null, возвращается ""|
| [get_double_value()](/cells/python-net/ru/aspose.cells.pivot/pivotitem/get_double_value/#) | Получает двойное значение элемента сводки<br/> Если значение равно null или не является числом, оно вернет 0|
| [get_date_time_value()](/cells/python-net/ru/aspose.cells.pivot/pivotitem/get_date_time_value/#) | Получает значение даты и времени элемента сводки<br/> Если значение равно null, оно вернет DateTime.MinValue.|
