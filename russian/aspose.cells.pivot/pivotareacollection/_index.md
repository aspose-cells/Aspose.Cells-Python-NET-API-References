##  PivotAreaCollection класс
Представляет собой список опорных областей.



Тип PivotAreaCollection предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells.pivot/pivotareacollection/capacity) | Возвращает или задает количество элементов, которые может содержать список массива.|
