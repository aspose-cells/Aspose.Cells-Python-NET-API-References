##  SxRng класс
Представляет диапазон группы в PivotField.



Тип SxRng предоставляет следующие члены:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [is_auto_start](/cells/python-net/ru/aspose.cells.pivot/sxrng/is_auto_start) | Указывает логическое значение, указывающее, будет ли приложение использовать исходные данные для установки значения начального диапазона.|
| [is_auto_end](/cells/python-net/ru/aspose.cells.pivot/sxrng/is_auto_end) | Указывает логическое значение, указывающее, будет ли приложение использовать исходные данные для установки конечного значения диапазона.|
| [start](/cells/python-net/ru/aspose.cells.pivot/sxrng/start) | Представляет начальный объект для диапазона группы.|
| [end](/cells/python-net/ru/aspose.cells.pivot/sxrng/end) | Представляет конечный объект для диапазона группы.|
| [by](/cells/python-net/ru/aspose.cells.pivot/sxrng/by) | Представляет объект интервала для диапазона группы.|
