##  PivotFieldType перечисление
Представляет тип поля сводной таблицы.



Тип PivotFieldType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| UNDEFINED | Представляет базовый тип сводного поля.|
| ROW | Представляет тип сводного поля строки.|
| COLUMN | Представляет тип сводного поля столбца.|
| PAGE | Представляет тип сводного поля страницы.|
| DATA | Представляет тип поля сводки данных.|
