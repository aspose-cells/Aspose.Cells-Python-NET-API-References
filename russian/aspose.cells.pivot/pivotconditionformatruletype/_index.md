##  PivotConditionFormatRuleType перечисление
Представляет тип правила форматирования условий сводной таблицы.



Тип PivotConditionFormatRuleType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Указывает, что условное форматирование Top N не оценивается.|
| ALL | Указывает, что условное форматирование Top N<br/> оценивается по всему диапазону охвата.|
| ROW | Указывает, что условное форматирование Top N оценивается для каждой строки.|
| COLUMN | Указывает, что условное форматирование Top N<br/> оценивается для каждого столбца.|
