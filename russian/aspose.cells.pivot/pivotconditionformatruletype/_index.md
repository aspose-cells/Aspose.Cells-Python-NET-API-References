##  PivotConditionFormatRuleType перечисление
Представляет тип правила форматирования условия сводной таблицы.



Тип PivotConditionFormatRuleType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Указывает, что условное форматирование Top N не оценивается|
| ALL | Указывает, что условное форматирование первых N<br/> оценивается во всем диапазоне охвата.|
| ROW | Указывает, что условное форматирование Top N оценивается для каждой строки.|
| COLUMN | Указывает, что условное форматирование первых N<br/> оценивается для каждого столбца.|
