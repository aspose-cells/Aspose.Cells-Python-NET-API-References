##  ConnectionParameterType перечисление
Указывает тип параметра внешнего соединения



Тип ConnectionParameterType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| CELL | Получите значение параметра из ячейки при каждом обновлении.|
| PROMPT | При каждом обновлении запрашивать у пользователя значение параметра.|
| VALUE | Используйте постоянное значение при каждом обновлении для значения параметра.|
