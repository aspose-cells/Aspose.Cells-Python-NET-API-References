##  ReConnectionMethodType перечисление
Указывает, что приложение для работы с электронными таблицами должно делать при сбое соединения.



Тип ReConnectionMethodType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| REQUIRED | При обновлении используйте существующую информацию о подключении, и если она окажется недействительной<br/> затем получите обновленную информацию о подключении, если она доступна из внешнего файла подключений.|
| ALWAYS | При каждом обновлении получайте обновленную информацию о подключении из внешнего файла подключений,<br/> если доступно, и использовать его вместо существующей информации о подключении.<br/>В этом случае обновление данных не будет выполнено, если файл внешнего соединения недоступен.|
| NEVER | Никогда не получайте обновленную информацию о подключении из внешнего файла подключений<br/> даже если он доступен и даже если существующая информация о подключении недействительна|
