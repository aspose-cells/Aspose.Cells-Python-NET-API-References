##  HtmlFormatHandlingType перечисление
Указывает, как обрабатывать форматирование из источника HTML.



Тип HtmlFormatHandlingType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| ALL | Перенесите все форматирование HTML на рабочий лист вместе с данными.|
| NONE | Введите данные в виде неформатированного текста (настройка типов данных все равно выполняется).|
| RTF |Переведите форматирование HTML в форматирование расширенного текста для данных, загруженных на рабочий лист.|
