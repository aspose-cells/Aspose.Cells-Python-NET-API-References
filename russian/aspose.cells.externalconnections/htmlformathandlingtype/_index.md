##  HtmlFormatHandlingType перечисление
Указывает, как обрабатывать форматирование из источника HTML.



Тип HtmlFormatHandlingType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| ALL | Перенесите все форматирование HTML на лист вместе с данными.|
| NONE | Внесите данные в виде неформатированного текста (установка типов данных все еще происходит).|
| RTF | Преобразуйте форматирование HTML в форматирование расширенного текста для данных, введенных на лист.|
