##  command_type недвижимость

 Указывает тип команды OLE DB.
1. Запрос указывает имя куба
2. Запрос определяет оператор SQL.
3. Запрос указывает имя таблицы.
4. Запрос указывает, что предоставлена информация по умолчанию, и поставщик должен сам решить, как ее интерпретировать.
5. Запрос выполняется через веб-поставщика данных списков.
