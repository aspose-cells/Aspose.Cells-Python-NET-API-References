**Aspose.Cells.ExternalConnections**Пространство имен предоставляет классы для внешних подключений с целью извлечения данных из внешних источников, таких как базы данных и Интернет.

###  Классы
| Сорт| Описание|
| :- | :- |
| [`ConnectionParameter`](/cells/python-net/ru/aspose.cells.externalconnections/connectionparameter) | Указывает свойства любых параметров, используемых с внешними подключениями к данным.<br/> Параметры действительны для ODBC и веб-запросов.|
| [`ConnectionParameterCollection`](/cells/python-net/ru/aspose.cells.externalconnections/connectionparametercollection) | Определяет коллекцию [`ConnectionParameter`](/cells/python-net/ru/aspose.cells.externalconnections/connectionparameter)|
| [`DBConnection`](/cells/python-net/ru/aspose.cells.externalconnections/dbconnection) | Указывает все свойства, связанные с подключением к внешним данным ODBC или OLE DB.|
| [`DataModelConnection`](/cells/python-net/ru/aspose.cells.externalconnections/datamodelconnection) | Указывает соединение с моделью данных|
| [`ExternalConnectionCollection`](/cells/python-net/ru/aspose.cells.externalconnections/externalconnectioncollection) | Определяет коллекцию ExternalConnection|
| [`WebQueryConnection`](/cells/python-net/ru/aspose.cells.externalconnections/webqueryconnection) | Задаёт свойства источника веб-запроса. Веб-запрос будет извлекать данные из таблиц HTML.<br/> и также может предоставить параметры HTTP "Get", которые будут обработаны веб-сервером при создании HTML<br/> включая параметры и элементы параметров.|
