**Aspose.Cells.ExternalConnections** namespace предоставляет классы для внешних подключений для извлечения данных из внешних источников, таких как базы данных и Интернет.

###  Классы
| Сорт| Описание|
| :- | :- |
| [ConnectionParameter](/cells/python-net/ru/aspose.cells.externalconnections/connectionparameter) | Указывает свойства любых параметров, используемых с внешними подключениями к данным.<br/> Параметры действительны для ODBC и веб-запросов.|
| [ConnectionParameterCollection](/cells/python-net/ru/aspose.cells.externalconnections/connectionparametercollection) | Указывает коллекцию [ConnectionParameter](/cells/python-net/ru/aspose.cells.externalconnections/connectionparameter)|
| [DBConnection](/cells/python-net/ru/aspose.cells.externalconnections/dbconnection) | Задает все свойства, связанные с подключением к внешним данным ODBC или OLE DB.|
| [DataModelConnection](/cells/python-net/ru/aspose.cells.externalconnections/datamodelconnection) | Указывает подключение к модели данных|
| [ExternalConnection](/cells/python-net/ru/aspose.cells.externalconnections/externalconnection) | Указывает внешнее подключение для передачи данных|
| [ExternalConnectionCollection](/cells/python-net/ru/aspose.cells.externalconnections/externalconnectioncollection) | Указывает коллекцию [ExternalConnection](/cells/python-net/ru/aspose.cells.externalconnections/externalconnection)|
| [WebQueryConnection](/cells/python-net/ru/aspose.cells.externalconnections/webqueryconnection) | Указывает свойства источника веб-запроса. Веб-запрос извлечет данные из HTML таблиц,<br/> а также может предоставлять параметры HTTP "Get" для обработки веб-сервером при создании HTML по<br/> включая параметры и элементы параметров.|
