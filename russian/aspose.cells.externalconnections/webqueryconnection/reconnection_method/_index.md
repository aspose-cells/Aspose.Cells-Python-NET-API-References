##  reconnection_method недвижимость

Указывает, что должно делать приложение для работы с электронными таблицами при сбое подключения.
Значение по умолчанию — ReConnectionMethodType.Required.

###  Примечания

 ПРИМЕЧАНИЕ. Это свойство устарело. Вместо,
используйте свойство ExternalConnection.ReconnectionMethodType.
 Этот ресурс будет удален через 12 месяцев, начиная с октября 2017 года.
Aspose приносит извинения за возможные неудобства.
