##  edit_page недвижимость

URL-адрес доступной пользователю веб-страницы, отображающей данные веб-запроса. Этот URL сохраняется
в случае, если sourceData="true" и URL-адрес были перенаправлены для ссылки на XML-файл.
Затем пользовательскую страницу можно отобразить в пользовательском интерфейсе и получить XML-данные.
за кулисами.

###  Примечания

 ПРИМЕЧАНИЕ. Это свойство устарело. Вместо,
используйте свойство WebQueryConnection.EditWebPage.
 Этот ресурс будет удален через 12 месяцев, начиная с октября 2017 года.
Aspose приносит извинения за возможные неудобства.
