##  refresh_on_change недвижимость

 Флаг, указывающий, должен ли запрос автоматически обновляться, когда содержимое
 ячейка, которая обеспечивает изменение значения параметра. Если true, то внешние данные обновляются
 используя новое значение параметра каждый раз при изменении. Если false, то внешние данные
обновляется только по запросу пользователя, или какое-либо другое событие запускает обновление (например, открытие рабочей книги).
###  Определение:
```python
@property
def refresh_on_change(self):
    ...
@refresh_on_change.setter
def refresh_on_change(self, value):
    ...
```
