##  refresh_on_change недвижимость

 Флаг, указывающий, должен ли запрос автоматически обновляться при обновлении содержимого
 Ячейка, в которой изменяется значение параметра. Если значение равно true, внешние данные обновляются.
 Используя новое значение параметра при каждом изменении. Если значение равно false, то внешние данные
обновляется только по запросу пользователя или при возникновении какого-либо другого события, вызывающего обновление (например, открытие рабочей книги).
###  Определение:
```python
@property
def refresh_on_change(self):
    ...
@refresh_on_change.setter
def refresh_on_change(self, value):
    ...
```
