##  SqlDataType перечисление
Указывает тип данных SQL параметра. Допустимо только для источников ODBC.



Тип SqlDataType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| SQL_UNSIGNED_OFFSET | беззнаковое смещение sql|
| SQL_SIGNED_OFFSET | смещение со знаком sql|
| SQL_GUID | руководство по sql|
| SQL_W_LONG_VARCHAR | sql с широкой длинной переменной char|
| SQL_W_VARCHAR | Широкая переменная char в SQL|
| SQL_W_CHAR | sql широкий символ|
| SQL_BIT | бит sql|
| SQL_TINY_INT | sql крошечный интервал|
| SQL_BIG_INT | sql большое целое|
| SQL_LONG_VAR_BINARY |Двоичный файл с длинной переменной SQL|
| SQL_VAR_BINARY | двоичная переменная SQL|
| SQL_BINARY | двоичный файл SQL|
| SQL_LONG_VAR_CHAR | SQL длинная переменная char|
| SQL_UNKNOWN_TYPE | sql неизвестный тип|
| SQL_CHAR | символ SQL|
| SQL_NUMERIC | SQL числовой|
| SQL_DECIMAL | sql десятичный|
| SQL_INTEGER | целое число SQL|
| SQL_SMALL_INT | sql маленькое целое|
| SQL_FLOAT | sql с плавающей запятой|
| SQL_REAL | sql реальный|
| SQL_DOUBLE | SQL двойной|
| SQL_TYPE_DATE | тип даты sql|
| SQL_TYPE_TIME | тип времени sql|
| SQL_TYPE_TIMESTAMP | тип метки времени sql|
| SQL_VAR_CHAR | символ переменной sql|
| SQL_INTERVAL_YEAR | SQL интервал год|
| SQL_INTERVAL_MONTH | месяц интервала sql|
| SQL_INTERVAL_DAY | день интервала sql|
| SQL_INTERVAL_HOUR | интервал sql час|
| SQL_INTERVAL_MINUTE | sql интервал минут|
| SQL_INTERVAL_SECOND | sql интервал секунды|
| SQL_INTERVAL_YEAR_TO_MONTH | Интервал sql из года в месяц|
| SQL_INTERVAL_DAY_TO_HOUR | интервал sql день в час|
| SQL_INTERVAL_DAY_TO_MINUTE | Интервал sql день в минуту|
| SQL_INTERVAL_DAY_TO_SECOND | Интервал sql день в секунду|
| SQL_INTERVAL_HOUR_TO_MINUTE | sql интервал час в минуту|
| SQL_INTERVAL_HOUR_TO_SECOND | sql интервал час в секунду|
| SQL_INTERVAL_MINUTE_TO_SECOND | интервал sql от минуты до секунды|
