##  SqlDataType перечисление
Указывает тип данных SQL для параметра. Действительно только для источников ODBC.



Тип SqlDataType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| SQL_UNSIGNED_OFFSET | беззнаковое смещение sql|
| SQL_SIGNED_OFFSET | смещение со знаком sql|
| SQL_GUID | sql guid|
| SQL_W_LONG_VARCHAR | sql wide длинная переменная char|
| SQL_W_VARCHAR | SQL-переменная типа char|
| SQL_W_CHAR | SQL Wide char|
| SQL_BIT | SQL-бит|
| SQL_TINY_INT | sql крошечный int|
| SQL_BIG_INT | sql большой int|
| SQL_LONG_VAR_BINARY | sql длинная переменная двоичная|
| SQL_VAR_BINARY | двоичная переменная SQL|
| SQL_BINARY | двоичный файл SQL|
| SQL_LONG_VAR_CHAR | sql длинная переменная char|
| SQL_UNKNOWN_TYPE | SQL неизвестный тип|
| SQL_CHAR | sql символ|
| SQL_NUMERIC | SQL числовой|
| SQL_DECIMAL | sql десятичная|
| SQL_INTEGER | SQL-целое число|
| SQL_SMALL_INT | sql small int|
| SQL_FLOAT | sql float|
| SQL_REAL | SQL реальный|
| SQL_DOUBLE | двойной SQL|
| SQL_TYPE_DATE | тип даты SQL|
| SQL_TYPE_TIME | тип времени SQL|
| SQL_TYPE_TIMESTAMP | тип временной метки sql|
| SQL_VAR_CHAR | SQL-переменная char|
| SQL_INTERVAL_YEAR | интервал SQL год|
| SQL_INTERVAL_MONTH | интервал sql месяц|
| SQL_INTERVAL_DAY | интервал sql день|
| SQL_INTERVAL_HOUR |интервал sql час|
| SQL_INTERVAL_MINUTE | интервал sql минута|
| SQL_INTERVAL_SECOND | интервал sql секунда|
| SQL_INTERVAL_YEAR_TO_MONTH | интервал SQL из года в месяц|
| SQL_INTERVAL_DAY_TO_HOUR | интервал SQL день-час|
| SQL_INTERVAL_DAY_TO_MINUTE | интервал SQL день в минуту|
| SQL_INTERVAL_DAY_TO_SECOND | SQL интервал день-секунда|
| SQL_INTERVAL_HOUR_TO_MINUTE | интервал SQL от часа до минуты|
| SQL_INTERVAL_HOUR_TO_SECOND | SQL интервал час в секунду|
| SQL_INTERVAL_MINUTE_TO_SECOND | интервал SQL от минуты до секунды|
