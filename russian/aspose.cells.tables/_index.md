**Aspose.Cells.Tables** namespace содержит классы, представляющие таблицы, строки, ячейки и их форматирование для файлов электронных таблиц Excel.

| :- | :- |

| [ListColumnCollection](/cells/python-net/ru/aspose.cells.tables/listcolumncollection) | Представляет коллекцию всех объектов [ListColumn](/cells/python-net/ru/aspose.cells.tables/listcolumn) в указанном объекте ListObject.|
| [ListObject](/cells/python-net/ru/aspose.cells.tables/listobject) | Представляет объект списка на листе.<br/> Объект ListObject является членом коллекции ListObjects.<br/> Коллекция ListObjects содержит все объекты списка на листе.|
| [ListObjectCollection](/cells/python-net/ru/aspose.cells.tables/listobjectcollection) | Представляет коллекцию из [ListObject](/cells/python-net/ru/aspose.cells.tables/listobject) объектов на листе.|
| [TableStyle](/cells/python-net/ru/aspose.cells.tables/tablestyle) | Представляет стиль таблицы.|

| [TableStyleElementCollection](/cells/python-net/ru/aspose.cells.tables/tablestyleelementcollection) |Представляет все элементы стиля таблицы.|
