##  TableStyleElementCollection класс
Представляет все элементы стиля таблицы.



Тип TableStyleElementCollection предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells.tables/tablestyleelementcollection/capacity) | Возвращает или задает количество элементов, которые может содержать список массива.|
