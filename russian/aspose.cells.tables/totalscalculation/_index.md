##  TotalsCalculation перечисление
Определяет тип расчета в строке Итоги столбца списка.



Тип TotalsCalculation предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| SUM | Представляет расчет итоговой суммы.|
| COUNT | Представляет вычисление итоговых значений Count.|
| AVERAGE |Представляет вычисление средних итогов.|
| MAX | Представляет вычисление максимальных сумм.|
| MIN | Представляет вычисление минимальных итогов.|
| VAR | Представляет вычисление итоговых значений Var.|
| COUNT_NUMS | Представляет вычисление итогов Count Nums.|
| STD_DEV | Представляет расчет итогов StdDev.|
| NONE | Представляет отсутствие расчета итогов.|
| CUSTOM | Представляет пользовательский расчет.|
