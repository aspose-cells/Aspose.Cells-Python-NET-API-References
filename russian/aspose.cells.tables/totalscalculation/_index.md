##  TotalsCalculation перечисление
Определяет тип расчета в строке «Итоги» столбца списка.



Тип TotalsCalculation предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| SUM | Представляет собой расчет итоговой суммы.|
| COUNT | Представляет собой подсчет итогов.|
| AVERAGE | Представляет собой расчет средних итоговых значений.|
| MAX | Представляет собой расчет максимальных итогов.|
| MIN | Представляет собой расчет минимальных итогов.|
| VAR | Представляет собой расчет итогов Var.|
| COUNT_NUMS | Представляет собой расчет итоговых значений Count Nums.|
| STD_DEV | Представляет собой расчет итогов StdDev.|
| NONE | Представляет собой отсутствие расчета итогов.|
| CUSTOM | Представляет собой индивидуальный расчет.|
