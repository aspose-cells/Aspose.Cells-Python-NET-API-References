##  TableStyleElement класс
Представляет элемент стиля таблицы.



Тип TableStyleElement предоставляет следующие члены:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [size](/cells/python-net/ru/aspose.cells.tables/tablestyleelement/size) | Количество строк или столбцов в одной полосе чередования.<br/> Применяется только для типов firstRowStripe, secondRowStripe, firstColumnStripe или secondColumnStripe.|
| [type](/cells/python-net/ru/aspose.cells.tables/tablestyleelement/type) | Получает тип элемента.|
