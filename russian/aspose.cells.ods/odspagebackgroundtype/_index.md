##  OdsPageBackgroundType перечисление
Представляет тип фона страницы ods.



Тип OdsPageBackgroundType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Нет фона.|
| COLOR | Форматирует фон цветом.|
| GRAPHIC | Форматирует фон с изображением.|
