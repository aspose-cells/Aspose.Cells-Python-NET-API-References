##  hidden_col_display_type недвижимость

Скрытый столбец (ширина этого столбца равна 0) в Excel, перед сохранением в формате HTML,
Если HtmlHiddenColDisplayType равно «Remove», скрытый столбец не будет выводиться,
Если значение «Скрыто», столбец будет выведен, но будет скрыт, значение по умолчанию — «Скрыто».
