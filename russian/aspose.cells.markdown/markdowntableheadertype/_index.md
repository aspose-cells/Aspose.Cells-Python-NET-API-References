##  MarkdownTableHeaderType перечисление
Представляет тип заголовка таблицы в файле разметки.



Тип MarkdownTableHeaderType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| FIRST_ROW | Первая строка как заголовок таблицы.|
| COLUMN_HEADER | Имя столбца (например, A,B,C...) в качестве заголовка таблицы.|
| EMPTY | Пустая строка заголовка.|
