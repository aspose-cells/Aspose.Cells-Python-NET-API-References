##  last_index_of {#aspose.cells.vba.VbaModule}
Ищет указанный объект и возвращает отсчитываемый от нуля индекс последнего вхождения во всем списке массива.


###  Возврат

Отсчитываемый от нуля индекс последнего вхождения значения во всем списке массива, если он найден; в противном случае -1.


```python
def last_index_of(self, item):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| item | [`VbaModule`](/cells/python-net/ru/aspose.cells.vba/vbamodule) | Объект, который необходимо найти в списке массивов. Значение может быть нулевым.|


##  last_index_of {#aspose.cells.vba.VbaModule-int}
Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс последнего вхождения в диапазоне элементов в списке массива, простирающемся от первого элемента до указанного индекса.


###  Возврат

Отсчитываемый от нуля индекс последнего вхождения значения в диапазоне элементов в списке массива, простирающемся от первого элемента до startIndex, если он найден; в противном случае -1.


```python
def last_index_of(self, item, index):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| item | [`VbaModule`](/cells/python-net/ru/aspose.cells.vba/vbamodule) | Объект, который необходимо найти в списке массивов. Значение может быть нулевым.|
| index | int | Начальный индекс обратного поиска, отсчитываемый от нуля.|


##  last_index_of {#aspose.cells.vba.VbaModule-int-int}
Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс последнего вхождения в диапазоне элементов в списке массива, который содержит указанное количество элементов и заканчивается указанным индексом.


###  Возврат

Отсчитываемый от нуля индекс последнего появления значения в диапазоне элементов в System.Collections. Список массивов, содержащий количество элементов и заканчивающийся на startIndex, если он найден; в противном случае -1.


```python
def last_index_of(self, item, index, count):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| item | [`VbaModule`](/cells/python-net/ru/aspose.cells.vba/vbamodule) | Объект, который необходимо найти в списке массивов. Значение может быть нулевым.|
| index | int | Начальный индекс обратного поиска, отсчитываемый от нуля.|
| count | int | Количество элементов в разделе для поиска.|
