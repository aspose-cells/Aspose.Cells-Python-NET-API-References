##  index_of {#aspose.cells.vba.VbaModule-int}
Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс первого вхождения в диапазоне элементов в списке массива, простирающемся от указанного индекса до последнего элемента.


###  Возврат

Отсчитываемый от нуля индекс первого вхождения значения в диапазоне элементов в списке массива, простирающемся от startIndex до последнего элемента, если он найден; в противном случае -1.


```python
def index_of(self, item, index):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| item | [`VbaModule`](/cells/python-net/ru/aspose.cells.vba/vbamodule) | Объект, который необходимо найти в списке массивов. Значение может быть нулевым.|
| index | int | Начальный индекс поиска, отсчитываемый от нуля. 0 (ноль) допустимо в пустом списке.|


##  index_of {#aspose.cells.vba.VbaModule-int-int}
Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс первого вхождения в диапазоне элементов в списке массивов, который начинается с указанного индекса и содержит указанное количество элементов.


###  Возврат

Отсчитываемый от нуля индекс первого вхождения значения в диапазоне элементов в списке массива, который начинается с startIndex и содержит количество элементов, если они найдены; в противном случае -1.


```python
def index_of(self, item, index, count):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| item | [`VbaModule`](/cells/python-net/ru/aspose.cells.vba/vbamodule) | Объект, который необходимо найти в списке массивов. Значение может быть нулевым.|
| index | int | Начальный индекс поиска, отсчитываемый от нуля. 0 (ноль) допустимо в пустом списке.|
| count | int | Количество элементов в разделе для поиска.|
