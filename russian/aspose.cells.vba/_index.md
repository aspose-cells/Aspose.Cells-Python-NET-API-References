| :- | :- |

| [`VbaModuleCollection`](/cells/python-net/ru/aspose.cells.vba/vbamodulecollection) | Представляет собой список [`VbaModule`](/cells/python-net/ru/aspose.cells.vba/vbamodule) |
| [`VbaProject`](/cells/python-net/ru/aspose.cells.vba/vbaproject) | Представляет проект VBA.|
| [`VbaProjectReference`](/cells/python-net/ru/aspose.cells.vba/vbaprojectreference) | Представляет собой ссылку на проект VBA.|
| [`VbaProjectReferenceCollection`](/cells/python-net/ru/aspose.cells.vba/vbaprojectreferencecollection) | Представляет все ссылки проекта VBA.|
