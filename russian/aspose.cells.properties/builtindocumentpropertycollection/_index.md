###  Примечания

Предоставляет доступ к объектам [DocumentProperty](/cells/python-net/ru/aspose.cells.properties/documentproperty) по их именам (с помощью индексатора) и
через набор типизированных свойств, которые возвращают значения соответствующих типов.
