##  PropertyType перечисление
Указывает тип данных свойства документа.



Тип PropertyType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| BOOLEAN | Свойство представляет собой логическое значение.|
| DATE_TIME | Свойство представляет собой значение даты и времени.|
| DOUBLE | Свойство представляет собой плавающее число.|
| NUMBER | Свойство представляет собой целое число.|
| STRING | Свойство представляет собой строковое значение.|
| BLOB | Свойство представляет собой массив байтов.|
