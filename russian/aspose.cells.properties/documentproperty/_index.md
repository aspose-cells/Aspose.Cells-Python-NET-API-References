##  DocumentProperty класс
Представляет пользовательское или встроенное свойство документа.



Тип DocumentProperty предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [name](/cells/python-net/ru/aspose.cells.properties/documentproperty/name) | Возвращает имя свойства.|
| [value](/cells/python-net/ru/aspose.cells.properties/documentproperty/value) | Получает или задает значение свойства.|
| [is_linked_to_content](/cells/python-net/ru/aspose.cells.properties/documentproperty/is_linked_to_content) | Указывает, связано ли это свойство с контентом|
| [source](/cells/python-net/ru/aspose.cells.properties/documentproperty/source) | Связанный источник контента.|
| [type](/cells/python-net/ru/aspose.cells.properties/documentproperty/type) | Получает тип данных свойства.|
| [is_generated_name](/cells/python-net/ru/aspose.cells.properties/documentproperty/is_generated_name) |Возвращает true, если это свойство не имеет имени в хранилище OLE2.<br/> и уникальное имя было создано только для публичного API.|


###  Методы
| Метод| Описание|
| :- | :- |
| [`to_int(self)`](/cells/python-net/ru/aspose.cells.properties/documentproperty/to_int/#) | Возвращает значение свойства как целое число.|
| [`to_double(self)`](/cells/python-net/ru/aspose.cells.properties/documentproperty/to_double/#) | Возвращает значение свойства как число двойной точности.|
| [`to_date_time(self)`](/cells/python-net/ru/aspose.cells.properties/documentproperty/to_date_time/#) | Возвращает значение свойства в виде DateTime в местном часовом поясе.|
| [`to_bool(self)`](/cells/python-net/ru/aspose.cells.properties/documentproperty/to_bool/#) | Возвращает значение свойства как bool.|
