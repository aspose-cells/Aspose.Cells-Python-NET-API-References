##  group_chr недвижимость

Задает символ (по умолчанию U+23DF).
Настоятельно рекомендуется использовать атрибут ChrType для установки символа ударения.
Используйте эту настройку свойства, если вы не можете найти нужный вам символ в известном типе.

###  Примечания

Следует отметить, что это свойство принимает только один символ, а если передано несколько символов, принимается только первый символ.

@property
def group_chr(self):
    ...

    ...
