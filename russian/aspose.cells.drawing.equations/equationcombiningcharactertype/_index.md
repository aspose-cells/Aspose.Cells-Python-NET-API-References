##  EquationCombiningCharacterType перечисление
Тип объединения символов.



Тип EquationCombiningCharacterType предоставляет следующие элементы:

| Поле| Описание|
| :- | :- |
| UNKNOWN | Используйте неизвестный тип, если его нет в существующем типе.|
| DOT_ABOVE | "̇" Unicode: u0307<br/> Объединение точек выше|
| DIAERESIS | "̈" Unicode: u0308<br/> Сочетание диэрезиса|
| THREE_DOTS_ABOVE | "⃛" Юникод: u20db<br/> Объединение трех точек выше|
| CIRCUMFLEX_ACCENT | "̂" Unicode: u0302<br/> Объединение циркумфлексного акцента|
| CARON | "̌" Unicode: u030c<br/> Сочетание Карона|
| ACUTE_ACCENT | "́" Unicode: u0301<br/> Сочетание острого акцента|
| GRAVE_ACCENT | "̀" Юникод: u0300<br/> Объединение грависного ударения|
| BREVE | "̆" Unicode: u0306<br/> Комбинирование Комбинирование Бреве|

| OVERLINE |"̅" Unicode: u0305<br/> Объединение надстрочных линий|
