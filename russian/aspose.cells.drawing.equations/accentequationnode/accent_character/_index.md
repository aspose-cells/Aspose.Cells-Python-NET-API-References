##  accent_character недвижимость

Этот атрибут определяет тип комбинационного диакритического знака, присоединяемого к основе функции ударения. Символ ударения по умолчанию — U+0302.
Настоятельно рекомендуется использовать атрибут AccentType для установки символа ударения.
Используйте эту настройку свойства, если вы не можете найти нужный вам символ в известном типе.

###  Примечания

Следует отметить, что это свойство принимает только один символ, а если передано несколько символов, принимается только первый символ.

    ...
