##  nary_operator недвижимость

n-арный оператор, например "∑".
Настоятельно рекомендуется использовать атрибут NaryOperatorType для задания n-арного оператора.
Используйте эту настройку свойства, если вы не можете найти нужный вам символ в известном типе.

###  Примечания

Следует отметить, что это свойство принимает только один символ, а если передано несколько символов, принимается только первый символ.
###  Определение:
```python
@property
def nary_operator(self):
    ...
@nary_operator.setter
def nary_operator(self, value):
    ...
```
