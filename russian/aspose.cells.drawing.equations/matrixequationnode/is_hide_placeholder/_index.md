##  is_hide_placeholder недвижимость

Этот атрибут определяет свойство «Скрыть заполнители» для матрицы. Если это свойство включено, заполнители не отображаются в матрице. По умолчанию заполнители отображаются, поэтому места, куда можно вставить текст, становятся видимыми.
