##  FractionEquationNode класс
Этот класс определяет уравнение дроби, состоящее из числителя и знаменателя, разделённых дробной чертой. Черта дроби может быть горизонтальной или диагональной, в зависимости от свойств дроби. Уравнение дроби также используется для представления стековой функции, которая размещает один элемент над другим без дробной черты.
