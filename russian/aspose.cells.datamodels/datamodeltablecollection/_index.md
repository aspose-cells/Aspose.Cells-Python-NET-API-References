##  DataModelTableCollection класс
Представляет список таблиц модели данных.



Тип DataModelTableCollection предоставляет следующие элементы:

| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells.datamodels/datamodeltablecollection/capacity) | Возвращает или задает количество элементов, которые может содержать список массива.|
