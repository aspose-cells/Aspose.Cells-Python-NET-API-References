##  DataModelRelationship класс
Представляет собой единичное отношение в модели данных электронной таблицы.



Тип DataModelRelationship предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [foreign_key_table](/cells/python-net/ru/aspose.cells.datamodels/datamodelrelationship/foreign_key_table) | Получает имя таблицы внешнего ключа для этого отношения.|
| [primary_key_table](/cells/python-net/ru/aspose.cells.datamodels/datamodelrelationship/primary_key_table) | Возвращает имя таблицы первичного ключа для этого отношения.|
| [foreign_key_column](/cells/python-net/ru/aspose.cells.datamodels/datamodelrelationship/foreign_key_column) | Возвращает имя столбца таблицы внешнего ключа для этого отношения.|
| [primary_key_column](/cells/python-net/ru/aspose.cells.datamodels/datamodelrelationship/primary_key_column) | Возвращает имя столбца таблицы первичного ключа для этого отношения.|
