##  SensitivityLabelAssignmentType перечисление
Представляет метод назначения метки чувствительности.



Тип SensitivityLabelAssignmentType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| STANDARD | Используется для любой метки чувствительности, которая не была нанесена непосредственно пользователем.|
| PRIVILEGED | Используется для любой метки чувствительности, нанесенной непосредственно пользователем.|
