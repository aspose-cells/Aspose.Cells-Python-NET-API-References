##  SlideViewType перечисление
Представляет тип при экспорте в слайды.



Тип SlideViewType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| VIEW | Экспорт в виде MS Excel.|
| PRINT | Экспорт в печать.|
