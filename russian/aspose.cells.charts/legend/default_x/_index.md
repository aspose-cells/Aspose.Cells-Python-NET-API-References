##  default_x недвижимость

Представляет x позиции по умолчанию в единицах, равных 1/4000 площади диаграммы.

###  Примечания

 ПРИМЕЧАНИЕ: этот элемент теперь устарел.
Вместо этого используйте свойство ChartFrame.DefaultXRatioToChart.
DefaultX = (целое)(DefaultXRatioToChart * 4000);
 Эта недвижимость будет снесена через 12 месяцев с февраля 2025 года.
Aspose приносит извинения за любые причиненные вам неудобства.
