##  offset недвижимость

 Получает и задает расстояние меток от оси категорий.
Только для оси категории (x).

###  Примечания

Расстояние по умолчанию составляет 100 процентов, что представляет собой интервал по умолчанию между метками осей и линией оси.
Значение может быть целым процентом от 0 до 1000 относительно размера шрифта метки оси.

```python
@property
def offset(self):
    ...
@offset.setter
def offset(self, value):
    ...
```
