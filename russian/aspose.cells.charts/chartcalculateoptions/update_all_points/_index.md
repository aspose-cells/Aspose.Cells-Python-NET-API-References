##  update_all_points недвижимость

 Следует ли обновлять все точки данных при выполнении расчета диаграммы. По умолчанию: Ложь.
Если вы хотите получить значение для каждой точки данных на диаграмме отдельно, установите для него значение true.
Если для этого параметра установлено значение True, новые точки данных могут быть созданы при расчете диаграммы. Это может увеличить размер файла Excel.
