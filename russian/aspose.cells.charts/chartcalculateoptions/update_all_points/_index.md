##  update_all_points недвижимость

 Обновлять ли все точки данных при расчёте диаграммы. Значение по умолчанию: False.
Если вы хотите получить значение для каждой конкретной точки данных на диаграмме, установите значение true.
Если этому параметру задано значение True, новые точки данных могут быть сгенерированы при расчёте диаграммы. Это может привести к увеличению размера файла Excel.
