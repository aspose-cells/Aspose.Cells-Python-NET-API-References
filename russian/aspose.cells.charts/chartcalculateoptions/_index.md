##  ChartCalculateOptions класс
Представляет варианты расчета диаграммы.



Тип ChartCalculateOptions предоставляет следующие элементы:

###  Конструкторы
| Конструктор| Описание|
| :- | :- |
| [`__init__(self)`](/cells/python-net/ru/aspose.cells.charts/chartcalculateoptions/__init__/#) | Создает параметры для расчета диаграммы.|


###  Характеристики
| Свойство| Описание|
| :- | :- |
| [update_all_points](/cells/python-net/ru/aspose.cells.charts/chartcalculateoptions/update_all_points) |Обновлять ли все точки данных при расчёте диаграммы. Значение по умолчанию: False.<br/>Если вы хотите получить значение для каждой конкретной точки данных на диаграмме, установите значение true.<br/> Если этому параметру задано значение True, новые точки данных могут быть сгенерированы при расчёте диаграммы. Это может привести к увеличению размера файла Excel.|
