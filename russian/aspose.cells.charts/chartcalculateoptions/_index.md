##  ChartCalculateOptions класс
Представляет параметры расчета диаграммы.



Тип ChartCalculateOptions предоставляет следующие элементы:

###  Конструкторы
| Конструктор| Описание|
| :- | :- |
| [__init__](/cells/python-net/ru/aspose.cells.charts/chartcalculateoptions/__init__/#) | Создает параметры для расчета диаграммы.|


###  Характеристики
| Свойство| Описание|
| :- | :- |
| [update_all_points](/cells/python-net/ru/aspose.cells.charts/chartcalculateoptions/update_all_points) | Следует ли обновлять все точки данных при выполнении расчета диаграммы. По умолчанию: Ложь.<br/>Если вы хотите получить значение для каждой точки данных на диаграмме отдельно, установите для него значение true.<br/> Если для этого параметра установлено значение True, новые точки данных могут быть созданы при расчете диаграммы. Это может увеличить размер файла Excel.|
