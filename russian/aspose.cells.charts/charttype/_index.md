##  ChartType перечисление
Перечисляет все типы диаграмм, используемые в Excel.



Тип ChartType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| AREA | Представляет собой площадную диаграмму.|
| AREA_STACKED | Представляет собой диаграмму с областями в стопке.|
| AREA_100_PERCENT_STACKED | Представляет собой 100%-ную составную диаграмму с областями.|
| AREA_3D | Представляет собой трехмерную диаграмму областей.|
| AREA_3D_STACKED | Представляет собой трехмерную диаграмму с областями в стопке.|
| AREA_3D100_PERCENT_STACKED | Представляет собой 3D-диаграмму с областями, составляющими 100%.|
| BAR | Представляет собой столбчатую диаграмму: кластеризованную столбчатую диаграмму.|
| BAR_STACKED | Представляет собой составную столбчатую диаграмму.|
| BAR_100_PERCENT_STACKED | Представляет собой 100%-ную столбчатую диаграмму с накоплением.|
| BAR_3D_CLUSTERED | Представляет собой трехмерную кластеризованную столбчатую диаграмму.|
| BAR_3D_STACKED | Представляет собой трехмерную столбчатую диаграмму с накоплением.|
| BAR_3D100_PERCENT_STACKED | Представляет собой 3D 100% составную столбчатую диаграмму.|
| BUBBLE | Представляет собой пузырьковую диаграмму.|
| BUBBLE_3D | Представляет собой 3D-пузырьковую диаграмму.|
| COLUMN | Представляет собой столбчатую диаграмму: кластеризованную столбчатую диаграмму.|
| COLUMN_STACKED | Представляет собой столбчатую диаграмму с накоплением.|
| COLUMN_100_PERCENT_STACKED | Представляет собой 100%-ную столбчатую диаграмму с накоплением.|
| COLUMN_3D | Представляет собой трехмерную столбчатую диаграмму.|
| COLUMN_3D_CLUSTERED | Представляет собой трехмерную кластеризованную столбчатую диаграмму.|
| COLUMN_3D_STACKED | Представляет собой трехмерную столбчатую диаграмму с накоплением.|
| COLUMN_3D100_PERCENT_STACKED | Представляет собой 3D 100% столбчатую диаграмму с накоплением.|
| CONE |Представляет собой конусную диаграмму.|
| CONE_STACKED | Представляет собой диаграмму с накоплением конусов.|
| CONE_100_PERCENT_STACKED | Представляет собой 100%-ную конусную диаграмму с накоплением.|
| CONICAL_BAR | Представляет собой коническую столбчатую диаграмму.|
| CONICAL_BAR_STACKED | Представляет собой штабелированную коническую столбчатую диаграмму.|
| CONICAL_BAR_100_PERCENT_STACKED | Представляет собой 100%-ную штабелированную коническую столбчатую диаграмму.|
| CONICAL_COLUMN_3D | Представляет собой трехмерную коническую столбчатую диаграмму.|
| CYLINDER | Представляет собой цилиндрическую диаграмму.|
| CYLINDER_STACKED | Представляет собой диаграмму сложенных цилиндров.|
| CYLINDER_100_PERCENT_STACKED | Представляет собой 100%-ную сложенную цилиндрическую диаграмму.|
| CYLINDRICAL_BAR | Представляет собой цилиндрическую столбчатую диаграмму.|
| CYLINDRICAL_BAR_STACKED | Представляет собой сложенную цилиндрическую столбчатую диаграмму.|
| CYLINDRICAL_BAR_100_PERCENT_STACKED | Представляет собой 100%-ную цилиндрическую столбчатую диаграмму.|
| CYLINDRICAL_COLUMN_3D | Представляет собой трехмерную цилиндрическую столбчатую диаграмму.|
| DOUGHNUT | Представляет собой кольцевую диаграмму.|
| DOUGHNUT_EXPLODED | Представляет собой взорванную кольцевую диаграмму.|
| LINE | Представляет собой линейную диаграмму.|
| LINE_STACKED | Представляет собой линейную диаграмму с накоплением.|
| LINE_100_PERCENT_STACKED | Представляет собой 100%-ную линейную диаграмму с накоплением.|
| LINE_WITH_DATA_MARKERS | Представляет собой линейную диаграмму с маркерами данных.|
| LINE_STACKED_WITH_DATA_MARKERS | Представляет собой линейную диаграмму с маркерами данных.|
| LINE_100_PERCENT_STACKED_WITH_DATA_MARKERS | Представляет собой 100%-ную линейную диаграмму с маркерами данных.|
| LINE_3D | Представляет собой трехмерную линейную диаграмму.|
| PIE |Представляет собой круговую диаграмму.|
| PIE_3D | Представляет собой трехмерную круговую диаграмму.|
| PIE_PIE | Представляет собой круговую диаграмму.|
| PIE_EXPLODED | Представляет собой развернутую круговую диаграмму.|
| PIE_3D_EXPLODED | Представляет собой трехмерную развернутую круговую диаграмму.|
| PIE_BAR | Представляет собой столбчатую или круговую диаграмму.|
| PYRAMID | Представляет собой пирамидальную диаграмму.|
| PYRAMID_STACKED | Представляет собой диаграмму в виде многоярусной пирамиды.|
| PYRAMID_100_PERCENT_STACKED | Представляет собой 100%-ную пирамидальную диаграмму.|
| PYRAMID_BAR | Представляет собой пирамидальную столбчатую диаграмму.|
| PYRAMID_BAR_STACKED | Представляет собой столбчатую диаграмму в виде пирамиды.|
| PYRAMID_BAR_100_PERCENT_STACKED | Представляет собой 100%-ную столбчатую пирамидальную диаграмму.|
| PYRAMID_COLUMN_3D | Представляет собой трехмерную пирамидальную столбчатую диаграмму.|
| RADAR | Представляет собой радарную диаграмму.|
| RADAR_WITH_DATA_MARKERS | Представляет собой радиарную диаграмму с маркерами данных.|
| RADAR_FILLED | Представляет собой заполненную радиарную диаграмму.|
| SCATTER | Представляет собой диаграмму рассеяния.|
| SCATTER_CONNECTED_BY_CURVES_WITH_DATA_MARKER | Представляет собой точечную диаграмму, соединенную кривыми, с маркерами данных.|
| SCATTER_CONNECTED_BY_CURVES_WITHOUT_DATA_MARKER | Представляет собой точечную диаграмму, соединенную кривыми, без маркеров данных.|
| SCATTER_CONNECTED_BY_LINES_WITH_DATA_MARKER | Представляет собой точечную диаграмму, соединенную линиями с маркерами данных.|
| SCATTER_CONNECTED_BY_LINES_WITHOUT_DATA_MARKER | Представляет собой точечную диаграмму, соединенную линиями, без маркеров данных.|
| STOCK_HIGH_LOW_CLOSE | Представляет собой график максимума-минимума-закрытия акций.|
| STOCK_OPEN_HIGH_LOW_CLOSE |Представляет собой биржевой график открытия-максимума-минимума-закрытия.|
| STOCK_VOLUME_HIGH_LOW_CLOSE | Представляет собой биржевой график «Объем-Максимум-Минимум-Закрытие».|
| STOCK_VOLUME_OPEN_HIGH_LOW_CLOSE | Представляет собой биржевой график «Объем-Открытие-Максимум-Минимум-Закрытие».|
| SURFACE_3D | Представляет собой поверхностную диаграмму: 3D поверхностная диаграмма.|
| SURFACE_WIREFRAME_3D | Представляет собой каркасную 3D-диаграмму поверхности.|
| SURFACE_CONTOUR | Представляет собой контурную диаграмму.|
| SURFACE_CONTOUR_WIREFRAME | Представляет собой каркасную контурную диаграмму.|
| BOX_WHISKER | Серия построена по принципу «ящик с усами».|
| FUNNEL | Серия выстроена в виде воронки.|
| PARETO_LINE | Серия выстроена в виде линий Парето.|
| SUNBURST | Серия имеет форму солнечных лучей.|
| TREEMAP | Серия представлена в виде древовидной карты.|
| WATERFALL | Серия построена по принципу водопада.|
| HISTOGRAM | Ряд представлен в виде гистограммы.|
| MAP | Серия представлена в виде карты региона.|
| RADIAL_HISTOGRAM | Ряд представлен в виде радиальной гистограммы. Он используется только для рендеринга.|
