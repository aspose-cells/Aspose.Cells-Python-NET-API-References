##  ChartSplitType перечисление
Представляет способ разделения двух частей круговой диаграммы или столбца круговой диаграммы.



Тип ChartSplitType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| POSITION | Представляет точки данных, которые должны быть разделены между круговой диаграммой<br/>и второй график, поместив последнюю позицию разделения<br/>точек данных на втором графике|
| VALUE | Представляет точки данных, которые должны быть разделены между круговой диаграммой<br/>и второй график, поместив точки данных с помощью<br/> значение меньше, чем Split Position на втором графике.|
| PERCENT_VALUE | Представляет точки данных, которые должны быть разделены между круговой диаграммой<br/>и второй график, поставив точки с<br/>процент меньше, чем процент разделенной позиции в<br/> второй график.|
| CUSTOM | Представляет точки данных, которые должны быть разделены между круговой диаграммой<br/>и второй график по Custom Split<br/> ценности.|
| AUTO | Представляет точки данных, которые должны быть разделены с использованием значения по умолчанию.<br/> механизм для этого типа диаграммы.|
