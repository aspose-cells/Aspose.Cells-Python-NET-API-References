##  ChartSplitType перечисление
Представляет собой способ разделения двух частей круговой диаграммы или столбчатой круговой диаграммы.



Тип ChartSplitType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| POSITION | Представляет точки данных, которые должны быть распределены между круговыми диаграммами<br/>и второй график, поместив последнюю позицию разделения<br/> точек данных на втором графике|
| VALUE | Представляет точки данных, которые должны быть распределены между круговыми диаграммами<br/>и второй график, поместив точки данных с<br/> значение меньше, чем Split Position во втором графике.|
| PERCENT_VALUE | Представляет точки данных, которые должны быть распределены между круговыми диаграммами<br/>и второй график, поставив точки с<br/>процент меньше, чем процент позиции разделения в<br/> вторая диаграмма.|
| CUSTOM | Представляет точки данных, которые должны быть распределены между круговыми диаграммами<br/>и второй график по Custom Split<br/> ценности.|
| AUTO |Представляет, что точки данных должны быть разделены с использованием значений по умолчанию.<br/> механизм для этого типа диаграммы.|
