##  ChartLineFormattingType перечисление
Представляет тип формата линии диаграммы.



Тип ChartLineFormattingType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| AUTOMATIC | Представляет собой тип автоматического форматирования.|
| SOLID | Представляет собой сплошной тип форматирования.|
| NONE | Представляет собой тип без форматирования.|
| GRADIENT | Градиент|
