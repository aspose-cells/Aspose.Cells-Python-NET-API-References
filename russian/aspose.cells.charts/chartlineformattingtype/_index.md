##  ChartLineFormattingType перечисление
Представляет тип линейного формата линии диаграммы.



Тип ChartLineFormattingType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| AUTOMATIC | Представляет тип автоматического форматирования.|
| SOLID | Представляет сплошной тип форматирования.|
| NONE | Не представляет тип форматирования.|
| GRADIENT | Градиент|
