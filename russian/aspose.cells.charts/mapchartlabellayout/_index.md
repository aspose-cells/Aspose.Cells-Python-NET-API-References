##  MapChartLabelLayout перечисление
Представляет собой макет надписей на карте.



Тип MapChartLabelLayout предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| BEST_FIT_ONLY | Только наиболее подходящий.|
| SHOW_ALL | Показывает все метки.|
| NONE | Никаких этикеток.|
