##  PlotDataByType перечисление
Представляет тип графика данных по строкам или столбцам.



Тип PlotDataByType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| ROW | По ряду.|
| COLUMN | По столбцам.|
