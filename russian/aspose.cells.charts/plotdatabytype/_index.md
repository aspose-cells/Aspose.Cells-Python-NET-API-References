##  PlotDataByType перечисление
Представляет тип графика данных по строке или столбцу.



Тип PlotDataByType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| ROW | По ряду.|
| COLUMN | По столбцу.|
