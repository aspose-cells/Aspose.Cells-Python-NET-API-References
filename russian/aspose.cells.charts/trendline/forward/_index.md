##  forward недвижимость

Возвращает или задает количество периодов (или единиц на диаграмме рассеяния), на которое распространяется линия тренда forward.
Количество периодов должно быть больше или равно нулю.
###  Определение:
```python
@property
def forward(self):
    ...
@forward.setter
def forward(self, value):
    ...
```
