```python

def copy_to(self, array):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| array | list | Одномерный массив, являющийся местом назначения для скопированных элементов. Индексация массива должна начинаться с нуля.|

Копирует диапазон элементов из списка массивов в совместимый список одномерных массивов, начиная с указанного индекса целевого списка массивов.



```python

def copy_to(self, index, array, array_index, count):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| index | int | Нулевой индекс в исходном списке массивов, с которого начинается копирование.|
| array | list |Одномерный массив list, в который копируются элементы из массива list. Индексация массива list должна начинаться с нуля.|
| array_index | int | Нулевой индекс в списке массивов, с которого начинается копирование.|
| count | int | Количество элементов для копирования.|
