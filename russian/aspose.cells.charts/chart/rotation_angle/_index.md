##  rotation_angle недвижимость

Представляет поворот трехмерного представления диаграммы (поворот области построения вокруг оси z в градусах).

###  Примечания

 Значение этого свойства должно быть от 0 до 360, за исключением 3-мерных столбчатых диаграмм, для которых значение должно быть от 0 до 44.
Значение по умолчанию — 20. Применимо только к трехмерным диаграммам.
