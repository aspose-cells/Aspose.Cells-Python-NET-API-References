##  elevation недвижимость

Представляет собой elevation трехмерного представления диаграммы в градусах.

###  Примечания

 Диаграмма elevation — это высота, на которой вы просматриваете диаграмму, в градусах.
 Для большинства типов диаграмм значение по умолчанию — 15.
Значение этого свойства должно быть в диапазоне от -90 до 90, за исключением трехмерных столбчатых диаграмм, для которых оно должно быть в диапазоне от 0 до 44.
