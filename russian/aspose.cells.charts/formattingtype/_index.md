##  FormattingType перечисление
Представляет тип форматирования, примененного к объекту [`Area`](/cells/python-net/ru/aspose.cells.drawing/area) или объекту [`Line`](/cells/python-net/ru/aspose.cells.drawing/line).



Тип FormattingType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| AUTOMATIC | Представляет собой тип автоматического форматирования.|
| CUSTOM | Представляет собой пользовательский тип форматирования.|
| NONE | Представляет собой тип без форматирования.|
