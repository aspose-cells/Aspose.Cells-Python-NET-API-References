##  FormattingType перечисление
Представляет тип форматирования, примененный к объекту [Area](/cells/python-net/ru/aspose.cells.drawing/area) или объекту [Line](/cells/python-net/ru/aspose.cells.drawing/line).



Тип FormattingType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| AUTOMATIC | Представляет тип автоматического форматирования.|
| CUSTOM | Представляет настраиваемый тип форматирования.|
| NONE | Не представляет тип форматирования.|
