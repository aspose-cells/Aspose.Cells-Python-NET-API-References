**Aspose.Cells.Timelines** namespace содержит классы, которые позволяют получать доступ и рисовать временные шкалы в файлах электронных таблиц Excel.

###  Классы
| Сорт| Описание|
| :- | :- |
| [Timeline](/cells/python-net/ru/aspose.cells.timelines/timeline) |Краткое описание просмотра временной шкалы<br/> Из-за MS Excel Excel 2003 не поддерживает временную шкалу.|
| [TimelineCollection](/cells/python-net/ru/aspose.cells.timelines/timelinecollection) | Задает коллекцию всех объектов временной шкалы на указанном листе.<br/> Из-за MS Excel Excel 2003 не поддерживает временную шкалу.|
