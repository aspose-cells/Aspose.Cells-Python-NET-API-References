##  DataLabelShapeType перечисление
Задает предустановленную геометрическую форму, которая будет использоваться для диаграммы.



Тип DataLabelShapeType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| RECT | Представляет собой форму прямоугольника.|
| ROUND_RECT | Представляет собой форму круглого прямоугольника.|
| ELLIPSE | Представляет собой форму эллипса.|
| RIGHT_ARROW_CALLOUT | Представляет собой выноску в виде стрелки, направленной вправо.|
| DOWN_ARROW_CALLOUT | Представляет собой выноску в виде стрелки, направленной вниз.|
| LEFT_ARROW_CALLOUT | Представляет собой выноску в виде стрелки, направленной влево.|
| UP_ARROW_CALLOUT | Представляет собой выноску в виде стрелки, направленной вверх.|
| WEDGE_RECT_CALLOUT | Представляет собой выноску в виде клиновидного прямоугольника.|
| WEDGE_ROUND_RECT_CALLOUT | Представляет собой выноску в виде скругленного прямоугольника с клином.|
| WEDGE_ELLIPSE_CALLOUT | Представляет собой выноску в виде клиновидного эллипса.|
| LINE_CALLOUT | Представляет собой форму выноски линии.|
| BENT_LINE_CALLOUT | Представляет собой выноску в виде изогнутой линии.|
| LINE_WITH_ACCENT_BAR_CALLOUT | Представляет собой линию с выноской в виде акцентной полосы.|
| BENT_LINE_WITH_ACCENT_BAR_CALLOUT | Представляет собой изогнутую линию с выноской в виде акцентной полосы.|
| LINE | Этот тип используется только для специальной обработки файлов.|
