##  get_option_index(self) {#}
Возвращает индекс параметра (на основе единицы) во всех переключателях GroupBox, содержащих данный переключатель.


###  Возврат




```python

def get_option_index(self):
    ...
```


###  Примечания

Если этот переключатель отсутствует в GroupBox, возвращает индекс параметра во всех переключателях, которые не находятся ни в одном GroupBox.
