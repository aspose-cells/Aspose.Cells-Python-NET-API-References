##  width_scale недвижимость

Получает и задает масштаб ширины в процентах от исходной ширины изображения.
Если фигура не является изображением, свойство WidthScale возвращает только 100;

###  Пример

```python

if shape.width_scale == 3:
    shape.width_scale = 1

```
