##  ShapePathPoint класс
Представляет собой координату xy в пространстве координат пути.



Тип ShapePathPoint предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [x](/cells/python-net/ru/aspose.cells.drawing/shapepathpoint/x) | Получает и задает координату x для данной координаты положения.|
| [y](/cells/python-net/ru/aspose.cells.drawing/shapepathpoint/y) | Получает координату y для данной координаты положения.|
