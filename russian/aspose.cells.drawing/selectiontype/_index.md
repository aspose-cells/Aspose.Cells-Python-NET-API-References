##  SelectionType перечисление
Тип выбора списка.



Тип SelectionType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| SINGLE | Одиночный тип выбора.|
| MULTI |Тип множественного выбора.|
| EXTEND | Расширить тип выбора.|
