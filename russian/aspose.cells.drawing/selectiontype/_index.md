##  SelectionType перечисление
Тип выбора списка.



Тип SelectionType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| SINGLE | Тип одиночного выбора.|
| MULTI | Тип множественного выбора.|
| EXTEND | Расширить тип выбора.|
