##  TextureType перечисление
Представляет предустановленный тип текстуры.



Тип TextureType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| BLUE_TISSUE_PAPER | Представляет собой тип текстуры «синяя папиросная бумага».|
| BOUQUET | Представляет собой тип текстуры «Букет».|
| BROWN_MARBLE | Представляет собой тип текстуры коричневого мрамора.|
| CANVAS | Представляет собой тип текстуры холста.|
| CORK | Представляет собой тип текстуры «Пробка».|
| DENIM | Представляет собой тип текстуры «Деним».|
| FISH_FOSSIL | Представляет собой тип текстуры окаменелости рыбы.|
| GRANITE | Представляет собой тип текстуры гранита.|
| GREEN_MARBLE | Представляет собой тип текстуры «Зеленый мрамор».|
| MEDIUM_WOOD | Представляет собой тип текстуры древесины средней твердости.|
| NEWSPRINT | Представляет собой тип фактуры газетной бумаги.|
| OAK | Представляет собой текстуру дуба.|
| PAPER_BAG | Представляет собой тип текстуры «Бумажный пакет».|
| PAPYRUS | Представляет собой тип текстуры папируса.|
| PARCHMENT |Представляет собой тип текстуры «Пергамент».|
| PINK_TISSUE_PAPER | Представляет собой текстуру розовой папиросной бумаги.|
| PURPLE_MESH | Представляет собой тип текстуры Purple Mesh.|
| RECYCLED_PAPER | Представляет собой тип текстуры переработанной бумаги.|
| SAND | Представляет собой тип текстуры «Песчаный».|
| STATIONERY | Представляет собой тип текстуры «Канцелярские товары».|
| WALNUT | Представляет собой тип текстуры «Капельки грецкого ореха».|
| WATER_DROPLETS | Представляет собой тип текстуры «Капли воды».|
| WHITE_MARBLE | Представляет собой тип текстуры белого мрамора.|
| WOVEN_MAT | Представляет собой тип текстуры «Тканый коврик».|
| UNKNOWN | Представляет неизвестный тип текстуры.|
