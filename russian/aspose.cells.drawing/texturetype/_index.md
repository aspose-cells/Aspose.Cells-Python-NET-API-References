##  TextureType перечисление
Представляет предустановленный тип текстуры.



Тип TextureType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| BLUE_TISSUE_PAPER | Представляет тип текстуры Blue Tissue Paper.|
| BOUQUET | Представляет тип текстуры Bouquet.|
| BROWN_MARBLE | Представляет тип текстуры «Коричневый мрамор».|
| CANVAS | Представляет тип текстуры Canvas.|
| CORK | Представляет тип текстуры Cork.|
| DENIM |Представляет тип текстуры Denim.|
| FISH_FOSSIL | Представляет тип текстуры Fish Fossil.|
| GRANITE | Представляет тип текстуры Granite.|
| GREEN_MARBLE | Представляет тип текстуры Green Marble.|
| MEDIUM_WOOD | Представляет тип текстуры Medium Wood.|
| NEWSPRINT | Представляет тип текстуры газетной бумаги.|
| OAK | Представляет тип текстуры Дуб.|
| PAPER_BAG | Представляет тип текстуры Paper Bag.|
| PAPYRUS | Представляет тип текстуры Papyrus.|
| PARCHMENT | Представляет тип текстуры пергамента.|
| PINK_TISSUE_PAPER | Представляет тип текстуры «Розовая папиросная бумага».|
| PURPLE_MESH | Представляет тип текстуры Purple Mesh.|
| RECYCLED_PAPER | Представляет тип текстуры переработанной бумаги.|
| SAND | Представляет тип текстуры Sand.|
| STATIONERY | Представляет тип текстуры Канцелярские товары.|
| WALNUT | Представляет тип текстуры «Капли грецкого ореха».|
| WATER_DROPLETS | Представляет тип текстуры Капли воды.|
| WHITE_MARBLE | Представляет тип текстуры «Белый мрамор».|
| WOVEN_MAT | Представляет тип текстуры Woven Mat.|
| UNKNOWN | Представляет неизвестный тип текстуры.|
