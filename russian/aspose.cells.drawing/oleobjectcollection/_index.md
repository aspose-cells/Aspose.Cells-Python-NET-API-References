##  OleObjectCollection класс
Представляет встроенные объекты OLE.



Тип OleObjectCollection предоставляет следующие члены:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells.drawing/oleobjectcollection/capacity) | Получает или задает количество элементов, которые может содержать список массивов.|
