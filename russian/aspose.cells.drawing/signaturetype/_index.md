##  SignatureType перечисление
Указывает тип подписи.



Тип SignatureType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| DEFAULT |Значение по умолчанию, соответствующее значению ProviderId, фиксировано: {0000000000-0000-0000-0000-0000000000}.|
| STAMP | Соответствующее значение ProviderId обычно равно {000CD6A4-0000-0000-C000-000000000046}.|
| CUSTOM | Соответствующее значение ProviderId обычно должно быть задано пользователем. Его следует получить из документации, поставляемой с поставщиком.|
