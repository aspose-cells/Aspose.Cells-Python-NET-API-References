##  arc_to(self, w_r, h_r, st_ang, sw_ang) {#float-float-float-float}
Добавляет эллиптическую дугу к текущей фигуре. Начальной точкой является конечная точка текущей фигуры.



```python

def arc_to(self, w_r, h_r, st_ang, sw_ang):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| w_r | float | Полуширина прямоугольной области эллипса, рисующего дугу (единица измерения: пиксель).|
| h_r | float | Половина высоты прямоугольной области эллипса, рисующего дугу (единица измерения: пиксель).|
| st_ang | float |Начальный угол дуги, измеряемый в градусах по часовой стрелке от оси x (единица измерения: градус).|
| sw_ang | float | Угол между начальным углом и концом дуги. (Единица измерения: градус)|
