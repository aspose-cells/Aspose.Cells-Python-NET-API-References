##  FillPattern перечисление
Перечисляет типы узоров заливки фигур.



Тип FillPattern предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Не представляет никакого фона.|
| SOLID |Представляет собой сплошной узор.|
| GRAY5 | Представляет собой 5%-ный серый узор.|
| GRAY10 | Представляет собой 10%-ный серый узор.|
| GRAY20 | Представляет собой 20%-ный серый узор.|
| GRAY30 | Представляет собой 30%-ный серый узор.|
| GRAY40 | Представляет собой 40%-ный серый узор.|
| GRAY50 | Представляет собой 50%-ный серый узор.|
| GRAY60 | Представляет собой 60%-ный серый узор.|
| GRAY70 | Представляет собой 70%-ный серый узор.|
| GRAY75 | Представляет собой 75%-ный серый узор.|
| GRAY80 | Представляет собой 80%-ный серый узор.|
| GRAY90 |Представляет собой 90%-ный серый узор.|
| GRAY25 | Представляет собой 25%-ный серый узор.|
| LIGHT_DOWNWARD_DIAGONAL | Представляет собой легкий диагональный рисунок, направленный вниз.|
| LIGHT_UPWARD_DIAGONAL | Представляет собой легкий восходящий диагональный узор.|
| DARK_DOWNWARD_DIAGONAL | Представляет собой темный диагональный рисунок, направленный вниз.|
| DARK_UPWARD_DIAGONAL | Представляет собой темный восходящий диагональный узор.|
| WIDE_DOWNWARD_DIAGONAL | Представляет собой широкий диагональный рисунок, направленный вниз.|
| WIDE_UPWARD_DIAGONAL | Представляет собой широкий восходящий диагональный узор.|
| LIGHT_VERTICAL | Представляет собой легкий вертикальный рисунок.|
| LIGHT_HORIZONTAL | Представляет собой светлый горизонтальный узор.|
| NARROW_VERTICAL | Представляет собой узкий вертикальный рисунок.|
| NARROW_HORIZONTAL | Представляет собой узкий горизонтальный узор.|
| DARK_VERTICAL | Представляет собой темный вертикальный узор.|
| DARK_HORIZONTAL | Представляет собой темный горизонтальный узор.|
| DASHED_DOWNWARD_DIAGONAL | Представляет собой пунктирную диагональную линию, направленную вниз.|
| DASHED_UPWARD_DIAGONAL | Представляет собой узор из пунктирных диагональных линий, направленных вверх.|
| DASHED_VERTICAL | Представляет собой пунктирный вертикальный узор.|
| DASHED_HORIZONTAL | Представляет собой пунктирный горизонтальный узор.|
| SMALL_CONFETTI | Представляет собой мелкий узор из конфетти.|
| LARGE_CONFETTI | Представляет собой крупный узор из конфетти.|
| ZIG_ZAG | Представляет собой зигзагообразный узор.|
| WAVE | Представляет собой волновой рисунок.|
| DIAGONAL_BRICK | Представляет собой диагональный рисунок кирпичной кладки.|
| HORIZONTAL_BRICK |Представляет собой горизонтальный рисунок кирпичной кладки.|
| WEAVE | Представляет собой узор переплетения.|
| PLAID | Представляет собой клетчатый узор.|
| DIVOT | Представляет собой узор из дивотов.|
| DOTTED_GRID | Представляет собой узор из точек.|
| DOTTED_DIAMOND | Представляет собой узор из ромбов с точками.|
| SHINGLE | Представляет собой рисунок в виде черепицы.|
| TRELLIS | Представляет собой решетчатый узор.|
| SPHERE | Представляет собой сферический узор.|
| SMALL_GRID | Представляет собой мелкую сетку.|
| LARGE_GRID | Представляет собой узор из крупной сетки.|
| SMALL_CHECKER_BOARD | Представляет собой мелкий шахматный узор.|
| LARGE_CHECKER_BOARD | Представляет собой узор в виде крупной шахматной доски.|
| OUTLINED_DIAMOND | Представляет собой контурный ромбовидный узор.|
| SOLID_DIAMOND | Представляет собой сплошной ромбовидный узор.|
| UNKNOWN | Представляет собой неизвестный шаблон.|
