##  top недвижимость

Представляет вертикальное смещение фигуры относительно ее строки top в пикселях.

###  Примечания

Если фигура присутствует на диаграмме, представляет собой вертикальное смещение фигуры от ее границы top.

###  Пример

```python

if shape.top == 3:
    shape.top = 1

```
