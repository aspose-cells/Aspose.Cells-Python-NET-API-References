##  LineJoinType перечисление
Представляет стили соединения линий.



Тип LineJoinType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| ROUND | Круглый сустав|
| BEVEL | Скошенное соединение|
| MITER | Соединение под углом|
| NONE | Ни один сустав|
