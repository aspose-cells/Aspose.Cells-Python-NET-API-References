##  LineJoinType перечисление
Представляет стили соединения линии.



Тип LineJoinType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| ROUND | Круглый сустав|
| BEVEL | Скошенное соединение|
| MITER | Митра сустав|
| NONE | Нет сустава|
