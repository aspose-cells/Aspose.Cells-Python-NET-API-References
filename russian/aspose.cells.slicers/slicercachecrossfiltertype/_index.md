##  SlicerCacheCrossFilterType перечисление
Представляет тип SlicerCacheCrossFilterType



Тип SlicerCacheCrossFilterType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Элемент стиля таблицы стиля слайсера для элементов слайсера<br/>без данных не применяется к элементам среза без данных и элементам среза<br/>без данных не сортируются отдельно в списке элементов среза в представлении среза|
| SHOW_ITEMS_WITH_DATA_AT_TOP | Элемент стиля таблицы стиля слайсера для элементов слайсера с<br/>никакие данные не применяются к элементам среза без данных, а элементы среза<br/> без данных сортируются внизу списка элементов среза в представлении среза|
| SHOW_ITEMS_WITH_NO_DATA | Элемент стиля таблицы в стиле среза для элементов среза без данных<br/> применяется к элементам среза без данных и элементам среза без данных<br/> не сортируются отдельно в списке элементов среза в представлении среза.|
