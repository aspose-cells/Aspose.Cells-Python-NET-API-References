##  SlicerCacheCrossFilterType перечисление
Представляют тип SlicerCacheCrossFilterType



Тип SlicerCacheCrossFilterType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Элемент стиля таблицы стиля слайсера для элементов слайсера<br/>без данных не применяется к элементам среза без данных, а элементы среза<br/> без данных не сортируются отдельно в списке элементов слайсера в представлении слайсера|
| SHOW_ITEMS_WITH_DATA_AT_TOP |Элемент стиля таблицы стиля слайсера для элементов слайсера с<br/>никакие данные не применяются к элементам среза без данных, а элементы среза<br/> без данных сортируются внизу в списке элементов слайсера в представлении слайсера|
| SHOW_ITEMS_WITH_NO_DATA | Элемент стиля таблицы стиля слайсера для элементов слайсера без данных<br/> применяется к элементам среза без данных и элементам среза без данных<br/> не сортируются отдельно в списке элементов слайсера в представлении слайсера.|
