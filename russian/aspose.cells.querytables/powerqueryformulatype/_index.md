##  PowerQueryFormulaType перечисление
Представляет тип формулы запроса мощности.



Тип PowerQueryFormulaType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| FORMULA | Формула запроса мощности формулы.|
| FUNCTION | Формула запроса мощности функции.|
| PARAMETER | Формула запроса мощности параметров.|
