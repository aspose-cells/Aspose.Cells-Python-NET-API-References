##  BulletType перечисление
Представляет тип пули.



Тип BulletType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Нет пули.|
| CHARACTER | Характерная пуля.|
| PICTURE | Образная пуля.|
| AUTO_NUMBERED | Автоматическая нумерованная пуля.|
