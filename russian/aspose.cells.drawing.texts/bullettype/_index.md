##  BulletType перечисление
Обозначает тип пули.



Тип BulletType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Пули нет.|
| CHARACTER | Характерный маркер.|
| PICTURE | Изображение маркера.|
| AUTO_NUMBERED | Автоматически пронумерованный маркер.|
