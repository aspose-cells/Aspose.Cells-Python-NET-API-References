##  last_index_of(self, item) {#Revision}
Выполняет поиск указанного объекта и возвращает нулевой индекс последнего вхождения во всем списке массива.


###  Возврат

Отсчитываемый от нуля индекс последнего вхождения значения во всем списке массива, если оно найдено; в противном случае -1.


```python

def last_index_of(self, item):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| item | Revision | Объект, который нужно найти в списке массивов. Значение может быть нулевым.|


##  last_index_of(self, item, index) {#Revision-int}
Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс последнего вхождения в диапазоне элементов списка массива, который простирается от первого элемента до указанного индекса.


###  Возврат

Отсчитываемый от нуля индекс последнего вхождения значения в диапазоне элементов списка массива, который простирается от первого элемента до startIndex, если найдено; в противном случае -1.


```python

def last_index_of(self, item, index):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| item | Revision | Объект, который нужно найти в списке массивов. Значение может быть нулевым.|
| index | int | Начальный индекс обратного поиска, отсчитываемый от нуля.|


##  last_index_of(self, item, index, count) {#Revision-int-int}
Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс последнего вхождения в диапазоне элементов списка массива, который содержит указанное количество элементов и заканчивается на указанном индексе.


###  Возврат

Отсчитываемый от нуля индекс последнего вхождения значения в диапазоне элементов в System.Collections. Список массивов, содержащий count элементов и заканчивающийся на startIndex, если найден; в противном случае -1.


```python

def last_index_of(self, item, index, count):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| item | Revision | Объект, который нужно найти в списке массивов. Значение может быть нулевым.|
| index | int | Начальный индекс обратного поиска, отсчитываемый от нуля.|
| count | int | Количество элементов в разделе для поиска.|
