##  LowCodeSaveOptions класс
Возможности сохранения сгенерированных результатов с минимальным написанием кода.



Тип LowCodeSaveOptions предоставляет следующие элементы:

###  Конструкторы
| Конструктор| Описание|
| :- | :- |
| [`__init__(self)`](/cells/python-net/ru/aspose.cells.lowcode/lowcodesaveoptions/__init__/#) | Создает новый экземпляр LowCodeSaveOptions|


###  Характеристики
| Свойство| Описание|
| :- | :- |
| [output_file](/cells/python-net/ru/aspose.cells.lowcode/lowcodesaveoptions/output_file) | Получает и задает файл (с путем, если необходимо) для сохранения сгенерированных данных.<br/> При установке этого свойства со значением, отличным от null или пустой строки, [`LowCodeSaveOptions.output_stream`](/cells/python-net/ru/aspose.cells.lowcode/lowcodesaveoptions#output_stream) будет игнорироваться.|
| [output_stream](/cells/python-net/ru/aspose.cells.lowcode/lowcodesaveoptions/output_stream) | Получает и задает поток для записи сгенерированных данных.<br/> При установке для этого свойства значения, отличного от null, [`LowCodeSaveOptions.output_file`](/cells/python-net/ru/aspose.cells.lowcode/lowcodesaveoptions#output_file) будет игнорироваться.|
| [save_format](/cells/python-net/ru/aspose.cells.lowcode/lowcodesaveoptions/save_format) | Получает и задает формат сохранения выходных данных.<br/>Как правило, для определенных процессов с низким объемом кода разрешены только некоторые определенные форматы.<br/>Пожалуйста, укажите правильный формат для соответствующего процесса, в противном случае результат будет неожиданным.<br/> или даже может быть вызвано исключение.|
