##  PdfConverter класс
Конвертер для преобразования файла шаблона в pdf.



Тип PdfConverter предоставляет следующие элементы:

###  Методы
| Метод| Описание|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/ru/aspose.cells.lowcode/pdfconverter/process/#str-str) | Конвертирует заданный файл шаблона в PDF.|
| [`process(, load_options, save_options)`](/cells/python-net/ru/aspose.cells.lowcode/pdfconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Конвертирует файл шаблона в PDF|



###  Пример

```python
from aspose.cells.lowcode import PdfConverter

PdfConverter.process("template.xlsx", "res.pdf")

```
