##  HtmlConverter класс
Конвертер для преобразования файлов html (html или mht) в другие форматы файлов электронных таблиц.



Тип HtmlConverter предоставляет следующие элементы:

###  Методы
| Метод| Описание|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/ru/aspose.cells.lowcode/htmlconverter/process/#str-str) | Конвертирует заданный файл шаблона между html и другими форматами.|
| [`process(, load_options, save_options)`](/cells/python-net/ru/aspose.cells.lowcode/htmlconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Конвертирует файлы между HTML и другими форматами электронных таблиц.|



###  Пример

```python
from aspose.cells.lowcode import HtmlConverter

HtmlConverter.process("Template.xlsx", "res.html")

```
