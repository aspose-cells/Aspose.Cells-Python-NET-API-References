##  TextConverter класс
Конвертер для преобразования текстовых форматов (csv, tsv, dif...) в другие форматы файлов электронных таблиц.



Тип TextConverter предоставляет следующие элементы:

###  Методы
| Метод| Описание|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/ru/aspose.cells.lowcode/textconverter/process/#str-str) | Конвертирует заданный файл шаблона между текстовыми файлами и другими форматами.|
| [`process(, load_options, save_options)`](/cells/python-net/ru/aspose.cells.lowcode/textconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Преобразует формат файла между текстовыми форматами и другими форматами файлов электронных таблиц.|



###  Пример

```python
from aspose.cells.lowcode import TextConverter

TextConverter.process("template.csv", "res.xlsx")

```
