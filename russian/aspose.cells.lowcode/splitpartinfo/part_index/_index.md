##  part_index недвижимость

Индекс текущей части в последовательности (начиная с 0).
-1 означает, что множественных частей нет, поэтому результат один.

###  Примечания

Если необходимо обработать несколько листов и каждый лист обрабатывается (разделяется)
отдельно индекс детали всегда начинается с 0 для каждого листа.
Например, при преобразовании рабочей книги в изображения,
он представляет собой индекс выходной страницы текущего обрабатываемого листа.
А -1 означает, что для текущего листа имеется только одна страница.
