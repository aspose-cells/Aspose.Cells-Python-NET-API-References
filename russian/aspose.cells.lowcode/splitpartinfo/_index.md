##  SplitPartInfo класс
Представляет информацию одного входа/выхода для нескольких входов/выходов,
например текущая страница, которая будет отображена при преобразовании электронной таблицы в изображение.



Тип SplitPartInfo предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [part_index](/cells/python-net/ru/aspose.cells.lowcode/splitpartinfo/part_index) | Индекс текущей части в последовательности (начиная с 0).<br/> -1 означает, что множественных частей нет, поэтому результат один.|
| [sheet_index](/cells/python-net/ru/aspose.cells.lowcode/splitpartinfo/sheet_index) | Индекс листа, на котором находится текущая деталь. -1 обозначает, что имеется только один лист.|
| [sheet_name](/cells/python-net/ru/aspose.cells.lowcode/splitpartinfo/sheet_name) | Имя листа, на котором находится текущая деталь.|
