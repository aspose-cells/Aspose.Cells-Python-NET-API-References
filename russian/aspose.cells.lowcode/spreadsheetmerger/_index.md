##  SpreadsheetMerger класс
Объединяет несколько файлов шаблонов в один.



Тип SpreadsheetMerger предоставляет следующие элементы:

###  Методы
| Метод| Описание|
| :- | :- |
| [`process(, template_files, result_file)`](/cells/python-net/ru/aspose.cells.lowcode/spreadsheetmerger/process/#list-str) | Объединить указанные файлы шаблонов.|
| [`process(, options)`](/cells/python-net/ru/aspose.cells.lowcode/spreadsheetmerger/process/#aspose.cells.lowcode.lowcodemergeoptions) | Объединяет несколько файлов шаблонов в один.|



###  Пример

```python
from aspose.cells.lowcode import SpreadsheetMerger

SpreadsheetMerger.process(["template2.xlsx", "template2.xlsx"], "res.xlsx")

```
