##  ImageConverter класс

Тип ImageConverter предоставляет следующие элементы:

###  Методы

| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/ru/aspose.cells.lowcode/imageconverter/process/#str-str) | Конвертирует файл шаблона в изображения.|
| [`process(, load_options, save_options)`](/cells/python-net/ru/aspose.cells.lowcode/imageconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Преобразует файл шаблона в изображения|
| [`process(, load_options, save_options, provider)`](/cells/python-net/ru/aspose.cells.lowcode/imageconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions-abstractlowcodesaveoptionsprovider) |  |
