##  SpreadsheetSplitter класс
Разделяет файл электронной таблицы на несколько частей.



Тип SpreadsheetSplitter предоставляет следующие элементы:

###  Методы
| Метод| Описание|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/ru/aspose.cells.lowcode/spreadsheetsplitter/process/#str-str) | Разбивает заданный файл шаблона на несколько частей.|
| [`process(, options)`](/cells/python-net/ru/aspose.cells.lowcode/spreadsheetsplitter/process/#aspose.cells.lowcode.lowcodesplitoptions) | Разделяет файл электронной таблицы на несколько частей.|



###  Пример

```python
from aspose.cells.lowcode import SpreadsheetSplitter

SpreadsheetSplitter.process("template.xlsx", "split.xlsx")

```
