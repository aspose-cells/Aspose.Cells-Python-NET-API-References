##  build_path_with_sheet_always недвижимость

Добавлять ли индекс листа или имя всегда к пути файла.
Значение по умолчанию — false, то есть, когда есть только один лист,
индекс листа (или имя) и соответствующий префикс не будут добавлены к пути к файлу.
