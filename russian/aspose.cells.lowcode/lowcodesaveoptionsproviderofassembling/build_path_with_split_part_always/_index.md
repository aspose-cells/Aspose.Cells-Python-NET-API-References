##  build_path_with_split_part_always недвижимость

Добавлять ли всегда индекс разделенной части к пути файла.
Значение по умолчанию — false, то есть, когда есть только одна разделенная часть,
индекс разделенной части и соответствующий префикс не будут добавлены к пути файла.
