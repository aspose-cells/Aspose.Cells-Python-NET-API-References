##  JsonConverter класс
Конвертер для преобразования структуры данных JSON в другие форматы файлов электронных таблиц.



Тип JsonConverter предоставляет следующие элементы:

###  Методы
| Метод| Описание|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/ru/aspose.cells.lowcode/jsonconverter/process/#str-str) | Конвертирует заданный файл шаблона между json и другими форматами.|
| [`process(, load_options, save_options)`](/cells/python-net/ru/aspose.cells.lowcode/jsonconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Конвертирует данные JSON в другие форматы файлов электронных таблиц и обратно.|



###  Пример

```python
from aspose.cells.lowcode import JsonConverter

JsonConverter.process("data.json", "res.xlsx")

```
