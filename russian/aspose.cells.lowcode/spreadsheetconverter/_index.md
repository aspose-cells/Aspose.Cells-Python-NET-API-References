##  SpreadsheetConverter класс
Конвертер для преобразования между различными форматами файлов электронных таблиц, такими как xls, xlsx, xlsb, электронные таблицы ml...



Тип SpreadsheetConverter предоставляет следующие элементы:

###  Методы
| Метод| Описание|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/ru/aspose.cells.lowcode/spreadsheetconverter/process/#str-str) | Конвертирует заданный файл шаблона между форматами файлов электронных таблиц.|
| [`process(, load_options, save_options)`](/cells/python-net/ru/aspose.cells.lowcode/spreadsheetconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Конвертирует файлы электронных таблиц между различными форматами.|



###  Пример

```python
from aspose.cells.lowcode import SpreadsheetConverter

SpreadsheetConverter.process("template.xlsx", "template.xlsb")

```
