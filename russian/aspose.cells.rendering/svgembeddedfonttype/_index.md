##  SvgEmbeddedFontType перечисление
Представляет встроенный тип шрифта в изображении SVG.



Тип SvgEmbeddedFontType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Не встраиваемый шрифт.|
| WOFF | Встроить шрифт WOFF.|
