##  PdfCompliance перечисление
Разрешение пользователю устанавливать совместимость преобразования PDF



Тип PdfCompliance предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Формат PDF совместим с PDF 1.4|
| PDF14 | Формат PDF совместим с PDF 1.4|
| PDF15 | Формат PDF совместим с PDF 1.5|
| PDF16 | Формат PDF совместим с PDF 1.6|
| PDF17 | Формат PDF совместим с PDF 1.7|
| PDF_A1B | Формат PDF совместим с PDF/A-1b(ISO 19005-1)|
| PDF_A1A | Формат PDF совместим с PDF/A-1a(ISO 19005-1)|
| PDF_A2B | Формат PDF совместим с PDF/A-2b(ISO 19005-2)|
| PDF_A2U | Формат PDF совместим с PDF/A-2u(ISO 19005-2)|
| PDF_A2A | Формат PDF совместим с PDF/A-2a(ISO 19005-2)|
| PDF_A3B | Формат PDF совместим с PDF/A-3b(ISO 19005-3)|
| PDF_A3U | Формат PDF совместим с PDF/A-3u(ISO 19005-3)|
| PDF_A3A |Формат PDF совместим с PDF/A-3a(ISO 19005-3)|
