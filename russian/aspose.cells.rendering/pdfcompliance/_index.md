##  PdfCompliance перечисление
Разрешение пользователю устанавливать совместимость преобразования PDF



Тип PdfCompliance предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Обычный формат пдф|
| PDF_A1B | Формат PDF, совместимый с PDFA-1b|
| PDF_A1A | Формат PDF, совместимый с PDFA-1a|
