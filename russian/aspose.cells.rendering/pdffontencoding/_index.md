##  PdfFontEncoding перечисление
Представляет собой кодировку встроенного шрифта PDF-файла.



Тип PdfFontEncoding предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| IDENTITY | Обозначает использование кодировки Identity-H для всех встроенных шрифтов в PDF.|
| ANSI_PREFER | Представляет предпочтение использовать WinAnsiEncoding для шрифтов TrueType с символами 32-127,<br/> в противном случае для встроенных шрифтов в PDF-файле будет использоваться кодировка Identity-H.|
