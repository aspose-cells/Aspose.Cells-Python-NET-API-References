##  default_font недвижимость

Если символы в Excel являются символами Unicode и им не задан правильный шрифт в стиле ячейки,
Они могут отображаться в виде блока в PDF-файле, изображения.
Установите шрифт по умолчанию, например MingLiu или MS Gothic, чтобы отобразить эти символы.
Если это свойство не задано, Aspose.Cells будет использовать системный шрифт по умолчанию для отображения этих символов Unicode.

```python
@property
def default_font(self):
    ...
@default_font.setter
def default_font(self, value):
    ...
```
