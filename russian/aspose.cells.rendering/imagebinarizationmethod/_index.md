##  ImageBinarizationMethod перечисление
Указывает метод, используемый для бинаризации изображения.



Тип ImageBinarizationMethod предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| THRESHOLD | Задает пороговый метод.|
| FLOYD_STEINBERG_DITHERING | Задает размывание с использованием метода диффузии ошибок Флойда-Стейнберга.|
