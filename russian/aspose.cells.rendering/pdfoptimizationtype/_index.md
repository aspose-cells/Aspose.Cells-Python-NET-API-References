##  PdfOptimizationType перечисление
Указывает тип оптимизации.



Тип PdfOptimizationType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| STANDARD |Высокое качество печати|
| MINIMUM_SIZE | Размер файла важнее качества печати|
