##  PdfCustomPropertiesExport перечисление
Указывает способ экспорта [CustomDocumentPropertyCollection](/cells/python-net/ru/aspose.cells.properties/customdocumentpropertycollection) в файл PDF.



Тип PdfCustomPropertiesExport предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Пользовательские свойства не экспортируются.|
| STANDARD | Пользовательские свойства экспортируются как записи в Info словарь.|
