##  horizontal_resolution недвижимость

Получает или задает разрешение по горизонтали для сгенерированных изображений в точках на дюйм.
Применяет метод генерации изображения, за исключением изображений в формате EMF.

###  Примечания

Значение по умолчанию — 96.
