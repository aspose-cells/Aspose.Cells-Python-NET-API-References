**Aspose.Cells.Rendering**namespace предоставляет классы для рендеринга изображений и pdf.

###  Классы
| Сорт| Описание|
| :- | :- |
| [DrawObject](/cells/python-net/ru/aspose.cells.rendering/drawobject) | DrawObject будет инициализирован и возвращен при рендеринге.|
| [DrawObjectEventHandler](/cells/python-net/ru/aspose.cells.rendering/drawobjecteventhandler) | Интерфейс для получения DrawObject и Bound при рендеринге.|
| [IPageSavingCallback](/cells/python-net/ru/aspose.cells.rendering/ipagesavingcallback) | Управление/указание хода процесса сохранения страницы.|
| [ImageOrPrintOptions](/cells/python-net/ru/aspose.cells.rendering/imageorprintoptions) | Позволяет указать параметры при рендеринге рабочего листа в изображения, печати рабочего листа или рендеринга диаграммы в изображение.|
| [PageEndSavingArgs](/cells/python-net/ru/aspose.cells.rendering/pageendsavingargs) | Информация для страницы завершает процесс сохранения.|
| [PageSavingArgs](/cells/python-net/ru/aspose.cells.rendering/pagesavingargs) | Информация для процесса сохранения страницы.|
| [PageStartSavingArgs](/cells/python-net/ru/aspose.cells.rendering/pagestartsavingargs) | Информация для страницы начинает процесс сохранения.|
| [PdfBookmarkEntry](/cells/python-net/ru/aspose.cells.rendering/pdfbookmarkentry) | PdfBookmarkEntry — это запись в закладке pdf.<br/>если свойство Text текущего экземпляра равно null или "",<br/> текущий экземпляр будет скрыт, а дочерние элементы будут вставлены на текущий уровень.|
| [SheetPrintingPreview](/cells/python-net/ru/aspose.cells.rendering/sheetprintingpreview) | Предварительный просмотр печати рабочего листа.|
| [SheetRender](/cells/python-net/ru/aspose.cells.rendering/sheetrender) | Представляет визуализацию рабочего листа, который может отображать рабочий лист для различных изображений, таких как (BMP, PNG, JPEG, TIFF..)<br/> Конструктор этого класса должен использоваться после модификации pagesetup, стиля ячейки.|
| [SheetSet](/cells/python-net/ru/aspose.cells.rendering/sheetset) | Описывает набор листов.|
| [WorkbookPrintingPreview](/cells/python-net/ru/aspose.cells.rendering/workbookprintingpreview) | Предварительный просмотр печати рабочей книги.|
| [WorkbookRender](/cells/python-net/ru/aspose.cells.rendering/workbookrender) | Представляет визуализацию рабочей книги.<br/> Конструктор этого класса должен использоваться после модификации pagesetup, стиля ячейки.|
