##  DrawObject класс
DrawObject будет инициализирован и возвращен при рендеринге.



Тип DrawObject предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [cell](/cells/python-net/ru/aspose.cells.rendering/drawobject/cell) | Указывает объект Cell при рендеринге.<br/> Доступны все свойства ячейки.|
| [shape](/cells/python-net/ru/aspose.cells.rendering/drawobject/shape) | Указывает объект Shape при рендеринге.<br/> Доступны все свойства формы.|
| [image_bytes](/cells/python-net/ru/aspose.cells.rendering/drawobject/image_bytes) | Указывает байты изображения визуализированной диаграммы, формы при визуализации.|
| [type](/cells/python-net/ru/aspose.cells.rendering/drawobject/type) | Указывает тип DrawObject.|
| [current_page](/cells/python-net/ru/aspose.cells.rendering/drawobject/current_page) | Указывает индекс страницы DrawObject.<br/>Индекс страниц основан на нулевом значении.<br/> При рендеринге один лист содержит несколько страниц.|
| [total_pages](/cells/python-net/ru/aspose.cells.rendering/drawobject/total_pages) | Показывает общее количество страниц в текущем рендеринге.|
| [sheet_index](/cells/python-net/ru/aspose.cells.rendering/drawobject/sheet_index) | Указывает текущий индекс листа DrawObject.|
