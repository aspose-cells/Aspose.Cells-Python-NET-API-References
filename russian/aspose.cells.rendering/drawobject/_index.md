##  DrawObject класс
DrawObject будет инициализирован и возвращен при рендеринге.



Тип DrawObject предоставляет следующие члены:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [cell](/cells/python-net/ru/aspose.cells.rendering/drawobject/cell) |Указывает объект Cell при рендеринге.<br/> Доступ ко всем свойствам ячейки.|
| [shape](/cells/python-net/ru/aspose.cells.rendering/drawobject/shape) | Указывает объект Shape при рендеринге.<br/> Доступ ко всем свойствам формы.|
| [image_bytes](/cells/python-net/ru/aspose.cells.rendering/drawobject/image_bytes) | Указывает байты изображения визуализируемой диаграммы, формы при рендеринге.|
| [type](/cells/python-net/ru/aspose.cells.rendering/drawobject/type) | Указывает тип DrawObject.|
| [current_page](/cells/python-net/ru/aspose.cells.rendering/drawobject/current_page) | Указывает индекс страницы DrawObject.<br/>Индекс страницы основан на нуле.<br/> Один лист содержит несколько страниц при рендеринге.|
| [total_pages](/cells/python-net/ru/aspose.cells.rendering/drawobject/total_pages) | Указывает общее количество страниц в текущем отображении.|
| [sheet_index](/cells/python-net/ru/aspose.cells.rendering/drawobject/sheet_index) | Указывает индекс текущего листа DrawObject.|
