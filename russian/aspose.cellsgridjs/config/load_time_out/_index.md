##  load_time_out недвижимость


Устанавливает/получает прерывание по тайм-ауту в миллисекундах при загрузке файла, когда период стоимости загрузки файла превышает ожидаемый, это вызывает исключение.
значение по умолчанию — -1, что означает, что прерывание по тайм-ауту не установлено.
