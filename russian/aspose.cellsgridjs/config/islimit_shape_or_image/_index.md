##  islimit_shape_or_image недвижимость


Устанавливает/получает, следует ли ограничивать общее количество форм отображения/изображений внутри одного листа, если установлено значение true,
GridJs ограничит общий размер формы/изображения дисплея внутри одного листа до MaxShapeOrImageCount.
Значение по умолчанию верно.
