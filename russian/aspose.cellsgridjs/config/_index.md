##  Config класс

Представляет все настройки GridJs.



Тип Config предоставляет следующие элементы:

###  Конструкторы
| Конструктор| Описание|
| :- | :- |
| [__init__](/cells/python-net/ru/aspose.cellsgridjs/config/__init__/#) | Создает новый экземпляр Config|


###  Характеристики
| Свойство| Описание|
| :- | :- |
| [save_html_as_zip](/cells/python-net/ru/aspose.cellsgridjs/config/save_html_as_zip) | Устанавливает/получает, сохранять ли html-файл в виде zip-архива, значение по умолчанию — false.|
| [same_image_detecting](/cells/python-net/ru/aspose.cellsgridjs/config/same_image_detecting) |
<br/>Значение по умолчанию верно. |
| [auto_optimize_for_large_cells](/cells/python-net/ru/aspose.cellsgridjs/config/auto_optimize_for_large_cells) |
<br/>игнорировать некоторые стили/границы, чтобы сократить время загрузки
<br/>Значение по умолчанию верно. |
| [islimit_shape_or_image](/cells/python-net/ru/aspose.cellsgridjs/config/islimit_shape_or_image) |
<br/>GridJs ограничит общий размер формы/изображения дисплея внутри одного листа до MaxShapeOrImageCount.
<br/>Значение по умолчанию верно. |
| [max_shape_or_image_count](/cells/python-net/ru/aspose.cellsgridjs/config/max_shape_or_image_count) |
<br/>значение по умолчанию — 100. |
| [max_total_shape_or_image_count](/cells/python-net/ru/aspose.cellsgridjs/config/max_total_shape_or_image_count) |
<br/>значение по умолчанию — 300. |
| [max_shape_or_image_width_or_height](/cells/python-net/ru/aspose.cellsgridjs/config/max_shape_or_image_width_or_height) |
<br/>значение по умолчанию — 10000. |
| [max_pdf_save_seconds](/cells/python-net/ru/aspose.cellsgridjs/config/max_pdf_save_seconds) |
<br/>значение по умолчанию — 10. |
| [ignore_empty_content](/cells/python-net/ru/aspose.cellsgridjs/config/ignore_empty_content) |
<br/> если последняя строка или столбец содержит ячейки без значения и формулы, но имеет собственный стиль
<br/>тогда мы не будем показывать эту строку/столбец, если это значение истинно.。
<br/>Значение по умолчанию верно . |
| [use_print_area](/cells/python-net/ru/aspose.cellsgridjs/config/use_print_area) |
<br/>Значение по умолчанию неверно . |
| [load_time_out](/cells/python-net/ru/aspose.cellsgridjs/config/load_time_out) |
<br/>значение по умолчанию — -1, что означает, что прерывание по тайм-ауту не установлено. |
| [show_chart_sheet](/cells/python-net/ru/aspose.cellsgridjs/config/show_chart_sheet) |
<br/>Значение по умолчанию неверно . |
| [page_size](/cells/python-net/ru/aspose.cellsgridjs/config/page_size) |
<br/>GridJs будет ограничивать размер строки на основе PageSize. Если PageSize равен -1, разбиение на страницы не будет выполняться.
<br/>значение по умолчанию — -1 |
| [empty_sheet_max_row](/cells/python-net/ru/aspose.cellsgridjs/config/empty_sheet_max_row) |
<br/>значение по умолчанию — 12. |
| [empty_sheet_max_col](/cells/python-net/ru/aspose.cellsgridjs/config/empty_sheet_max_col) |
<br/>значение по умолчанию — 15. |
| [picture_cache_directory](/cells/python-net/ru/aspose.cellsgridjs/config/picture_cache_directory) |
<br/>путь по умолчанию будет «_piccache» внутри FileCacheDirectory. |
| [file_cache_directory](/cells/python-net/ru/aspose.cellsgridjs/config/file_cache_directory) |
<br/>Нам нужно установить для него определенный путь, прежде чем мы будем использовать GridJs. |
