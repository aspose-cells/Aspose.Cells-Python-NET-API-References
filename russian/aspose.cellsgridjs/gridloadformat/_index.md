##  GridLoadFormat перечисление

Представляет формат загружаемого файла.



Тип GridLoadFormat предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| AUTO | Означает автоматическое распознавание формата.|
| CSV | Представляет текстовый файл со значениями, разделенными запятыми (CSV).|
| XLSX | Представляет книгу или шаблон Office Open XML электронная таблицаML с макросами или без них.|
| TSV | Представляет файл TSV (файл значений, разделенных табуляцией).|
| TAB_DELIMITED | Представляет текстовый файл с разделителями табуляцией, аналогичный [`GridLoadFormat.TSV`](/cells/python-net/ru/aspose.cellsgridjs/gridloadformat#TSV).|
| HTML | Представляет HTML-файл.|
| M_HTML | Представляет файл MHTML.|
| ODS | Представляет файл открытого листа документа (ODS).|
| EXCEL_97_TO_2003 | Представляет файл Excel97-2003 xls.|
| SPREADSHEET_ML |Представляет XML-файл Excel 2003.|
| XLSB | Представляет файл xlsb.|
| NUMBERS | Представляет файл чисел.|
| FODS | Представляет формат файла электронной таблицы OpenDocument Flat XML (.fods).|
| SXC | Представляет формат файла электронной таблицы StarOffice Calc (.sxc).|
| UNKNOWN | Представляет нераспознанный формат, не может быть загружен.|
