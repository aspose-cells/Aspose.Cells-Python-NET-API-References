##  id недвижимость

Указывает GUID, который может быть связан с GUID строки подписи, сохраненной в содержимом документа.
Значение по умолчанию — пусто (все нули) Guid.

###  Примечания

При установке он связывает [`SignatureLine`](/cells/python-net/ru/aspose.cells.drawing/signatureline) с соответствующим [`DigitalSignature`](/cells/python-net/ru/aspose.cells.digitalsignatures/digitalsignature).
###  Определение:
```python
@property
def id(self):
    ...
@id.setter
def id(self, value):
    ...
```
