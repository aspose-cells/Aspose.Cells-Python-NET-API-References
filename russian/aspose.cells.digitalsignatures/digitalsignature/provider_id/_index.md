##  provider_id недвижимость

Указывает идентификатор класса поставщика подписи.
Значение по умолчанию — Пусто (все нули) Guid.

###  Примечания

Поставщик криптографических услуг (CSP) — это независимый программный модуль, который фактически выполняет криптографические алгоритмы для аутентификации, кодирования и шифрования.
Microsoft Office резервирует значение {00000000-0000-0000-0000-000000000000} для своего поставщика подписи по умолчанию,
и {000CD6A4-0000-0000-C000-000000000046} для поставщика подписи из Восточной Азии.

GUID дополнительно установленного провайдера следует получить из документации, поставляемой с провайдером.
