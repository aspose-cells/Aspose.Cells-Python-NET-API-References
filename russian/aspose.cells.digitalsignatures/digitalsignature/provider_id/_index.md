##  provider_id недвижимость

Указывает идентификатор класса поставщика подписи.
Значение по умолчанию — пусто (все нули) Guid.

###  Примечания

Поставщик криптографических услуг (CSP) — это независимый программный модуль, который фактически реализует криптографические алгоритмы для аутентификации, кодирования и шифрования.
Microsoft Office резервирует значение {00000000-0000-0000-0000-0000000000000} для своего поставщика подписей по умолчанию,
и {000CD6A4-0000-0000-C000-000000000046} для его поставщика подписей в Восточной Азии.

GUID дополнительно установленного поставщика следует получить из документации, поставляемой вместе с поставщиком.
