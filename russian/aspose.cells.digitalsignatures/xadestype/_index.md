##  XAdESType перечисление
Тип XML-расширенной электронной подписи (XAdES).



Тип XAdESType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | XAdES отключен.|
| X_AD_ES | Базовый XAdES.|
