##  XAdESType перечисление
Тип расширенной электронной подписи XML (XAdES).



Тип XAdESType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | XAdES отключен.|
| X_AD_ES | Базовый XAdES.|
