##  set_local_function_name(self, standard_name, local_name, bidirectional) {#str-str-bool}
Устанавливает имя функции, зависящее от локали, соответствующее заданному стандартному имени функции.



```python

def set_local_function_name(self, standard_name, local_name, bidirectional):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| standard_name | str | Стандартное (локаль en-US) имя функции.|
| local_name | str | Имя функции, зависящее от локали|
| bidirectional | bool | Будет ли автоматически сопоставлять имя локальной функции со именем стандартной функции.<br/>Если true, локальное имя будет автоматически сопоставлено со стандартным именем.<br/>поэтому пользователю не нужно снова звонить по номеру [`SettableGlobalizationSettings.set_standard_function_name`](/cells/python-net/ru/aspose.cells/settableglobalizationsettings/set_standard_function_name)<br/> для одной и той же пары стандартных и локальных имен|
