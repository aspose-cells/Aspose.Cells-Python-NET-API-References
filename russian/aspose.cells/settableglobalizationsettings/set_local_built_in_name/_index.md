##  set_local_built_in_name(self, standard_name, local_name, bidirectional) {#str-str-bool}
Устанавливает зависящий от локали текст для встроенного имени с заданным стандартным текстом имени.



```python

def set_local_built_in_name(self, standard_name, local_name, bidirectional):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| standard_name | str | Стандартный (локаль en-US) текст имени встроенного имени.|
| local_name | str | Текст имени, зависящий от локали|
| bidirectional | bool |Будет ли автоматически сопоставлять локальный текст имени со стандартным текстом имени.<br/>Если значение равно true, текст локального имени будет автоматически сопоставлен со стандартным текстом имени.<br/>поэтому пользователю не нужно снова звонить по номеру [`SettableGlobalizationSettings.set_standard_built_in_name`](/cells/python-net/ru/aspose.cells/settableglobalizationsettings/set_standard_built_in_name)<br/> для одной и той же пары стандартных и локальных имен|
