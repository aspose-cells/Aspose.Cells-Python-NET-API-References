##  CellValueType перечисление
Указывает тип значения ячейки.



Тип CellValueType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| IS_UNKNOWN | Тип значения Cell неизвестен.|
| IS_NULL |Пустая ячейка. Соответствующее значение должно быть нулевым.|
| IS_NUMERIC | Значение Cell является числовым. Соответствующее значение должно быть типа int или double.|
| IS_DATE_TIME | Значение Cell — datetime. Соответствующее значение должно быть DateTime.|
| IS_STRING | Значение Cell — строка. Соответствующее значение должно быть строкой.|
| IS_BOOL | Значение Cell — логическое. Соответствующее значение должно быть логическим.|
| IS_ERROR | Cell содержит значение ошибки. Соответствующее значение должно быть строкой ошибки.|
