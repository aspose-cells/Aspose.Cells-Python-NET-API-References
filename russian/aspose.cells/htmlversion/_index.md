##  HtmlVersion перечисление
Указывает, что при сохранении в форматах HTML используется версия HTML.



Тип HtmlVersion предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| DEFAULT | Сохраните документ в соответствии с экспортом MS Excel HTML.|
| X_HTML |Сохраняет документ в соответствии со стандартом XHTML 1.0 Transitional.|
| HTML5 | Сохраняет документ в соответствии со стандартом HTML 5.|
