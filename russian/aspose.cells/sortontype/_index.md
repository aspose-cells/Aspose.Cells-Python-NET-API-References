##  SortOnType перечисление
Тип сортированного значения.



Тип SortOnType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| VALUE | Сортировка по значению ячеек.|
| CELL_COLOR | Сортировка по цвету ячеек.|
| FONT_COLOR | Сортировка по цвету шрифта ячеек.|
| ICON | Сортировка по условному значку.|
