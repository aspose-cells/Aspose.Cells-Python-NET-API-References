##  SaveFormat перечисление
Представляет формат, в котором сохранена книга.



Тип SaveFormat предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| CSV |Текстовый файл со значениями, разделенными запятыми (CSV).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [SaveFormat.CSV](/cells/python-net/ru/aspose.cells/saveformat#CSV).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| XLSX | Представляет файл xlsx.|
| XLSM | Представляет файл xlsm, который включает макросы.|
| XLTX | Представляет xltx-файл.|
| XLTM | Представляет файл xltm, который включает макросы.|
| XLAM | Представляет файл xltm, который включает макросы надстроек.|
| TSV | Текстовый файл значений, разделенных табуляцией (TSV).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [SaveFormat.TSV](/cells/python-net/ru/aspose.cells/saveformat#TSV).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| TAB_DELIMITED | Представляет текстовый файл с разделителями табуляции, то же самое с [SaveFormat.TSV](/cells/python-net/ru/aspose.cells/saveformat#TSV).|
| HTML | Представляет html-файл.|
| M_HTML | Представляет файл mhtml.|
| ODS | Откройте файл Лист документа (ODS).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [SaveFormat.ODS](/cells/python-net/ru/aspose.cells/saveformat#ODS).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| EXCEL_97_TO_2003 | Представляет файл Excel97-2003 xls.|
| SPREADSHEET_ML | Представляет XML-файл Excel 2003.|
| XLSB | Представляет файл xlsb.|
| AUTO | При сохранении файла на диск формат файла соответствует расширению имени файла.<br/> При сохранении файла в поток используется формат xlsx.|
| UNKNOWN | Представляет нераспознанный формат, не может быть сохранен.|
| PDF | Представляет файл PDF.|
| XPS |XPS (XML Paper Specification).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [SaveFormat.XPS](/cells/python-net/ru/aspose.cells/saveformat#XPS).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| TIFF | Представляет файл TIFF.<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [SaveFormat.TIFF](/cells/python-net/ru/aspose.cells/saveformat#TIFF).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| SVG | SVG файл.<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [SaveFormat.SVG](/cells/python-net/ru/aspose.cells/saveformat#SVG).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| DIF | Формат обмена данными.|
| OTS | Откройте файл шаблона документа (OTS).|
| XLT | Файл шаблона Excel 97-2003.|
| XML | Представляет собой простой XML-файл.|
| NUMBERS | Представляет файл чисел.|
| MARKDOWN | Представляет документ уценки.|
| FODS | Представляет формат файла OpenDocument Flat XML Spreadsheet (.fods).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [SaveFormat.FODS](/cells/python-net/ru/aspose.cells/saveformat#FODS).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| SXC | Представляет формат файла электронной таблицы StarOffice Calc (.sxc).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [SaveFormat.SXC](/cells/python-net/ru/aspose.cells/saveformat#SXC).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| PPTX | Представляет файл .pptx.|
| DOCX |Представляет файл .docx.|
| EMF | Windows Расширенный метафайл.|
| JPG | JPEG ЖФИФ.|
| PNG | Портативная сетевая графика.|
| BMP | Windows Растровое изображение|
| GIF | гифка|
| JSON | Джейсон|
| SQL_SCRIPT | SQL|
