##  SaveFormat перечисление
Представляет формат, в котором сохранена рабочая книга.



Тип SaveFormat предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| CSV | Текстовый файл «Значения, разделенные запятыми» (CSV).<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`SaveFormat.CSV`](/cells/python-net/ru/aspose.cells/saveformat#CSV).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| XLSX | Представляет собой файл xlsx.|
| XLSM | Представляет собой xlsm-файл, включающий макросы.|
| XLTX |Представляет собой файл xltx.|
| XLTM | Представляет собой xltm-файл, включающий макросы.|
| XLAM | Представляет собой xltm-файл, который позволяет добавлять макросы.|
| TSV | Текстовый файл «Значения, разделенные табуляцией» (TSV).<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`SaveFormat.TSV`](/cells/python-net/ru/aspose.cells/saveformat#TSV).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| TAB_DELIMITED | Представляет собой текстовый файл с разделителями-табуляциями, такой же, как и [`SaveFormat.TSV`](/cells/python-net/ru/aspose.cells/saveformat#TSV).|
| HTML | Представляет собой HTML-файл.|
| M_HTML |Представляет собой mhtml-файл.|
| ODS | Откройте файл Лист документа (ODS).<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`SaveFormat.ODS`](/cells/python-net/ru/aspose.cells/saveformat#ODS).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| EXCEL_97_TO_2003 | Представляет собой xls-файл Excel97-2003.|
| SPREADSHEET_ML | Представляет собой XML-файл Excel 2003.|
| XLSB | Представляет собой файл xlsb.|
| AUTO | При сохранении файла на диск формат файла соответствует расширению имени файла.<br/> При сохранении файла в потоке формат файла — xlsx.|
| UNKNOWN | Представляет собой неизвестный формат, не может быть сохранен.|
| PDF | Представляет собой PDF-файл.|
| XPS | Формат XPS (спецификация XML-документа).<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`SaveFormat.XPS`](/cells/python-net/ru/aspose.cells/saveformat#XPS).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| TIFF | Представляет файл TIFF.<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`SaveFormat.TIFF`](/cells/python-net/ru/aspose.cells/saveformat#TIFF).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| SVG | файл SVG.<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`SaveFormat.SVG`](/cells/python-net/ru/aspose.cells/saveformat#SVG).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| DIF | Формат обмена данными.|
| OTS | Откройте файл шаблона документа (OTS).|
| XLT | Файл шаблона Excel 97-2003.|
| XML | Представляет собой простой XML-файл.|
| NUMBERS | Представляет файл чисел.|
| MARKDOWN | Представляет собой документ с уценкой.|
| FODS | Представляет формат файла OpenDocument Flat XML Spreadsheet (.fods).<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`SaveFormat.FODS`](/cells/python-net/ru/aspose.cells/saveformat#FODS).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| SXC | Представляет формат файла электронной таблицы StarOffice Calc (.sxc).<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`SaveFormat.SXC`](/cells/python-net/ru/aspose.cells/saveformat#SXC).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| PPTX | Представляет собой файл .pptx.|
| DOCX | Представляет собой файл .docx.|
| EMF | Windows Расширенный метафайл.|
| JPG | JPEG ЙФИФ.|
| PNG | Переносимая сетевая графика.|
| BMP | Windows Растровое изображение|
| GIF | Гифка|
| JSON | Json|
| SQL_SCRIPT | SQL|
| X_HTML | Представляет собой XHtml-файл.<br/>пожалуйста, используйте свойство [`HtmlSaveOptions.html_version`](/cells/python-net/ru/aspose.cells/htmlsaveoptions#html_version).<br/> Эта недвижимость будет снесена через 6 месяцев, с марта 2025 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| EPUB | Представляет собой файл Epub.|
| AZW3 | Представляет файл Azw3.|
| PCL | PCL (язык команд принтера)|
| DBF | Файл данных Xbase|
