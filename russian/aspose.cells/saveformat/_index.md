##  SaveFormat перечисление
Представляет формат, в котором сохраняется книга.



Тип SaveFormat предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| CSV | Текстовый файл со значениями, разделенными запятыми (CSV).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`SaveFormat.CSV`](/cells/python-net/ru/aspose.cells/saveformat#CSV).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| XLSX | Представляет файл xlsx.|
| XLSM | Представляет файл xlsm, включающий макросы.|
| XLTX | Представляет файл xltx.|
| XLTM | Представляет файл xltm, который включает макросы.|
| XLAM | Представляет файл xltm, который включает макросы надстроек.|
| TSV | Текстовый файл со значениями, разделенными табуляцией (TSV).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`SaveFormat.TSV`](/cells/python-net/ru/aspose.cells/saveformat#TSV).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| TAB_DELIMITED | Представляет текстовый файл с разделителями табуляцией, аналогичный [`SaveFormat.TSV`](/cells/python-net/ru/aspose.cells/saveformat#TSV).|
| HTML | Представляет HTML-файл.|
| M_HTML | Представляет файл MHTML.|
| ODS | Откройте файл «Лист документа» (ODS).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`SaveFormat.ODS`](/cells/python-net/ru/aspose.cells/saveformat#ODS).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| EXCEL_97_TO_2003 | Представляет файл xls Excel97-2003.|
| SPREADSHEET_ML | Представляет XML-файл Excel 2003.|
| XLSB | Представляет файл xlsb.|
| AUTO | При сохранении файла на диск формат файла соответствует расширению имени файла.<br/>При сохранении файла в потоке формат файла — xlsx.|
| UNKNOWN | Представляет нераспознанный формат, не может быть сохранен.|
| PDF | Представляет PDF-файл.|
| XPS | Формат XPS (спецификация бумаги XML).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`SaveFormat.XPS`](/cells/python-net/ru/aspose.cells/saveformat#XPS).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| TIFF | Представляет файл TIFF.<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`SaveFormat.TIFF`](/cells/python-net/ru/aspose.cells/saveformat#TIFF).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| SVG | Файл SVG.<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`SaveFormat.SVG`](/cells/python-net/ru/aspose.cells/saveformat#SVG).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| DIF | Формат обмена данными.|
| OTS | Откройте файл листа шаблона документа (OTS).|
| XLT | Файл шаблона Excel 97-2003.|
| XML | Представляет простой XML-файл.|
| NUMBERS | Представляет файл чисел.|
| MARKDOWN | Представляет документ уценки.|
| FODS | Представляет формат файла электронной таблицы OpenDocument Flat XML (.fods).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`SaveFormat.FODS`](/cells/python-net/ru/aspose.cells/saveformat#FODS).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| SXC | Представляет формат файла электронной таблицы StarOffice Calc (.sxc).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`SaveFormat.SXC`](/cells/python-net/ru/aspose.cells/saveformat#SXC).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| PPTX | Представляет файл .pptx.|
| DOCX | Представляет файл .docx.|
| EMF | Windows Расширенный метафайл.|
| JPG | JPEG JFIF.|
