##  extend_to_next_sheet недвижимость

Расширяет ли данные на следующий лист, когда количество строк или столбцов данных превышает лимит.
По умолчанию — ложь.

###  Примечания

Если это свойство имеет значение true, дополнительные данные будут помещены на следующий лист после текущего.
(если текущий лист является последним, новый лист будет добавлен в текущую книгу).
Если это свойство имеет значение false, данные, превышающие лимит, будут игнорироваться.
###  Определение:
```python
@property
def extend_to_next_sheet(self):
    ...
@extend_to_next_sheet.setter
def extend_to_next_sheet(self, value):
    ...
```
