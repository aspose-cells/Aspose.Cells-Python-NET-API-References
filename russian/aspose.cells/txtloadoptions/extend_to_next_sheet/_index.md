##  extend_to_next_sheet недвижимость

Распространяет ли данные на следующий лист, если строки или столбцы данных превышают лимит.
Значение по умолчанию — false.

###  Примечания

Если это свойство истинно, дополнительные данные будут помещены на следующий лист после текущего.
(если текущий лист последний, новый лист будет добавлен к текущей рабочей книге).
Если это свойство имеет значение false, данные, превышающие лимит, будут игнорироваться.
###  Определение:
```python
@property
def extend_to_next_sheet(self):
    ...
@extend_to_next_sheet.setter
def extend_to_next_sheet(self, value):
    ...
```
