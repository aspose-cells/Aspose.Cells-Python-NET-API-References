##  TxtLoadStyleStrategy перечисление
Указывает, как применять стиль для проанализированных значений при преобразовании строкового значения в число или дату и время.



Тип TxtLoadStyleStrategy предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Не устанавливает стиль для анализируемого значения.|
| BUILT_IN |Установите стиль как встроенное число/дата-время, когда анализируемое значение представляет собой простые числовые значения/дату-время.|
| EXACT_FORMAT | Установите точный пользовательский формат для проанализированного значения, чтобы отформатированное значение совпадало с исходным входным.|
