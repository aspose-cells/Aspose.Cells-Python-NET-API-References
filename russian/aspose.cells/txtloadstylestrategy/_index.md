##  TxtLoadStyleStrategy перечисление
Указывает, как применять стиль к проанализированным значениям при преобразовании строкового значения в число или дату-время.



Тип TxtLoadStyleStrategy предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Не задает стиль для проанализированного значения.|
| BUILT_IN | Установите стиль как встроенный число/дата-время, когда проанализированное значение представляет собой простые числовые значения/значения даты-времени.|
| EXACT_FORMAT | Установите точный пользовательский формат для проанализированного значения, чтобы отформатированное значение совпадало с исходным входным значением.|
