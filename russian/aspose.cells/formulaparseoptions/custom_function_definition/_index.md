##  custom_function_definition недвижимость

Определение для анализа пользовательских функций.

###  Примечания

Для некоторых особых требований, например, при вычислении пользовательской функции в пользовательском движке,
некоторые его параметры необходимо вычислять в режиме массива, используя это свойство, можно пометить эти параметры
как режим массива при разборе формулы. В противном случае пользователю необходимо обновить эти пользовательские функции позже.
[`Workbook.update_custom_function_definition`](/cells/python-net/ru/aspose.cells/workbook/update_custom_function_definition), чтобы получить тот же результат.
###  Определение:
```python
@property
def custom_function_definition(self):
    ...
@custom_function_definition.setter
def custom_function_definition(self, value):
    ...
```
