##  parse недвижимость

Указана ли формула parse? Значение по умолчанию — true.
Если значение равно false, то заданная строка формулы будет сохранена для ячейки до тех пор, пока пользователь не вызовет другие методы для нее.
или проанализированные данные формулы требуются для других операций, таких как вычисление формул.
###  Определение:
```python
@property
def parse(self):
    ...
@parse.setter
def parse(self, value):
    ...
```
