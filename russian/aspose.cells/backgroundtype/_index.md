##  BackgroundType перечисление
Перечисляет типы фоновых узоров ячеек.



Тип BackgroundType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| DIAGONAL_CROSSHATCH | Представляет собой диагональную штриховку.|
| DIAGONAL_STRIPE | Представляет собой рисунок диагональных полос.|
| GRAY6 | Представляет 6,25% серого рисунка.|
| GRAY12 | Представляет 12,5% серого рисунка.|
| GRAY25 | Представляет 25% серого рисунка.|
| GRAY50 | Представляет 50% серого рисунка.|
| GRAY75 | Представляет 75% серого рисунка.|
| HORIZONTAL_STRIPE | Представляет собой узор горизонтальных полос.|
| NONE |Не представляет никакого фона.|
| REVERSE_DIAGONAL_STRIPE | Представляет собой узор из обратных диагональных полос.|
| SOLID | Представляет собой сплошной узор.|
| THICK_DIAGONAL_CROSSHATCH | Представляет собой толстую диагональную штриховку.|
| THIN_DIAGONAL_CROSSHATCH | Представляет собой тонкую диагональную штриховку.|
| THIN_DIAGONAL_STRIPE | Представляет собой узор из тонких диагональных полос.|
| THIN_HORIZONTAL_CROSSHATCH | Представляет собой тонкую горизонтальную штриховку.|
| THIN_HORIZONTAL_STRIPE | Представляет собой узор из тонких горизонтальных полос.|
| THIN_REVERSE_DIAGONAL_STRIPE | Представляет собой узор из тонких обратных диагональных полос.|
| THIN_VERTICAL_STRIPE | Представляет собой узор из тонких вертикальных полос.|
| VERTICAL_STRIPE | Представляет собой рисунок вертикальных полос.|
