##  BackgroundType перечисление
Перечисляет типы фоновых узоров ячеек.



Тип BackgroundType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| DIAGONAL_CROSSHATCH | Представляет диагональную штриховку.|
| DIAGONAL_STRIPE | Представляет собой рисунок диагональной полосы.|
| GRAY6 |Представляет 6,25% серого узора|
| GRAY12 | Представляет 12,5% серый узор|
| GRAY25 | Представляет 25% серый узор.|
| GRAY50 | Представляет 50% серый узор.|
| GRAY75 | Представляет 75% серый узор.|
| HORIZONTAL_STRIPE | Представляет собой рисунок с горизонтальной полосой.|
| NONE | Представляет отсутствие фона.|
| REVERSE_DIAGONAL_STRIPE | Представляет собой узор из перевернутых диагональных полос.|
| SOLID | Представляет собой сплошной узор.|
| THICK_DIAGONAL_CROSSHATCH | Представляет толстую диагональную штриховку.|
| THIN_DIAGONAL_CROSSHATCH | Представляет собой тонкую диагональную штриховку.|
| THIN_DIAGONAL_STRIPE | Представляет собой узор из тонких диагональных полос.|
| THIN_HORIZONTAL_CROSSHATCH | Представляет собой тонкую горизонтальную штриховку.|
| THIN_HORIZONTAL_STRIPE | Представляет собой узор из тонких горизонтальных полос.|
| THIN_REVERSE_DIAGONAL_STRIPE | Представляет собой узор из тонких обратных диагональных полос.|
| THIN_VERTICAL_STRIPE | Представляет собой тонкую вертикальную полоску.|
| VERTICAL_STRIPE | Представляет собой узор из вертикальных полос.|
