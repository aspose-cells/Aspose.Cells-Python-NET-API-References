##  BackgroundType перечисление
Перечисляет типы узоров фона ячеек.



Тип BackgroundType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| DIAGONAL_CROSSHATCH | Представляет собой узор в виде диагональной штриховки.|
| DIAGONAL_STRIPE | Представляет собой узор из диагональных полос.|
| GRAY6 | Представляет собой 6,25% серый узор|
| GRAY12 | Представляет собой 12,5% серый узор|
| GRAY25 | Представляет собой 25%-ный серый узор.|
| GRAY50 | Представляет собой 50%-ный серый узор.|
| GRAY75 | Представляет собой 75%-ный серый узор.|
| HORIZONTAL_STRIPE | Представляет собой узор из горизонтальных полос.|
| NONE | Не представляет никакого фона.|
| REVERSE_DIAGONAL_STRIPE | Представляет собой узор из перевернутых диагональных полос.|
| SOLID |Представляет собой сплошной узор.|
| THICK_DIAGONAL_CROSSHATCH | Представляет собой толстый диагональный узор с перекрестной штриховкой.|
| THIN_DIAGONAL_CROSSHATCH | Представляет собой узор из тонких диагональных перекрестных штрихов.|
| THIN_DIAGONAL_STRIPE | Представляет собой узор из тонких диагональных полос.|
| THIN_HORIZONTAL_CROSSHATCH | Представляет собой узор из тонких горизонтальных штриховок.|
| THIN_HORIZONTAL_STRIPE | Представляет собой узор из тонких горизонтальных полос.|
| THIN_REVERSE_DIAGONAL_STRIPE | Представляет собой узор из тонких обратных диагональных полос.|
| THIN_VERTICAL_STRIPE | Представляет собой узор из тонких вертикальных полос.|
| VERTICAL_STRIPE | Представляет собой узор из вертикальных полос.|
