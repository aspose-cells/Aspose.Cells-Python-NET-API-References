##  ErrorCellValueType перечисление
Представляет значение ячейки, содержащее ошибку.



Тип ErrorCellValueType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| BLOCKED | Представляет значение ячейки, содержащей ошибку #BLOCKED!.|
| BUSY |Представляет значение ячейки, содержащей ошибку #BUSY!.|
| CALC | Представляет значение ячейки, содержащей ошибку #CALC!.|
| CONNECT | Представляет значение ячейки, содержащей ошибку #CONNECT!.|
| NAME | Представляет значение ячейки, содержащей ошибку #NAME?.|
| FIELD | Представляет значение ячейки, содержащей ошибку #FIELD!.|
| SPILL | Представляет значение ячейки, содержащей ошибку #SPILL!.|
| UNKNOWN | Представляет значение ячейки, содержащей ошибку #UNKNOWN!.|
| TIME_OUT | Представляет значение ячейки, содержащей ошибку #TIMEOUT!.|
| EXTERNAL | Представляет значение ячейки, содержащей ошибку #EXTERNAL!.|
