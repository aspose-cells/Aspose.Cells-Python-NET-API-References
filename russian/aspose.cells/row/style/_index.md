##  style недвижимость

Представляет style этой строки.

###  Примечания

 ПРИМЕЧАНИЕ. Этот элемент устарел.
Вместо этого используйте метод Row.GetStyle(), чтобы получить значение по умолчанию style для этой строки.
 Это свойство будет удалено через 12 месяцев, начиная с декабря 2022 года.
Aspose приносит извинения за возможные неудобства.
