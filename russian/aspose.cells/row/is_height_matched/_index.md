##  is_height_matched недвижимость

Указывает, соответствует ли высота строки текущей настройке шрифта по умолчанию для рабочей книги.
Значение True этого свойства также означает, что высота строки является «автоматической» и не требует установки пользователем специального значения высоты.

###  Примечания

Если это свойство истинно, то при изменении содержимого этой строки
обычно высоту строки необходимо пересчитать (например, [`Worksheet.auto_fit_rows`](/cells/python-net/ru/aspose.cells/worksheet/auto_fit_rows))
чтобы получить тот же результат, что отображается в MS Excel при открытии в нем рабочей книги.
