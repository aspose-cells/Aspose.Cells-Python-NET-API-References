##  LoadFormat перечисление
Представляет формат загружаемого файла.



Тип LoadFormat предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| AUTO | Означает автоматическое распознавание формата.|
| CSV | Текстовый файл «Значения, разделенные запятыми» (CSV).<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`LoadFormat.CSV`](/cells/python-net/ru/aspose.cells/loadformat#CSV).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| XLSX | Представляет книгу или шаблон Office Open XML электронная таблицаML с макросами или без них.|
| TSV | Текстовый файл «Значения, разделенные табуляцией» (TSV).<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`LoadFormat.TSV`](/cells/python-net/ru/aspose.cells/loadformat#TSV).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| TAB_DELIMITED | Представляет собой текстовый файл с разделителями-табуляциями, такой же, как и [`LoadFormat.TSV`](/cells/python-net/ru/aspose.cells/loadformat#TSV).|
| HTML | Представляет собой HTML-файл.|
| M_HTML |Представляет собой mhtml-файл.|
| ODS | Откройте файл Лист документа (ODS).<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`LoadFormat.ODS`](/cells/python-net/ru/aspose.cells/loadformat#ODS).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| EXCEL_97_TO_2003 | Представляет собой xls-файл Excel97-2003.|
| SPREADSHEET_ML | Представляет собой XML-файл Excel 2003.|
| XLSB | Представляет собой файл xlsb.|
| OTS | Откройте файл шаблона документа (OTS).|
| NUMBERS | Представляет файл чисел.|
| FODS | Представляет формат файла OpenDocument Flat XML Spreadsheet (.fods).<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`LoadFormat.FODS`](/cells/python-net/ru/aspose.cells/loadformat#FODS).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| SXC | Представляет формат файла электронной таблицы StarOffice Calc (.sxc).<br/> ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,<br/>пожалуйста, используйте свойство [`LoadFormat.SXC`](/cells/python-net/ru/aspose.cells/loadformat#SXC).<br/> Эта недвижимость будет снесена через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за любые причиненные вам неудобства.|
| XML | Представляет собой простой XML-файл.|
| EPUB | Представляет файл EPUB.|
| AZW3 | Представляет собой файл AZW3.|
| CHM | Представляет собой CHM-файл.|
| MARKDOWN | Представляет собой файл Markdown.|
| UNKNOWN |Представляет нераспознанный формат, не может быть загружен.|
| IMAGE | Изображение|
| JSON | Json|
| DIF | Формат обмена данными.|
| DBF | Файл данных Xbase|
