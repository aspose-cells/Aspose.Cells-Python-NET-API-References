##  LoadFormat перечисление
Представляет формат загружаемого файла.



Тип LoadFormat предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| AUTO | Означает автоматическое распознавание формата.|
| CSV | Текстовый файл со значениями, разделенными запятыми (CSV).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`LoadFormat.CSV`](/cells/python-net/ru/aspose.cells/loadformat#CSV).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| XLSX | Представляет книгу или шаблон Office Open XML электронная таблицаML с макросами или без них.|
| TSV | Текстовый файл со значениями, разделенными табуляцией (TSV).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`LoadFormat.TSV`](/cells/python-net/ru/aspose.cells/loadformat#TSV).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| TAB_DELIMITED | Представляет текстовый файл с разделителями табуляцией, аналогичный [`LoadFormat.TSV`](/cells/python-net/ru/aspose.cells/loadformat#TSV).|
| HTML | Представляет HTML-файл.|
| M_HTML | Представляет файл MHTML.|
| ODS | Откройте файл «Лист документа» (ODS).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`LoadFormat.ODS`](/cells/python-net/ru/aspose.cells/loadformat#ODS).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| EXCEL_97_TO_2003 | Представляет файл xls Excel97-2003.|
| SPREADSHEET_ML | Представляет XML-файл Excel 2003.|
| XLSB | Представляет файл xlsb.|
| OTS | Откройте файл листа шаблона документа (OTS).|
| NUMBERS | Представляет файл чисел.|
| FODS | Представляет формат файла электронной таблицы OpenDocument Flat XML (.fods).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`LoadFormat.FODS`](/cells/python-net/ru/aspose.cells/loadformat#FODS).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| SXC | Представляет формат файла электронной таблицы StarOffice Calc (.sxc).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [`LoadFormat.SXC`](/cells/python-net/ru/aspose.cells/loadformat#SXC).<br/> Этот объект недвижимости будет удален через 6 месяцев, начиная с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| XML | Представляет простой XML-файл.|
| EPUB | Представляет файл EPUB.|
| AZW3 |Представляет файл AZW3.|
| UNKNOWN | Представляет нераспознанный формат, не может быть загружен.|
| IMAGE | Изображение|
| JSON | Json|
