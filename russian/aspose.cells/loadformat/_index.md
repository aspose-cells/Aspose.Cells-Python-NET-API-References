##  LoadFormat перечисление
Представляет формат файла загрузки.



Тип LoadFormat предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| AUTO | Представляет автоматическое распознавание формата.|
| CSV |Текстовый файл со значениями, разделенными запятыми (CSV).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [LoadFormat.CSV](/cells/python-net/ru/aspose.cells/loadformat#CSV).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| XLSX | Представляет книгу или шаблон электронной таблицы Office Open XML с макросами или без них.|
| TSV | Текстовый файл значений, разделенных табуляцией (TSV).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [LoadFormat.TSV](/cells/python-net/ru/aspose.cells/loadformat#TSV).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| TAB_DELIMITED | Представляет текстовый файл с разделителями табуляции, то же самое с [LoadFormat.TSV](/cells/python-net/ru/aspose.cells/loadformat#TSV).|
| HTML | Представляет html-файл.|
| M_HTML | Представляет файл mhtml.|
| ODS | Откройте файл Лист документа (ODS).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [LoadFormat.ODS](/cells/python-net/ru/aspose.cells/loadformat#ODS).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| EXCEL_97_TO_2003 | Представляет файл Excel97-2003 xls.|
| SPREADSHEET_ML | Представляет XML-файл Excel 2003.|
| XLSB | Представляет файл xlsb.|
| OTS | Откройте файл шаблона документа (OTS).|
| NUMBERS | Представляет файл чисел.|
| FODS | Представляет формат файла OpenDocument Flat XML Spreadsheet (.fods).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [LoadFormat.FODS](/cells/python-net/ru/aspose.cells/loadformat#FODS).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| SXC | Представляет формат файла электронной таблицы StarOffice Calc (.sxc).<br/> ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,<br/>пожалуйста, используйте свойство [LoadFormat.SXC](/cells/python-net/ru/aspose.cells/loadformat#SXC).<br/> Это свойство будет удалено через 6 месяцев с апреля 2021 года.<br/> Aspose приносит извинения за возможные неудобства.|
| XML | Представляет собой простой XML-файл.|
| UNKNOWN | Представляет нераспознанный формат, не может быть загружен.|
| IMAGE | Изображение|
| JSON | Джейсон|
