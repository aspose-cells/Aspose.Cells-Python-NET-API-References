##  EmfRenderSetting перечисление
Настройка для рендеринга метафайла EMF.



Тип EmfRenderSetting предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| EMF_ONLY | Только рендеринг записей Emf.|
| EMF_PLUS_PREFER | Предпочитает рендеринг записей EmfPlus.|
