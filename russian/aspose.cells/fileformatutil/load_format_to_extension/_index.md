##  load_format_to_extension(, load_format){#aspose.cells.LoadFormat}

Возвращаемое расширение представляет собой строку строчными буквами с точкой в начале.


```python

@staticmethod

    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| load_format | [`LoadFormat`](/cells/python-net/ru/aspose.cells/loadformat) | Формат загруженного файла.|
