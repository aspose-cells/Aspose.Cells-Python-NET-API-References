##  get_table_row_type_of_current() {#}
Получает имя типа строк таблицы, состоящих из текущей строки в таблице, на которую ссылаются.
По умолчанию используется «Эта строка», поэтому в формуле «# Эта строка» представляет текущую строку в ссылочной таблице.
