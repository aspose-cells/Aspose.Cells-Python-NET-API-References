##  get_table_row_type_of_current(self) {#}
Возвращает имя типа строк таблицы, состоящей из текущей строки в указанной таблице.
По умолчанию используется значение «Эта строка», поэтому в формуле «#Эта строка» представляет текущую строку в указанной таблице.
