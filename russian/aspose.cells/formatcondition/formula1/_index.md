##  formula1 недвижимость

Возвращает и задает значение или выражение, связанное с условным форматированием.

###  Примечания

Пожалуйста, добавьте все области перед установкой формулы.
При задании формулы для этого условия, если входное значение начинается с «=», то оно будет принято как формула.
В противном случае оно будет воспринято как обычное значение (текст, число, логическое значение). Текстовое значение, начинающееся с символа «=», пользователь может ввести как формулу в формате: «=\"=...\"».
