##  formula1 недвижимость

Получает и задает значение или выражение, связанное с условным форматированием.

###  Примечания

Пожалуйста, добавьте все области перед установкой формулы.
Для установки формулы для этого условия, если входное значение начинается с '=', то оно будет принято как формула.
В противном случае оно будет принято как простое значение (текст, число, логическое значение). Для текстового значения, которое начинается с '=', пользователь может ввести его как формулу в формате: "=\"=...\"".
