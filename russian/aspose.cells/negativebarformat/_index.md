##  NegativeBarFormat класс
Представляет настройки цвета гистограмм для отрицательных значений, определенных правилом условного форматирования гистограмм.



Тип NegativeBarFormat предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [border_color](/cells/python-net/ru/aspose.cells/negativebarformat/border_color) |Получает или задает объект FormatColor, который можно использовать для указания цвета границы для отрицательных гистограмм.|
| [border_color_type](/cells/python-net/ru/aspose.cells/negativebarformat/border_color_type) | Определяет, следует ли использовать тот же цвет границы, что и положительные гистограммы.|
| [color](/cells/python-net/ru/aspose.cells/negativebarformat/color) | Получает или задает объект FormatColor, который можно использовать для указания цвета заливки для отрицательных гистограмм.|
| [color_type](/cells/python-net/ru/aspose.cells/negativebarformat/color_type) | Получает или задает, следует ли использовать тот же цвет заливки, что и положительные гистограммы.|
