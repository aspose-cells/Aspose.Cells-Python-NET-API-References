##  NegativeBarFormat класс
Представляет настройки цвета гистограмм для отрицательных значений, которые определяются правилом условного форматирования гистограммы.



Тип NegativeBarFormat предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [border_color](/cells/python-net/ru/aspose.cells/negativebarformat/border_color) | Возвращает или задает объект FormatColor, который можно использовать для указания цвета границы для отрицательных столбцов данных.|
| [border_color_type](/cells/python-net/ru/aspose.cells/negativebarformat/border_color_type) | Определяет, использовать ли тот же цвет границы, что и для положительных столбцов данных.|
| [color](/cells/python-net/ru/aspose.cells/negativebarformat/color) | Возвращает или задает объект FormatColor, который можно использовать для указания цвета заливки для отрицательных столбцов данных.|
| [color_type](/cells/python-net/ru/aspose.cells/negativebarformat/color_type) | Возвращает или задает, следует ли использовать тот же цвет заливки, что и для положительных столбцов данных.|
