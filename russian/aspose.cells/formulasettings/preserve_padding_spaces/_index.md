##  preserve_padding_spaces недвижимость

Указывает, следует ли сохранять пробелы и разрывы строк, заполняемые между токенами формулы.
при получении и установке формул.
Значение по умолчанию — ложь.

###  Примечания

Обычно эти пробелы и разрывы строк предназначены только для визуальных целей.
Сохранение их или нет не влияет на рассчитанный результат.
Для рассмотрения производительности, если нет особых требований,
их лучше не сохранять при обработке формул.
