##  preserve_padding_spaces недвижимость

Указывает, следует ли сохранять пробелы и переносы строк, которые находятся между токенами формулы.
при получении и установке формул.
Значение по умолчанию — false.

###  Примечания

Обычно эти пробелы и переносы строк нужны только для визуального эффекта,
Их сохранение или нет не влияет на результат расчета.
Для оценки производительности, если нет особых требований,
при обработке формул их лучше не сохранять.
