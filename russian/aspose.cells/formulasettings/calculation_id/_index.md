##  calculation_id недвижимость

Указывает версию механизма вычислений, используемого для вычисления значений в книге.

###  Примечания

Это свойство предназначено только для сохранения настроек в результирующий файл электронной таблицы.
чтобы другие приложения (такие как MS Excel) могли действовать соответствующим образом при загрузке и манипулировании результирующим файлом.
В случае MS Excel, если значение этого свойства меньше, чем связанный идентификатор механизма пересчета
с приложением, открывающим результирующий файл, приложение пересчитает результаты всех формул
в этой книге сразу после загрузки файла.
Из соображений производительности для большинства пользовательских приложений мы не вычисляем никакие формулы в рабочей книге автоматически,
независимо от того, какое значение было установлено для этого свойства.
