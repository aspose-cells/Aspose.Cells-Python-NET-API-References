##  ColorScale класс
 Опишите правило условного форматирования ColorScale.
Это правило условного форматирования создает в ячейках градуированную цветовую шкалу.



Тип ColorScale предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [is_3_color_scale](/cells/python-net/ru/aspose.cells/colorscale/is_3_color_scale) | Указывает, является ли условное форматирование трехцветной шкалой.|
| [min_cfvo](/cells/python-net/ru/aspose.cells/colorscale/min_cfvo) | Получите или установите объект минимального значения этого объекта ColorScale.<br/> Невозможно установить для него значение null или CFValueObject с типом FormatConditionValueType.Max.|
| [mid_cfvo](/cells/python-net/ru/aspose.cells/colorscale/mid_cfvo) | Получите или установите объект среднего значения этого ColorScale.<br/> Невозможно установить для него CFValueObject с типом FormatConditionValueType.Max или FormatConditionValueType.Min.|
| [max_cfvo](/cells/python-net/ru/aspose.cells/colorscale/max_cfvo) | Получите или установите максимальное значение объекта ColorScale.<br/> Невозможно установить для него значение null или CFValueObject с типом FormatConditionValueType.Min.|
| [min_color](/cells/python-net/ru/aspose.cells/colorscale/min_color) | Получите или установите цвет градиента для минимального значения в диапазоне.|
| [mid_color](/cells/python-net/ru/aspose.cells/colorscale/mid_color) |Получите или установите цвет градиента для среднего значения в диапазоне.|
| [max_color](/cells/python-net/ru/aspose.cells/colorscale/max_color) | Получите или установите цвет градиента для максимального значения в диапазоне.|
