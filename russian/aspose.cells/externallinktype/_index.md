##  ExternalLinkType перечисление
Представляет тип внешней ссылки.



Тип ExternalLinkType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| DDE_LINK | Представляет собой связь DDE.|
| EXTERNAL | Представляет внешнюю ссылку.|
