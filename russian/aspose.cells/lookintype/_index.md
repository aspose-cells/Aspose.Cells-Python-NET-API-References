##  LookInType перечисление
Представляет вид в типе.



Тип LookInType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| FORMULAS |Если ячейка содержит формулу, найти объект по формуле, иначе найти его по значению.|
| VALUES | Только найти объект из отформатированных значений.|
| VALUES_EXCLUDE_FORMULA_CELL | Только найти объект из значений ячеек, которые не содержат формулы.|
| COMMENTS | Найти объект только из комментариев.|
| ONLY_FORMULAS | Только найти объект из формул.|
| ORIGINAL_VALUES | Только найти объект из исходных значений.|
