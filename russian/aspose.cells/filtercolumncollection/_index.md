##  FilterColumnCollection класс
Коллекция объектов Filter, представляющая все фильтры в автофильтруемом диапазоне.



Тип FilterColumnCollection предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells/filtercolumncollection/capacity) | Получает или задает количество элементов, которые может содержать список массива.|

| [index_of](/cells/python-net/ru/aspose.cells/filtercolumncollection/index_of/#aspose.cells.FilterColumn-int-int) |Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс первого вхождения в диапазоне элементов в списке массивов, который начинается с указанного индекса и содержит указанное количество элементов.|

| [get_by_index](/cells/python-net/ru/aspose.cells/filtercolumncollection/get_by_index/#int) | Возвращает один объект Filter из коллекции.|
