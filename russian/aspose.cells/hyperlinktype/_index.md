##  HyperlinkType перечисление
Представляет тип гиперссылки.



Тип HyperlinkType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| URL | URL |
| FILE | Ссылка на внешний файл.|
| CELL | Свяжите ячейку книги.|
| UNKNOWN | Неизвестный тип.|
