##  cached_file_folder недвижимость

Папка для временных файлов, которые могут использоваться в качестве кэша данных.

###  Примечания

Если папка не указана,
значение по умолчанию — [`CellsHelper.get_cache_folder`](/cells/python-net/ru/aspose.cells/cellshelper/get_cache_folder).
Если его значение по умолчанию равно null или пусто, или было указано как null или пусто,
то при сохранении книги кэш-файл использоваться не будет.
