##  FilterCategory перечисление
Представляет категорию фильтра.



Тип FilterCategory предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Без фильтра.|
| LABEL | Фильтр субтитров.|
| NUMBER_VALUE | Фильтр числовых значений.|
| DATE | Фильтр по значению даты.|
| TOP10 | Фильтр Top10 по ценности.|
