##  ThreadedCommentCollection класс
Представляет список цепочек комментариев.



Тип ThreadedCommentCollection предоставляет следующие члены:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells/threadedcommentcollection/capacity) | Получает или задает количество элементов, которые может содержать список массивов.|
