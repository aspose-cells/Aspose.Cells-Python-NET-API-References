##  ThreadedCommentCollection класс
Представляет собой список связанных комментариев.



Тип ThreadedCommentCollection предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells/threadedcommentcollection/capacity) | Возвращает или задает количество элементов, которые может содержать список массива.|
