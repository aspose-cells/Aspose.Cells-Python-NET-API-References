##  name недвижимость

Получает или задает name рабочего листа.

###  Примечания

 Максимальная длина листа name равна 31. И вы не можете назначить один и тот же name (без учета регистра) двум листам.
Например, вы не можете установить «SheetName1» для первого рабочего листа и установить «SHEETNAME1» для второго рабочего листа.
###  Определение:
```python
@property
def name(self):
    ...
@name.setter
def name(self, value):
    ...
```
