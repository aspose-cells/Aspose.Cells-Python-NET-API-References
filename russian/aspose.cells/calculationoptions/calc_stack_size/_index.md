##  calc_stack_size недвижимость

Размер стека для рекурсивного вычисления ячеек. Значение по умолчанию — 200.

###  Примечания

При наличии большого количества ячеек необходимо рекурсивно рассчитываться в дереве зависимостей,
StackOverflowException может возникнуть в процессе вычислений.
В этом случае пользователю следует указать меньшее значение для этого свойства.
В такой ситуации пользователь должен определить правильное значение этого свойства в соответствии с фактическими формулами и данными.
Однако слишком маленькое значение может привести к снижению производительности расчета по формуле, а значение меньше 2.
сделает невозможным вычисление формулы, которая зависит от другой. Поэтому, если указанное значение меньше 2,
оно будет сброшено до 2.
