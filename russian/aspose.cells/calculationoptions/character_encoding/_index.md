##  character_encoding недвижимость

Указывает кодировку, используемую для кодирования/декодирования символов при вычислении формул.
Для таких функций, как CHAR, CODE, рассчитанный результат зависит от настроек региона и кодировки среды по умолчанию.
С помощью этого свойства пользователь может указать правильную кодировку, используемую для этих функций, чтобы получить ожидаемый результат.
###  Определение:
```python
@property
def character_encoding(self):
    ...
@character_encoding.setter
def character_encoding(self, value):
    ...
```
