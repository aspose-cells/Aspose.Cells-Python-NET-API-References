##  custom_function недвижимость

Функции вычисления пользовательских формул для расширения механизма вычислений.

###  Примечания

 ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,
пожалуйста, используйте свойство CustomEngine, AbstractCalculationEngine предоставляет более удобные и гибкие API
для управления пользовательскими функциями.
 Это свойство будет удалено через 12 месяцев, начиная с августа 2020 года.
Aspose приносит извинения за возможные неудобства.
