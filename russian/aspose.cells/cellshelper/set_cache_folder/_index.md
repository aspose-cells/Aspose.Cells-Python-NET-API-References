##  set_cache_folder(, кэш){#str}
Задает папку для временных файлов, которые могут использоваться в качестве кэша данных.



```python

@staticmethod
def set_cache_folder(cache):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| cache | str | Папка для временных файлов, которые могут использоваться в качестве кэша данных.|
###  Примечания

Файлы кэша обычно используются для некоторых функций с целью оценки производительности памяти,
например, сохранение больших наборов данных в файл xls,
или с использованием режима памяти с файловым кэшем для модели ячеек.
