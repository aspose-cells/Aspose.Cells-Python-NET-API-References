##  is_template недвижимость

Указывает, сохраняется ли файл шаблона.

###  Примечания

 ПРИМЕЧАНИЕ: Этот элемент устарел. Вместо этого,
пожалуйста, создайте XlsSaveOptions с соответствующим форматом сохранения (xlt для true и xls для false).
 Этот метод будет удален через 12 месяцев, начиная с августа 2020 года.
Aspose приносит извинения за любые причиненные вам неудобства.
