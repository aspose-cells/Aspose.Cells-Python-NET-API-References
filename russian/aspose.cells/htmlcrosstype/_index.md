##  HtmlCrossType перечисление
Представляет пять типов перекрестной строки HTML.



Тип HtmlCrossType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| DEFAULT | Отображать как MS Excel, зависит от следующей ячейки.<br/>Если следующая ячейка пуста, строка будет пересекаться или будет усечена.|
| MS_EXPORT | Отобразите строку, как MS Excel, экспортирующую html.|
| CROSS | Отобразите HTML перекрестную строку, эта производительность для создания больших файлов html будет более чем в десять раз быстрее, чем установка значения по умолчанию или FitToCell.|
| CROSS_HIDE_RIGHT | Отображать перекрестную строку HTML и скрывать правую строку, когда тексты перекрываются.|
| FIT_TO_CELL | Отображает только строку в пределах ширины ячейки.|
