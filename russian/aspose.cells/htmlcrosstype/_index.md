##  HtmlCrossType перечисление
Представляет пять типов перекрестных строк HTML.



Тип HtmlCrossType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| DEFAULT | Отображение как в MS Excel, зависит от следующей ячейки.<br/> Если следующая ячейка нулевая, строка будет пересечена или усечена.|
| MS_EXPORT | Отобразите строку как MS Excel, экспортируя html.|
| CROSS |Отображение перекрестной строки HTML, такая производительность при создании больших HTML-файлов будет более чем в десять раз выше, чем при установке значения Default или FitToCell.|
| CROSS_HIDE_RIGHT | Отобразить перекрестную строку HTML и скрыть правую строку при перекрытии текстов.|
| FIT_TO_CELL | Отображение строки только в пределах ширины ячейки.|
