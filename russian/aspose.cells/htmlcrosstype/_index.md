##  HtmlCrossType перечисление
Представляет пять типов перекрестных строк HTML.



Тип HtmlCrossType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| DEFAULT | Отображение как в MS Excel, зависит от следующей ячейки.<br/> Если следующая ячейка равна нулю, строка пересечется или будет усечена.|
| MS_EXPORT | Отобразите строку, например, MS Excel, экспортирующую HTML.|
| CROSS | Отобразите перекрестную строку HTML, такая производительность при создании больших HTML-файлов будет более чем в десять раз быстрее, чем при установке значения Default или FitToCell.|
| CROSS_HIDE_RIGHT | Отобразить перекрестную строку HTML и скрыть правую строку, когда тексты перекрываются.|
| FIT_TO_CELL | Отображение строки только в пределах ширины ячейки.|
