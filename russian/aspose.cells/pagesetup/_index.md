##  PageSetup класс
 Инкапсулирует объект, представляющий описание настроек страницы.
Объект PageSetup содержит все параметры настройки страницы.



Тип PageSetup предоставляет следующие элементы:

###  Пример

```python
from aspose.cells import Workbook

workbook = Workbook()
sheets = workbook.worksheets
# Add a worksheet
sheets.add()
sheet = sheets[1]
pageSetup = sheet.page_setup
pageSetup.print_area = "D1:K13"

```
