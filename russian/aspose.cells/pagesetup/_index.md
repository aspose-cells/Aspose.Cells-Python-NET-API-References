##  PageSetup класс
 Инкапсулирует объект, представляющий описание настройки страницы.
Объект PageSetup содержит все параметры настройки страницы.

###  Пример

```python
from aspose.cells import Workbook

workbook = Workbook()
sheets = workbook.worksheets
# Add a worksheet
sheets.add()
sheet = sheets[1]
pageSetup = sheet.page_setup
pageSetup.print_area = "D1:K13"

```
