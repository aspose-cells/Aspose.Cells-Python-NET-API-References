##  set_first_page_header(section, header_script) {#int-str}
Задает сценарий, форматирующий заголовок первой страницы файла Excel.
Только эффект в Excel 2007, когда IsHFDiffFirst имеет значение true.

def set_first_page_header(self, section, header_script):
    ...

| Параметр| Тип| Описание|

| header_script | str | Скрипт формата заголовка.|
