##  set_even_footer(self, section, footer_script) {#int-str}
Устанавливает скрипт форматирования нижнего колонтитула четной страницы файла Excel.
Действует только в Excel 2007, когда IsHFDiffOddEven имеет значение true.

def set_even_footer(self, section, footer_script):
    ...

| Параметр| Тип| Описание|
| :- | :- | :- |
| section | int |0: Левая секция, 1: Центральная секция, 2: Правая секция.|
| footer_script | str | Скрипт форматирования нижнего колонтитула.|
