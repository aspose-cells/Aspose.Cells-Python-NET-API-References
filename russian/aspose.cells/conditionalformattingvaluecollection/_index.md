##  ConditionalFormattingValueCollection класс
Описывает коллекцию CFValueObject.
Использовать только для наборов иконок.



Тип ConditionalFormattingValueCollection предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells/conditionalformattingvaluecollection/capacity) | Возвращает или задает количество элементов, которые может содержать список массива.|
