##  ConditionalFormattingValueCollection класс
Описывает коллекцию CFValueObject.
Используйте только для наборов значков.



Тип ConditionalFormattingValueCollection предоставляет следующие элементы:

| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells/conditionalformattingvaluecollection/capacity) | Получает или задает количество элементов, которые может содержать список массива.|
