##  merged_cells_shrink_type недвижимость

Указывает, как обрабатывать объединенные ячейки при удалении пустых строк/столбцов.

###  Примечания

Для числа [`MergedCellsShrinkType.KEEP_HEADER_ONLY`](/cells/python-net/ru/aspose.cells/mergedcellsshrinktype#KEEP_HEADER_ONLY) все ячейки в нём будут считаться пустыми, за исключением непустой верхней левой ячейки. Это значение свойства по умолчанию.

Для числа [`MergedCellsShrinkType.NONE`](/cells/python-net/ru/aspose.cells/mergedcellsshrinktype#NONE) все ячейки в нем будут считаться непустыми.

Для [`MergedCellsShrinkType.SHRINK_TO_FIT`](/cells/python-net/ru/aspose.cells/mergedcellsshrinktype#SHRINK_TO_FIT) все ячейки за пределами области отображения содержимого будут считаться пустыми.
