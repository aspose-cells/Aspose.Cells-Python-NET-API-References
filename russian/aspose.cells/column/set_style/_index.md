##  set_style(self, style) {#aspose.cells.Style}
Задает стиль этого столбца.



```python

def set_style(self, style):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| style | [`Style`](/cells/python-net/ru/aspose.cells/style) | стиль, который будет использоваться в качестве стиля по умолчанию для ячеек в этом столбце.|
###  Примечания

Этот метод только устанавливает заданный стиль как стиль по умолчанию для этого столбца,
не изменяя настройки стиля для существующих ячеек в этом столбце.
Чтобы одновременно обновить настройки стиля существующих ячеек до указанного стиля,
пожалуйста, используйте [`Column.apply_style`](/cells/python-net/ru/aspose.cells/column/apply_style)
