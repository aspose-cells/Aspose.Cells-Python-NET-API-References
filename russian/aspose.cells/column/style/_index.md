##  style недвижимость

Получает style этого столбца.

###  Примечания

 ПРИМЕЧАНИЕ. Этот элемент устарел.
Вместо этого используйте метод Column.GetStyle(), чтобы получить значение по умолчанию style для этого столбца.
 Это свойство будет удалено через 12 месяцев, начиная с декабря 2022 года.
Aspose приносит извинения за возможные неудобства.
