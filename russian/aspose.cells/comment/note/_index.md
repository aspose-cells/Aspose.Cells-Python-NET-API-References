##  note недвижимость

Представляет содержимое комментария.

###  Примечания

Если это цепочный комментарий, то note нельзя изменить, иначе MS Excel не сможет обработать его как цепочный комментарий.

###  Пример

```python

comment1.note = "First note."

```
###  Определение:
```python
@property
def note(self):
    ...
@note.setter
def note(self, value):
    ...
```
