**Aspose.Cells** Пространство имен предоставляет фундаментальные и базовые классы для создания, преобразования, изменения, рендеринга и печати электронных таблиц без использования Microsoft Excel.
Например, класс [`Workbook`](/cells/python-net/ru/aspose.cells/workbook) является основной записью для создания или загрузки, а затем управления файлом электронной таблицы,
Класс [`Cells`](/cells/python-net/ru/aspose.cells/cells) — это основной класс для управления всеми данными и свойствами на листе и т. д.
