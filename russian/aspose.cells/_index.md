**Aspose.Cells** Пространство имен предоставляет фундаментальные и базовые классы для создания, преобразования, изменения, рендеринга и печати электронных таблиц без использования Microsoft Excel.
Например, класс [`Workbook`](/cells/python-net/ru/aspose.cells/workbook) — это основная запись для создания или загрузки и последующего управления файлом электронной таблицы.
Класс [`Cells`](/cells/python-net/ru/aspose.cells/cells) — это основной класс для управления всеми данными и свойствами листа и т. д.
