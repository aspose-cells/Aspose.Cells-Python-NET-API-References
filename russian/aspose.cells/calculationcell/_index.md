##  CalculationCell класс
Представляет собой соответствующие расчету данные об одной ячейке, для которой выполняется расчет.



Тип CalculationCell предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [workbook](/cells/python-net/ru/aspose.cells/calculationcell/workbook) | Получает объект Workbook.|
| [worksheet](/cells/python-net/ru/aspose.cells/calculationcell/worksheet) | Получает объект Worksheet, в котором находится ячейка.|
| [cell_row](/cells/python-net/ru/aspose.cells/calculationcell/cell_row) | Получает индекс строки ячейки.|
| [cell_column](/cells/python-net/ru/aspose.cells/calculationcell/cell_column) | Получает индекс столбца ячейки.|
| [cell](/cells/python-net/ru/aspose.cells/calculationcell/cell) | Получает объект Cell, который вычисляется.|


###  Методы
| Метод| Описание|
| :- | :- |
| [`set_calculated_value(self, v)`](/cells/python-net/ru/aspose.cells/calculationcell/set_calculated_value/#any) | Задает расчетное значение для ячейки.|



###  Примечания

Все объекты, предоставляемые этим классом, предназначены только для чтения.
Пользователь не должен изменять какие-либо данные в Рабочей книге в процессе расчета формулы.
В противном случае может возникнуть непредвиденный результат или исключение.
