##  ValidationType перечисление
Представляет тип проверки данных.



Тип ValidationType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| ANY_VALUE | Любой тип проверки значения.|
| WHOLE_NUMBER | Тип проверки целого числа.|
| DECIMAL | Десятичный тип проверки.|
| LIST | Тип проверки списка.|
| DATE | Тип проверки даты.|
| TIME | Тип проверки времени.|
| TEXT_LENGTH | Тип проверки длины текста.|
| CUSTOM | Пользовательский тип проверки.|
