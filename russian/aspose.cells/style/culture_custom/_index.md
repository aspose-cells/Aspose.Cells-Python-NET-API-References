##  culture_custom недвижимость

Получает и задает строку шаблона, зависящую от языка и региональных параметров, для числового формата.
Если для данного объекта не установлен числовой формат, будет возвращено значение null.
Если формат числа встроен, будет возвращена строка шаблона, соответствующая встроенному числу.

###  Примечания

Для встроенного числового формата, как шаблонное содержимое (например, один встроенный формат даты — «м/д/г» для некоторых локалей,
но для некоторых других локалей это становится "д/м/г") и спецификатор формата (например,
(в некоторых локалях для обозначения части года при форматировании даты используется символ, отличный от «y»)
зависят от культуры;
Для указанного пользователем формата изменяются только спецификаторы формата в соответствии с культурой,
остальные части шаблона форматирования не будут изменены.
