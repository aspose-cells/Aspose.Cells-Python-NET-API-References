##  culture_custom недвижимость

Получает и задает строку шаблона, зависящую от языка и региональных параметров, для числового формата.
Если для этого объекта не задан числовой формат, будет возвращено значение NULL.
Если числовой формат является встроенным, будет возвращена строка шаблона, соответствующая встроенному числу.

###  Примечания

Для встроенного числового формата содержимое шаблона (например, один встроенный формат даты — «m/d/y» для некоторых локалей,
но для некоторых других локалей это становится "d/m/y") и спецификатором формата (например,
некоторые локали используют символ, отличный от «y», для представления части года для форматирования даты)
зависят от культуры;
Для заданного пользователем пользовательского формата изменяются только спецификаторы формата в соответствии с культурой,
другие части шаблона форматирования не будут изменены.
