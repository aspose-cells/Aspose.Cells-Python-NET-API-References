##  FilterType перечисление
Тип фильтра.



Тип FilterType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| COLOR_FILTER | Фильтрация по цвету заливки ячейки.|
| CUSTOM_FILTERS | Пользовательский тип фильтра.|
| DYNAMIC_FILTER | Тип динамического фильтра.|
| MULTIPLE_FILTERS | Если для фильтрации выбрано несколько значений или если для фильтрации выбрана группа значений даты,<br/> этот элемент группирует эти критерии вместе.|
| ICON_FILTER | Фильтрация по значку условного форматирования.|
| TOP10 | Топ-10 фильтров.|
| NONE | Нет фильтра.|
