##  FilterType перечисление
Тип фильтра.



Тип FilterType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| COLOR_FILTER |Фильтр по цвету заливки ячейки.|
| CUSTOM_FILTERS | Пользовательский тип фильтра.|
| DYNAMIC_FILTER | Тип фильтра — динамический.|
| MULTIPLE_FILTERS | Если для фильтрации выбрано несколько значений или группа значений дат,<br/> этот элемент группирует эти критерии вместе.|
| ICON_FILTER | Фильтр по значку условного форматирования.|
| TOP10 | Фильтр Топ 10.|
| NONE | Фильтра нет.|
