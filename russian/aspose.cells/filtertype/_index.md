##  FilterType перечисление
Тип фильтра.



Тип FilterType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| COLOR_FILTER | Фильтр по цвету заливки ячейки.|
| CUSTOM_FILTERS | Пользовательский тип фильтра.|
| DYNAMIC_FILTER | Тип динамического фильтра.|
| MULTIPLE_FILTERS | Когда для фильтрации выбрано несколько значений или когда для фильтрации выбрана группа значений даты,<br/> этот элемент группирует эти критерии вместе.|
| ICON_FILTER | Фильтр по иконке условного форматирования.|
| TOP10 | Топ 10 фильтр.|
| NONE | Нет фильтра.|
