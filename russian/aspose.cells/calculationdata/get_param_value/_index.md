##  get_param_value(index) {#int}
Получает представленный объект значения параметра по заданному индексу.


###  Возвращает

Если параметр представляет собой простое значение, то возвращает простое значение.
Если параметр является ссылкой, то возвращает объект ReferredArea.
Если параметр ссылается на несколько наборов данных, то возвращает массив объектов.

```python
def get_param_value(self, index):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| index | int | индекс параметра (на основе 0)|
