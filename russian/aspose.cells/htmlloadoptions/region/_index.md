##  region недвижимость

Возвращает или задает региональные настройки, используемые для рабочей книги, которая будет загружена.

###  Примечания

Региональные настройки могут быть использованы для инициализации некоторых функций рабочей книги.
такие как шрифты, темы и т. д.
Для текстовых форматов файлов, таких как CSV, HTML, ..., региональные настройки
также будет использоваться для определения числовых форматов и преобразования текстовых значений в числовые.
или значения даты и времени для ячеек.
Эта настройка будет сохранена для экземпляра рабочей книги позже, то есть
[`WorkbookSettings.region`](/cells/python-net/ru/aspose.cells/workbooksettings#region) рабочей книги будет использовать тот же region
с этой недвижимостью.
###  Определение:
```python
@property
def region(self):
    ...
@region.setter
def region(self, value):
    ...
```
