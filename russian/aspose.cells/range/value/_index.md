##  value недвижимость

Получает и задает значение value диапазона.

###  Примечания

Если диапазон содержит несколько ячеек, возвращаемый/применяемый объект должен быть двумерным объектом массива.
###  Определение:
```python
@property
def value(self):
    ...
@value.setter
def value(self, value):
    ...
```
