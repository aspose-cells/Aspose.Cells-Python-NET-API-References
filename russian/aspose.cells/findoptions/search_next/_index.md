##  search_next недвижимость

Порядок поиска. Верно: поиск далее. False: искать предыдущий.

###  Примечания

 ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,
 используйте свойство FindOptions.SearchBackward.
 Этот ресурс будет удален через 12 месяцев, начиная с ноября 2018 года.
Aspose приносит извинения за возможные неудобства.
