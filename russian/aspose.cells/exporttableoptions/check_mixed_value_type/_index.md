##  check_mixed_value_type недвижимость

Ложь, Aspose.Cells установит тип DataColumn по типу значения первой строки для производительности.
 True, Aspose.Cells проверит, являются ли типы значений в столбце смешанными, прежде чем задать тип DataColumn.
А типы значений смешанные, тип DataColumn будет строковым.
