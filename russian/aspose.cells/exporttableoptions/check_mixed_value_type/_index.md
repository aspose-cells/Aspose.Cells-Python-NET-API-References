##  check_mixed_value_type недвижимость

False, Aspose.Cells установит тип DataColumn по типу значения первой строки для повышения производительности.
 Правда, Aspose.Cells проверит, смешаны ли типы значений в столбце, прежде чем установить тип DataColumn.
И тип значения смешанный, тип DataColumn будет строковым.
