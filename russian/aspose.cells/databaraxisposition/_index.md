##  DataBarAxisPosition перечисление
Задает положение оси для диапазона ячеек с условным форматированием в виде гистограмм.



Тип DataBarAxisPosition предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| AUTOMATIC | Отобразить ось в переменной позиции на основе отношения минимального отрицательного значения к максимальному положительному значению в диапазоне.<br/> Положительные значения отображаются слева направо.<br/> Отрицательные значения отображаются справа налево.<br/> Если все значения положительные или все значения отрицательные, ось не отображается.|
| MIDPOINT | Отобразить ось в середине ячейки независимо от набора значений в диапазоне.<br/> Положительные значения отображаются слева направо.<br/> Отрицательные значения отображаются справа налево.|
| NONE | Ось не отображается, а положительные и отрицательные значения отображаются в направлении слева направо.|
