##  allow_deleting_row недвижимость

Указывает, разрешено ли удаление строк на защищенном листе.

###  Примечания

Строки, содержащие удаляемые ячейки, должны быть разблокированы, когда лист защищен,
и опция «Выбрать разблокированные ячейки» должна быть включена.
###  Определение:
```python
@property
def allow_deleting_row(self):
    ...
@allow_deleting_row.setter
def allow_deleting_row(self, value):
    ...
```
