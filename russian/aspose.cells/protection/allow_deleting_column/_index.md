##  allow_deleting_column недвижимость

Указывает, разрешено ли удаление столбцов на защищенном листе.

###  Примечания

Столбцы, содержащие ячейки, которые нужно удалить, должны быть разблокированы, если лист защищен.
и опция «Выбрать разблокированные ячейки» должна быть включена.
###  Определение:
```python
@property
def allow_deleting_column(self):
    ...
@allow_deleting_column.setter
def allow_deleting_column(self, value):
    ...
```
