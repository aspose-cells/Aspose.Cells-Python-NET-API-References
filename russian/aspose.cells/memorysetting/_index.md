##  MemorySetting перечисление
Режимы использования памяти для модели данных ячеек.



Тип MemorySetting предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NORMAL | Режим по умолчанию для модели ячеек.|
| MEMORY_PREFERENCE | Предпочтительно производительность памяти.|
| FILE_CACHE | Предпочтительно производительность памяти и использование файла вместо памяти<br/> для сохранения данных ячеек.|
