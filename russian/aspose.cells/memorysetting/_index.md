##  MemorySetting перечисление
Варианты использования памяти.



Тип MemorySetting предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NORMAL | Вариант по умолчанию для модели ячеек.|
| MEMORY_PREFERENCE | Предпочтительна производительность памяти.<br/>При использовании этого параметра данные будут храниться в компактном формате, поэтому для распространенных сценариев это может привести к снижению затрат на память.<br/> Однако в некоторых особых случаях этот параметр также может немного снизить производительность чтения/записи.|
