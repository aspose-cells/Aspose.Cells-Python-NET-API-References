##  MemorySetting перечисление
Варианты использования памяти.



Тип MemorySetting предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| NORMAL | Параметр по умолчанию для модели ячеек.|
| MEMORY_PREFERENCE | Желательна производительность памяти.<br/>С этой опцией данные будут храниться в компактном формате, поэтому для обычных сценариев это может снизить затраты памяти.<br/> Однако в некоторых особых случаях эта опция также может немного снизить производительность чтения/записи.|
