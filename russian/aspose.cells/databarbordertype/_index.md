##  DataBarBorderType перечисление
Задает тип границы панели данных.



Тип DataBarBorderType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Панель данных не имеет границ.|
| SOLID | Панель данных имеет сплошную границу.|
