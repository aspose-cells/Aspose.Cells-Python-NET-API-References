##  DataBarBorderType перечисление
Указывает тип границы панели данных.



Тип DataBarBorderType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NONE | Панель данных не имеет границы.|
| SOLID | Панель данных имеет сплошную рамку.|
