##  html_cross_string_type недвижимость

Указывает, будет ли строка между ячейками отображаться так же, как в MS Excel, при сохранении файла Excel в формате html.
По умолчанию используется значение «По умолчанию», поэтому для строк между ячейками разница между html-файлами, созданными Aspose.Cells, и MS Excel незначительна.
Но производительность при создании больших html-файлов при установке значения Cross будет в несколько раз выше, чем при установке значения Default или Fit2Cell.
