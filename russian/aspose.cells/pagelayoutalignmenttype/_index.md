##  PageLayoutAlignmentType перечисление
Перечисляет типы выравнивания макета страницы.



Тип PageLayoutAlignmentType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| BOTTOM | Представляет собой выравнивание макета нижней части страницы.|
| CENTER | Представляет собой выравнивание макета страницы по центру.|
| LEFT | Представляет собой выравнивание макета страницы по левому краю.|
| RIGHT | Представляет собой выравнивание макета страницы по правому краю.|
| TOP | Представляет собой выравнивание верхнего края страницы.|
