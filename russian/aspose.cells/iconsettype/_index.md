##  IconSetType перечисление
 Тип набора значков для условного форматирования.
Пороговые значения для срабатывания различных значков в наборе:
настраивается, а порядок значков можно изменить.



Тип IconSetType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| ARROWS3 | Набор иконок из 3 стрелок.|
| ARROWS_GRAY3 | Набор иконок из 3 серых стрелок.|
| FLAGS3 | Набор иконок из 3 флагов.|
| SIGNS3 | Набор иконок из 3 знаков.|
| SYMBOLS3 | Набор из 3 значков символов (обведены).|
| SYMBOLS32 | Набор иконок из 3 символов (не обведены).|
| TRAFFIC_LIGHTS31 | Набор из 3 значков светофора (без оправы).|
| TRAFFIC_LIGHTS32 | Набор из 3 значков светофора с толстой черной рамкой.|
| ARROWS4 | Набор иконок из 4 стрелок.|
| ARROWS_GRAY4 | Набор иконок из 4 серых стрелок.|
| RATING4 | Набор из 4 значков оценок.|
| RED_TO_BLACK4 | Набор из 4 значков «от красного к черному».|
| TRAFFIC_LIGHTS4 | Набор из 4 значков светофора.|
| ARROWS5 | Набор иконок из 5 стрелок.|
| ARROWS_GRAY5 | Набор иконок из 5 серых стрелок.|
| QUARTERS5 | Набор иконок «5 четвертей».|
| RATING5 | Набор значков с рейтингом 5.|
| STARS3 | 3 звезды установлены|
| BOXES5 | Набор из 5 коробок|
| TRIANGLES3 | Набор из 3 треугольников|
| NONE | Никто|
| CUSTOM_SET | ПользовательскийНабор.<br/> Этот элемент доступен только для чтения.|
| SMILIES3 | 3 смайлика.<br/> Только для .ods.|
| COLOR_SMILIES3 | цветных смайлика.<br/> Только для .ods.|
