##  IconSetType перечисление
 Тип набора значков для условного форматирования.
Пороговые значения для запуска различных значков в наборе:
настраивается, а порядок значков является обратимым.



Тип IconSetType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| ARROWS3 | Набор значков из 3 стрелок.|
| ARROWS_GRAY3 | Набор значков 3 серых стрелок.|
| FLAGS3 | Набор значков из 3 флагов.|
| SIGNS3 | Набор иконок из 3 знаков.|
| SYMBOLS3 | Набор значков из 3 символов (обведен).|
| SYMBOLS32 | Набор значков из 3 символов (без кружка).|
| TRAFFIC_LIGHTS31 | Набор значков из 3 светофоров (без оправы).|
| TRAFFIC_LIGHTS32 | 3 значка светофора с толстой черной рамкой.|
| ARROWS4 | Набор значков с 4 стрелками.|
| ARROWS_GRAY4 | Набор значков из 4 серых стрелок.|
| RATING4 | 4 набора значков рейтингов.|
| RED_TO_BLACK4 |Набор из 4 значков «от красного к черному».|
| TRAFFIC_LIGHTS4 | Набор значков из 4 светофоров.|
| ARROWS5 | Набор значков из 5 стрел.|
| ARROWS_GRAY5 | Набор значков из 5 серых стрелок.|
| QUARTERS5 | Набор иконок из 5 четвертей.|
| RATING5 | Набор из 5 значков рейтинга.|
| STARS3 | 3-звездочный набор|
| BOXES5 | набор из 5 коробок|
| TRIANGLES3 | Набор из 3 треугольников|
| NONE | Никто|
| CUSTOM_SET | ПользовательскийНабор.<br/> Этот элемент доступен только для чтения.|
| SMILIES3 | 3 смайлика.<br/> Только для .ods.|
| COLOR_SMILIES3 | 3 цветных смайлика.<br/> Только для .ods.|
