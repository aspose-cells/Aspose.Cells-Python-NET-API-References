##  auto_filter недвижимость

Указывает, выполняется ли автоматическая фильтрация данных при загрузке файлов.

###  Примечания

Иногда, хотя автофильтр установлен, соответствующие строки не скрываются в файле.
Теперь работает только для файла SpreadSheetML.
###  Определение:
```python
@property
def auto_filter(self):
    ...
@auto_filter.setter
def auto_filter(self, value):
    ...
```
