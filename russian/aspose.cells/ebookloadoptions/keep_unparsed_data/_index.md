##  keep_unparsed_data недвижимость

Сохранять ли неанализированные данные в памяти для книги при ее загрузке из файла шаблона. По умолчанию верно.

###  Примечания

Для сценариев, когда пользователю нужно только прочитать некоторое содержимое из файла шаблона и не нужно сохранять книгу обратно,
установите для этого свойства значение false, это может повысить производительность, особенно при использовании его вместе с каким-либо LoadFilter,
