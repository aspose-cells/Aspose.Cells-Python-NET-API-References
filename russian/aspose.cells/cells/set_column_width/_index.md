##  set_column_width {#int-float}
Устанавливает ширину указанного столбца в обычном виде.



```python
def set_column_width(self, column, width):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| column | int | Индекс столбца.|
| width | float | Ширина столбца. Ширина столбца должна быть в диапазоне от 0 до 255.|
###  Примечания

Чтобы скрыть столбец, установите ширину столбца равной нулю.
