Устанавливает ширину указанного столбца в обычном режиме просмотра.



```python

def set_column_width(self, column, width):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| column | int | Индекс столбца.|
| width | float | Ширина столбца в символах. Ширина столбца должна быть от 0 до 255.|
###  Примечания

В электронной таблице ширина столбца измеряется как количество символов.
максимальной ширины цифр от 0 до 9, отображаемых шрифтом обычного стиля.
