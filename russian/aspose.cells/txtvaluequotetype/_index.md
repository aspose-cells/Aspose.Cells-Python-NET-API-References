##  TxtValueQuoteType перечисление
Указывает тип использования кавычек для значений в файлах текстового формата.



Тип TxtValueQuoteType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NORMAL | Все значения, содержащие специальные символы, такие как кавычки, разделители, будут заключены в кавычки.<br/>То же самое с поведением MS Excel при экспорте текстового файла.|
| ALWAYS | Все значения всегда будут указаны.|
| MINIMUM | Заключайте значения в кавычки только при необходимости. Например, если одно значение содержит кавычки, но кавычки не находятся в начале этого значения, это значение не будет заключено в кавычки.|
| NEVER | Все значения не будут заключены в кавычки. Экспортированный текстовый файл этого типа может быть прочитан неправильно из-за отсутствия необходимых кавычек.|
