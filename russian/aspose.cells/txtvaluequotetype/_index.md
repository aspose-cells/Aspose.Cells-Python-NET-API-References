##  TxtValueQuoteType перечисление
Указывает тип использования кавычек для значений в файлах текстового формата.



Тип TxtValueQuoteType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| NORMAL | Все значения, содержащие специальные символы, такие как кавычки и символы-разделители, будут заключены в кавычки.<br/> То же самое и с поведением MS Excel при экспорте текстового файла.|
| ALWAYS | Все значения будут всегда указаны.|
| MINIMUM | Цитируйте значения только при необходимости. Например, если одно значение содержит кавычку, но кавычка не находится в начале этого значения, это значение не будет заключено в кавычки.|
| NEVER |Все значения не будут указаны. Экспортированный текстовый файл этого типа может быть прочитан неправильно из-за отсутствия необходимых кавычек.|
