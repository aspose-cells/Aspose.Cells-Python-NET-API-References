##  refer_to_sheet_with_same_name недвижимость

При копировании листа в другую книгу, если лист содержит формулы, которые ссылаются на другие листы в MS Excel,
скопированные формулы должны ссылаться на исходную книгу.
Но иногда мы копировали другие рабочие листы и надеемся, что скопированные формулы ссылаются на другие рабочие листы с тем же именем в той же рабочей книге,
установите для этого свойства значение true.

###  Примечания

Значение по умолчанию верно.

```python
@property
def refer_to_sheet_with_same_name(self):
    ...
@refer_to_sheet_with_same_name.setter
def refer_to_sheet_with_same_name(self, value):
    ...
```
