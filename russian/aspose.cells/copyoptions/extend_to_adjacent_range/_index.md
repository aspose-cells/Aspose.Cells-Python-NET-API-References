##  extend_to_adjacent_range недвижимость

Указывает, следует ли расширять диапазоны при копировании диапазона в соседний диапазон.

###  Примечания

Если значение равно true, то только расширяется диапазон гиперссылки, а не добавляется новая гиперссылка при копировании гиперссылок в соседние строки.

```python
@property
def extend_to_adjacent_range(self):
    ...
@extend_to_adjacent_range.setter
def extend_to_adjacent_range(self, value):
    ...
```
