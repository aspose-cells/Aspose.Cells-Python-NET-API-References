##  NameCollection класс
Представляет собой коллекцию всех объектов [`Name`](/cells/python-net/ru/aspose.cells/name) в электронной таблице.



Тип NameCollection предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells/namecollection/capacity) | Возвращает или задает количество элементов, которые может содержать список массива.|
