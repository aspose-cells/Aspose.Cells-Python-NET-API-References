##  AutoFillType перечисление
Представляет тип автоматического заполнения.



Тип AutoFillType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| COPY | Копирует значение и формат исходной области в целевую область.|
| DEFAULT | Автоматически заполняет целевую область значением и форматом.|
| FORMATS |Копирует только формат исходной области в целевую область.|
| SERIES | Расширьте значение из исходной области до целевой области в виде серии и скопируйте формат в целевую область.|
| VALUES | Копирует только значение исходной области в целевую область.|
