##  FormatConditionValueType перечисление
Тип значения условия.



Тип FormatConditionValueType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| FORMULA | Минимальное/среднее/максимальное значение для<br/> градиент определяется по формуле.|
| MAX | Указывает, что максимальное значение в диапазоне должно быть<br/> используется как максимальное значение градиента.|
| MIN | Указывает, что минимальное значение в диапазоне должно быть<br/> используется как минимальное значение градиента.|
| NUMBER | Указывает, что минимум/средняя точка/максимум<br/>значение градиента задается константой<br/> числовое значение.|
| PERCENT | Значение указывает процент между минимальным<br/>и максимальные значения в диапазоне должны использоваться в качестве<br/> минимальное/среднее/максимальное значение градиента.|
| PERCENTILE |Значение указывает, что процентильный рейтинг в диапазоне должен<br/>использоваться в качестве минимального/среднего/максимального значения<br/> для градиента.|
| AUTOMATIC_MAX | Указывает, что автоматическое максимальное значение в диапазоне должно быть<br/> используется в качестве автоматического максимального значения градиента.|
| AUTOMATIC_MIN | Указывает, что автоматическое минимальное значение в диапазоне должно быть<br/> используется в качестве автоматического минимального значения градиента.|
