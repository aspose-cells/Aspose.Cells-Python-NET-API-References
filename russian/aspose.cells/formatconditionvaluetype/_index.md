##  FormatConditionValueType перечисление
Тип значения условия.



Тип FormatConditionValueType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| FORMULA | Минимальное/среднее/максимальное значение для<br/> градиент определяется по формуле.|
| MAX | Указывает, что максимальное значение в диапазоне должно быть<br/> используется как максимальное значение для градиента.|
| MIN | Указывает, что минимальное значение в диапазоне должно быть<br/> используется как минимальное значение для градиента.|
| NUMBER | Указывает, что минимум / средняя точка / максимум<br/>значение градиента задается константой<br/> числовое значение.|
| PERCENT | Значение указывает процент между минимальным<br/>и максимальные значения в диапазоне должны использоваться в качестве<br/> минимальное / среднее / максимальное значение градиента.|
| PERCENTILE | Значение указывает процентиль рейтинга в диапазоне<br/>использоваться как минимальное / среднее / максимальное значение<br/> для градиента.|
| AUTOMATIC_MAX | Указывает, что автоматическое максимальное значение в диапазоне должно быть<br/>используется как автоматическое максимальное значение для градиента.|
| AUTOMATIC_MIN | Указывает, что автоматическое минимальное значение в диапазоне должно быть<br/> используется как автоматическое минимальное значение для градиента.|
