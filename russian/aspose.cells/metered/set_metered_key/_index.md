##  set_metered_key {#str-str}
Устанавливает измеренный открытый и закрытый ключ.
 Если вы приобретаете лимитную лицензию, при запуске приложения следует позвонить по этому номеру API, обычно этого достаточно.
 Однако, если данные о потреблении не загружаются и превышают 24 часа, лицензия будет переведена в оценочный статус.
Чтобы избежать такого случая, вам следует регулярно проверять статус лицензии. Если это оценочный статус, позвоните еще раз по этому номеру API.



```python
def set_metered_key(self, public_key, private_key):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| public_key | str | открытый ключ|
| private_key | str | закрытый ключ|
