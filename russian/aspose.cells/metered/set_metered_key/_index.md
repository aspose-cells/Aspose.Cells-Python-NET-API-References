##  set_metered_key(public_key, private_key) {#str-str}
Устанавливает измеренный открытый и закрытый ключ.
Если вы покупаете лимитную лицензию, при запуске приложения следует позвонить по этому номеру API, обычно этого достаточно. Однако, если всегда не удается загрузить данные о потреблении и превышает 24 часа, лицензия будет установлена в статусе оценки, чтобы избежать этого, вы должны регулярно проверять статус лицензии, если это статус оценки, снова звоните по этому номеру API.



```python
def set_metered_key(self, public_key, private_key):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| public_key | str | открытый ключ|
| private_key | str | закрытый ключ|
