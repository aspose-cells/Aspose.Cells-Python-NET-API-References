##  FilterColumn класс
Представляет фильтр для одного столбца. Объект Filter является членом коллекции Filters.



Тип FilterColumn предоставляет следующие члены:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [is_dropdown_visible](/cells/python-net/ru/aspose.cells/filtercolumn/is_dropdown_visible) | Указывает, видна ли кнопка автофильтра для этого столбца.|
| [visibledropdown](/cells/python-net/ru/aspose.cells/filtercolumn/visibledropdown) | Указывает, видна ли кнопка автофильтра для этого столбца.|
| [filter](/cells/python-net/ru/aspose.cells/filtercolumn/filter) | Получает и задает условие фильтрации данных.|
| [filter_type](/cells/python-net/ru/aspose.cells/filtercolumn/filter_type) | Получает и задает тип для фильтрации данных.|
| [field_index](/cells/python-net/ru/aspose.cells/filtercolumn/field_index) | Получает и задает смещение столбца в диапазоне.|
