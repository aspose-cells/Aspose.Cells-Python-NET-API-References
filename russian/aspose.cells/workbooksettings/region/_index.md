##  region недвижимость

Получает или задает региональные настройки для книги.

###  Примечания

1. Региональные настройки, используемые компонентом Aspose.Cells для рабочей книги, загруженной из файла шаблона:
i). Для файла XLS определены поля для региональных настроек, и MS Excel сохраняет данные региональных настроек в файл при сохранении файла XLS.
Итак, мы используем сохраненный region в файле шаблона для рабочей книги.
Если вы не хотите использовать код region, сохраненный в файле XLS, пожалуйста, смените его на ожидаемый (например, CountryCode.Default) после загрузки файла шаблона.
И мы также сохраняем указанное пользователем значение (этим методом) в файле при сохранении файла XLS.
ii). Для других форматов файлов, таких как XLSX, XLSB... и т. д., в спецификации формата файла не определено поле для региональных настроек.
Поэтому мы используем региональные настройки среды приложения для рабочей книги.
И указанное пользователем значение (этим методом) не может быть сохранено для созданных файлов с этими форматами файлов.
2. Для эффекта просмотра в MS Excel:
Примененные здесь региональные настройки могут вступить в силу только во время выполнения компонента Aspose.Cells, а не при просмотре сгенерированного файла в MS Excel.
Даже для созданного файла XLS, в котором сохранены указанные данные региональных настроек, при просмотре/редактировании его в MS Excel,
используемый region для выполнения форматирования MS Excel всегда является региональными настройками по умолчанию среды, в которой работает MS Excel,
Не тот, который сохранён в файле. Это поведение MS Excel, и его нельзя изменить кодом.
