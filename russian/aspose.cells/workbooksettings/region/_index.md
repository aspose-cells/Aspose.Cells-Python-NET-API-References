##  region недвижимость

Получает или задает региональные параметры книги.

###  Примечания

1. Региональные настройки, используемые компонентом Aspose.Cells для книги, загруженной из файла шаблона:
я). Для файла XLS есть поля, определенные для региональных настроек, и MS Excel сохраняет данные региональных настроек в файл при сохранении файла XLS.
Итак, мы используем сохраненный region в файле шаблона для книги.
Если вы не хотите использовать region, сохраненный в файле XLS, сбросьте его на ожидаемый (например, CountryCode.Default) после загрузки файла шаблона.
И мы сохраняем указанное пользователем значение (этим методом) в файл также при сохранении файла XLS.
ii). Для других форматов файлов, таких как XLSX, XLSB... и т. д., в спецификации формата файла не определено поле для региональных настроек.
Итак, мы используем региональные настройки среды приложения для книги.
Кроме того, указанное пользователем значение (с помощью этого метода) не может быть сохранено для созданных файлов этих форматов.
2. Для эффекта просмотра в MS Excel:
Примененные здесь региональные настройки могут вступить в силу только во время выполнения с компонентом Aspose.Cells, а не при просмотре сгенерированного файла в MS Excel.
Даже для сгенерированного файла XLS, в котором сохранены указанные данные региональных настроек, при просмотре/редактировании его в MS Excel,
используемый для форматирования MS Excel номер region всегда является региональными настройками по умолчанию для среды, в которой работает MS Excel,
не тот, который сохранен в файле. Это поведение MS Excel, и его нельзя изменить с помощью кода.
