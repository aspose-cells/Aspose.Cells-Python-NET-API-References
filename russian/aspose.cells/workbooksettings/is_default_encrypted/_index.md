##  is_default_encrypted недвижимость

Указывает, следует ли шифровать книгу с паролем по умолчанию, если структура и Windows книги заблокированы.

###  Примечания

Значение по умолчанию теперь равно false. Это то же самое, что MS Excel 2013.
