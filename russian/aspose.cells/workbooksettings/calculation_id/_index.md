##  calculation_id недвижимость

Указывает версию механизма вычислений, используемого для вычисления значений в книге.

ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо этого используйте FormulaSettings.CalculationId.

Aspose приносит извинения за возможные неудобства.
