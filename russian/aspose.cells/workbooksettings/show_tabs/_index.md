##  show_tabs недвижимость

Получить или задать значение, указывающее, отображаются ли вкладки рабочей книги.

###  Примечания

Значение по умолчанию — true.

###  Пример

Следующий код скрывает вкладки листов и кнопки прокрутки вкладок для электронной таблицы.

```python

#  Hide the spreadsheet tabs.
workbook.settings.show_tabs = False

```
