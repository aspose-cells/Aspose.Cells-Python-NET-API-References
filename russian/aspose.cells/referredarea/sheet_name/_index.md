##  sheet_name недвижимость

Указывает, на каком листе находится данная ссылка.

###  Примечания

Если он ссылается на несколько рабочих листов,
возвращаемое значение аналогично выражению диапазона в формуле.
Например, «Лист1:Лист3» для ссылки в формуле «=СУММ(Лист1:Лист3!$A$1:$B$2)».

def sheet_name(self):
    ...
