##  built_in_preference недвижимость

Указывает, является ли свойство числового формата предпочтительным, когда стиль определяет как встроенное число, так и пользовательский шаблон.
Значение по умолчанию — false, что означает, что по умолчанию для форматирования значений будет использоваться пользовательский шаблон, если только он не является пустым для одного стиля.

###  Примечания

При загрузке рабочей книги из существующего файла шаблона для одного стиля могут быть определены как встроенный номер, так и пользовательский шаблон.
Это свойство определяет, следует ли использовать встроенное число или пользовательский шаблон при форматировании значений с помощью стиля.

    ...
