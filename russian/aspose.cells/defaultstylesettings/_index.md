##  DefaultStyleSettings класс
Параметры значений по умолчанию свойств стиля книги.



Тип DefaultStyleSettings предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [font_name](/cells/python-net/ru/aspose.cells/defaultstylesettings/font_name) | Получает/устанавливает имя шрифта по умолчанию для книги.|
| [font_size](/cells/python-net/ru/aspose.cells/defaultstylesettings/font_size) | Получает/устанавливает стандартный размер шрифта по умолчанию для книги.|
| [horizontal_alignment](/cells/python-net/ru/aspose.cells/defaultstylesettings/horizontal_alignment) | Получает/устанавливает значение по умолчанию для горизонтального выравнивания.|
| [vertical_alignment](/cells/python-net/ru/aspose.cells/defaultstylesettings/vertical_alignment) | Получает/устанавливает значение по умолчанию для вертикального выравнивания.|
