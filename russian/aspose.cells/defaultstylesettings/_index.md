##  DefaultStyleSettings класс
Настройки значений по умолчанию для свойств стиля рабочей книги.



Тип DefaultStyleSettings предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [built_in_preference](/cells/python-net/ru/aspose.cells/defaultstylesettings/built_in_preference) | Указывает, является ли свойство числового формата предпочтительным, когда стиль определяет как встроенное число, так и пользовательский шаблон.<br/> Значение по умолчанию — false, что означает, что по умолчанию для форматирования значений будет использоваться пользовательский шаблон, если только он не является пустым для одного стиля.|
| [font_name](/cells/python-net/ru/aspose.cells/defaultstylesettings/font_name) | Получает/задает имя шрифта по умолчанию для рабочей книги|
| [font_size](/cells/python-net/ru/aspose.cells/defaultstylesettings/font_size) | Возвращает/задает стандартный размер шрифта по умолчанию для рабочей книги.|
| [horizontal_alignment](/cells/python-net/ru/aspose.cells/defaultstylesettings/horizontal_alignment) | Получает/задает значение по умолчанию для горизонтального выравнивания|
| [vertical_alignment](/cells/python-net/ru/aspose.cells/defaultstylesettings/vertical_alignment) | Получает/задает значение по умолчанию для вертикального выравнивания|
