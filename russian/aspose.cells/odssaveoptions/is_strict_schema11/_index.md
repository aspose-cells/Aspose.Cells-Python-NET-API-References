##  is_strict_schema11 недвижимость

Указывает, следует ли сохранять файл ods в формате ODF версии 1.1. По умолчанию — ложь.

###  Примечания

 ПРИМЕЧАНИЕ. Этот элемент устарел. Вместо,
используйте свойство OdsSaveOptions.OdfStrictVersion.
 Этот метод будет удален через 12 месяцев, с февраля 2024 года.
Aspose приносит извинения за возможные неудобства.
