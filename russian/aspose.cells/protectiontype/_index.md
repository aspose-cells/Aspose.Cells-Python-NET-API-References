##  ProtectionType перечисление
Представляет тип защиты книги/листа.



Тип ProtectionType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| ALL | Олицетворяет защиту всех.|
| CONTENTS | Представляет собой защиту содержимого, используемого в защите рабочего листа.|
| OBJECTS | Представляет собой защиту объектов, используемых в защите рабочего листа.|
| SCENARIOS | Представляет сценарии защиты, используемые в защите рабочего листа.|
| STRUCTURE | Представляет защитную структуру, используемую при защите книги.|
| WINDOWS | Представляет окно защиты, используемое при защите книги.|
| NONE | Не представляет никакой защиты. Только для недвижимости в Ридинге.|
