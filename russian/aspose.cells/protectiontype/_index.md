##  ProtectionType перечисление
Представляет собой тип защиты рабочей книги/рабочего листа.



Тип ProtectionType предоставляет следующие элементы:

###  Поля
| Поле| Описание|
| :- | :- |
| ALL | Олицетворяет защиту всех.|
| CONTENTS | Представляет собой защиту содержимого, используемого для защиты рабочего листа.|
| OBJECTS | Представляет собой защиту объектов, используется для защиты рабочего листа.|
| SCENARIOS | Представляет собой защиту сценариев, используемых при защите рабочего листа.|
| STRUCTURE | Представляет собой защитную структуру, используемую для защиты рабочей книги.|
| WINDOWS | Представляет собой защитное окно, используемое для защиты рабочей книги.|
| NONE | Не представляет никакой защиты. Только для объекта Reading.|
