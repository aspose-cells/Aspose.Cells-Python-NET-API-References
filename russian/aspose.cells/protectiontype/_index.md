##  ProtectionType перечисление
Представляет тип защиты книги/листа.



Тип ProtectionType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| ALL | Представляет для защиты всех.|
| CONTENTS | Представляет для защиты содержимого, используемого в защите рабочего листа.|
| OBJECTS | Представляет для защиты объектов, используемых в защите рабочего листа.|
| SCENARIOS | Представляет сценарии защиты, используемые в защите рабочего листа.|
| STRUCTURE | Представляет структуру защиты, используемую в защите рабочей книги.|
| WINDOWS | Представляет окно защиты, используемое в защите рабочей книги.|
| NONE | Не представляет защиты. Только для свойства чтения.|
