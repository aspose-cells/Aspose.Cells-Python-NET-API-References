##  AutoFitWrappedTextType перечисление
Представляет тип автоподбора обернутого текста.



Тип AutoFitWrappedTextType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| DEFAULT | Работает как MS Excel.|
| PARAGRAPH | Автоподбор ширины по самому длинному абзацу.|
