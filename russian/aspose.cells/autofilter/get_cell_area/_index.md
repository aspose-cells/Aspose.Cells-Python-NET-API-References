##  get_cell_area(self, refresh_applied_range) {#bool}
Возвращает [`CellArea`](/cells/python-net/ru/aspose.cells/cellarea), к которому применяется указанный автофильтр.


###  Возврат

область, к которой применяется этот фильтр


```python

def get_cell_area(self, refresh_applied_range):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| refresh_applied_range | bool | Обновлять ли применяемый диапазон.<br/>Для примененного диапазона автоматического фильтра последняя строка может измениться при изменении данных в ячейках.<br/> Если этот флаг равен true, то последняя строка диапазона будет пересчитана в соответствии с текущими данными ячеек.|
