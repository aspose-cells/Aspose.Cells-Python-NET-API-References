##  get_leafs() {#}
Получите все ячейки, которые ссылаются на эту ячейку напрямую и должны быть обновлены при изменении этой ячейки.


###  Возвращает

Перечислитель для перечисления всех иждивенцев (Cell)


```python
def get_leafs(self):
    ...
```


###  Примечания

ПРИМЕЧАНИЕ. Этот класс устарел. Вместо,
пожалуйста, используйте Cell.GetDependentsInCalculation(bool), чтобы получить все иждивенцы в цепочке вычислений.
Это свойство будет удалено через 12 месяцев, начиная с мая 2022 года.
Aspose приносит извинения за возможные неудобства.

##  get_leafs(recursive) {#bool}
Получить все ячейки, которые будут обновлены при изменении этой ячейки.


###  Возвращает

Перечислитель для перечисления всех иждивенцев (Cell)


```python
def get_leafs(self, recursive):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| recursive | bool | Возвращает ли те листы, которые не ссылаются на эту ячейку напрямую<br/> но ссылка на другие листы этой ячейки|
###  Примечания

ПРИМЕЧАНИЕ. Этот класс устарел. Вместо,
пожалуйста, используйте Cell.GetDependentsInCalculation(bool), чтобы получить все иждивенцы в цепочке вычислений.
Это свойство будет удалено через 12 месяцев, начиная с мая 2022 года.
Aspose приносит извинения за возможные неудобства.
