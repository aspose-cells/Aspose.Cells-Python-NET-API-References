##  get_leafs(self) {#}
Получить все ячейки, которые напрямую ссылаются на данную ячейку и должны быть обновлены при изменении данной ячейки.


###  Возврат

Перечислитель для перечисления всех иждивенцев (Cell)


```python

def get_leafs(self):
    ...
```


###  Примечания

ПРИМЕЧАНИЕ: Этот класс устарел. Вместо этого
используйте Cell.GetDependentsInCalculation(bool) для получения всех зависимых элементов в цепочке вычислений.
Эта недвижимость будет снесена через 12 месяцев с мая 2022 года.
Aspose приносит извинения за любые причиненные вам неудобства.

##  get_leafs(self, recursive) {#bool}
Получить все ячейки, которые будут обновлены при изменении данной ячейки.


###  Возврат

Перечислитель для перечисления всех иждивенцев (Cell)


```python

def get_leafs(self, recursive):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| recursive | bool | Возвращает ли те листья, которые не ссылаются на эту ячейку напрямую<br/> но ссылка на другие листы этой ячейки|
###  Примечания

ПРИМЕЧАНИЕ: Этот класс устарел. Вместо этого
используйте Cell.GetDependentsInCalculation(bool) для получения всех зависимых элементов в цепочке вычислений.
Эта недвижимость будет снесена через 12 месяцев с мая 2022 года.
Aspose приносит извинения за любые причиненные вам неудобства.
