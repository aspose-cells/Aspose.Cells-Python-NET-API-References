##  copy_to(array) {#list}
Копирует весь список массивов в совместимый список одномерных массивов, начиная с начала целевого списка массивов.



```python
def copy_to(self, array):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| array | list | Список одномерных массивов, который является местом назначения списка скопированных массивов элементов. Список массивов должен иметь индексацию с отсчетом от нуля.|


##  copy_to(index, array, array_index, count) {#int-list-int-int}
Копирует диапазон элементов из списка массивов в совместимый список одномерных массивов, начиная с указанного индекса целевого списка массивов.



```python
def copy_to(self, index, array, array_index, count):
    ...
```


| Параметр| Тип| Описание|
| :- | :- | :- |
| index | int | Отсчитываемый от нуля индекс в списке исходных массивов, с которого начинается копирование.|
| array | list | Список одномерных массивов, который является местом назначения элементов, скопированных из списка массивов. Список массивов должен иметь индексацию с отсчетом от нуля.|
| array_index | int |Отсчитываемый от нуля индекс в списке массивов, с которого начинается копирование.|
| count | int | Количество элементов для копирования.|
