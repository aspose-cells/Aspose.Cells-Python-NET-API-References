##  CustomXmlPartCollection класс
Представляет часть хранилища пользовательских данных XML (пользовательские данные XML в пакете).



Тип CustomXmlPartCollection предоставляет следующие члены:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection/capacity) | Получает или задает количество элементов, которые может содержать список массивов.|


###  Методы
| Метод| Описание|
| :- | :- |
| [copy_to(array)](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection/copy_to/#list) | Копирует весь список массивов в совместимый список одномерных массивов, начиная с начала целевого списка массивов.|
| [copy_to(index, array, array_index, count)](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection/copy_to/#int-list-int-int) |Копирует диапазон элементов из списка массивов в совместимый список одномерных массивов, начиная с указанного индекса целевого списка массивов.|
| [index_of(item, index)](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection/index_of/#CustomXmlPart-int) | Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс первого вхождения в диапазоне элементов в списке массивов, который простирается от указанного индекса до последнего элемента.|
| [index_of(item, index, count)](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection/index_of/#CustomXmlPart-int-int) | Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс первого вхождения в диапазоне элементов в списке массивов, который начинается с указанного индекса и содержит указанное количество элементов.|
| [last_index_of(item)](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection/last_index_of/#CustomXmlPart) | Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс последнего вхождения во всем списке массивов.|
| [last_index_of(item, index)](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection/last_index_of/#CustomXmlPart-int) | Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс последнего вхождения в диапазоне элементов в списке массивов, который простирается от первого элемента до указанного индекса.|
| [last_index_of(item, index, count)](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection/last_index_of/#CustomXmlPart-int-int) |Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс последнего вхождения в диапазоне элементов в списке массивов, который содержит указанное количество элементов и заканчивается указанным индексом.|
| [add(data, shema_data)](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection/add/#bytes-bytes) | Добавляет элемент в коллекцию.|
| [select_by_id(id)](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection/select_by_id/#str) | Получает элемент по идентификатору.|
| [binary_search(item)](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection/binary_search/#CustomXmlPart) | Ищет элемент во всем отсортированном списке массивов, используя компаратор по умолчанию, и возвращает отсчитываемый от нуля индекс элемента.|
