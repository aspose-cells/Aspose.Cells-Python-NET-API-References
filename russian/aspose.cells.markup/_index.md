**Aspose.Cells.Markup** пространство имен содержит классы, которые представляют определяемую пользователем семантику в документе: смарт-теги, пользовательский XML и другие теги разметки.

###  Классы
| Сорт| Описание|
| :- | :- |
| [CustomXmlPart](/cells/python-net/ru/aspose.cells.markup/customxmlpart) | Представляет часть хранилища пользовательских данных XML (пользовательские данные XML в пакете).|
| [CustomXmlPartCollection](/cells/python-net/ru/aspose.cells.markup/customxmlpartcollection) | Представляет часть хранилища пользовательских данных XML (пользовательские данные XML в пакете).|
| [SmartTag](/cells/python-net/ru/aspose.cells.markup/smarttag) | Представляет смарт-тег.|
| [SmartTagCollection](/cells/python-net/ru/aspose.cells.markup/smarttagcollection) | Представляет все смарт-теги в ячейке.|
| [SmartTagOptions](/cells/python-net/ru/aspose.cells.markup/smarttagoptions) | Представляет параметры смарт-тега.|
| [SmartTagProperty](/cells/python-net/ru/aspose.cells.markup/smarttagproperty) | Представляет свойство смарт-тега ячейки.|
| [SmartTagPropertyCollection](/cells/python-net/ru/aspose.cells.markup/smarttagpropertycollection) |Представляет все свойства смарт-тега ячейки.|
| [SmartTagSetting](/cells/python-net/ru/aspose.cells.markup/smarttagsetting) | Представляет все объекты [SmartTagCollection](/cells/python-net/ru/aspose.cells.markup/smarttagcollection) на листе.|
