##  WebExtensionTaskPane класс
Представляет собой сохраненный объект области задач.



Тип WebExtensionTaskPane предоставляет следующие элементы:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [web_extension](/cells/python-net/ru/aspose.cells.webextensions/webextensiontaskpane/web_extension) | Получает и задает часть веб-расширения, связанную с экземпляром области задач.|
| [dock_state](/cells/python-net/ru/aspose.cells.webextensions/webextensiontaskpane/dock_state) | Получает и задает последнее закрепленное местоположение этого объекта области задач.|
| [is_visible](/cells/python-net/ru/aspose.cells.webextensions/webextensiontaskpane/is_visible) |Указывает, отображается ли панель задач по умолчанию при открытии документа.|
| [is_locked](/cells/python-net/ru/aspose.cells.webextensions/webextensiontaskpane/is_locked) | Указывает, заблокирована ли область задач для документа в пользовательском интерфейсе и не может ли пользователь ее закрыть.|
| [width](/cells/python-net/ru/aspose.cells.webextensions/webextensiontaskpane/width) | Получает и задает значение ширины по умолчанию для этого экземпляра области задач.|
| [row](/cells/python-net/ru/aspose.cells.webextensions/webextensiontaskpane/row) | Возвращает и задает индекс, перечисляя снаружи внутрь, этой области задач среди других сохраненных областей задач, закрепленных в том же месте по умолчанию.|
