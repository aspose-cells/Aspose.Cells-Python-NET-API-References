##  WebExtensionCollection класс
Представляет список веб-расширений.



Тип WebExtensionCollection предоставляет следующие элементы:

| Свойство| Описание|
| :- | :- |
| [capacity](/cells/python-net/ru/aspose.cells.webextensions/webextensioncollection/capacity) | Возвращает или задает количество элементов, которые может содержать список массива.|
