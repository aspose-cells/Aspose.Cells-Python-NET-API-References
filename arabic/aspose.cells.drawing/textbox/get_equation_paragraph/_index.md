##  get_equation_paragraph(self) {#}
يحصل على الفقرة الرياضية الأولى من خاصية TextBody الخاصة بكائن TextBox.


###  عائدات

إذا كان هناك فقرة رياضية، يتم إرجاع الفقرة الأولى، وإلا يتم إرجاع قيمة null.


```python

def get_equation_paragraph(self):
    ...
```




##  get_equation_paragraph(self, index) {#int}
احصل على فقرة الرياضيات المحددة من خاصية TextBody الخاصة بكائن TextBox.
يلاحظ:
(1) يتم إرجاع NULL عندما يكون الفهرس خارج الحدود أو غير موجود.
(2) يتم أيضًا إرجاع NULL إذا لم يكن موضع الفهرس المحدد فقرة رياضية.


###  عائدات

إرجاع الفقرة الرياضية المحددة بواسطة الفهرس.


```python

def get_equation_paragraph(self, index):
    ...
```


| معامل| يكتب| وصف|
| :- | :- | :- |
| index | int | مؤشر موضع الفقرة الرياضية، يبدأ من 0.|
