##  id fastighet

Anger ett GUID som kan korsrefereras med GUID:t för signaturraden som lagras i dokumentinnehållet.
Standardvärdet är tomt (bara nollor) Guid.

###  Anmärkningar

När den är inställd associerar den [`SignatureLine`](/cells/python-net/sv/aspose.cells.drawing/signatureline) med motsvarande [`DigitalSignature`](/cells/python-net/sv/aspose.cells.digitalsignatures/digitalsignature).
