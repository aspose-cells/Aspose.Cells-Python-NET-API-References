##  html_cross_string_type fastighet

Anger om en sträng mellan celler kommer att visas på samma sätt som i MS Excel när en Excel-fil sparas i html-format.
Som standard är värdet Standard, så för strängar som sträcker sig över flera celler är det liten skillnad mellan html-filerna som skapas av Aspose.Cells och MS Excel.
Men prestandan för att skapa stora html-filer, om värdet ställs in på Cross, skulle vara flera gånger snabbare än att ställa in det på Default eller Fit2Cell.
