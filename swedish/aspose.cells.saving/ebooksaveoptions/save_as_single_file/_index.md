##  save_as_single_file fastighet

Anger om html-filen ska sparas som en enda fil.
Standardvärdet är falskt.

###  Anmärkningar

Om det finns flera arbetsblad eller andra nödvändiga resurser, till exempel bilder, i arbetsboken,
 Vanligtvis behöver dessa arbetsblad och andra resurser sparas i separata filer.
I vissa fall kan användaren bara behöva hämta en resulterande fil, till exempel för att underlätta överföringen.
Om så är fallet kan användaren ange den här egenskapen som sann.
