##  part_index fastighet

Index för aktuell del i sekvens (0-baserat).
-1 betyder att det inte finns några flera delar så resultatet är ett enda.

###  Anmärkningar

Om flera ark behöver bearbetas och varje ark bearbetas (delas)
separat börjar delindexet alltid från 0 för varje ark.
Till exempel, när man konverterar arbetsboken till bilder,
Den representerar utdatasidans index för det ark som för närvarande bearbetas.
Och -1 anger att det bara finns en sida för det aktuella arket.
