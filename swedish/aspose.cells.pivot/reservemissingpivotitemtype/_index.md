##  ReserveMissingPivotItemType uppräkning
Representerar hur man behåller de saknade pivotobjekten.



Typen ReserveMissingPivotItemType avslöjar följande medlemmar:

###  Fält
| Fält| Beskrivning|
| :- | :- |
| DEFAULT | Tar bort gamla saknade pivotobjekt och reserverar synliga objekt som den aktuella datakällan inte innehåller som saknade objekt.|
| ALL | Reserverar alla saknade föremål.|
| NONE | Tar bort alla saknade pivotobjekt.|
