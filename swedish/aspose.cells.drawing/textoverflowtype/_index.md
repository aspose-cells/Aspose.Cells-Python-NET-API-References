##  TextOverflowType uppräkning
Representerar hur texten flyter över vertikalt eller horisontellt.



Typen TextOverflowType avslöjar följande medlemmar:

###  Fält
| Fält| Beskrivning|
| :- | :- |
| CLIP | Var uppmärksam på övre och nedre barriärer.<br/> Ange ingen indikation på att det finns text som inte är synlig.|
| ELLIPSIS | Var uppmärksam på övre och nedre barriärer.<br/>Använd en ellips för att ange att det finns text som inte är synlig.<br/> Endast för vertikalt överflöde.|
| OVERFLOW | Överfyll texten och var inte uppmärksam på övre och nedre barriärer.|
