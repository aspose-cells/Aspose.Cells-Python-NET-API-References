##  DrawObject klass
DrawObject kommer att initieras och returneras vid rendering.



Typen DrawObject avslöjar följande medlemmar:

###  Egenskaper
| Fast egendom| Beskrivning|
| :- | :- |
| [cell](/cells/python-net/sv/aspose.cells.rendering/drawobject/cell) |Indikerar objektet Cell vid rendering.<br/> Alla egenskaper för cellen kan nås.|
| [shape](/cells/python-net/sv/aspose.cells.rendering/drawobject/shape) | Indikerar Shape-objektet vid rendering.<br/> Alla formegenskaper kan nås.|
| [image_bytes](/cells/python-net/sv/aspose.cells.rendering/drawobject/image_bytes) | Indikerar bildbytes av renderat diagram, form vid rendering.|
| [type](/cells/python-net/sv/aspose.cells.rendering/drawobject/type) | Indikerar typen av DrawObject.|
| [current_page](/cells/python-net/sv/aspose.cells.rendering/drawobject/current_page) | Indikerar sidindexet för DrawObject.<br/>Sidindex baseras på noll.<br/> Ett ark innehåller flera sidor vid rendering.|
| [total_pages](/cells/python-net/sv/aspose.cells.rendering/drawobject/total_pages) | Indikerar totalt antal sidor i aktuell rendering.|
| [sheet_index](/cells/python-net/sv/aspose.cells.rendering/drawobject/sheet_index) | Indikerar aktuellt arkindex för DrawObject.|
