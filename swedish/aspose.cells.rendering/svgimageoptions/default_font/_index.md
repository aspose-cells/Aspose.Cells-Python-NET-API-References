##  default_font fastighet

När tecken i Excel är Unicode och inte är inställda med rätt typsnitt i cellstilen,
De kan visas som block i pdf-filen eller bilden.
Ställ in standardteckensnittet, till exempel MingLiu eller MS Gothic, för att visa dessa tecken.
Om den här egenskapen inte är inställd kommer Aspose.Cells att använda systemets standardteckensnitt för att visa dessa Unicode-tecken.
