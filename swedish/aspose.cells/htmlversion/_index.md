##  HtmlVersion uppräkning
Anger att version HTML används när man sparar i HTML-format.



Typen HtmlVersion avslöjar följande medlemmar:

###  Fält
| Fält| Beskrivning|
| :- | :- |
| DEFAULT | Spara dokumentet i enlighet med MS Excel-exporten HTML.|
| X_HTML |Sparar dokumentet i enlighet med XHTML 1.0 Transitional-standarden.|
| HTML5 | Sparar dokumentet i enlighet med standarden HTML 5.|
