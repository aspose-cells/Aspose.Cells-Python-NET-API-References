##  get_pivot_total_name {#}
Hämtar namnet på etiketten "Totalt" i pivottabellen.
Du måste åsidosätta den här metoden när pivottabellen innehåller två eller flera pivotfält i dataområdet.


###  Returnerar

Namnet på etiketten "Totalt".


```python
def get_pivot_total_name(self):
    ...
```

använd metoden PivotGlobalizationSettings.GetColumnLabelsOfPivotTable().
