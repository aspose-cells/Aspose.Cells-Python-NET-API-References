##  region fastighet

Hämtar eller anger de regionala inställningar som används för arbetsboken som ska läsas in.

###  Anmärkningar

De regionala inställningarna kan användas för att initiera vissa funktioner i arbetsboken.
såsom teckensnitt, teman och så vidare.
För textbaserade filformat, såsom CSV, HTML, ..., den regionala inställningen
kommer också att användas för att upptäcka talformat och analysera textvärden till numeriska värden
eller datum- och tidsvärden för celler.
Den här inställningen kommer att sparas för den instansierade arbetsboken senare, det vill säga,
[`WorkbookSettings.region`](/cells/python-net/sv/aspose.cells/workbooksettings#region) i arbetsboken kommer att använda samma region
med denna egendom.
