##  set_style(self, style) {#aspose.cells.Style}
Anger stilen för den här kolumnen.



```python

def set_style(self, style):
    ...
```


| Parameter| Typ| Beskrivning|
| :- | :- | :- |
| style | [`Style`](/cells/python-net/sv/aspose.cells/style) | stilen som ska användas som standardstil för celler i den här kolumnen.|
###  Anmärkningar

Den här metoden anger endast den givna stilen som standardstil för den här kolumnen,
utan att ändra stilinställningarna för befintliga celler i den här kolumnen.
För att uppdatera stilinställningarna för befintliga celler till den angivna stilen samtidigt,
vänligen använd [`Column.apply_style`](/cells/python-net/sv/aspose.cells/column/apply_style)
