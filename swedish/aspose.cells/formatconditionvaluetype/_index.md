##  FormatConditionValueType uppräkning
Villkor värde typ.



Typen FormatConditionValueType avslöjar följande medlemmar:

###  Fält
| Fält| Beskrivning|
| :- | :- |
| FORMULA | Minsta/mittpunkts-/maxvärde för<br/> gradient bestäms av en formel.|
| MAX | Indikerar att maxvärdet i intervallet ska vara<br/> används som maxvärde för gradienten.|
| MIN | Indikerar att minimivärdet i intervallet ska vara<br/> används som minimivärde för gradienten.|
| NUMBER | Indikerar att minimum / mittpunkt / maximum<br/>värdet för gradienten anges av en konstant<br/> numeriskt värde.|
| PERCENT | Värde anger en procentandel mellan minimum<br/>och maximala värden i intervallet ska användas som<br/> minimum / mittpunkt / maxvärde för gradienten.|
| PERCENTILE |Värde indikerar en percentilrankning i intervallet ska<br/>användas som minimum/mittpunkt/maximivärde<br/> för gradienten.|
| AUTOMATIC_MAX | Indikerar att det automatiska maxvärdet i intervallet ska vara<br/> används som det automatiska maxvärdet för gradienten.|
| AUTOMATIC_MIN | Indikerar att det automatiska minimivärdet i intervallet ska vara<br/> används som det automatiska minimivärdet för gradienten.|
