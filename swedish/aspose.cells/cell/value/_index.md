##  value fastighet

Boolean,


Datum Tid,


Dubbel,


Heltal


Sträng.


För int value kan det returneras som ett heltalsobjekt eller ett dubbelobjekt.
Och det finns ingen garanti för att det returnerade value alltid kommer att behållas som samma typ av objekt.
