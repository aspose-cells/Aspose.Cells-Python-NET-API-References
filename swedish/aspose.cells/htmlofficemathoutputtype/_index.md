##  HtmlOfficeMathOutputType uppräkning
Representerar hur man exporterar OfficeMath till HTML.



Typen HtmlOfficeMathOutputType avslöjar följande medlemmar:

###  Fält
| Fält| Beskrivning|
| :- | :- |
| IMAGE | Konverterar OfficeMath till HTML enligt bilden som anges av<img> märka.|
| MATH_ML | Konverterar OfficeMath till HTML med MathML.|
