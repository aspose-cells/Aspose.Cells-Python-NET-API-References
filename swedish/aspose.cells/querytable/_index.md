##  QueryTable klass
Representerar informationen QueryTable.



Typen QueryTable avslöjar följande medlemmar:

###  Egenskaper
| Fast egendom| Beskrivning|
| :- | :- |
| [connection_id](/cells/python-net/sv/aspose.cells/querytable/connection_id) | Hämtar anslutnings-ID:t för frågetabellen.|
| [external_connection](/cells/python-net/sv/aspose.cells/querytable/external_connection) | Hämtar den relaterade externa anslutningen.|
| [name](/cells/python-net/sv/aspose.cells/querytable/name) |Hämtar namnet på frågetabellen.|
| [result_range](/cells/python-net/sv/aspose.cells/querytable/result_range) | Hämtar resultatets intervall.|
| [preserve_formatting](/cells/python-net/sv/aspose.cells/querytable/preserve_formatting) | Returnerar eller anger objektets PreserveFormatting.|
| [adjust_column_width](/cells/python-net/sv/aspose.cells/querytable/adjust_column_width) | Returnerar eller anger objektets AdjustColumnWidth.|



###  Exempel

```python
from aspose.cells import Workbook

# Instantiating a Workbook object
workbook = Workbook()
# Obtaining the reference of the first worksheet
worksheet = workbook.worksheets[0]

```
