##  ProtectionType uppräkning
Representerar skyddstypen för arbetsbok/kalkylblad.



Typen ProtectionType avslöjar följande medlemmar:

###  Fält
| Fält| Beskrivning|
| :- | :- |
| ALL | Representerar för att skydda alla.|
| CONTENTS | Representerar för att skydda innehåll, används i kalkylbladsskydd.|
| OBJECTS | Representerar för att skydda objekt, används i kalkylbladsskydd.|
| SCENARIOS | Representerar för att skydda scenarier, används i kalkylbladsskydd.|
| STRUCTURE | Representerar för att skydda struktur, används i arbetsboksskydd.|
| WINDOWS | Representerar att skydda fönstret, används i arbetsboksskydd.|
| NONE | Representerar inget skydd. Endast för läsegenskapen.|
