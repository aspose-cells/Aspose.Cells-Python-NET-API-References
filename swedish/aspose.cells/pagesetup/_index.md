##  PageSetup klass
 Inkapslar objektet som representerar beskrivningen av sidinställningarna.
Objektet PageSetup innehåller alla alternativ för sidinställningar.

###  Exempel

```python
from aspose.cells import Workbook

workbook = Workbook()
sheets = workbook.worksheets
# Add a worksheet
sheets.add()
sheet = sheets[1]
pageSetup = sheet.page_setup
pageSetup.print_area = "D1:K13"

```
