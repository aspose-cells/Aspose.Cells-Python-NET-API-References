##  Enumerazione GridLoadFormat

Rappresenta il formato del file di caricamento.



Il tipo GridLoadFormat espone i seguenti membri:

###  Campi
| Campo| Descrizione|
| :- | :- |
| AUTO | Rappresenta il riconoscimento automatico del formato.|
| CSV | Rappresenta il file di testo con valori separati da virgole (CSV).|
| XLSX | Rappresenta la cartella di lavoro o il modello Office Open XML spreadsheetML, con o senza macro.|
| TSV | Rappresenta il file TSV (file con valori separati da tabulazioni).|
| TAB_DELIMITED | Rappresenta un file di testo delimitato da tabulazioni, lo stesso con [`GridLoadFormat.TSV`](/cells/python-net/it/aspose.cellsgridjs/gridloadformat#TSV).|
| HTML | Rappresenta il file html.|
| M_HTML | Rappresenta il file mhtml.|
| ODS | Rappresenta il file Apri documento foglio (ODS).|
| EXCEL_97_TO_2003 | Rappresenta il file xls Excel97-2003.|
| SPREADSHEET_ML | Rappresenta il file xml di Excel 2003.|
| XLSB | Rappresenta il file xlsb.|
| NUMBERS | Rappresenta il file di numeri.|
| FODS | Rappresenta il formato file OpenDocument Flat XML Spreadsheet (.fods).|
| SXC | Rappresenta il formato file StarOffice Calc Spreadsheet (.sxc).|
| UNKNOWN |Rappresenta un formato non riconosciuto, non pu√≤ essere caricato.|
