##  WebQueryConnection classe
 Specifica le proprietà per una sorgente di query web. Una query web recupererà i dati dalle tabelle HTML,
 e può anche fornire parametri HTTP "Get" da elaborare dal server web nella generazione di HTML da
compresi i parametri e gli elementi parametrici.
