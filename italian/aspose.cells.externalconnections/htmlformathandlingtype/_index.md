##  HtmlFormatHandlingType enumerazione
Specifica come gestire la formattazione dall'origine HTML



Il tipo HtmlFormatHandlingType espone i membri seguenti:

###  Campi
| Campo| Descrizione|
| :- | :- |
| ALL | Trasferisci tutta la formattazione HTML nel foglio di lavoro insieme ai dati.|
| NONE | Inserisci i dati come testo non formattato (l'impostazione dei tipi di dati si verifica ancora).|
| RTF | Tradurre la formattazione HTML in formato RTF sui dati inseriti nel foglio di lavoro.|
