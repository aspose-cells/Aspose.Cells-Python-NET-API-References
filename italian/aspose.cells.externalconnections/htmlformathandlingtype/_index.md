##  Enumerazione HtmlFormatHandlingType
Specifica come gestire la formattazione dalla sorgente HTML



Il tipo HtmlFormatHandlingType espone i seguenti membri:

###  Campi
| Campo| Descrizione|
| :- | :- |
| ALL | Trasferisci tutta la formattazione HTML nel foglio di lavoro insieme ai dati.|
| NONE | Importa i dati come testo non formattato (l'impostazione dei tipi di dati avviene comunque).|
| RTF |Tradurre la formattazione HTML in formato RTF sui dati inseriti nel foglio di lavoro.|
