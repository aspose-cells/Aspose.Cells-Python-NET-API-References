##  Enumerazione PropertyType
Specifica il tipo di dati di una proprietà del documento.



Il tipo PropertyType espone i seguenti membri:

###  Campi
| Campo| Descrizione|
| :- | :- |
| BOOLEAN | La proprietà è un valore booleano.|
| DATE_TIME | La proprietà è un valore data/ora.|
| DOUBLE | La proprietà è un numero mobile.|
| NUMBER | La proprietà è un numero intero.|
| STRING | La proprietà è un valore stringa.|
| BLOB | La proprietà è un array di byte.|
