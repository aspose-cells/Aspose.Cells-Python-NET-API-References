##  PdfCompressionCore enumerazione
Specifica un tipo di compressione applicato a tutto il contenuto nel file PDF tranne le immagini.



Il tipo PdfCompressionCore espone i membri seguenti:

###  Campi
| Campo| Descrizione|
| :- | :- |
| NONE | Nessuno|
| RLE | Rle|
| LZW | Lzw|
