##  PdfCustomPropertiesExport enumerazione
Specifica il modo in cui [CustomDocumentPropertyCollection](/cells/python-net/it/aspose.cells.properties/customdocumentpropertycollection) vengono esportati nel file PDF.



Il tipo PdfCustomPropertiesExport espone i membri seguenti:

###  Campi
| Campo| Descrizione|
| :- | :- |
| NONE | Nessuna proprietà personalizzata viene esportata.|
| STANDARD | Le proprietà personalizzate vengono esportate come voci nel dizionario Info.|
