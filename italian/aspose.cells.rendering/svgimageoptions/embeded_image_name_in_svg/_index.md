##  embeded_image_name_in_svg proprietà

Indica il nome del file dell'immagine incorporata in svg.
Dovrebbe essere il percorso completo con la directory come "c:\\xpsEmbedded"

###  Osservazioni

 NOTA: questo membro è ora obsoleto. Invece,
Si prega di rimuovere questa proprietà perché ora le immagini vengono sempre incorporate in Svg con formato base64.
 Questa proprietà verrà rimossa 12 mesi dopo, a partire da aprile 2025.
Aspose si scusa per ogni eventuale disagio arrecato.
