##  is_optimized proprietà

Indica se ottimizzare gli elementi di output.

###  Osservazioni

 Il valore predefinito è falso.
Attualmente, quando questa proprietà è impostata su true, verranno eseguite le seguenti ottimizzazioni:
1. ottimizzare le linee di confine.
2. Ottimizzare le dimensioni del file durante il rendering dell'immagine Svg.
