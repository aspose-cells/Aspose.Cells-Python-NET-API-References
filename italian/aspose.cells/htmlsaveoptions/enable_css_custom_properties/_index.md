##  enable_css_custom_properties proprietà

Ottimizza l'output HTML utilizzando le proprietà CSS personalizzate. Ad esempio, nel caso in cui un'immagine base64 presenti più occorrenze, con le proprietà personalizzate i dati dell'immagine devono essere salvati una sola volta, migliorando così le prestazioni del codice HTML risultante.
Il valore predefinito è falso.
