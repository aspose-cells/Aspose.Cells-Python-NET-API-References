##  export_images_as_base64 proprietà

Specifica se le immagini vengono salvate nel formato Base64 HTML, MHTML o EPUB.

###  Osservazioni

Quando questa proprietà è impostata su true, i dati dell'immagine vengono esportati direttamente sul
Non vengono creati elementi img e file separati.
