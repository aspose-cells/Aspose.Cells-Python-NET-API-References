##  html_cross_string_type proprietà

Indica se una stringa tra celle verrà visualizzata nello stesso modo di MS Excel quando si salva un file Excel in formato HTML.
Per impostazione predefinita il valore è Predefinito, quindi, per le stringhe tra celle, c'è poca differenza tra i file HTML creati da Aspose.Cells e MS Excel.
Tuttavia, le prestazioni nella creazione di file HTML di grandi dimensioni, impostando il valore su Cross, sarebbero molto più veloci rispetto all'impostazione su Default o Fit2Cell.
