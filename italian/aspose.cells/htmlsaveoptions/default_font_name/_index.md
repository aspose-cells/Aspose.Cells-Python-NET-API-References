##  default_font_name proprietà

Specifica il nome del carattere predefinito per l'esportazione di html, il carattere predefinito verrà utilizzato quando il carattere dello stile non esiste,
Se questa proprietà è nulla, Aspose.Cells utilizzerà font universali che hanno la stessa famiglia del font originale,
il valore predefinito è nullo.
