##  get_local_function_name {#str}
Ottiene il nome della funzione dipendente dalle impostazioni locali in base al nome della funzione standard specificato.


###  ritorna

Nome della funzione dipendente dalla lingua. Le impostazioni locali sono state specificate dalla cartella di lavoro per la quale vengono utilizzate queste impostazioni.


```python
def get_local_function_name(self, standard_name):
    ...
```


| Parametro| Tipo| Descrizione|
| :- | :- | :- |
| standard_name | str | Nome della funzione standard (impostazioni locali en-US).|
