##  display_doc_title proprietà

Indica se la barra del titolo della finestra deve visualizzare il titolo del documento.

###  Osservazioni

Se falso, la barra del titolo dovrebbe invece visualizzare il nome del file PDF.
Il valore predefinito è falso.
