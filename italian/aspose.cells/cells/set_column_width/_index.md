##  set_column_width(self, column, width) {#int-float}
Imposta la larghezza della colonna specificata nella visualizzazione normale.



```python

def set_column_width(self, column, width):

```


| Parametro| Tipo| Descrizione|
| :- | :- | :- |
| column | int | Indice delle colonne.|
| width | float | Larghezza della colonna in unità di caratteri. La larghezza della colonna deve essere compresa tra 0 e 255.|
###  Osservazioni

Per i fogli di calcolo, la larghezza della colonna è misurata come il numero di caratteri
della larghezza massima delle cifre dei numeri da 0 a 9, come renderizzati nel font dello stile normale.
