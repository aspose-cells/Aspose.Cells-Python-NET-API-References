##  password proprietà

Rappresenta lo password per proteggere il foglio di lavoro.

###  Osservazioni

Se password è impostato su null o stringa vuota, è possibile rimuovere la protezione del foglio di lavoro o della cartella di lavoro senza usare password. In caso contrario, è necessario specificare password per rimuovere la protezione del foglio di lavoro o della cartella di lavoro.
