##  password proprietà

Rappresenta password per proteggere il foglio di lavoro.

###  Osservazioni

Se password è impostato su una stringa nulla o vuota,
è possibile rimuovere la protezione dal foglio di lavoro o dalla cartella di lavoro senza utilizzare password.
In caso contrario, è necessario specificare password per rimuovere la protezione dal foglio di lavoro o dalla cartella di lavoro.
