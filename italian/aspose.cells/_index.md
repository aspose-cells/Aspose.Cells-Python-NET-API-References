 IL**Aspose.Cells** namespace fornisce classi fondamentali e di base per generare, convertire, modificare, eseguire il rendering e stampare fogli di calcolo senza utilizzare Microsoft Excel.
Ad esempio, la classe [`Workbook`](/cells/python-net/it/aspose.cells/workbook) è la voce principale per creare o caricare e quindi manipolare un file di foglio di calcolo,
La classe [`Cells`](/cells/python-net/it/aspose.cells/cells) è la classe principale per manipolare tutti i dati e le proprietà in un foglio, ... ecc.
