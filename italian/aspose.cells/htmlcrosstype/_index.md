##  Enumerazione HtmlCrossType
Rappresenta cinque tipi di stringhe incrociate HTML.



Il tipo HtmlCrossType espone i seguenti membri:

###  Campi
| Campo| Descrizione|
| :- | :- |
| DEFAULT | Visualizza come MS Excel, dipende dalla cella successiva.<br/> Se la cella successiva è nulla, la stringa verrà incrociata o troncata|
| MS_EXPORT | Visualizza la stringa come se MS Excel esportasse il codice HTML.|
| CROSS |Visualizza HTML stringa incrociata, questa prestazione per la creazione di file HTML di grandi dimensioni sarà più di dieci volte più veloce rispetto all'impostazione del valore su Default o FitToCell.|
| CROSS_HIDE_RIGHT | Visualizza la stringa incrociata HTML e nasconde la stringa giusta quando i testi si sovrappongono.|
| FIT_TO_CELL | Visualizza solo la stringa contenuta nella larghezza della cella.|
