##  ignore_pivot_tables proprietà

Indica se salvare le tabelle pivot.

###  Osservazioni

La funzionalità tabella pivot di OpenOffice ha meno impostazioni rispetto a Excel, quindi i risultati delle tabelle pivot sono molto diversi.
A volte è meglio ignorare queste tabelle pivot quando si salva il file .ods.
