##  is_strict_schema11 proprietà

Indica se il file ods deve essere salvato in formato ODF versione 1.1. Il valore predefinito è "false".

###  Osservazioni

 NOTA: questo membro è ora obsoleto. Invece,
si prega di utilizzare la proprietà OdsSaveOptions.OdfStrictVersion.
 Questo metodo verrà rimosso 12 mesi dopo, a partire da febbraio 2024.
Aspose si scusa per ogni eventuale disagio arrecato.
