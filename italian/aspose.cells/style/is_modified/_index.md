##  is_modified(modify_flag) {#StyleModifyFlag}
Controlla se le proprietà specificate dello stile sono state modificate.
Utilizzato per lo stile di ConditionalFormattings per verificare se le proprietà specificate di questo stile devono essere utilizzate quando si applicano ConditionalFormattings a una cella.


###  ritorna
