##  treat_quote_prefix_as_value proprietà

Indica se il simbolo di virgoletta singola iniziale deve essere considerato parte del valore di una cella.
Il valore predefinito è true. Se è false, l'apostrofo singolo iniziale verrà rimosso dal valore della cella corrispondente.
e [`Style.quote_prefix`](/cells/python-net/it/aspose.cells/style#quote_prefix) verrà impostato come vero per la cella.
