##  parsing_pivot_cached_records proprietà

Indica se l'analisi dei record memorizzati nella cache pivot durante il caricamento del file.
Il valore predefinito è falso.

###  Osservazioni

Si applica solo ai file Excel Xlsx, Xltx, Xltm , Xlsm e xlsb
