##  note proprietà

Rappresenta il contenuto del commento.

###  Osservazioni

Se si tratta di un commento con thread, note non può essere modificato, altrimenti MS Excel non potrebbe elaborarlo come commento con thread.

###  Esempio

```python

comment1.note = "First note."

```
