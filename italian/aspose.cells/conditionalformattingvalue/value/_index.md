##  value proprietà

Ottenere o impostare il value di questo oggetto di formattazione condizionale value.
Dovrebbe essere usato insieme a Type.

###  Osservazioni

Se value è una stringa e inizia con "=", verrà elaborata come una formula,
in caso contrario lo elaboreremo come un semplice value.
###  Definizione:
```python
@property
def value(self):
    ...
@value.setter
def value(self, value):
    ...
```
