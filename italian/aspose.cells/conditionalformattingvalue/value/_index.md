##  value proprietà

Ottieni o imposta il value di questo oggetto di formattazione condizionale value.
Dovrebbe essere utilizzato insieme a Type.

###  Osservazioni

Se value è una stringa e inizia con "=", verrà elaborato come una formula,
altrimenti lo elaboreremo come un semplice value.
###  Definizione:
```python
@property
def value(self):
    ...
@value.setter
def value(self, value):
    ...
```
