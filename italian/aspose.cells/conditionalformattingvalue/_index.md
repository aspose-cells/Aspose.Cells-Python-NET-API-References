##  classe ConditionalFormattingValue
Descrive i valori dei punti di interpolazione in una scala gradiente, dataBar o iconSet.



Il tipo ConditionalFormattingValue espone i membri seguenti:

###  Proprietà
| Proprietà| Descrizione|
| :- | :- |
| [value](/cells/python-net/it/aspose.cells/conditionalformattingvalue/value) |Ottenere o impostare il valore di questo oggetto valore di formattazione condizionale.<br/> Dovrebbe essere usato insieme a Type.|
| [type](/cells/python-net/it/aspose.cells/conditionalformattingvalue/type) | Ottenere o impostare il tipo di questo oggetto valore di formattazione condizionale.<br/> Impostare il tipo su FormatConditionValueType.Min o FormatConditionValueType.Max<br/> imposterà automaticamente "Valore" su null.|
| [is_gte](/cells/python-net/it/aspose.cells/conditionalformattingvalue/is_gte) | Ottenere o impostare il flag Maggiore di o uguale.<br/> Usa solo per i set di icone, determina se questo valore di soglia viene utilizzato<br/> l'operatore maggiore o uguale a.<br/>false' indica che viene utilizzato 'maggiore di' invece di 'maggiore o uguale a'.<br/> Il valore predefinito è vero.|
