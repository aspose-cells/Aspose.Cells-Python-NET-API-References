##  get_param_value(index) {#int}
Ottiene l'oggetto valore rappresentato del parametro in corrispondenza dell'indice specificato.


###  ritorna

Se il parametro è un valore normale, restituisce il valore normale.
Se il parametro è riferimento, restituisce l'oggetto ReferredArea.
Se il parametro fa riferimento a più set di dati, restituisce un array di oggetti.
