##  get_named_ranges(self) {#}
Ottiene tutti gli intervalli denominati predefiniti nel foglio di calcolo.


###  ritorna

Un array di oggetti Range.
Se il riferimento del Nome definito è esterno o ha più intervalli, per questo Nome non verrà restituito alcun oggetto Intervallo.



Restituisce null se l'intervallo denominato non esiste.
