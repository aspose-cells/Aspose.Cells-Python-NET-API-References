##  get_cache_folder() {#}
Ottiene la cartella per i file temporanei che possono essere utilizzati come cache dati.


###  ritorna

Cartella specificata per i file di cache.
Se non è stato specificato, verrà restituito null
e il percorso temporaneo del sistema verrà utilizzato quando necessario.


```python

@staticmethod
def get_cache_folder():
    ...
```


###  Osservazioni

I file di cache vengono generalmente utilizzati per alcune funzionalità per la considerazione delle prestazioni della memoria,
come il salvataggio di grandi set di dati in file xls,
oppure utilizzando la modalità memoria con cache di file per il modello di celle.
