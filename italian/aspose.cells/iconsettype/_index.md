##  Enumerazione IconSetType
 Tipo di set di icone per la formattazione condizionale.
I valori soglia per l'attivazione delle diverse icone all'interno di un set sono
configurabile e l'ordine delle icone è reversibile.



Il tipo IconSetType espone i seguenti membri:

###  Campi
| Campo| Descrizione|
| :- | :- |
| ARROWS3 | Set di 3 icone di frecce.|
| ARROWS_GRAY3 | Set di 3 icone di frecce grigie.|
| FLAGS3 | Set di 3 icone di bandiere.|
| SIGNS3 | Set di icone con 3 segni.|
| SYMBOLS3 | Set di 3 icone di simboli (cerchiati).|
| SYMBOLS32 | Set di 3 icone di simboli (non cerchiati).|
| TRAFFIC_LIGHTS31 | Set di 3 icone di semafori (senza bordo).|
| TRAFFIC_LIGHTS32 | Set di 3 icone di semafori con bordo nero spesso.|
| ARROWS4 | Set di 4 icone di frecce.|
| ARROWS_GRAY4 | Set di 4 icone di frecce grigie.|
| RATING4 | Set di 4 icone di valutazione.|
| RED_TO_BLACK4 | Set di 4 icone 'dal rosso al nero'.|
| TRAFFIC_LIGHTS4 | Set di 4 icone di semafori.|
| ARROWS5 | Set di 5 icone di frecce.|
| ARROWS_GRAY5 | Set di 5 icone di frecce grigie.|
| QUARTERS5 | Set di icone da 5 quarti.|
| RATING5 | Set di 5 icone di valutazione.|
| STARS3 | 3 stelle impostate|
| BOXES5 | Set di 5 scatole|
| TRIANGLES3 | Set di 3 triangoli|
| NONE | Nessuno|
| CUSTOM_SET | Set personalizzato.<br/> Questo elemento è di sola lettura.|
| SMILIES3 | 3 faccine.<br/> Solo per .ods.|
| COLOR_SMILIES3 |Faccine a 3 colori.<br/> Solo per .ods.|
