##  IconSetType enumerazione
 Tipo di set di icone per la formattazione condizionale.
I valori di soglia per l'attivazione delle diverse icone all'interno di un set sono
configurabile e l'ordine delle icone è reversibile.



Il tipo IconSetType espone i membri seguenti:

###  Campi
| Campo| Descrizione|
| :- | :- |
| ARROWS3 | Insieme dell'icona di 3 frecce.|
| ARROWS_GRAY3 | Insieme dell'icona di 3 frecce grigie.|
| FLAGS3 | Insieme dell'icona di 3 bandiere.|
| SIGNS3 | insieme dell'icona di 3 segni.|
| SYMBOLS3 | Insieme dell'icona di 3 simboli (cerchiato).|
| SYMBOLS32 | Insieme dell'icona di 3 simboli (non cerchiato).|
| TRAFFIC_LIGHTS31 | Set di icone a 3 semafori (senza bordo).|
| TRAFFIC_LIGHTS32 | Icona di 3 semafori con bordo nero spesso.|
| ARROWS4 | Insieme dell'icona di 4 frecce.|
| ARROWS_GRAY4 | Insieme dell'icona di 4 frecce grigie.|
| RATING4 | 4 set di icone di rating.|
| RED_TO_BLACK4 | set di icone "da rosso a nero".|
| TRAFFIC_LIGHTS4 | Insieme dell'icona di 4 semafori.|
| ARROWS5 | Insieme dell'icona di 5 frecce.|
| ARROWS_GRAY5 | Insieme dell'icona di 5 frecce grigie.|
| QUARTERS5 | set di icone di 5 quarti.|
| RATING5 | 5 set di icone di valutazione.|
| STARS3 | 3 stelle impostate|
| BOXES5 | Set di 5 scatole|
| TRIANGLES3 | Set di 3 triangoli|
| NONE | Nessuno|
| CUSTOM_SET | CustomSet.<br/> Questo elemento è di sola lettura.|
| SMILIES3 | 3 faccine.<br/> Solo per .ods.|
| COLOR_SMILIES3 | 3 faccine colorate.<br/> Solo per .ods.|
