##  LoadFormat enumerazione
Rappresenta il formato del file di caricamento.



Il tipo LoadFormat espone i membri seguenti:

###  Campi
| Campo| Descrizione|
| :- | :- |
| AUTO | Rappresenta il riconoscimento automatico del formato.|
| CSV |File di testo con valori separati da virgola (CSV).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [LoadFormat.CSV](/cells/python-net/it/aspose.cells/loadformat#CSV).<br/> Questa proprietà verrà rimossa 6 mesi dopo da aprile 2021.<br/> Aspose si scusa per gli eventuali disagi causati.|
| XLSX | Rappresenta la cartella di lavoro o il modello di foglio di calcolo XML Office Open XML, con o senza macro.|
| TSV | File di testo Valori separati da tabulazioni (TSV).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [LoadFormat.TSV](/cells/python-net/it/aspose.cells/loadformat#TSV).<br/> Questa proprietà verrà rimossa 6 mesi dopo da aprile 2021.<br/> Aspose si scusa per gli eventuali disagi causati.|
| TAB_DELIMITED | Rappresenta un file di testo delimitato da tabulazioni, uguale a [LoadFormat.TSV](/cells/python-net/it/aspose.cells/loadformat#TSV).|
| HTML | Rappresenta un file html.|
| M_HTML | Rappresenta un file mhtml.|
| ODS | Apri il file Foglio documento (ODS).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [LoadFormat.ODS](/cells/python-net/it/aspose.cells/loadformat#ODS).<br/> Questa proprietà verrà rimossa 6 mesi dopo da aprile 2021.<br/> Aspose si scusa per gli eventuali disagi causati.|
| EXCEL_97_TO_2003 | Rappresenta un file xls Excel97-2003.|
| SPREADSHEET_ML | Rappresenta un file xml di Excel 2003.|
| XLSB | Rappresenta un file xlsb.|
| OTS | Apri il file Document Template Sheet (OTS).|
| NUMBERS | Rappresenta un file di numeri.|
| FODS | Rappresenta il formato di file OpenDocument Flat XML Spreadsheet (.fods).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [LoadFormat.FODS](/cells/python-net/it/aspose.cells/loadformat#FODS).<br/> Questa proprietà verrà rimossa 6 mesi dopo da aprile 2021.<br/> Aspose si scusa per gli eventuali disagi causati.|
| SXC | Rappresenta il formato di file StarOffice Calc Spreadsheet (.sxc).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [LoadFormat.SXC](/cells/python-net/it/aspose.cells/loadformat#SXC).<br/> Questa proprietà verrà rimossa 6 mesi dopo da aprile 2021.<br/> Aspose si scusa per gli eventuali disagi causati.|
| XML | Rappresenta un semplice file xml.|
| UNKNOWN | Rappresenta un formato non riconosciuto, non può essere caricato.|
| IMAGE | Immagine|
| JSON | Json|
