##  Enumerazione LoadFormat
Rappresenta il formato del file di caricamento.



Il tipo LoadFormat espone i seguenti membri:

###  Campi
| Campo| Descrizione|
| :- | :- |
| AUTO | Rappresenta il riconoscimento automatico del formato.|
| CSV | File di testo con valori separati da virgole (CSV).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`LoadFormat.CSV`](/cells/python-net/it/aspose.cells/loadformat#CSV).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| XLSX | Rappresenta la cartella di lavoro o il modello Office Open XML spreadsheetML, con o senza macro.|
| TSV | File di testo con valori separati da tabulazioni (TSV).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`LoadFormat.TSV`](/cells/python-net/it/aspose.cells/loadformat#TSV).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| TAB_DELIMITED | Rappresenta un file di testo delimitato da tabulazioni, lo stesso con [`LoadFormat.TSV`](/cells/python-net/it/aspose.cells/loadformat#TSV).|
| HTML | Rappresenta un file html.|
| M_HTML | Rappresenta un file Mhtml.|
| ODS | Aprire il file Foglio documenti (ODS).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`LoadFormat.ODS`](/cells/python-net/it/aspose.cells/loadformat#ODS).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| EXCEL_97_TO_2003 | Rappresenta un file xls Excel97-2003.|
| SPREADSHEET_ML | Rappresenta un file xml di Excel 2003.|
| XLSB | Rappresenta un file xlsb.|
| OTS | Aprire il file Document Template Sheet (OTS).|
| NUMBERS | Rappresenta un file di numeri.|
| FODS | Rappresenta il formato file OpenDocument Flat XML Spreadsheet (.fods).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`LoadFormat.FODS`](/cells/python-net/it/aspose.cells/loadformat#FODS).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| SXC | Rappresenta il formato file StarOffice Calc Spreadsheet (.sxc).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`LoadFormat.SXC`](/cells/python-net/it/aspose.cells/loadformat#SXC).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| XML | Rappresenta un semplice file xml.|
| EPUB | Rappresenta un file EPUB.|
| AZW3 |Rappresenta un file AZW3.|
| UNKNOWN | Rappresenta un formato non riconosciuto, non può essere caricato.|
| IMAGE | Immagine|
| JSON | Json|
