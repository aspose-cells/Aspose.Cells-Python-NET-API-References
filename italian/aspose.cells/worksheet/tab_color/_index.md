##  tab_color proprietà

Rappresenta il colore della scheda del foglio di lavoro.

###  Osservazioni

Questa funzionalità è supportata solo in ExcelXP (Excel2002) e versioni successive.
Se si salva il file in formato Excel97 o Excel2000, verrà omesso.
