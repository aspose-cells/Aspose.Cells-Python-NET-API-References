##  Enumerazione SaveFormat
Rappresenta il formato in cui viene salvata la cartella di lavoro.



Il tipo SaveFormat espone i seguenti membri:

###  Campi
| Campo| Descrizione|
| :- | :- |
| CSV | File di testo con valori separati da virgole (CSV).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.CSV`](/cells/python-net/it/aspose.cells/saveformat#CSV).<br/> Questa proprietà verrà rimossa 6 mesi dopo, a partire da aprile 2021.<br/> Aspose si scusa per ogni eventuale disagio arrecato.|
| XLSX | Rappresenta un file xlsx.|
| XLSM | Rappresenta un file xlsm che abilita le macro.|
| XLTX |Rappresenta un file xltx.|
| XLTM | Rappresenta un file xltm che abilita le macro.|
| XLAM | Rappresenta un file xltm che abilita le macro aggiuntive.|
| TSV | File di testo Valori separati da tabulazione (TSV).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.TSV`](/cells/python-net/it/aspose.cells/saveformat#TSV).<br/> Questa proprietà verrà rimossa 6 mesi dopo, a partire da aprile 2021.<br/> Aspose si scusa per ogni eventuale disagio arrecato.|
| TAB_DELIMITED | Rappresenta un file di testo delimitato da tabulazioni, lo stesso di [`SaveFormat.TSV`](/cells/python-net/it/aspose.cells/saveformat#TSV).|
| HTML | Rappresenta un file html.|
| M_HTML |Rappresenta un file mhtml.|
| ODS | Apri il file Document Sheet(ODS).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.ODS`](/cells/python-net/it/aspose.cells/saveformat#ODS).<br/> Questa proprietà verrà rimossa 6 mesi dopo, a partire da aprile 2021.<br/> Aspose si scusa per ogni eventuale disagio arrecato.|
| EXCEL_97_TO_2003 | Rappresenta un file xls di Excel97-2003.|
| SPREADSHEET_ML | Rappresenta un file xml di Excel 2003.|
| XLSB | Rappresenta un file xlsb.|
| AUTO | Se si salva il file sul disco, il formato del file dipende dall'estensione del nome del file.<br/> Se si salva il file nel flusso, il formato del file è xlsx.|
| UNKNOWN | Rappresenta un formato non riconosciuto, non può essere salvato.|
| PDF | Rappresenta un file Pdf.|
| XPS | Formato XPS (XML Paper Specification).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.XPS`](/cells/python-net/it/aspose.cells/saveformat#XPS).<br/> Questa proprietà verrà rimossa 6 mesi dopo, a partire da aprile 2021.<br/> Aspose si scusa per ogni eventuale disagio arrecato.|
| TIFF | Rappresenta un file TIFF.<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.TIFF`](/cells/python-net/it/aspose.cells/saveformat#TIFF).<br/> Questa proprietà verrà rimossa 6 mesi dopo, a partire da aprile 2021.<br/> Aspose si scusa per ogni eventuale disagio arrecato.|
| SVG | SVG file.<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.SVG`](/cells/python-net/it/aspose.cells/saveformat#SVG).<br/> Questa proprietà verrà rimossa 6 mesi dopo, a partire da aprile 2021.<br/> Aspose si scusa per ogni eventuale disagio arrecato.|
| DIF | Formato di interscambio dati.|
| OTS | Aprire il file Document Template Sheet (OTS).|
| XLT | File modello Excel 97-2003.|
| XML | Rappresenta un semplice file xml.|
| NUMBERS | Rappresenta un file di numeri.|
| MARKDOWN | Rappresenta il documento markdown.|
| FODS | Rappresenta il formato di file OpenDocument Flat XML Spreadsheet (.fods).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.FODS`](/cells/python-net/it/aspose.cells/saveformat#FODS).<br/> Questa proprietà verrà rimossa 6 mesi dopo, a partire da aprile 2021.<br/> Aspose si scusa per ogni eventuale disagio arrecato.|
| SXC | Rappresenta il formato file StarOffice Calc Spreadsheet (.sxc).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.SXC`](/cells/python-net/it/aspose.cells/saveformat#SXC).<br/> Questa proprietà verrà rimossa 6 mesi dopo, a partire da aprile 2021.<br/> Aspose si scusa per ogni eventuale disagio arrecato.|
| PPTX | Rappresenta il file .pptx.|
| DOCX | Rappresenta il file .docx.|
| EMF | Windows Metafile migliorato.|
| JPG | JPEG JFIF.|
| PNG | Grafica di rete portatile.|
| BMP | Windows Bitmap|
| GIF | Gif|
| JSON | Json|
| SQL_SCRIPT | SQL|
| X_HTML | Rappresenta il file XHtml.<br/>si prega di utilizzare la proprietà [`HtmlSaveOptions.html_version`](/cells/python-net/it/aspose.cells/htmlsaveoptions#html_version).<br/> Questa proprietà verrà rimossa 6 mesi dopo, a partire da marzo 2025.<br/> Aspose si scusa per ogni eventuale disagio arrecato.|
| EPUB | Rappresenta il file Epub.|
| AZW3 | Rappresenta il file Azw3.|
| PCL | PCL (linguaggio di comando della stampante)|
| DBF | File di dati Xbase|
