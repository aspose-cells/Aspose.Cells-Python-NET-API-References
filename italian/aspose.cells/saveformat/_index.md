##  Enumerazione SaveFormat
Rappresenta il formato in cui viene salvata la cartella di lavoro.



Il tipo SaveFormat espone i seguenti membri:

###  Campi
| Campo| Descrizione|
| :- | :- |
| CSV | File di testo con valori separati da virgole (CSV).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.CSV`](/cells/python-net/it/aspose.cells/saveformat#CSV).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| XLSX | Rappresenta un file xlsx.|
| XLSM | Rappresenta un file xlsm che abilita le macro.|
| XLTX | Rappresenta un file xltx.|
| XLTM | Rappresenta un file xltm che abilita le macro.|
| XLAM | Rappresenta un file xltm che abilita l'aggiunta di macro.|
| TSV | File di testo con valori separati da tabulazioni (TSV).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.TSV`](/cells/python-net/it/aspose.cells/saveformat#TSV).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| TAB_DELIMITED | Rappresenta un file di testo delimitato da tabulazioni, lo stesso con [`SaveFormat.TSV`](/cells/python-net/it/aspose.cells/saveformat#TSV).|
| HTML | Rappresenta un file html.|
| M_HTML | Rappresenta un file Mhtml.|
| ODS | Aprire il file Foglio documenti (ODS).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.ODS`](/cells/python-net/it/aspose.cells/saveformat#ODS).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| EXCEL_97_TO_2003 | Rappresenta un file xls Excel97-2003.|
| SPREADSHEET_ML | Rappresenta un file xml di Excel 2003.|
| XLSB | Rappresenta un file xlsb.|
| AUTO | Se si salva il file su disco, il formato del file corrisponde all'estensione del nome del file.<br/>Se si salva il file nello stream, il formato file è xlsx.|
| UNKNOWN | Rappresenta un formato non riconosciuto, non può essere salvato.|
| PDF | Rappresenta un file PDF.|
| XPS | Formato XPS (Specifica carta XML).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.XPS`](/cells/python-net/it/aspose.cells/saveformat#XPS).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| TIFF | Rappresenta un file TIFF.<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.TIFF`](/cells/python-net/it/aspose.cells/saveformat#TIFF).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| SVG | SVG cartella.<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.SVG`](/cells/python-net/it/aspose.cells/saveformat#SVG).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| DIF | Formato di scambio dati.|
| OTS | Aprire il file Document Template Sheet (OTS).|
| XLT | File modello Excel 97-2003.|
| XML | Rappresenta un semplice file xml.|
| NUMBERS | Rappresenta un file di numeri.|
| MARKDOWN | Rappresenta il documento di ribasso.|
| FODS | Rappresenta il formato file OpenDocument Flat XML Spreadsheet (.fods).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.FODS`](/cells/python-net/it/aspose.cells/saveformat#FODS).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| SXC | Rappresenta il formato file StarOffice Calc Spreadsheet (.sxc).<br/> NOTA: questo membro è ora obsoleto. Invece,<br/>si prega di utilizzare la proprietà [`SaveFormat.SXC`](/cells/python-net/it/aspose.cells/saveformat#SXC).<br/> Questa proprietà verrà rimossa 6 mesi dopo a partire da aprile 2021.<br/> Aspose si scusa per eventuali disagi riscontrati.|
| PPTX | Rappresenta il file .pptx.|
| DOCX | Rappresenta il file .docx.|
| EMF | Windows Metafile avanzato.|
| JPG | JPEG JFIF.|
| PNG | Grafica di rete portatile.|
| BMP | Windows Bitmap|
| GIF | Gif|
| JSON | Json|
| SQL_SCRIPT | Sql|
| X_HTML | Rappresenta il file XHtml.|
| EPUB | Rappresenta il file Epub.|
