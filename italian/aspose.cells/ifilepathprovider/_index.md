##  IFilePathProvider classe
Rappresenta il provider del percorso del file esportato.



Il tipo IFilePathProvider espone i seguenti membri:

###  Metodi
| Metodo| Descrizione|
| :- | :- |
| [get_full_name](/cells/python-net/it/aspose.cells/ifilepathprovider/get_full_name/#str) | Ottiene il percorso completo del file in base al nome del foglio di lavoro quando si esporta il foglio di lavoro in HTML separatamente.<br/> Quindi i riferimenti tra i fogli di lavoro possono essere esportati correttamente.|
