##  PageSetup classe
 Incapsula l'oggetto che rappresenta la descrizione dell'impostazione della pagina.
L'oggetto PageSetup contiene tutte le opzioni di impostazione della pagina.

###  Esempio

```python
from aspose.cells import Workbook

workbook = Workbook()
sheets = workbook.worksheets
# Add a worksheet
sheets.add()
sheet = sheets[1]
pageSetup = sheet.page_setup
pageSetup.print_area = "D1:K13"

```
