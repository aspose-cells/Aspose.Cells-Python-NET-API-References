##  parse_formulas(self, ignore_error) {#bool}
Analizza tutte le formule che non sono state analizzate quando sono state caricate dal file modello o impostate in una cella.

| Parametro| Tipo| Descrizione|
| :- | :- | :- |
| ignore_error | bool | Ignora l'errore per la formula non valida.<br/>Per una formula non valida, se si ignora l'errore, questa formula verrà ignorata<br/> e il processo continuerà ad analizzare altre formule, altrimenti verrà generata un'eccezione.|
