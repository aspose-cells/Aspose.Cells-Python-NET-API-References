##  custom_path proprietà

Percorso personalizzato (URL) dell'utente salvato nel file HTML generato per la sorgente di riferimento. Se non definito dall'utente, verrà utilizzato DefaultPath.
Ad esempio, se i dati del foglio verranno salvati dall'utente in d:/sheet001.htm, l'URL utilizzato nel file HTML principale dovrebbe essere "d:/sheet001.htm" o un altro percorso relativo valido a cui possa accedere il file HTML principale.
