##  default_path proprietà

Percorso predefinito (URL) salvato nel file HTML generato per la sorgente a cui si fa riferimento.
Ad esempio, i dati del foglio salvati in xxx_files/sheet001.htm, l'URL utilizzato nel file HTML principale dovrebbe essere simile a "src="xxx_files/sheet001.htm""
