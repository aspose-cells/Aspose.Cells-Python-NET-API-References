##  set_standard_built_in_name {#str-str-bool}
Imposta il nome della funzione dipendente dalla locale in base al nome della funzione standard specificato.



```python
def set_standard_built_in_name(self, local_name, standard_name, bidirectional):
    ...
```


| Parametro| Tipo| Descrizione|
| :- | :- | :- |
| local_name | str | Nome della funzione dipendente dalla lingua|
| standard_name | str | Nome della funzione standard (impostazioni locali en-US).|
| bidirectional | bool | Se mappa automaticamente il testo del nome standard sul testo del nome locale.<br/>Se vero, il testo del nome standard verrà mappato automaticamente al testo del nome locale<br/>quindi l'utente non dovrà chiamare nuovamente lo [`SettableGlobalizationSettings.set_local_built_in_name`](/cells/python-net/it/aspose.cells/settableglobalizationsettings/set_local_built_in_name)<br/> per la stessa coppia di nomi standard e locali|
