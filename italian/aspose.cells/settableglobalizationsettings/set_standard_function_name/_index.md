##  set_standard_function_name {#str-str-bool}
Imposta il nome della funzione dipendente dalla locale in base al nome della funzione standard specificato.



```python
def set_standard_function_name(self, local_name, standard_name, bidirectional):
    ...
```


| Parametro| Tipo| Descrizione|
| :- | :- | :- |
| local_name | str | Nome della funzione dipendente dalla lingua|
| standard_name | str | Nome della funzione standard (impostazioni locali en-US).|
| bidirectional | bool |Se mappa automaticamente il nome della funzione standard sul nome della funzione locale.<br/>Se vero, il nome standard verrà mappato automaticamente al nome locale<br/>quindi l'utente non dovrà chiamare nuovamente lo [`SettableGlobalizationSettings.set_local_function_name`](/cells/python-net/it/aspose.cells/settableglobalizationsettings/set_local_function_name)<br/> per la stessa coppia di nomi standard e locali|
