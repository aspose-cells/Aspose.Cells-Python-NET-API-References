##  type proprietà

Ottenere o impostare l'icona impostata type da visualizzare.
L'impostazione di type verificherà automaticamente se il conteggio Cfvos corrente è
 in accordo con il nuovo type. Se non in accordo, i vecchi Cfvo verranno ripuliti e
verranno aggiunti Cfvo predefiniti.
###  Definizione:
```python
@property
def type(self):
    ...
@type.setter
def type(self, value):
    ...
```
