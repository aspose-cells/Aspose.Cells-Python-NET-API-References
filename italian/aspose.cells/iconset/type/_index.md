##  type proprietà

Ottieni o imposta il set di icone type da visualizzare.
Impostando type verrà verificato automaticamente se il conteggio Cfvos corrente è
 in accordo con il nuovo type. In caso contrario, i vecchi Cfvos verranno puliti e
Verranno aggiunti i Cfvos predefiniti.
###  Definizione:
```python
@property
def type(self):
    ...
@type.setter
def type(self, value):
    ...
```
