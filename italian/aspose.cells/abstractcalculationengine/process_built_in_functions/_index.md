##  process_built_in_functions proprietà

Se le funzioni integrate sono state supportate dal motore integrato
dovrebbero essere controllati ed elaborati da questa implementazione.
L'impostazione predefinita è falsa.

###  Osservazioni

Se l'utente ha bisogno di modificare la logica di calcolo di alcune funzioni integrate, questa proprietà deve essere impostata su true.
In caso contrario, lasciare questa proprietà come false per considerazioni sulle prestazioni.
