##  Enumerazione FileFormatType
Rappresenta i tipi di formato file.



Il tipo FileFormatType espone i seguenti membri:

| Campo| Descrizione|
| :- | :- |
| CSV |File di testo con valori separati da virgole (CSV).|
| XLSX | File Office Open XML SpreadsheetML (senza macro).|
| XLSM | Office Open XML SpreadsheetML File con attivazione macro.|
| XLTX | Modello Office Open XML SpreadsheetML (senza macro).|
| XLTM | Modello Office Open XML SpreadsheetML con attivazione macro.|
| XLAM | File Office Open XML SpreadsheetML aggiuntivo con abilitazione macro.|
| TSV | File di testo con valori separati da tabulazioni (TSV).|
