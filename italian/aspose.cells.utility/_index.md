 IL**Aspose.Cells.Utility** Lo spazio dei nomi contiene classi di utilità che elaborano i formati JSON per i file di fogli di calcolo Excel.

###  Classi
| Classe| Descrizione|
| :- | :- |
| [`ConversionUtility`](/cells/python-net/it/aspose.cells.utility/conversionutility) | Rappresenta un'utilità per convertire i file in altri formati.|
| [`ExportRangeToJsonOptions`](/cells/python-net/it/aspose.cells.utility/exportrangetojsonoptions) | Indica le opzioni per l'esportazione dell'intervallo in JSON.|
| [`JsonLayoutOptions`](/cells/python-net/it/aspose.cells.utility/jsonlayoutoptions) | Rappresenta le opzioni del tipo di layout JSON.|
| [`JsonUtility`](/cells/python-net/it/aspose.cells.utility/jsonutility) | Rappresenta la classe di utilità dell'elaborazione JSON.|
