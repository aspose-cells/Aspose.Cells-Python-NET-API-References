##  classe CustomXmlPart
Rappresenta una parte di archiviazione dati XML personalizzata (dati XML personalizzati all'interno di un pacchetto).



Il tipo CustomXmlPart espone i membri seguenti:

###  Proprietà
| Proprietà| Descrizione|
| :- | :- |
| [data](/cells/python-net/it/aspose.cells.markup/customxmlpart/data) | Ottiene o imposta il contenuto XML di questa parte di archiviazione dati XML personalizzata.|
| [schema_data](/cells/python-net/it/aspose.cells.markup/customxmlpart/schema_data) | Ottiene o imposta il contenuto XML di questa parte di archiviazione dei dati dello schema XML personalizzato.|
| [id](/cells/python-net/it/aspose.cells.markup/customxmlpart/id) |Ottiene e imposta l'ID della parte XML personalizzata.|
