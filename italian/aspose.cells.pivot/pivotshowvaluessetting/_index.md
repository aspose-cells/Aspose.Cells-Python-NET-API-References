##  PivotShowValuesSetting classe
Rappresenta le impostazioni relative alla visualizzazione dei valori quando è in uso il calcolo ShowDataAs.



Il tipo PivotShowValuesSetting espone i seguenti membri:

###  Proprietà
| Proprietà| Descrizione|
| :- | :- |
| [calculation_type](/cells/python-net/it/aspose.cells.pivot/pivotshowvaluessetting/calculation_type) | Rappresenta il modo in cui visualizzare i valori di un campo dati nel report pivot.|
| [base_field_index](/cells/python-net/it/aspose.cells.pivot/pivotshowvaluessetting/base_field_index) | Rappresenta il campo base per un calcolo ShowDataAs quando il calcolo ShowDataAs è in uso.|
| [base_item_position_type](/cells/python-net/it/aspose.cells.pivot/pivotshowvaluessetting/base_item_position_type) | Rappresenta il tipo dell'elemento pivot di base nel campo di base quando è in uso il calcolo ShowDataAs.<br/> Valido solo per i campi dati.<br/>Poiché PivotItemPosition.Custom è solo per la lettura, se è necessario impostare PivotItemPosition.Custom,<br/> impostare l'attributo PivotField.BaseItemIndex.|
| [base_item_index](/cells/python-net/it/aspose.cells.pivot/pivotshowvaluessetting/base_item_index) | Rappresenta l'indice personalizzato dell'elemento pivot nel campo base quando è in uso il calcolo ShowDataAs.<br/> Valido solo per i campi dati.|
