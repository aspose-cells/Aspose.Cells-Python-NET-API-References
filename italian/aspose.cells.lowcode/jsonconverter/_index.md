##  JsonConverter classe
Convertitore per la conversione tra la struttura dati JSON e altri formati di file di fogli di calcolo.



Il tipo JsonConverter espone i seguenti membri:

###  Metodi
| Metodo| Descrizione|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/it/aspose.cells.lowcode/jsonconverter/process/#str-str) | Converte il file modello specificato tra json e altri formati.|
| [`process(, load_options, save_options)`](/cells/python-net/it/aspose.cells.lowcode/jsonconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Converte i dati tra JSON e altri formati di file di fogli di calcolo.|



###  Esempio

```python
from aspose.cells.lowcode import JsonConverter

JsonConverter.process("data.json", "res.xlsx")

```
