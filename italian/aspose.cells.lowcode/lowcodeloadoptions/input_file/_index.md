##  input_file proprietà

Ottiene e imposta il file (con percorso se necessario) del modello.

###  Osservazioni

Quando si imposta un percorso non nullo e non vuoto per questa proprietà,
il valore impostato in precedenza per [`LowCodeLoadOptions.input_stream`](/cells/python-net/it/aspose.cells.lowcode/lowcodeloadoptions#input_stream) verrà ignorato.
