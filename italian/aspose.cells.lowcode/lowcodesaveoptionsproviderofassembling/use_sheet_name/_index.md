##  use_sheet_name proprietà

Specifica se il percorso del file viene creato con il nome del foglio anziché con l'indice del foglio. Il valore predefinito è "false".

###  Osservazioni

Il nome del foglio non verrà mai ricostruito automaticamente.
Quindi, quando lo imposti su true, assicurati che non ci sia un nome di foglio speciale
che può causare percorsi o nomi di file non validi.
