##  HtmlConverter classe
Convertitore per la conversione tra file HTML (html o mht) e altri formati di file di fogli di calcolo.



Il tipo HtmlConverter espone i seguenti membri:

###  Metodi
| Metodo| Descrizione|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/it/aspose.cells.lowcode/htmlconverter/process/#str-str) | Converte il file modello specificato tra HTML e altri formati.|
| [`process(, load_options, save_options)`](/cells/python-net/it/aspose.cells.lowcode/htmlconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Converte i file tra HTML e altri formati di fogli di calcolo.|



###  Esempio

```python
from aspose.cells.lowcode import HtmlConverter

HtmlConverter.process("Template.xlsx", "res.html")

```
