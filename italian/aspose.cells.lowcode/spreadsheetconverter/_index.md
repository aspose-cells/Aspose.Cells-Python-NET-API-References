##  SpreadsheetConverter classe
Convertitore per la conversione tra diversi formati di file di fogli di calcolo, come xls, xlsx, xlsb, spreadsheet ml...



Il tipo SpreadsheetConverter espone i seguenti membri:

###  Metodi
| Metodo| Descrizione|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/it/aspose.cells.lowcode/spreadsheetconverter/process/#str-str) | Converte il file modello specificato tra formati di file di foglio di calcolo.|
| [`process(, load_options, save_options)`](/cells/python-net/it/aspose.cells.lowcode/spreadsheetconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Converte tra diversi formati di file di fogli di calcolo.|



###  Esempio

```python
from aspose.cells.lowcode import SpreadsheetConverter

SpreadsheetConverter.process("template.xlsx", "template.xlsb")

```
