##  TextConverter classe
Convertitore per la conversione tra formati basati su testo (csv, tsv, dif...) e altri formati di file di fogli di calcolo.



Il tipo TextConverter espone i seguenti membri:

###  Metodi
| Metodo| Descrizione|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/it/aspose.cells.lowcode/textconverter/process/#str-str) | Converte il file modello specificato tra file basati su testo e altri formati.|
| [`process(, load_options, save_options)`](/cells/python-net/it/aspose.cells.lowcode/textconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Converte il formato del file tra formati basati su testo e altri formati di file di fogli di calcolo|



###  Esempio

```python
from aspose.cells.lowcode import TextConverter

TextConverter.process("template.csv", "res.xlsx")

```
