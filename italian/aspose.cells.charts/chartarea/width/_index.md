##  width proprietà

Ottiene o imposta lo scostamento orizzontale dalla colonna dell'angolo inferiore destro, in unità di 1/4000 dell'area del grafico.

###  Osservazioni

 NOTA: questo membro è ormai obsoleto.
Utilizzare invece la proprietà ChartArea.WidthRatioToChart.
Larghezza = WidthRatioToChart * 4000.
 Questa proprietà verrà rimossa 12 mesi dopo, a partire da febbraio 2025.
Aspose si scusa per ogni eventuale disagio arrecato.
