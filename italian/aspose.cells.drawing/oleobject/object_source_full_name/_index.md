##  object_source_full_name proprietà

Restituisce il nome completo di origine del file di origine per l'oggetto OLE collegato.

###  Osservazioni

Supporta solo l'impostazione del nome completo dell'origine quando il tipo di file è OleFileType.Unknown.
Come file wav, file avi, ecc.
