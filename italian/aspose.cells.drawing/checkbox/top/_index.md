##  top proprietà

Rappresenta l'offset verticale della forma dalla riga top, in unità di pixel.

###  Osservazioni

Se la forma è nel grafico, rappresenta l'offset verticale della forma dal bordo top.

###  Esempio

```python

if shape.top == 3:
    shape.top = 1
```
