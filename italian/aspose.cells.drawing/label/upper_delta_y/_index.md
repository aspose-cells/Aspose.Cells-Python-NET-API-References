##  upper_delta_y proprietà

Ottiene o imposta l'offset verticale della forma dalla riga dell'angolo superiore sinistro.

###  Osservazioni

L'intervallo di valori è compreso tra 0 e 256.

###  Esempio

```python

if shape.upper_delta_y == 3:
    shape.upper_delta_y = 1
```
