##  lower_delta_x proprietà

Ottiene o imposta l'offset orizzontale della forma dalla colonna dell'angolo inferiore destro.

###  Osservazioni

L'intervallo di valori è compreso tra 0 e 1024.

```python

if shape.lower_delta_x == 3:
```
