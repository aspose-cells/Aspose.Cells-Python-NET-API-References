##  get_equation_paragraph(self, index) {#int}
Ottieni il paragrafo matematico specificato dalla proprietà TextBody dell'oggetto TextBox.
Avviso:
(1) Restituisce NULL quando l'indice è fuori dai limiti o non viene trovato.
(2) Restituisce anche NULL se la posizione dell'indice specificata non è un paragrafo matematico.


###  ritorna

Restituisce il paragrafo matematico specificato dall'indice.


```python

    ...
```


| Parametro| Tipo| Descrizione|
| :- | :- | :- |
| index | int | Indice di posizione del paragrafo matematico, a partire da 0.|
