##  upper_left_row proprietà

Rappresenta l'indice della riga superiore.

###  Osservazioni

Se la forma è nella forma o nel gruppo, UpperLeftRow verrà ignorato.

###  Esempio

```python

if shape.upper_left_row == 3:
    shape.upper_left_row = 1
```
