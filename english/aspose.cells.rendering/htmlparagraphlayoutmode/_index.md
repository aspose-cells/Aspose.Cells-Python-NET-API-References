## HtmlParagraphLayoutMode enumeration

Specifies how HTML <p> elements are rendered when loading HTML.



The HtmlParagraphLayoutMode type exposes the following members:

### Fields
| Field | Description |
| :- | :- |
| NORMAL | Treat <p> inside <td> as inline content in a single cell. |
| AS_ROW | Starts a new row for each <p> element. |
