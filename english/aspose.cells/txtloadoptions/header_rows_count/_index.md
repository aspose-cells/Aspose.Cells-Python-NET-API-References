## header_rows_count property


The count of header rows to be repeated for extended sheets.

### Remarks 


The header rows specified by this property will be duplicated for those extended sheets.
This property only takes effect when [`TxtLoadOptions.extend_to_next_sheet`](/cells/python-net/aspose.cells/txtloadoptions#extend_to_next_sheet) is true.
