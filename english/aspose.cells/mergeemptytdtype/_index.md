## MergeEmptyTdType enumeration

Represents the merge type for an empty TD element when exporting a file to HTML.



The MergeEmptyTdType type exposes the following members:

### Fields
| Field | Description |
| :- | :- |
| DEFAULT | Display like MS Excel. |
| NONE | Empty TD elements will not be merged when exporting file to html. <br/>This will generate a significantly larger html file. |
| MERGE_FORCELY | Merging empty TD element forcedly when exporting file to html. <br/>The size of html file will be reduced significantly after setting value to true.<br/>If you want to import the html file to excel or export perfect grid lines when saving file to html, <br/>please keep the default value. |
