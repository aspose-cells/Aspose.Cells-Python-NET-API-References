## VisibilityType enumeration

Represents the states for sheet visibility.



The VisibilityType type exposes the following members:

### Fields
| Field | Description |
| :- | :- |
| VISIBLE | Indicates the sheet is visible. |
| HIDDEN | Indicates the sheet is hidden, but can be shown by the user via the user interface. |
| VERY_HIDDEN | Indicates the sheet is hidden and cannot be shown in the user interface (UI). <br/>This state is only available programmatically. |
