## DataBarNegativeColorType enumeration

Specifies whether to use the same border and fill color as positive data bars.



The DataBarNegativeColorType type exposes the following members:

### Fields
| Field | Description |
| :- | :- |
| COLOR | Use the color specified in the Negative Value and Axis Setting dialog box <br/>or by using the ColorType and BorderColorType properties of the NegativeBarFormat object. |
| SAME_AS_POSITIVE | Use the same color as positive data bars. |
