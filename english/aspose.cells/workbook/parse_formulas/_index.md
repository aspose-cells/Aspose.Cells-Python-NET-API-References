## parse_formulas(self, ignore_error) {#bool}

Parses all formulas which have not been parsed when they were loaded from template file or set to a cell.



```python

    ...
```


| Parameter | Type | Description |
| :- | :- | :- |
| ignore_error | bool | Whether ignore error for invalid formula.<br/>For one invalid formula, if ignore error then this formula will be ignored<br/>and the process will continue to parse other formulas, otherwise exception will be thrown. |
