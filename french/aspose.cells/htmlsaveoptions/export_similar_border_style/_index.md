##  export_similar_border_style propriété

Indique si l'exportation du style de bordure similaire lorsque le style de bordure n'est pas pris en charge par les navigateurs.
Si vous souhaitez importer le fichier html ou mht vers Excel, veuillez conserver la valeur par défaut.
La valeur par défaut est fausse.
