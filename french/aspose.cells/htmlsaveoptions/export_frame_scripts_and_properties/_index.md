##  export_frame_scripts_and_properties propriété

Indique si les scripts de cadre et les propriétés du document doivent être exportés. La valeur par défaut est « true ». Si vous souhaitez importer le fichier HTML ou MHT
pour exceller, veuillez conserver la valeur par défaut.
