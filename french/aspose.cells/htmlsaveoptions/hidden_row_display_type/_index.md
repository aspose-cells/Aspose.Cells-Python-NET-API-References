##  hidden_row_display_type propriété

Ligne cachée (la hauteur de cette ligne est 0) dans Excel, avant de l'enregistrer au format HTML,
si HtmlHiddenRowDisplayType est "Remove", la ligne masquée ne sera pas affichée,
si la valeur est "Cachée", la ligne serait affichée, mais était masquée, la valeur par défaut est "Cachée"
