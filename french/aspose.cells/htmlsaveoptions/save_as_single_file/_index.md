##  save_as_single_file propriété

Indique si le code HTML doit être enregistré sous forme de fichier unique.
La valeur par défaut est false.

###  Remarques

S'il y a plusieurs feuilles de travail ou d'autres ressources requises telles que des images dans le classeur,
 généralement, ces feuilles de travail et autres ressources doivent être enregistrées dans des fichiers séparés.
Dans certains scénarios, l'utilisateur peut avoir besoin d'obtenir un seul fichier résultant, par exemple pour faciliter le transfert.
Si tel est le cas, l’utilisateur peut définir cette propriété sur true.
