##  default_font_name propriété

Spécifiez le nom de police par défaut pour l'exportation HTML, la police par défaut sera utilisée lorsque la police de style n'existe pas,
Si cette propriété est nulle, Aspose.Cells utilisera une police universelle qui a la même famille que la police d'origine,
la valeur par défaut est null.
