##  enable_css_custom_properties propriété

Optimisez la sortie HTML en utilisant des propriétés CSS personnalisées. Par exemple, dans le cas où une image base64 présente plusieurs occurrences, les données de l'image ne doivent être enregistrées qu'une seule fois grâce à la propriété personnalisée, ce qui améliore les performances du code HTML obtenu.
La valeur par défaut est false.
