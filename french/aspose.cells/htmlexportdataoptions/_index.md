##  HtmlExportDataOptions énumération
Représente les options d'exportation de données HTML.



Le type HtmlExportDataOptions expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| TABLE | Exportez le fichier au format HTML qui ne contient qu'une partie du tableau.|
| ALL | Exportez toutes les données au format HTML.|
