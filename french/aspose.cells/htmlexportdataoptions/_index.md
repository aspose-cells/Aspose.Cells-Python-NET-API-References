##  HtmlExportDataOptions dénombrement
Représente les options d'exportation de données HTML.



Le type HtmlExportDataOptions expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| TABLE | Exporter le fichier au format HTML qui ne contient qu'une partie du tableau.|
| ALL | Exportez toutes les données en html.|
