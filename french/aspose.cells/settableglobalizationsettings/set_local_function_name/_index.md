##  set_local_function_name(self, standard_name, local_name, bidirectional) {#str-str-bool}
Définit le nom de la fonction dépendante des paramètres régionaux correspondant au nom de fonction standard donné.



```python

def set_local_function_name(self, standard_name, local_name, bidirectional):
    ...
```


| Paramètre| Taper| Description|
| :- | :- | :- |
| standard_name | str | Nom de fonction standard (locale en-US).|
| local_name | str | Nom de fonction dépendant des paramètres régionaux|
| bidirectional | bool | Si le nom de la fonction locale doit être mappé automatiquement au nom de la fonction standard.<br/>Si vrai, le nom local sera automatiquement mappé au nom standard<br/>l'utilisateur n'a donc pas besoin d'appeler à nouveau le [`SettableGlobalizationSettings.set_standard_function_name`](/cells/python-net/fr/aspose.cells/settableglobalizationsettings/set_standard_function_name)<br/> pour la même paire de noms standard et locaux|
