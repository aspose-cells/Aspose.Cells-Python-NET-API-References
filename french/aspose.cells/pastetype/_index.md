##  PasteType énumération
Représente le type spécial de pâte.



Le type PasteType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| ALL | Copie toutes les données de la plage.|
| DEFAULT |Cela fonctionne comme le comportement « Tout » de MS Excel.|
| ALL_EXCEPT_BORDERS | Copie toutes les données de la plage sans la plage.|
| DEFAULT_EXCEPT_BORDERS | Cela fonctionne comme le comportement « Tout sauf les bordures » de MS Excel.|
| COLUMN_WIDTHS | Copie uniquement les largeurs de la plage.|
| ROW_HEIGHTS | Copie uniquement les hauteurs de la plage.|
| COMMENTS | Copie uniquement les commentaires dans la plage.|
| FORMATS | Copie uniquement les formats de la plage.|
| FORMULAS | Copie uniquement les formules de la plage.|
| FORMULAS_AND_NUMBER_FORMATS | Copie uniquement les formules et les formats numériques de la plage.|
| VALIDATION | Copie uniquement les validations dans la plage.|
| VALUES | Copie uniquement les valeurs de la plage.|
| VALUES_AND_FORMATS | Copie uniquement les valeurs et les formats de la plage.|
| VALUES_AND_NUMBER_FORMATS | Copie uniquement les valeurs et les formats numériques de la plage.|
