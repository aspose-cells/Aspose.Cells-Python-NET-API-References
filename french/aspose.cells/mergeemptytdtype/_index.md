##  MergeEmptyTdType énumération
Représente le type de fusion pour l'élément TD vide lors de l'exportation du fichier au format HTML.



Le type MergeEmptyTdType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| DEFAULT | Afficher comme MS Excel.|
| NONE | Les éléments TD vides ne seront pas fusionnés lors de l'exportation du fichier au format HTML.<br/> Cela générera un fichier HTML beaucoup plus volumineux.|
| MERGE_FORCELY | Fusion forcée d'un élément TD vide lors de l'exportation d'un fichier au format HTML.<br/>La taille du fichier HTML sera considérablement réduite après avoir défini la valeur sur true.<br/> Si vous souhaitez importer le fichier HTML vers Excel ou exporter des lignes de grille parfaites lors de l'enregistrement du fichier au format HTML,<br/> veuillez conserver la valeur par défaut.|
