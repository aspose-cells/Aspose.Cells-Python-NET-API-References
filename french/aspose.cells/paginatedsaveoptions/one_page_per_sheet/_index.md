##  one_page_per_sheet propriété

 Si OnePagePerSheet est true , tout le contenu d'une feuille sortira sur une seule page dans le résultat.
 Le format de papier de pagesetup sera invalide, et les autres paramètres de pagesetup
prendra toujours effet.
