##  TxtLoadStyleStrategy énumération
Spécifie comment appliquer le style aux valeurs analysées lors de la conversion d'une valeur de chaîne en nombre ou en datetime.



Le type TxtLoadStyleStrategy expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| NONE |Ne définit pas le style pour la valeur analysée.|
| BUILT_IN | Définissez le style comme nombre/dateheure intégré lorsque les valeurs analysées sont des valeurs numériques/dateheure simples.|
| EXACT_FORMAT | Définissez le format personnalisé exact pour la valeur analysée afin que la valeur formatée soit identique à celle d'entrée d'origine.|
