##  ConditionalFormattingValue classe
Décrit les valeurs des points d'interpolation dans une échelle de dégradé, une barre de données ou un ensemble d'icônes.



Le type ConditionalFormattingValue expose les membres suivants :

###  Propriétés
| Propriété| Description|
| :- | :- |
| [value](/cells/python-net/fr/aspose.cells/conditionalformattingvalue/value) |Obtenez ou définissez la valeur de cet objet de valeur de mise en forme conditionnelle.<br/> Il doit être utilisé conjointement avec Type.|
| [type](/cells/python-net/fr/aspose.cells/conditionalformattingvalue/type) | Obtenez ou définissez le type de cet objet de valeur de mise en forme conditionnelle.<br/> Définition du type sur FormatConditionValueType.Min ou FormatConditionValueType.Max<br/> définira automatiquement "Valeur" sur null.|
| [is_gte](/cells/python-net/fr/aspose.cells/conditionalformattingvalue/is_gte) | Obtenir ou définir l'indicateur Supérieur ou égal.<br/> Utiliser uniquement pour les ensembles d'icônes, détermine si cette valeur de seuil utilise<br/> l'opérateur supérieur ou égal à.<br/>false' indique que 'supérieur à' est utilisé à la place de 'supérieur ou égal à'.<br/> La valeur par défaut est true.|
