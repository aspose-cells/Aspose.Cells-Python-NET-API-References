##  ColorScale classe
 Décrivez la règle de mise en forme conditionnelle ColorScale.
Cette règle de mise en forme conditionnelle crée une échelle de couleurs graduée sur les cellules.



Le type ColorScale expose les membres suivants :

| :- | :- |
| [is_3_color_scale](/cells/python-net/fr/aspose.cells/colorscale/is_3_color_scale) | Indique si la mise en forme conditionnelle est à 3 échelles de couleurs.|
| [min_cfvo](/cells/python-net/fr/aspose.cells/colorscale/min_cfvo) | Obtenez ou définissez l'objet de valeur minimale de ce ColorScale.<br/> Impossible de définir null ou CFValueObject avec le type FormatConditionValueType.Max.|

| [max_cfvo](/cells/python-net/fr/aspose.cells/colorscale/max_cfvo) | Obtenez ou définissez l'objet de valeur maximale de ce ColorScale.<br/> Impossible de définir null ou CFValueObject avec le type FormatConditionValueType.Min.|
