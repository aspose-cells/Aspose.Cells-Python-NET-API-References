##  set_style(self, style) {#aspose.cells.Style}
Définit le style de cette ligne.



```python

def set_style(self, style):
    ...
```


| Paramètre| Taper| Description|
| :- | :- | :- |
| style | [`Style`](/cells/python-net/fr/aspose.cells/style) | le style à utiliser comme style par défaut pour les cellules de cette ligne.|
###  Remarques

Cette méthode définit uniquement le style donné comme style par défaut pour cette ligne,
sans modifier les paramètres de style des cellules existantes dans cette ligne.
Pour mettre à jour les paramètres de style des cellules existantes selon le style spécifié en même temps,
veuillez utiliser le [`Row.apply_style`](/cells/python-net/fr/aspose.cells/row/apply_style)
