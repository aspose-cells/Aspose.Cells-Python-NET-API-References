##  ShiftType énumération
Représente les options de décalage lors de la suppression d’une plage de cellules.



Le type ShiftType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| DOWN | Déplacez les cellules vers le bas.|
| LEFT | Décaler les cellules vers la gauche.|
| NONE | Ne déplacez pas les cellules.|
| RIGHT | Déplacez les cellules vers la droite.|
| UP | Déplacez les cellules vers le haut.|
