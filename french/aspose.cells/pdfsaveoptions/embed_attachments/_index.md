##  embed_attachments propriété

Indique s'il faut intégrer une pièce jointe pour les objets Ole dans Excel.

###  Remarques

La valeur par défaut est « false ». Elle doit être « false » lorsque la conformité PDF/A est définie ou que le chiffrement PDF est activé.
