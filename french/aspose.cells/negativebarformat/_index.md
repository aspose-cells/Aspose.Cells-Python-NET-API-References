##  NegativeBarFormat classe
Représente les paramètres de couleur des barres de données pour les valeurs négatives définies par une règle de mise en forme conditionnelle de la barre de données.



Le type NegativeBarFormat expose les membres suivants :

###  Propriétés
| Propriété| Description|
| :- | :- |
| [border_color](/cells/python-net/fr/aspose.cells/negativebarformat/border_color) |Obtient ou définit un objet FormatColor que vous pouvez utiliser pour spécifier la couleur de bordure des barres de données négatives.|
| [border_color_type](/cells/python-net/fr/aspose.cells/negativebarformat/border_color_type) | Obtient s'il faut utiliser la même couleur de bordure que les barres de données positives.|
| [color](/cells/python-net/fr/aspose.cells/negativebarformat/color) | Obtient ou définit un objet FormatColor que vous pouvez utiliser pour spécifier la couleur de remplissage des barres de données négatives.|
| [color_type](/cells/python-net/fr/aspose.cells/negativebarformat/color_type) | Obtient ou définit s'il faut utiliser la même couleur de remplissage que les barres de données positives.|
