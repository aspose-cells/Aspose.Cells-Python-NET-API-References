##  PageSetup classe
 Encapsule l'objet qui représente la description de la mise en page.
L'objet PageSetup contient toutes les options de mise en page.



Le type PageSetup expose les membres suivants :

###  Exemple

```python
from aspose.cells import Workbook

workbook = Workbook()
sheets = workbook.worksheets
# Add a worksheet
sheets.add()
sheet = sheets[1]
pageSetup = sheet.page_setup
pageSetup.print_area = "D1:K13"

```
