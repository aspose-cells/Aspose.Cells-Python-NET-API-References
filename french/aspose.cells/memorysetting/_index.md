##  MemorySetting dénombrement
Options d'utilisation de la mémoire.



Le type MemorySetting expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| NORMAL | Option par défaut pour le modèle de cellules.|
| MEMORY_PREFERENCE | Performances de la mémoire préférables.<br/>Avec cette option, les données seront conservées dans un format compact, donc pour les scénarios courants, cela peut réduire le coût de la mémoire.<br/> Cependant, cette option peut également dégrader un peu les performances R/W dans certains cas particuliers.|
