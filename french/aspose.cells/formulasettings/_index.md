##  FormulaSettings classe
Paramètres des formules et calcul.



Le type FormulaSettings expose les membres suivants :

###  Propriétés
| Propriété| Description|
| :- | :- |
| [calculate_on_open](/cells/python-net/fr/aspose.cells/formulasettings/calculate_on_open) | Indique si l'application doit effectuer un calcul complet lorsque le classeur est ouvert.|
| [calculate_on_save](/cells/python-net/fr/aspose.cells/formulasettings/calculate_on_save) | Indique si le classeur doit être recalculé avant d'enregistrer le document, en mode de calcul manuel.|
| [force_full_calculation](/cells/python-net/fr/aspose.cells/formulasettings/force_full_calculation) | Indique si toutes les formules sont calculées à chaque fois qu'un calcul est déclenché.|
| [calculation_mode](/cells/python-net/fr/aspose.cells/formulasettings/calculation_mode) | Obtient ou définit le mode de calcul du classeur dans MS Excel.|
| [calculation_id](/cells/python-net/fr/aspose.cells/formulasettings/calculation_id) | Spécifie la version du moteur de calcul utilisé pour calculer les valeurs dans le classeur.|
| [enable_iterative_calculation](/cells/python-net/fr/aspose.cells/formulasettings/enable_iterative_calculation) | Indique si le calcul itératif doit être activé pour résoudre les références circulaires.|
| [max_iteration](/cells/python-net/fr/aspose.cells/formulasettings/max_iteration) | Le nombre maximal d'itérations pour résoudre une référence circulaire.|
| [max_change](/cells/python-net/fr/aspose.cells/formulasettings/max_change) | Le changement maximal pour résoudre une référence circulaire.|
| [precision_as_displayed](/cells/python-net/fr/aspose.cells/formulasettings/precision_as_displayed) |Si la précision du résultat calculé doit être définie telle qu'elle est affichée lors du calcul des formules|
| [enable_calculation_chain](/cells/python-net/fr/aspose.cells/formulasettings/enable_calculation_chain) | Activer la chaîne de calcul pour les formules. La valeur par défaut est « false ».|
| [preserve_padding_spaces](/cells/python-net/fr/aspose.cells/formulasettings/preserve_padding_spaces) | Indique si les espaces et les sauts de ligne qui sont remplis entre les jetons de formule doivent être conservés<br/>tout en obtenant et en définissant des formules.<br/> La valeur par défaut est faux.|
