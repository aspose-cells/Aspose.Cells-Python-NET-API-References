##  SaveFormat énumération
Représente le format dans lequel le classeur est enregistré.



Le type SaveFormat expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| CSV | Fichier texte à valeurs séparées par des virgules (CSV).<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`SaveFormat.CSV`](/cells/python-net/fr/aspose.cells/saveformat#CSV).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| XLSX | Représente un fichier xlsx.|
| XLSM | Représente un fichier xlsm qui active les macros.|
| XLTX |Représente un fichier xltx.|
| XLTM | Représente un fichier xltm qui active les macros.|
| XLAM | Représente un fichier xltm qui active les macros complémentaires.|
| TSV | Fichier texte de valeurs séparées par des tabulations (TSV).<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`SaveFormat.TSV`](/cells/python-net/fr/aspose.cells/saveformat#TSV).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| TAB_DELIMITED | Représente un fichier texte délimité par des tabulations, identique à [`SaveFormat.TSV`](/cells/python-net/fr/aspose.cells/saveformat#TSV).|
| HTML | Représente un fichier html.|
| M_HTML |Représente un fichier mhtml.|
| ODS | Ouvrir le fichier Document Sheet(ODS).<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`SaveFormat.ODS`](/cells/python-net/fr/aspose.cells/saveformat#ODS).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| EXCEL_97_TO_2003 | Représente un fichier xls Excel97-2003.|
| SPREADSHEET_ML | Représente un fichier XML Excel 2003.|
| XLSB | Représente un fichier xlsb.|
| AUTO | Si vous enregistrez le fichier sur le disque, le format du fichier correspond à l'extension du nom du fichier.<br/> Si vous enregistrez le fichier dans le flux, le format de fichier est xlsx.|
| UNKNOWN | Représente un format non reconnu, ne peut pas être enregistré.|
| PDF | Représente un fichier PDF.|
| XPS | Format XPS (spécification papier XML).<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`SaveFormat.XPS`](/cells/python-net/fr/aspose.cells/saveformat#XPS).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| TIFF | Représente un fichier TIFF.<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`SaveFormat.TIFF`](/cells/python-net/fr/aspose.cells/saveformat#TIFF).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| SVG | Fichier SVG.<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`SaveFormat.SVG`](/cells/python-net/fr/aspose.cells/saveformat#SVG).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| DIF | Format d'échange de données.|
| OTS | Ouvrez le fichier de modèle de document (OTS).|
| XLT | Fichier modèle Excel 97-2003.|
| XML | Représente un fichier XML simple.|
| NUMBERS | Représente un fichier de nombres.|
| MARKDOWN | Représente le document Markdown.|
| FODS | Représente le format de fichier OpenDocument Flat XML Spreadsheet (.fods).<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`SaveFormat.FODS`](/cells/python-net/fr/aspose.cells/saveformat#FODS).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| SXC | Représente le format de fichier StarOffice Calc Spreadsheet (.sxc).<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`SaveFormat.SXC`](/cells/python-net/fr/aspose.cells/saveformat#SXC).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| PPTX | Représente le fichier .pptx.|
| DOCX | Représente le fichier .docx.|
| EMF | Windows Métafichier amélioré.|
| JPG | JPEG JFIF.|
| PNG | Graphiques réseau portables.|
| BMP | Windows Bitmap|
| GIF | Gif|
| JSON | Json|
| SQL_SCRIPT | SQL|
| X_HTML | Représente le fichier XHtml.<br/>veuillez utiliser la propriété [`HtmlSaveOptions.html_version`](/cells/python-net/fr/aspose.cells/htmlsaveoptions#html_version).<br/> Cette propriété sera supprimée 6 mois plus tard soit mars 2025.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| EPUB | Représente le fichier Epub.|
| AZW3 | Représente le fichier Azw3.|
| PCL | PCL (langage de commande d'imprimante)|
| DBF | Fichier de données Xbase|
