##  SaveFormat dénombrement
Représente le format dans lequel le classeur est enregistré.



Le type SaveFormat expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| CSV |Fichier texte de valeurs séparées par des virgules (CSV).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [SaveFormat.CSV](/cells/python-net/fr/aspose.cells/saveformat#CSV).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| XLSX | Représente un fichier xlsx.|
| XLSM | Représente un fichier xlsm qui active les macros.|
| XLTX | Représente un fichier xltx.|
| XLTM | Représente un fichier xltm qui active les macros.|
| XLAM | Représente un fichier xltm qui active les macros complémentaires.|
| TSV | Fichier texte de valeurs séparées par des tabulations (TSV).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [SaveFormat.TSV](/cells/python-net/fr/aspose.cells/saveformat#TSV).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| TAB_DELIMITED | Représente un fichier texte délimité par des tabulations, identique à [SaveFormat.TSV](/cells/python-net/fr/aspose.cells/saveformat#TSV).|
| HTML | Représente un fichier html.|
| M_HTML | Représente un fichier mhtml.|
| ODS | Ouvrez le fichier Feuille de document (ODS).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [SaveFormat.ODS](/cells/python-net/fr/aspose.cells/saveformat#ODS).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| EXCEL_97_TO_2003 | Représente un fichier xls Excel97-2003.|
| SPREADSHEET_ML | Représente un fichier xml Excel 2003.|
| XLSB | Représente un fichier xlsb.|
| AUTO | Si vous enregistrez le fichier sur le disque, le format du fichier correspond à l'extension du nom du fichier.<br/> Si vous enregistrez le fichier dans le flux, le format de fichier est xlsx.|
| UNKNOWN | Représente un format non reconnu, ne peut pas être enregistré.|
| PDF | Représente un fichier Pdf.|
| XPS |XPS (spécification papier XML).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [SaveFormat.XPS](/cells/python-net/fr/aspose.cells/saveformat#XPS).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| TIFF | Représente un fichier TIFF.<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [SaveFormat.TIFF](/cells/python-net/fr/aspose.cells/saveformat#TIFF).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| SVG | fichier SVG.<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [SaveFormat.SVG](/cells/python-net/fr/aspose.cells/saveformat#SVG).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| DIF | Format d'échange de données.|
| OTS | Ouvrez le fichier de feuille de modèle de document (OTS).|
| XLT | Fichier modèle Excel 97-2003.|
| XML | Représente un simple fichier xml.|
| NUMBERS | Représente un fichier de nombres.|
| MARKDOWN | Représente le document de démarque.|
| FODS | Représente le format de fichier OpenDocument Flat XML Spreadsheet (.fods).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [SaveFormat.FODS](/cells/python-net/fr/aspose.cells/saveformat#FODS).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| SXC | Représente le format de fichier de feuille de calcul StarOffice Calc (.sxc).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [SaveFormat.SXC](/cells/python-net/fr/aspose.cells/saveformat#SXC).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| PPTX | Représente le fichier .pptx.|
| DOCX |Représente le fichier .docx.|
| EMF | Windows Métafichier amélioré.|
| JPG | JPEG JFIF.|
| PNG | Portable Network Graphics.|
| BMP | Windows Bitmap|
| GIF | GIF|
| JSON | Json|
| SQL_SCRIPT | SQL|
