##  SaveFormat énumération
Représente le format dans lequel le classeur est enregistré.



Le type SaveFormat expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| CSV | Fichier texte de valeurs séparées par des virgules (CSV).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [`SaveFormat.CSV`](/cells/python-net/fr/aspose.cells/saveformat#CSV).<br/> Ce bien sera supprimé 6 mois plus tard depuis avril 2021.<br/> Le Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| XLSX | Représente un fichier xlsx.|
| XLSM | Représente un fichier xlsm qui active les macros.|
| XLTX | Représente un fichier XLTX.|
| XLTM | Représente un fichier XLTM qui active les macros.|
| XLAM | Représente un fichier XLTM qui active les macros complémentaires.|
| TSV | Fichier texte Valeurs séparées par des tabulations (TSV).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [`SaveFormat.TSV`](/cells/python-net/fr/aspose.cells/saveformat#TSV).<br/> Ce bien sera supprimé 6 mois plus tard depuis avril 2021.<br/> Le Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| TAB_DELIMITED | Représente un fichier texte délimité par des tabulations, identique à [`SaveFormat.TSV`](/cells/python-net/fr/aspose.cells/saveformat#TSV).|
| HTML | Représente un fichier HTML.|
| M_HTML | Représente un fichier mhtml.|
| ODS | Ouvrez le fichier Feuille de document (ODS).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [`SaveFormat.ODS`](/cells/python-net/fr/aspose.cells/saveformat#ODS).<br/> Ce bien sera supprimé 6 mois plus tard depuis avril 2021.<br/> Le Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| EXCEL_97_TO_2003 | Représente un fichier xls Excel97-2003.|
| SPREADSHEET_ML | Représente un fichier XML Excel 2003.|
| XLSB | Représente un fichier xlsb.|
| AUTO | Si vous enregistrez le fichier sur le disque, le format de fichier correspond à l'extension du nom de fichier.<br/>Si vous enregistrez le fichier dans le flux, le format de fichier est xlsx.|
| UNKNOWN | Représente un format non reconnu, ne peut pas être enregistré.|
| PDF | Représente un fichier PDF.|
| XPS | Format XPS (spécification papier XML).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [`SaveFormat.XPS`](/cells/python-net/fr/aspose.cells/saveformat#XPS).<br/> Ce bien sera supprimé 6 mois plus tard depuis avril 2021.<br/> Le Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| TIFF | Représente un fichier TIFF.<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [`SaveFormat.TIFF`](/cells/python-net/fr/aspose.cells/saveformat#TIFF).<br/> Ce bien sera supprimé 6 mois plus tard depuis avril 2021.<br/> Le Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| SVG | Fichier SVG.<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [`SaveFormat.SVG`](/cells/python-net/fr/aspose.cells/saveformat#SVG).<br/> Ce bien sera supprimé 6 mois plus tard depuis avril 2021.<br/> Le Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| DIF | Format d'échange de données.|
| OTS | Ouvrez le fichier de feuille de modèle de document (OTS).|
| XLT | Fichier modèle Excel 97-2003.|
| XML | Représente un simple fichier XML.|
| NUMBERS | Représente un fichier de nombres.|
| MARKDOWN | Représente un document de démarque.|
| FODS | Représente le format de fichier de feuille de calcul OpenDocument Flat XML (.fods).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [`SaveFormat.FODS`](/cells/python-net/fr/aspose.cells/saveformat#FODS).<br/> Ce bien sera supprimé 6 mois plus tard depuis avril 2021.<br/> Le Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| SXC | Représente le format de fichier StarOffice Calc Spreadsheet (.sxc).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [`SaveFormat.SXC`](/cells/python-net/fr/aspose.cells/saveformat#SXC).<br/> Ce bien sera supprimé 6 mois plus tard depuis avril 2021.<br/> Le Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| PPTX | Représente le fichier .pptx.|
| DOCX | Représente le fichier .docx.|
| EMF | Windows Métafichier amélioré.|
| JPG | JPEG JFIF.|
| PNG | Portable Network Graphics.|
| BMP | Windows Bitmap|
| GIF | Gifs|
| JSON | Json|
| SQL_SCRIPT | SQL|
| X_HTML | Représente un fichier XHtml.|
| EPUB | Représente le fichier Epub.|
