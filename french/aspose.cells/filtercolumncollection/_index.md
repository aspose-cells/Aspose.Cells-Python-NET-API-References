##  FilterColumnCollection classe
Collection d'objets Filter qui représente tous les filtres d'une plage filtrée automatiquement.



Le type FilterColumnCollection expose les membres suivants :

###  Propriétés
| Propriété| Description|
| :- | :- |
| [capacity](/cells/python-net/fr/aspose.cells/filtercolumncollection/capacity) | Obtient ou définit le nombre d'éléments que la liste de tableau peut contenir.|
