##  header_columns_count propriété

Nombre de colonnes d'en-tête à répéter pour les feuilles étendues.

###  Remarques

Les colonnes d'en-tête spécifiées par cette propriété seront dupliquées pour ces feuilles étendues.
Cette propriété ne prend effet que lorsque [`TxtLoadOptions.extend_to_next_sheet`](/cells/python-net/fr/aspose.cells/txtloadoptions#extend_to_next_sheet) est vrai.
