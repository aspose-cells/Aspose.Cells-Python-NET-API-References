Définit le style de cette colonne.

    ...

| Paramètre| Taper| Description|
| :- | :- | :- |
| style | [`Style`](/cells/python-net/fr/aspose.cells/style) | le style à utiliser comme style par défaut pour les cellules de cette colonne.|
###  Remarques

Cette méthode définit uniquement le style donné comme style par défaut pour cette colonne,
sans modifier les paramètres de style des cellules existantes dans cette colonne.
Pour mettre à jour les paramètres de style des cellules existantes selon le style spécifié en même temps,
veuillez utiliser le [`Column.apply_style`](/cells/python-net/fr/aspose.cells/column/apply_style)
