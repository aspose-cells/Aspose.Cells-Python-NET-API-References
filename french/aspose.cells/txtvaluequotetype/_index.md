##  TxtValueQuoteType énumération
Spécifie le type d'utilisation de guillemets pour les valeurs dans les fichiers au format texte.



Le type TxtValueQuoteType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| NORMAL | Toutes les valeurs contenant des caractères spéciaux tels que des guillemets ou un caractère séparateur seront citées.<br/> Idem avec le comportement de MS Excel pour l'exportation d'un fichier texte.|
| ALWAYS | Toutes les valeurs seront toujours citées.|
| MINIMUM | Ne citez les valeurs que lorsque cela est nécessaire. Par exemple, si une valeur contient un guillemet mais que le guillemet n’est pas au début de cette valeur, cette valeur ne sera pas citée.|
| NEVER |Toutes les valeurs ne seront pas citées. Le fichier texte exporté avec ce type risque de ne pas être relu correctement car les guillemets nécessaires sont absents.|
