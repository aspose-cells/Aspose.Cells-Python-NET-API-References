##  TxtValueQuoteType énumération
Spécifie le type d'utilisation des guillemets pour les valeurs dans les fichiers au format texte.



Le type TxtValueQuoteType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| NORMAL | Toutes les valeurs contenant des caractères spéciaux tels que des guillemets ou des caractères séparateurs seront entre guillemets.<br/>Même chose avec le comportement de ms excel pour l'exportation de fichiers texte.|
| ALWAYS | Toutes les valeurs seront toujours citées.|
| MINIMUM | N'utilisez des guillemets que si nécessaire. Par exemple, si une valeur contient des guillemets, mais que ceux-ci ne sont pas placés au début de cette valeur, celle-ci ne sera pas mise entre guillemets.|
| NEVER | Toutes les valeurs ne seront pas entre guillemets. Le fichier texte exporté avec ce type risque de ne pas être lu correctement, car les guillemets nécessaires sont absents.|
