##  TxtValueQuoteType dénombrement
Spécifie le type d'utilisation des guillemets pour les valeurs dans les fichiers au format texte.



Le type TxtValueQuoteType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| NORMAL | Toutes les valeurs qui contiennent des caractères spéciaux tels que des guillemets, des caractères de séparation seront citées.<br/> Idem avec le comportement de ms excel pour l'exportation de fichier texte.|
| ALWAYS | Toutes les valeurs seront toujours citées.|
| MINIMUM | Citez uniquement les valeurs lorsque cela est nécessaire. Par exemple, si une valeur contient des guillemets mais que le guillemet n'est pas au début de cette valeur, cette valeur ne sera pas citée.|
| NEVER | Toutes les valeurs ne seront pas citées. Le fichier texte exporté avec ce type peut ne pas être relu correctement car les guillemets nécessaires sont absents.|
