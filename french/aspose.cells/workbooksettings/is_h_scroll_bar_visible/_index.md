##  is_h_scroll_bar_visible propriété

Obtient ou définit une valeur indiquant si la feuille de calcul générée contiendra une barre de défilement horizontale.

###  Remarques

La valeur par défaut est vraie.

###  Exemple

Le code suivant rend la barre de défilement horizontale invisible pour la feuille de calcul.

```python

#  Hide the horizontal scroll bar of the Excel file.
settings.is_h_scroll_bar_visible = False
```
