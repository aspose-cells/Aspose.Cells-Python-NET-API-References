##  memory_setting propriété

Obtient ou définit les options d'utilisation de la mémoire. La nouvelle option sera considérée comme l'option par défaut pour les feuilles de calcul nouvellement créées, mais ne prend pas effet pour les feuilles de calcul existantes.
