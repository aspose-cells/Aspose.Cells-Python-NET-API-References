##  check_compatibility propriété

Indique si la compatibilité avec les versions antérieures doit être vérifiée lors de l'enregistrement du classeur.

###  Remarques

La valeur par défaut est vrai.
Uniquement pour les fichiers xls ou xlt Excel97-2003.
