##  show_tabs propriété

Obtenir ou définir une valeur indiquant si les onglets du classeur sont affichés.

###  Remarques

La valeur par défaut est vrai.

###  Exemple

Le code suivant masque les onglets de feuille et les boutons de défilement des onglets de la feuille de calcul.

```python

#  Hide the spreadsheet tabs.
workbook.settings.show_tabs = False

```
