##  CellValueType énumération
Spécifie un type de valeur de cellule.



Le type CellValueType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| IS_UNKNOWN | Le type de valeur Cell est inconnu.|
| IS_NULL | Cellule vide. La valeur correspondante doit être nulle.|
| IS_NUMERIC | La valeur Cell est numérique. La valeur correspondante doit être un entier ou un double.|
| IS_DATE_TIME | La valeur Cell est datetime. La valeur correspondante doit être DateTime.|
| IS_STRING | La valeur Cell est une chaîne. La valeur correspondante doit être une chaîne.|
| IS_BOOL | La valeur Cell est booléenne. La valeur correspondante doit être booléenne.|
| IS_ERROR | Cell contient une valeur d'erreur. La valeur correspondante doit être une chaîne d'erreur.|
