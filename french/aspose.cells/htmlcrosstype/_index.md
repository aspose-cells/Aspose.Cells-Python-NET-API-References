##  HtmlCrossType dénombrement
Représente cinq types de chaîne croisée html.



Le type HtmlCrossType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| DEFAULT | Afficher comme MS Excel, dépend de la cellule suivante.<br/>Si la cellule suivante est nulle, la chaîne se croisera ou sera tronquée|
| MS_EXPORT | Affichez la chaîne comme MS Excel exportant html.|
| CROSS | Afficher HTML chaîne croisée, cette performance pour créer de gros fichiers html sera plus de dix fois plus rapide que de définir la valeur sur Default ou FitToCell.|
| CROSS_HIDE_RIGHT | Affichez la chaîne croisée HTML et masquez la chaîne de droite lorsque les textes se chevauchent.|
| FIT_TO_CELL | Afficher uniquement la chaîne dans la largeur de la cellule.|
