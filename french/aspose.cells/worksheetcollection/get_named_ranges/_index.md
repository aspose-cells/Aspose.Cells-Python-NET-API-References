##  get_named_ranges() {#}
Obtient toutes les plages nommées prédéfinies dans la feuille de calcul.


###  Retour

Un tableau d'objets Range.
Si la référence du nom défini est externe ou comporte plusieurs plages, aucun objet Range ne sera renvoyé pour ce nom.



Renvoie null si la plage nommée n'existe pas.
