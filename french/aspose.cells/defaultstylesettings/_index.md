##  DefaultStyleSettings classe
Paramètres des valeurs par défaut des propriétés de style du classeur.



Le type DefaultStyleSettings expose les membres suivants :

###  Propriétés
| Propriété| Description|
| :- | :- |
| [built_in_preference](/cells/python-net/fr/aspose.cells/defaultstylesettings/built_in_preference) | Indique si la propriété pour le format numérique est préférable lorsque le style définit à la fois un nombre intégré et un modèle personnalisé.<br/> La valeur par défaut est false, ce qui signifie que par défaut, un modèle personnalisé sera utilisé pour formater les valeurs tant qu'il n'est pas vide pour un style.|
| [font_name](/cells/python-net/fr/aspose.cells/defaultstylesettings/font_name) | Obtient/Définit le nom de police par défaut pour le classeur|
| [font_size](/cells/python-net/fr/aspose.cells/defaultstylesettings/font_size) | Obtient/définit la taille de police standard par défaut pour le classeur.|
| [horizontal_alignment](/cells/python-net/fr/aspose.cells/defaultstylesettings/horizontal_alignment) | Obtient/définit la valeur par défaut pour l'alignement horizontal|
| [vertical_alignment](/cells/python-net/fr/aspose.cells/defaultstylesettings/vertical_alignment) | Obtient/définit la valeur par défaut pour l'alignement vertical|
