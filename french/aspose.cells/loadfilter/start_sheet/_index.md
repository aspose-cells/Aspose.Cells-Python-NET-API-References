##  start_sheet(self, sheet) {#aspose.cells.Worksheet}
Prépare les options de filtre avant de charger la feuille de calcul donnée.
L'implémentation de LoadFilter par l'utilisateur peut modifier les options LoadDataFilter ici
pour indiquer comment charger les données pour cette feuille de calcul.



```python

def start_sheet(self, sheet):
    ...
```


| Paramètre| Taper| Description|

Les propriétés disponibles sont :<br/> Nom, Index, VisibilityType|
