##  custom propriété

Représente la chaîne de format numérique custom de cet objet de style.
Si le format numérique custom n'est pas défini (par exemple, le format numérique est intégré), "" sera renvoyé.

###  Remarques

La chaîne custom renvoyée est indépendante de la culture.
