##  rotation_angle propriété

Représente l'angle de rotation du texte.

###  Remarques

0 : Pas de rotation.


255 : de haut en bas.


-90 : Vers le bas.


90 : Vers le haut.


Vous pouvez définir 255 ou une valeur comprise entre -90 et 90.
