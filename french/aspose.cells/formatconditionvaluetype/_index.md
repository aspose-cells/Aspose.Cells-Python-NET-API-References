##  FormatConditionValueType dénombrement
Type de valeur de condition.



Le type FormatConditionValueType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| FORMULA | La valeur minimale/médiane/maximale de la<br/> gradient est déterminé par une formule.|
| MAX | Indique que la valeur maximale de la plage doit être<br/> utilisé comme valeur maximale pour le gradient.|
| MIN | Indique que la valeur minimale de la plage doit être<br/> utilisé comme valeur minimale pour le gradient.|
| NUMBER | Indique que le minimum / milieu / maximum<br/>la valeur du gradient est spécifiée par une constante<br/> valeur numérique.|
| PERCENT | La valeur indique un pourcentage entre le minimum<br/>et les valeurs maximales de la plage doivent être utilisées comme<br/> valeur minimale / médiane / maximale du gradient.|
| PERCENTILE | La valeur indique un classement centile dans la plage doit<br/>être utilisé comme valeur minimale / médiane / maximale<br/>pour le dégradé.|
| AUTOMATIC_MAX | Indique que la valeur maximale automatique dans la plage doit être<br/> utilisé comme valeur maximale automatique pour le dégradé.|
| AUTOMATIC_MIN | Indique que la valeur minimale automatique dans la plage doit être<br/> utilisé comme valeur minimale automatique pour le dégradé.|
