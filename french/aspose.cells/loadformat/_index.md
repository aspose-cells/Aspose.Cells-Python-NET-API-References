##  LoadFormat énumération
Représente le format du fichier de chargement.



Le type LoadFormat expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| AUTO | Représente la reconnaissance automatique du format.|
| CSV | Fichier texte à valeurs séparées par des virgules (CSV).<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`LoadFormat.CSV`](/cells/python-net/fr/aspose.cells/loadformat#CSV).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| XLSX | Représente un classeur ou un modèle Office Open XML spreadsheetML, avec ou sans macros.|
| TSV | Fichier texte de valeurs séparées par des tabulations (TSV).<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`LoadFormat.TSV`](/cells/python-net/fr/aspose.cells/loadformat#TSV).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| TAB_DELIMITED | Représente un fichier texte délimité par des tabulations, identique à [`LoadFormat.TSV`](/cells/python-net/fr/aspose.cells/loadformat#TSV).|
| HTML | Représente un fichier html.|
| M_HTML |Représente un fichier mhtml.|
| ODS | Ouvrir le fichier Document Sheet(ODS).<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`LoadFormat.ODS`](/cells/python-net/fr/aspose.cells/loadformat#ODS).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| EXCEL_97_TO_2003 | Représente un fichier xls Excel97-2003.|
| SPREADSHEET_ML | Représente un fichier XML Excel 2003.|
| XLSB | Représente un fichier xlsb.|
| OTS | Ouvrez le fichier de modèle de document (OTS).|
| NUMBERS | Représente un fichier de nombres.|
| FODS | Représente le format de fichier OpenDocument Flat XML Spreadsheet (.fods).<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`LoadFormat.FODS`](/cells/python-net/fr/aspose.cells/loadformat#FODS).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| SXC | Représente le format de fichier StarOffice Calc Spreadsheet (.sxc).<br/> REMARQUE : Ce membre est désormais obsolète. À la place,<br/>veuillez utiliser la propriété [`LoadFormat.SXC`](/cells/python-net/fr/aspose.cells/loadformat#SXC).<br/> Cette propriété sera supprimée 6 mois plus tard soit avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.|
| XML | Représente un fichier XML simple.|
| EPUB | Représente un fichier EPUB.|
| AZW3 | Représente un fichier AZW3.|
| CHM | Représente un fichier CHM.|
| MARKDOWN | Représente un fichier Markdown.|
| UNKNOWN |Représente un format non reconnu, ne peut pas être chargé.|
| IMAGE | Image|
| JSON | Json|
| DIF | Format d'échange de données.|
| DBF | Fichier de données Xbase|
