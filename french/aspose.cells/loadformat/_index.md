##  LoadFormat dénombrement
Représente le format du fichier de chargement.



Le type LoadFormat expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| AUTO | Représente la reconnaissance automatique du format.|
| CSV |Fichier texte de valeurs séparées par des virgules (CSV).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [LoadFormat.CSV](/cells/python-net/fr/aspose.cells/loadformat#CSV).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| XLSX | Représente le classeur ou le modèle Office Open XML spreadsheetML, avec ou sans macros.|
| TSV | Fichier texte de valeurs séparées par des tabulations (TSV).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [LoadFormat.TSV](/cells/python-net/fr/aspose.cells/loadformat#TSV).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| TAB_DELIMITED | Représente un fichier texte délimité par des tabulations, identique à [LoadFormat.TSV](/cells/python-net/fr/aspose.cells/loadformat#TSV).|
| HTML | Représente un fichier html.|
| M_HTML | Représente un fichier mhtml.|
| ODS | Ouvrez le fichier Feuille de document (ODS).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [LoadFormat.ODS](/cells/python-net/fr/aspose.cells/loadformat#ODS).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| EXCEL_97_TO_2003 | Représente un fichier xls Excel97-2003.|
| SPREADSHEET_ML | Représente un fichier xml Excel 2003.|
| XLSB | Représente un fichier xlsb.|
| OTS | Ouvrez le fichier de feuille de modèle de document (OTS).|
| NUMBERS | Représente un fichier de nombres.|
| FODS | Représente le format de fichier OpenDocument Flat XML Spreadsheet (.fods).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [LoadFormat.FODS](/cells/python-net/fr/aspose.cells/loadformat#FODS).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| SXC | Représente le format de fichier de feuille de calcul StarOffice Calc (.sxc).<br/> REMARQUE : ce membre est désormais obsolète. Plutôt,<br/>veuillez utiliser la propriété [LoadFormat.SXC](/cells/python-net/fr/aspose.cells/loadformat#SXC).<br/> Cette propriété sera supprimée 6 mois plus tard depuis avril 2021.<br/> Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.|
| XML | Représente un simple fichier xml.|
| UNKNOWN | Représente un format non reconnu, ne peut pas être chargé.|
| IMAGE | Image|
| JSON | Json|
