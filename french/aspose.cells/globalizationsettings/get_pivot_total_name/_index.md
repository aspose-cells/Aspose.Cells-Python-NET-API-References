##  get_pivot_total_name() {#}
Obtient le nom de l'étiquette "Total" dans le tableau croisé dynamique.
Vous devez remplacer cette méthode lorsque le tableau croisé dynamique contient au moins deux champs croisés dynamiques dans la zone de données.


###  Retour

Le nom de l'étiquette "Total"

    ...
