##  get_table_row_type_of_totals(self) {#}
Obtient le nom de type des lignes de table qui se composent de la ligne totale de la table référencée.
La valeur par défaut est « Totaux », donc dans la formule « #Totaux » représente la ligne totale de la table référencée.


###  Retour

le nom du type des lignes de la table
