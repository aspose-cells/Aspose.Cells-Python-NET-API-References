##  get_table_row_type_of_totals {#}
Obtient le nom de type des lignes de table qui constituent la ligne totale de la table référencée.
La valeur par défaut est "Totaux", donc dans la formule "#Totaux" représente la ligne totale du tableau référencé.


###  Retour

le nom du type des lignes du tableau
