##  get_table_row_type_of_current {#}
Obtient le nom de type des lignes de table constituées de la ligne actuelle dans la table référencée.
La valeur par défaut est "This Row", donc dans la formule "#This Row" représente la ligne actuelle dans la table référencée.


###  Retour

le nom du type des lignes du tableau
