##  get_table_row_type_of_current() {#}
Obtient le nom de type des lignes de table qui se compose de la ligne actuelle dans la table référencée.
La valeur par défaut est "Cette ligne", donc dans la formule "#Cette ligne" représente la ligne actuelle dans la table référencée.
