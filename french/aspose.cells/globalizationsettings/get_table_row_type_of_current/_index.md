##  get_table_row_type_of_current(self) {#}
Obtient le nom de type des lignes de table qui se composent de la ligne actuelle dans la table référencée.
La valeur par défaut est « Cette ligne », donc dans la formule « #Cette ligne » représente la ligne actuelle dans la table référencée.


###  Retour

le nom du type des lignes de la table
