##  check_mixed_value_type propriété

Faux, Aspose.Cells définira le type de DataColumn par le type de valeur de la première ligne pour les performances.
 Vrai, Aspose.Cells vérifiera si le type de valeur dans la colonne est mixte avant de définir le type de DataColumn
Et le type de valeur est mixte, le type de DataColumn sera une chaîne.
