##  HtmlVersion énumération
Indique que la version HTML est utilisée lors de l'enregistrement aux formats HTML.



Le type HtmlVersion expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| DEFAULT | Enregistrez le document conformément à l'exportation MS Excel HTML.|
| X_HTML |Enregistre le document conformément à la norme XHTML 1.0 Transitional.|
| HTML5 | Enregistre le document conformément à la norme HTML 5.|
