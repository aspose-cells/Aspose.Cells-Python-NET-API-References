##  value propriété

Obtient le value contenu dans cette cellule.

###  Remarques

Type possible :

nul,


booléen,


DateHeure,


Double,


Entier


Chaîne.


Pour int value, il peut être renvoyé sous la forme d'un objet Integer ou d'un objet Double.
Et il n'y a aucune garantie que le value retourné sera toujours conservé comme le même type d'objet.

    ...

    ...
