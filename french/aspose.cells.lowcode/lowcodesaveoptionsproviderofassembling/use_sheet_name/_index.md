##  use_sheet_name propriété

Le chemin d'accès au fichier est généré avec le nom de la feuille plutôt qu'avec son index. La valeur par défaut est « false ».

###  Remarques

Le nom de la feuille ne sera jamais reconstruit automatiquement.
Donc, lorsque vous le définissez sur vrai, assurez-vous qu'il n'y a pas de nom de feuille spécial
cela peut entraîner un chemin ou un nom de fichier non valide.
