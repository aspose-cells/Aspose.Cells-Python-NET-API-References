##  HtmlConverter classe
Convertisseur pour la conversion entre les fichiers HTML (html ou mht) et d'autres formats de fichiers de feuille de calcul.



Le type HtmlConverter expose les membres suivants :

###  Méthodes
| Méthode| Description|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/fr/aspose.cells.lowcode/htmlconverter/process/#str-str) | Convertit le fichier modèle donné entre HTML et d'autres formats.|
| [`process(, load_options, save_options)`](/cells/python-net/fr/aspose.cells.lowcode/htmlconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Convertit le fichier entre HTML et d'autres formats de fichiers de feuille de calcul.|



###  Exemple

```python
from aspose.cells.lowcode import HtmlConverter

HtmlConverter.process("Template.xlsx", "res.html")
```
