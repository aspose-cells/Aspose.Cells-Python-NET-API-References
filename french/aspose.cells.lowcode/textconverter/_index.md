##  TextConverter classe
Convertisseur pour la conversion entre les formats basés sur du texte (csv, tsv, dif...) et d'autres formats de fichiers de feuille de calcul.



Le type TextConverter expose les membres suivants :

###  Méthodes
| Méthode| Description|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/fr/aspose.cells.lowcode/textconverter/process/#str-str) | Convertit le fichier modèle donné entre des fichiers texte et d'autres formats.|
| [`process(, load_options, save_options)`](/cells/python-net/fr/aspose.cells.lowcode/textconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Convertit le format de fichier entre les formats texte et d'autres formats de fichiers de feuille de calcul|



###  Exemple

```python
from aspose.cells.lowcode import TextConverter

TextConverter.process("template.csv", "res.xlsx")

```
