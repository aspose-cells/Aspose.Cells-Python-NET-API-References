##  SpreadsheetConverter classe
Convertisseur pour la conversion entre différents formats de fichiers de feuille de calcul, tels que xls, xlsx, xlsb, tableur ml...



Le type SpreadsheetConverter expose les membres suivants :

| Méthode| Description|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/fr/aspose.cells.lowcode/spreadsheetconverter/process/#str-str) | Convertit le fichier modèle donné entre les formats de fichier de feuille de calcul.|
| [`process(, load_options, save_options)`](/cells/python-net/fr/aspose.cells.lowcode/spreadsheetconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Convertit entre différents formats de fichiers de feuille de calcul.|
