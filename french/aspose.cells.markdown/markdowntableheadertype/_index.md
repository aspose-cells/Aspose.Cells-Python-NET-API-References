##  MarkdownTableHeaderType énumération
Représente le type d'en-tête du tableau dans le fichier Markdown.



Le type MarkdownTableHeaderType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| FIRST_ROW | Première ligne comme en-tête du tableau.|
| COLUMN_HEADER | Nom de la colonne (tel que A, B, C...) comme en-tête du tableau.|
| EMPTY | Une ligne d'en-tête vide.|
