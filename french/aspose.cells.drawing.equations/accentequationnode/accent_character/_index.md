##  accent_character propriété

Cet attribut spécifie le type de signe diacritique combiné attaché à la base de la fonction d'accentuation. Le caractère d'accentuation par défaut est U+0302.
Il est fortement recommandé d'utiliser l'attribut AccentType pour définir le caractère accentué.
Utilisez ce paramètre de propriété si vous ne trouvez pas le caractère dont vous avez besoin dans un type connu.

###  Remarques

Il convient de noter que cette propriété n'accepte qu'un seul caractère et que si plusieurs caractères sont transmis, seul le premier caractère est accepté.
