##  group_chr propriété

Spécifie un symbole (par défaut U+23DF).
Il est fortement recommandé d'utiliser l'attribut ChrType pour définir le caractère accentué.
Utilisez ce paramètre de propriété si vous ne trouvez pas le caractère dont vous avez besoin dans un type connu.

###  Remarques

Il convient de noter que cette propriété n'accepte qu'un seul caractère et que si plusieurs caractères sont transmis, seul le premier caractère est accepté.
