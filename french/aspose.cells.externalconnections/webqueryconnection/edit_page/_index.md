##  edit_page propriété

L'URL de la page web destinée à l'utilisateur affichant les données de la requête. Cette URL est persistante.
dans le cas où sourceData="true" et l'url ont été redirigés pour référencer un fichier XML.
Ensuite, la page destinée à l'utilisateur peut être affichée dans l'interface utilisateur et les données XML peuvent être récupérées.
dans les coulisses.

###  Remarques

 REMARQUE : Cette propriété est désormais obsolète. À la place,
veuillez utiliser la propriété WebQueryConnection.EditWebPage.
 Cette propriété sera supprimée 12 mois plus tard soit en octobre 2017.
Aspose s'excuse pour tout inconvénient que vous avez pu rencontrer.
