##  is_xl97 propriété

Cet indicateur existe pour la compatibilité descendante avec les anciens fichiers de feuille de calcul existants et est défini
à vrai si cette requête Web a été créée dans Microsoft Excel 97.
Il s’agit d’un attribut facultatif qui peut être ignoré.
