##  WebQueryConnection classe
 Spécifie les propriétés d'une source de requête Web. Une requête Web récupère les données des tables HTML.
 et peut également fournir des paramètres HTTP « Get » à traiter par le serveur Web lors de la génération du HTML par
y compris les paramètres et les éléments de paramètres.
