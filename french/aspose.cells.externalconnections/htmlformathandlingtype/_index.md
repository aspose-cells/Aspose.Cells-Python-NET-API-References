##  HtmlFormatHandlingType énumération
Spécifie comment gérer le formatage à partir de la source HTML



Le type HtmlFormatHandlingType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| ALL | Transférez tout le formatage HTML dans la feuille de calcul avec les données.|
| NONE | Importez les données sous forme de texte non formaté (la définition des types de données se produit toujours).|
| RTF |Traduisez le formatage HTML en formatage de texte enrichi sur les données introduites dans la feuille de calcul.|
