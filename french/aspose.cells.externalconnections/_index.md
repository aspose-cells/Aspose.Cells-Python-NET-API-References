 Le**Aspose.Cells.ExternalConnections**L'espace de noms fournit des classes pour les connexions externes afin de récupérer des données à partir de sources externes telles que des bases de données et le Web.

###  Des classes
| Classe| Description|
| :- | :- |
| [`ConnectionParameter`](/cells/python-net/fr/aspose.cells.externalconnections/connectionparameter) | Spécifie les propriétés de tous les paramètres utilisés avec les connexions de données externes<br/> Les paramètres sont valides pour les requêtes ODBC et Web.|
| [`ConnectionParameterCollection`](/cells/python-net/fr/aspose.cells.externalconnections/connectionparametercollection) | Spécifie la collection [`ConnectionParameter`](/cells/python-net/fr/aspose.cells.externalconnections/connectionparameter)|
| [`DBConnection`](/cells/python-net/fr/aspose.cells.externalconnections/dbconnection) | Spécifie toutes les propriétés associées à une connexion de données externes ODBC ou OLE DB.|
| [`DataModelConnection`](/cells/python-net/fr/aspose.cells.externalconnections/datamodelconnection) | Spécifie une connexion au modèle de données|
| [`ExternalConnectionCollection`](/cells/python-net/fr/aspose.cells.externalconnections/externalconnectioncollection) | Spécifie la collection ExternalConnection|
| [`WebQueryConnection`](/cells/python-net/fr/aspose.cells.externalconnections/webqueryconnection) | Spécifie les propriétés d'une source de requête Web. Une requête Web récupère les données des tables HTML.<br/> et peut également fournir des paramètres HTTP « Get » à traiter par le serveur Web lors de la génération du HTML par<br/> y compris les paramètres et les éléments de paramètres.|
