##  command_type propriété

 Spécifie le type de commande OLE DB.
1. La requête spécifie un nom de cube
2. La requête spécifie une instruction SQL
3. La requête spécifie un nom de table
4. La requête précise que des informations par défaut ont été fournies et qu'il appartient au fournisseur de les interpréter.
5. La requête est adressée à un fournisseur de données de liste basé sur le Web.
