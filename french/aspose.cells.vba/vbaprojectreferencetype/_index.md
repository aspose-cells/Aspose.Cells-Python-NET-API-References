##  VbaProjectReferenceType dénombrement
Représente le type de référence de projet VBA.



Le type VbaProjectReferenceType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| REGISTERED | Spécifie une référence à une bibliothèque de types Automation.|
| CONTROL | Spécifie une référence à une bibliothèque de types modifiée et à sa bibliothèque de types étendue.|
| PROJECT | Spécifie une référence à un projet VBA externe.|
