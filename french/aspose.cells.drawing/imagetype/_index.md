##  ImageType dénombrement
Spécifie le type (format) d'une image.



Le type ImageType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| UNKNOWN | Un type d'image inconnu.|
| EMF | Windows Métafichier amélioré.|
| WMF | Windows Métafichier.|
| PICT | Macintosh PICT.|
| JPEG | JPEG JFIF.|
| PNG | Portable Network Graphics.|
| BMP | Windows Bitmap|
| GIF | GIF|
| TIFF | Tif|
| SVG | SVG|
| SVM | SVM|
| GLTF | glTF|
| OFFICE_COMPATIBLE_EMF | Windows Enhanced Metafile qui est plus compatible avec Office.|
| WEB_P |Format d'image Weppy|
