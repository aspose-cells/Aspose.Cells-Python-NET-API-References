##  upper_left_row propriété

Représente l’index de la ligne du coin supérieur gauche.

###  Remarques

Si la forme est dans la forme ou dans le groupe, UpperLeftRow sera ignoré.

###  Exemple

```python

if shape.upper_left_row == 3:
    shape.upper_left_row = 1
```
