##  input_range propriété

Obtient ou définit la plage de feuille de calcul utilisée pour remplir la zone de liste déroulante spécifiée.

###  Exemple

```python

if shape.input_range == "$B$6:$B10":
    shape.input_range = "$A$1:$A$5"
```
