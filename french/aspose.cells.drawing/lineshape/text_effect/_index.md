##  text_effect propriété

 Renvoie un objet TextEffectFormat qui contient les propriétés de mise en forme des effets de texte pour la forme spécifiée.
S'applique aux objets Shape qui représentent WordArt.

###  Exemple

```python

if shape.is_word_art:
    textEffectFormat = shape.text_effect
```
