##  signature_line_type propriété

Obtient ou définit le type de signature.
Par défaut - Lorsque la valeur par défaut est définie, la valeur ProviderId correspondante est fixée à {0000000000-0000-0000-0000-0000000000}.
Timbre - Lorsque la valeur est Timbre, la valeur ProviderId correspondante est généralement {000CD6A4-0000-0000-C000-000000000046}.
Personnalisé - Lorsque la valeur est Personnalisé, la valeur ProviderId correspondante doit généralement être définie par l'utilisateur. Elle doit être obtenue à partir de la documentation fournie avec le fournisseur.
