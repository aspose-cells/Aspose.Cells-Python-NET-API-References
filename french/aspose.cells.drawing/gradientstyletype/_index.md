##  GradientStyleType dénombrement
Représente le style d'ombrage dégradé.



Le type GradientStyleType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| DIAGONAL_DOWN | Style d'ombrage diagonal vers le bas|
| DIAGONAL_UP | Style d'ombrage diagonal vers le haut|
| FROM_CENTER | Du style d'ombrage central|
| FROM_CORNER | Du style d'ombrage d'angle|
| HORIZONTAL | Style d'ombrage horizontal|
| VERTICAL | Style d'ombrage vertical|
| UNKNOWN |Style d'ombrage inconnu.Uniquement pour le style d'ombrage (qui n'est pour aucun membre du GradientStyleType) dans le fichier de modèle.|
