##  GradientStyleType énumération
Représente le style d'ombrage dégradé.



Le type GradientStyleType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| DIAGONAL_DOWN | Style d'ombrage diagonal vers le bas|
| DIAGONAL_UP | Style d'ombrage diagonal vers le haut|
| FROM_CENTER | Style d'ombrage du centre|
| FROM_CORNER | Du style d'ombrage des coins|
| HORIZONTAL | Style d'ombrage horizontal|
| VERTICAL | Style d'ombrage vertical|
| UNKNOWN | Style d'ombrage inconnu. Uniquement pour le style d'ombrage (qui n'est destiné à aucun membre du GradientStyleType) dans le fichier modèle.|
