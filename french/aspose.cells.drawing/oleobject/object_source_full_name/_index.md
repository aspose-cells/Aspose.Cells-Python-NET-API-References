##  object_source_full_name propriété

Renvoie le nom complet du fichier source de l'objet OLE lié.

###  Remarques

Prend uniquement en charge la définition du nom complet de la source lorsque le type de fichier est OleFileType.Unknown.
Tels que les fichiers wav, les fichiers avi, etc.
