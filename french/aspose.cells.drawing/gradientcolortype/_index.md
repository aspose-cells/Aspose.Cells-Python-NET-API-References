##  GradientColorType dénombrement
Représente le type de couleur de dégradé pour le remplissage spécifié.



Le type GradientColorType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| NONE | Pas de dégradé de couleur|
| ONE_COLOR | Une couleur dégradée|
| PRESET_COLORS | Couleurs de dégradé prédéfinies|
| TWO_COLORS | Deux couleurs dégradées|
