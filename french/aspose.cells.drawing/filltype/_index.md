##  FillType énumération
Type de format de remplissage.



Le type FillType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| AUTOMATIC | Représente le type de formatage automatique.|
| NONE | Représente un type sans formatage.|
| SOLID | Format de remplissage solide.|
| GRADIENT | Format de remplissage dégradé.|
| TEXTURE | Format de remplissage de texture (inclut le remplissage d'image).|
| PATTERN | Format de remplissage de motif.|
| GROUP | Hériter des propriétés de remplissage du groupe.|
