##  id propriété

Spécifie un GUID qui peut être référencé avec le GUID de la ligne de signature stockée dans le contenu du document.
La valeur par défaut est Vide (tous les zéros) Guid.

###  Remarques

Lorsqu'il est défini, il associe [`SignatureLine`](/cells/python-net/fr/aspose.cells.drawing/signatureline) au [`DigitalSignature`](/cells/python-net/fr/aspose.cells.digitalsignatures/digitalsignature) correspondant.
