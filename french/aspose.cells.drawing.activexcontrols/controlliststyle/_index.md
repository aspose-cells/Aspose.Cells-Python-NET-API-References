##  ControlListStyle énumération
Représente l'apparence visuelle de la liste dans une ListBox ou une ComboBox.



Le type ControlListStyle expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| PLAIN | Affiche une liste dans laquelle l'arrière-plan d'un élément est mis en surbrillance lorsqu'il est sélectionné.|
| OPTION | Affiche une liste dans laquelle un bouton d'option ou une case à cocher à côté de chaque entrée affiche l'état de sélection de cet élément.|
