##  ActiveXPersistenceType dénombrement
Représente la méthode de persistance pour conserver un contrôle ActiveX.



Le type ActiveXPersistenceType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| PROPERTY_BAG | Les données sont stockées sous forme de données xml.|
| STORAGE | Les données sont stockées sous forme de données binaires de stockage.|
| STREAM | Les données sont stockées sous la forme d'un flux de données binaires.|
| STREAM_INIT | Les données sont stockées sous forme de données binaires streaminit.|
