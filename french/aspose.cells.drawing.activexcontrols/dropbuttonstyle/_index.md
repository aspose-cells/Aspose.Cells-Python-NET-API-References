##  DropButtonStyle énumération
Représente le symbole affiché sur le bouton de dépôt.



Le type DropButtonStyle expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| PLAIN | Affiche un bouton sans symbole.|
| ARROW | Affiche un bouton avec une flèche vers le bas.|
| ELLIPSIS | Affiche un bouton avec des points de suspension (...).|
| REDUCE | Affiche un bouton avec une ligne horizontale comme un caractère de soulignement.|
