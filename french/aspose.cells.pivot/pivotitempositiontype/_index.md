##  PivotItemPositionType énumération
Représente le type de position de l'élément de base du pivot dans le champ de base lorsque le calcul ShowDataAs est utilisé.



Le type PivotItemPositionType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| PREVIOUS | Représente l'élément pivot précédent dans le champ PivotField.|
| NEXT | Représente l'élément pivot suivant dans le champ PivotField.|
| CUSTOM | Affiche les valeurs sous un format différent en fonction de l'index de l'élément pivot dans le champ pivot.|
