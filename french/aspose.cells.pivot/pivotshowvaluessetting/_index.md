##  PivotShowValuesSetting classe
Représente les paramètres d'affichage des valeurs lorsque le calcul ShowDataAs est utilisé.



Le type PivotShowValuesSetting expose les membres suivants :

###  Propriétés
| Propriété| Description|
| :- | :- |
| [calculation_type](/cells/python-net/fr/aspose.cells.pivot/pivotshowvaluessetting/calculation_type) | Représente comment afficher les valeurs d'un champ de données dans le rapport croisé dynamique.|
| [base_field_index](/cells/python-net/fr/aspose.cells.pivot/pivotshowvaluessetting/base_field_index) | Représente le champ de base pour un calcul ShowDataAs lorsque le calcul ShowDataAs est en cours d'utilisation.|
| [base_item_position_type](/cells/python-net/fr/aspose.cells.pivot/pivotshowvaluessetting/base_item_position_type) | Représente le type de l'élément pivot de base dans le champ de base lorsque le calcul ShowDataAs est utilisé.<br/> Valable uniquement pour les champs de données.<br/>Étant donné que PivotItemPosition.Custom est uniquement destiné à la lecture, si vous devez définir PivotItemPosition.Custom,<br/> veuillez définir l'attribut PivotField.BaseItemIndex.|
| [base_item_index](/cells/python-net/fr/aspose.cells.pivot/pivotshowvaluessetting/base_item_index) | Représente l'index personnalisé de l'élément pivot dans le champ de base lorsque le calcul ShowDataAs est utilisé.<br/> Valable uniquement pour les champs de données.|
