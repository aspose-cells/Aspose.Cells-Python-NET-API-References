##  PivotConditionFormatRuleType dénombrement
Représente le type de règle de mise en forme de condition de tableau croisé dynamique.



Le type PivotConditionFormatRuleType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| NONE | Indique que la mise en forme conditionnelle Top N n'est pas évaluée|
| ALL | Indique que la mise en forme conditionnelle Top N est<br/> évalué sur l'ensemble de la plage de portée.|
| ROW | Indique que la mise en forme conditionnelle Top N est évaluée pour chaque ligne.|
| COLUMN | Indique que la mise en forme conditionnelle Top N est<br/> évaluée pour chaque colonne.|
