##  PivotConditionFormatRuleType énumération
Représente le type de règle de formatage de condition de tableau croisé dynamique.



Le type PivotConditionFormatRuleType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| NONE | Indique que la mise en forme conditionnelle Top N n'est pas évaluée|
| ALL | Indique que le formatage conditionnel Top N est<br/> évalué sur l’ensemble de la gamme de portée.|
| ROW | Indique que la mise en forme conditionnelle Top N est évaluée pour chaque ligne.|
| COLUMN | Indique que le formatage conditionnel Top N est<br/> évalué pour chaque colonne.|
