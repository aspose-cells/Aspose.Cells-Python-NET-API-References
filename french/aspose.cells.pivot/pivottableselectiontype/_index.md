##  PivotTableSelectionType énumération
Spécifie ce qui peut être sélectionné dans un tableau croisé dynamique lors d'une sélection structurée.
Ces constantes peuvent être combinées pour sélectionner plusieurs types.



Le type PivotTableSelectionType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| DATA_AND_LABEL | Données et étiquettes|
| DATA_ONLY | Sélectionne uniquement les données|
| LABEL_ONLY | Sélectionne uniquement les étiquettes|
