##  PivotTableSourceType énumération
Représente le type de source de données du tableau croisé dynamique.



Le type PivotTableSourceType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| SHEET | Spécifie que les données source sont une plage.|
| EXTERNAL |Spécifie que des données source externes sont utilisées.|
| CONSOLIDATION | Spécifie que plusieurs plages de consolidation sont utilisées comme données source.|
| SCENARIO | Les données sources sont renseignées à partir d’une structure interne temporaire.|
| UNKNOWN | Source de données inconnue.|
