##  PivotItemPosition dénombrement
Représente la position suivante/précédente/toute de l'élément de base du tableau croisé dynamique dans le champ de base .



Le type PivotItemPosition expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| PREVIOUS | Représente l'élément de pivot précédent dans PivotField.|
| NEXT | Représente l'élément pivot suivant dans PivotField.|
| CUSTOM | Représente un index d'élément pivot, tel que spécifié par Pivot Items, qui spécifie un élément pivot dans PivotField.<br/> lire seulement|
