##  hidden_row_display_type propriété

Ligne cachée (la hauteur de cette ligne est 0) dans Excel, avant de l'enregistrer au format HTML,
si HtmlHiddenRowDisplayType est « Supprimer », la ligne masquée ne sera pas affichée,
si la valeur est « Masqué », la ligne aurait été affichée, mais elle était masquée, la valeur par défaut est « Masqué »
