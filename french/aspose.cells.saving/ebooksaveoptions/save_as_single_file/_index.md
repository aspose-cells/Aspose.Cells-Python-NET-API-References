##  save_as_single_file propriété

Indique si vous enregistrez le code HTML sous forme de fichier unique.
La valeur par défaut est fausse.

###  Remarques

S'il y a plusieurs feuilles de calcul ou d'autres ressources requises telles que des images dans le classeur,
 généralement, ces feuilles de calcul et autres ressources doivent être enregistrées dans des fichiers séparés.
Pour certains scénarios, l'utilisateur n'aura peut-être besoin d'obtenir qu'un seul fichier résultant, par exemple pour faciliter le transfert.
Si tel est le cas, l'utilisateur peut définir cette propriété sur true.
