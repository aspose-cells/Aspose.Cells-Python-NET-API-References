##  TextFontAlignType énumération
Représente les différents types d'alignement de police.



Le type TextFontAlignType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| AUTOMATIC | Lorsque le flux de texte est horizontal ou vertical simple, comme dans fontBaseline<br/> mais pour les autres modes verticaux, même chose que fontCenter.|
| BOTTOM | Les lettres sont ancrées tout en bas d’une seule ligne.|
| BASELINE | Les lettres sont ancrées à la ligne de base inférieure d'une seule ligne.|
| CENTER | Les lettres sont ancrées entre les deux lignes de base d'une même ligne.|
| TOP | Les lettres sont ancrées à la ligne de base supérieure d'une seule ligne.|
