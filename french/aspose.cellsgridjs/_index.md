 Le**Aspose.Cells.GridJs** L'espace de noms contient toutes les classes de GridJ qui fournissent des opérations de base pour gérer des feuilles de calcul sans utiliser Microsoft Excel.

###  Des classes
| Classe| Description|
| :- | :- |
| [`Config`](/cells/python-net/fr/aspose.cellsgridjs/config) | Représente tous les paramètres de GridJs|
| [`GridAbstractCalculationEngine`](/cells/python-net/fr/aspose.cellsgridjs/gridabstractcalculationengine) | Représente le moteur de calcul personnalisé de l'utilisateur pour étendre le moteur de calcul par défaut de Aspose.Cells.|
| [`GridCacheForStream`](/cells/python-net/fr/aspose.cellsgridjs/gridcacheforstream) | Cette classe contient les opérations de cache pour GridJs. L'utilisateur doit implémenter sa propre logique métier pour le stockage basé sur celle-ci.|
| [`GridCalculationData`](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata) | Représente les données requises lors du calcul d'une fonction, telles que le nom de la fonction, les paramètres, ... etc.|
| [`GridCellException`](/cells/python-net/fr/aspose.cellsgridjs/gridcellexception) | L'exception levée lorsque l'erreur spécifiée par GridJs se produit.|
| [`GridJsWorkbook`](/cells/python-net/fr/aspose.cellsgridjs/gridjsworkbook) |Représente la classe d'entrée principale pour GridJs|
