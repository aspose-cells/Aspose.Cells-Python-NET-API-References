##  get_values {#}

Si cette zone n'est pas valide, "#REF!" sera restitué ;
Si cette zone est une seule cellule, renvoyez l'objet valeur de cellule ;
Sinon, renvoie un tableau pour toutes les valeurs de cette zone.
