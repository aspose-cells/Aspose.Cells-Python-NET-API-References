##  GridCalculationData classe

Représente les données requises lors du calcul d'une fonction, telles que le nom de la fonction, les paramètres, ... etc.



Le type GridCalculationData expose les membres suivants :

###  Propriétés
| Propriété| Description|
| :- | :- |
| [calculated_value](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata/calculated_value) | Obtient/définit la valeur calculée pour cette fonction.|
| [row](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata/row) | Obtient l'index de ligne Cell dans lequel se trouve la fonction.|
| [column](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata/column) | Obtient l’index de colonne Cell dans lequel se trouve la fonction.|
| [string_value](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata/string_value) | Obtient le Cell DisplayStringValue dans lequel se trouve la fonction.|
| [value](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata/value) | Obtient la valeur Cell dans laquelle se trouve la fonction.|
| [formula](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata/formula) | Obtient la formule Cell dans laquelle se trouve la fonction.|
| [sheet_name](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata/sheet_name) | Obtient le nom de la feuille de calcul dans laquelle se trouve la fonction.|
| [function_name](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata/function_name) | Obtient le nom de la fonction à calculer.|
| [param_count](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata/param_count) | Obtient le nombre de paramètres.|


###  Méthodes
| Méthode| Description|
| :- | :- |
| [get_param_value](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata/get_param_value/#int) |Obtient l'objet valeur représenté du paramètre à un index donné.|
| [get_param_text](/cells/python-net/fr/aspose.cellsgridjs/gridcalculationdata/get_param_text/#int) | Obtient le texte littéral du paramètre à l'index donné.|



###  Remarques


Tous les objets fournis par cette classe sont uniquement destinés à la « lecture ».
L'utilisateur ne doit modifier aucune donnée dans le classeur pendant le processus de calcul de la formule.
Sinon, un résultat inattendu ou une exception peut être provoqué.
