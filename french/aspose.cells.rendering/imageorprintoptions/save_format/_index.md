##  save_format propriété

Obtient ou définit le type de format du fichier de sortie
Assistance Tiff/XPS

###  Remarques

 REMARQUE : ce membre est désormais obsolète. Plutôt,
Pour Tiff/Svg, utilisez [ImageOrPrintOptions.image_type](/cells/python-net/fr/aspose.cells.rendering/imageorprintoptions#image_type) ; Pour Xps, utilisez [Workbook.save(file_name, save_format)](/cells/python-net/fr/aspose.cells/workbook/save) avec [XpsSaveOptions](/cells/python-net/fr/aspose.cells/xpssaveoptions).
 Cette propriété sera supprimée 12 mois plus tard depuis août 2022.
Aspose s'excuse pour tout inconvénient que vous pourriez avoir rencontré.
