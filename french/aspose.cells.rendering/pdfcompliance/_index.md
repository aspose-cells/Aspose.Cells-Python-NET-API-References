##  PdfCompliance énumération
Permettre à l'utilisateur de définir la compatibilité de la conversion PDF



Le type PdfCompliance expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| NONE | Format PDF compatible avec PDF 1.4|
| PDF14 | Format PDF compatible avec PDF 1.4|
| PDF15 | Format PDF compatible avec PDF 1.5|
| PDF16 | Format PDF compatible avec PDF 1.6|
| PDF17 | Format PDF compatible avec PDF 1.7|
| PDF_A1B | Format PDF compatible avec PDF/A-1b (ISO 19005-1)|
| PDF_A1A | Format PDF compatible avec PDF/A-1a(ISO 19005-1)|
| PDF_A2B | Format PDF compatible avec PDF/A-2b (ISO 19005-2)|
| PDF_A2U | Format PDF compatible avec PDF/A-2u(ISO 19005-2)|
| PDF_A2A | Format PDF compatible avec PDF/A-2a(ISO 19005-2)|
| PDF_A3B | Format PDF compatible avec PDF/A-3b (ISO 19005-3)|
| PDF_A3U | Format PDF compatible avec PDF/A-3u(ISO 19005-3)|
| PDF_A3A |Format PDF compatible avec PDF/A-3a(ISO 19005-3)|
