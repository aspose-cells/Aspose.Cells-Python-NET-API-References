##  PdfCompliance dénombrement
Permettre à l'utilisateur de définir la compatibilité de la conversion PDF



Le type PdfCompliance expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| NONE | Format pdf normal|
| PDF_A1B | Format PDF compatible avec PDFA-1b|
| PDF_A1A | Format PDF compatible avec PDFA-1a|
