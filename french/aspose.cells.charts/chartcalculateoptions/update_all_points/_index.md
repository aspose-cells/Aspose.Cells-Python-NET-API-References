##  update_all_points propriété

 Si mettre à jour tous les points de données lors de l'exécution du calcul du graphique. Par défaut : Faux.
Lorsque vous souhaitez obtenir spécifiquement la valeur de chaque point de données du graphique, définissez-la sur true.
Si ce paramètre est défini sur True, les nouveaux points de données peuvent être générés lors du calcul du graphique. Cela pourrait rendre le fichier Excel plus volumineux.
