##  update_all_points propriété

 Mettre à jour tous les points de données lors du calcul du graphique. Valeur par défaut : Faux.
Lorsque vous souhaitez obtenir la valeur de chaque point de données du graphique spécifiquement, définissez-la sur vrai.
Si ce paramètre est défini sur « Vrai », de nouveaux points de données peuvent être générés lors du calcul du graphique. Cela peut accroître la taille du fichier Excel.
