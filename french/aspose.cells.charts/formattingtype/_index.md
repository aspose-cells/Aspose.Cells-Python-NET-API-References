##  FormattingType énumération
Représente le type de formatage appliqué à un objet [`Area`](/cells/python-net/fr/aspose.cells.drawing/area) ou à un objet [`Line`](/cells/python-net/fr/aspose.cells.drawing/line).



Le type FormattingType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| AUTOMATIC | Représente le type de formatage automatique.|
| CUSTOM | Représente le type de formatage personnalisé.|
| NONE | Représente un type sans formatage.|
