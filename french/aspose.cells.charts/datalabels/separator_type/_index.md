##  separator_type propriété

Obtient ou définit le type de séparateur utilisé pour les étiquettes de données sur un graphique.

###  Remarques

Pour définir un séparateur personnalisé, définissez la propriété [DataLabels.separator_type](/cells/python-net/fr/aspose.cells.charts/datalabels#separator_type) sur [DataLabelsSeparatorType.CUSTOM](/cells/python-net/fr/aspose.cells.charts/datalabelsseparatortype#CUSTOM), puis spécifiez la valeur attendue pour [DataLabels.separator_value](/cells/python-net/fr/aspose.cells.charts/datalabels#separator_value).
