##  TickLabelAlignmentType énumération
Représente le type d'alignement du texte pour les étiquettes de graduation sur l'axe



Le type TickLabelAlignmentType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| CENTER | Représente que le texte doit être centré.|
| LEFT | Représente que le texte doit être justifié à gauche.|
| RIGHT | Représente que le texte doit être justifié à droite.|
