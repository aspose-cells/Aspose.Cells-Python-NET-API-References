##  ChartSplitType énumération
Représente la manière dont les deux sections d'un graphique à secteurs ou d'un graphique à barres sont divisées.



Le type ChartSplitType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| POSITION | Représente les points de données qui doivent être répartis entre le graphique à secteurs<br/>et le deuxième graphique en mettant la dernière position fractionnée<br/> des points de données dans le deuxième graphique|
| VALUE | Représente les points de données qui doivent être répartis entre le graphique à secteurs<br/>et le deuxième graphique en mettant les points de données avec<br/> valeur inférieure à la position fractionnée dans le deuxième graphique.|
| PERCENT_VALUE | Représente les points de données qui doivent être répartis entre le graphique à secteurs<br/>et le deuxième graphique en mettant les points avec<br/>pourcentage inférieur au pourcentage de position fractionnée dans le<br/> deuxième graphique.|
| CUSTOM | Représente les points de données qui doivent être répartis entre le graphique à secteurs<br/>et le deuxième graphique selon le Custom Split<br/> valeurs.|
| AUTO |Représente les points de données qui doivent être divisés en utilisant la valeur par défaut<br/> mécanisme pour ce type de graphique.|
