##  ChartSplitType dénombrement
Représente la façon dont les deux sections d'un camembert ou d'une barre de camembert sont divisées.



Le type ChartSplitType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| POSITION | Représente les points de données doivent être répartis entre la tarte<br/>et le deuxième graphique en mettant la dernière position de fractionnement<br/> des points de données dans le deuxième graphique|
| VALUE | Représente les points de données doivent être répartis entre la tarte<br/>et le deuxième graphique en mettant les points de données avec<br/> valeur inférieure à Split Position dans le deuxième graphique.|
| PERCENT_VALUE | Représente les points de données doivent être répartis entre la tarte<br/>et le deuxième graphique en mettant les points avec<br/>pourcentage inférieur au pourcentage de position partagée dans le<br/> deuxième tableau.|
| CUSTOM | Représente les points de données doivent être répartis entre la tarte<br/>et le deuxième graphique selon le partage personnalisé<br/> valeurs.|
| AUTO | Représente les points de données qui doivent être divisés à l'aide de la valeur par défaut<br/> mécanisme pour ce type de graphique.|
