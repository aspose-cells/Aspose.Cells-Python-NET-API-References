##  SparklineAxisMinMaxType énumération
Représente les types de valeurs minimales et maximales pour l'axe vertical du graphique sparkline.



Le type SparklineAxisMinMaxType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| AUTO_INDIVIDUAL | Automatique pour chaque sparkline.|
| GROUP | Idem pour tous les sparklines du groupe.|
| CUSTOM | Valeur personnalisée pour le sparkline.|
