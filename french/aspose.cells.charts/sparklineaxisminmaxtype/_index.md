##  SparklineAxisMinMaxType dénombrement
Représente les types de valeurs minimum et maximum pour l'axe vertical du graphique sparkline.



Le type SparklineAxisMinMaxType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| AUTO_INDIVIDUAL | Automatique pour chaque sparkline.|
| GROUP | Idem pour toutes les sparklines du groupe.|
| CUSTOM | Valeur personnalisée pour le graphique sparkline.|
