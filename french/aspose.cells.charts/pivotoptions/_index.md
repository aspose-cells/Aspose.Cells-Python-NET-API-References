##  PivotOptions classe
Représente un type complexe qui spécifie les contrôles pivot qui apparaissent sur le graphique



Le type PivotOptions expose les membres suivants :

###  Propriétés
| Propriété| Description|
| :- | :- |
| [drop_zone_filter](/cells/python-net/fr/aspose.cells.charts/pivotoptions/drop_zone_filter) | Spécifie si un contrôle pour chaque champ de tableau croisé dynamique sur l'axe de la page du tableau croisé dynamique<br/> du tableau croisé dynamique source apparaît sur le graphique lorsque dropZonesVisible est défini sur true.|
| [drop_zone_categories](/cells/python-net/fr/aspose.cells.charts/pivotoptions/drop_zone_categories) | Spécifie si un contrôle pour chaque champ de tableau croisé dynamique sur l'axe des lignes du tableau croisé dynamique<br/> du tableau croisé dynamique source apparaît sur le graphique lorsque dropZonesVisible est défini sur true.|
| [drop_zone_data](/cells/python-net/fr/aspose.cells.charts/pivotoptions/drop_zone_data) |Spécifie si un contrôle pour chaque champ de tableau croisé dynamique sur l'axe de données du tableau croisé dynamique<br/> du tableau croisé dynamique source apparaît sur le graphique lorsque dropZonesVisible est défini sur true.|
| [drop_zone_series](/cells/python-net/fr/aspose.cells.charts/pivotoptions/drop_zone_series) | Spécifie si un contrôle pour chaque champ de tableau croisé dynamique sur l'axe des colonnes du tableau croisé dynamique<br/> du tableau croisé dynamique source apparaît sur le graphique lorsque dropZonesVisible est défini sur true.|
| [drop_zones_visible](/cells/python-net/fr/aspose.cells.charts/pivotoptions/drop_zones_visible) | Spécifie si des contrôles de pivot peuvent apparaître sur le graphique croisé dynamique.|
