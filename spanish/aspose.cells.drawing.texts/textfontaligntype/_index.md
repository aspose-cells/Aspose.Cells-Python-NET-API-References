##  TextFontAlignType enumeración
Representa los diferentes tipos de alineación de fuentes.



El tipo TextFontAlignType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| AUTOMATIC | Cuando el flujo de texto es horizontal o vertical simple igual que fontBaseline<br/> pero para otros modos verticales igual que fontCenter.|
| BOTTOM | Las letras están ancladas en la parte inferior de una sola línea.|
| BASELINE | Las letras están ancladas a la línea de base inferior de una sola línea.|
| CENTER | Las letras están ancladas entre las dos líneas base de una sola línea.|
| TOP | Las letras están ancladas a la línea de base superior de una sola línea.|
