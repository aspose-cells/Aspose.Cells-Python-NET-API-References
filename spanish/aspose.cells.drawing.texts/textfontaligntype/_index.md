##  TextFontAlignType enumeración
Representa los diferentes tipos de alineación de fuentes.



El tipo TextFontAlignType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| AUTOMATIC | Cuando el flujo del texto es horizontal o vertical simple igual que fontBaseline<br/> pero para otros modos verticales igual que fontCenter.|
| BOTTOM | Las letras están ancladas al final de una sola línea.|
| BASELINE | Las letras están ancladas a la línea base inferior de una sola línea.|
| CENTER | Las letras están ancladas entre las dos líneas base de una misma línea.|
| TOP | Las letras están ancladas a la línea base superior de una sola línea.|
