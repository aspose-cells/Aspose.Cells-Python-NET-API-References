##  id propiedad

Especifica un GUID que puede compararse con el GUID de la línea de firma almacenada en el contenido del documento.
El valor predeterminado es Vacío (todos ceros) Guid.

###  Observaciones

Cuando se configura, asocia [`SignatureLine`](/cells/python-net/es/aspose.cells.drawing/signatureline) con el [`DigitalSignature`](/cells/python-net/es/aspose.cells.digitalsignatures/digitalsignature) correspondiente.
