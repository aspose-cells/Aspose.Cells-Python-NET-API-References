##  sheet_set propiedad

Obtiene o establece las hojas para representar. El valor predeterminado es todas las hojas visibles en el libro de trabajo: [SheetSet.visible](/cells/python-net/es/aspose.cells.rendering/sheetset#visible).

###  Observaciones
