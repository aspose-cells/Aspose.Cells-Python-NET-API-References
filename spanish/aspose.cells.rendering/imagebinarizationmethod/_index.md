##  ImageBinarizationMethod enumeración
Especifica el método utilizado para binarizar la imagen.



El tipo ImageBinarizationMethod expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| THRESHOLD | Especifica el método de umbral.|
| FLOYD_STEINBERG_DITHERING | Especifica el tramado mediante el método de difusión de errores Floyd-Steinberg.|
