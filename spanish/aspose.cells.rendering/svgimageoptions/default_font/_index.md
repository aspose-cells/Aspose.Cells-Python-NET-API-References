##  default_font propiedad

Cuando los caracteres en Excel son Unicode y no están configurados con la fuente correcta en el estilo de celda,
Pueden aparecer como bloques en formato PDF o imagen.
Establezca la fuente predeterminada, como MingLiu o MS Gothic, para mostrar estos caracteres.
Si esta propiedad no está configurada, Aspose.Cells utilizará la fuente predeterminada del sistema para mostrar estos caracteres Unicode.
