##  ConnectionDataSourceType enumeración
Especifica el tipo de fuente de base de datos externa



El tipo ConnectionDataSourceType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| ODBC_BASED_SOURCE | Fuente basada en ODBC|
| DAO_BASED_SOURCE | Fuente basada en DAO|
| FILE_BASED_DATA_BASE_SOURCE | Fuente de base de datos basada en archivos|
| WEB_QUERY | Consulta web|
| OLEDB_BASED_SOURCE | Fuente basada en OLE DB|
| TEXT_BASED_SOURCE | Fuente basada en texto|
| ADO_RECORD_SET | Conjunto de registros ADO|
| DSP | DSP |
| OLEDB_DATA_MODEL | Fuente de datos OLE DB creada por el modelo de datos de hoja de cálculo.|
| DATA_FEED_DATA_MODEL |Fuente de datos de alimentación de datos creada por el modelo de datos de la hoja de cálculo.|
| WORKSHEET_DATA_MODEL | Fuente de datos de la hoja de trabajo creada por el modelo de datos de la hoja de cálculo.|
| TABLE | Fuente de datos de la hoja de trabajo creada por el modelo de datos de la hoja de cálculo.|
| TEXT_DATA_MODEL | Fuente de datos de texto creada por el modelo de datos de hoja de cálculo.|
| UNKNOWN | Fuente de datos de texto creada por el modelo de datos de hoja de cálculo.|
