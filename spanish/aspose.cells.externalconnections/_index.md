 El**Aspose.Cells.ExternalConnections**El espacio de nombres proporciona clases para conexiones externas para recuperar datos de fuentes externas, como bases de datos y la Web.

###  Clases
| Clase| Descripción|
| :- | :- |
| [`ConnectionParameter`](/cells/python-net/es/aspose.cells.externalconnections/connectionparameter) | Especifica propiedades sobre cualquier parámetro utilizado con conexiones de datos externas<br/> Los parámetros son válidos para consultas ODBC y web.|
| [`ConnectionParameterCollection`](/cells/python-net/es/aspose.cells.externalconnections/connectionparametercollection) | Especifica la colección [`ConnectionParameter`](/cells/python-net/es/aspose.cells.externalconnections/connectionparameter)|
| [`DBConnection`](/cells/python-net/es/aspose.cells.externalconnections/dbconnection) | Especifica todas las propiedades asociadas con una conexión de datos externos ODBC u OLE DB.|
| [`DataModelConnection`](/cells/python-net/es/aspose.cells.externalconnections/datamodelconnection) | Especifica una conexión de modelo de datos|
| [`ExternalConnectionCollection`](/cells/python-net/es/aspose.cells.externalconnections/externalconnectioncollection) | Especifica la colección ExternalConnection|
| [`WebQueryConnection`](/cells/python-net/es/aspose.cells.externalconnections/webqueryconnection) | Especifica las propiedades de una fuente de consulta web. Una consulta web recuperará datos de HTML tablas.<br/> y también puede proporcionar parámetros HTTP "Get" para que el servidor web los procese al generar el HTML<br/> Incluyendo los parámetros y elementos de parámetros.|
