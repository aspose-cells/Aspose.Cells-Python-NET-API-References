##  WebQueryConnection clase
 Especifica las propiedades de una fuente de consulta web. Una consulta web recuperará datos de HTML tablas.
 y también puede proporcionar parámetros HTTP "Get" para que el servidor web los procese al generar el HTML
Incluyendo los parámetros y elementos de parámetros.
