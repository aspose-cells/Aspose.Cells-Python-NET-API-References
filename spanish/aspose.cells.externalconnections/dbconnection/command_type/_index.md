##  command_type propiedad

 Especifica el tipo de comando OLE DB.
1. La consulta especifica un nombre de cubo
2. La consulta especifica una declaración SQL
3. La consulta especifica un nombre de tabla
4. La consulta especifica que se ha proporcionado información predeterminada y depende del proveedor cómo interpretarla.
5. La consulta se realiza contra un proveedor de datos de listas basado en web.
