##  WebExtensionStoreType enumeración
Representa el tipo de tienda de la extensión web.



El tipo WebExtensionStoreType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| OMEX | Especifica que el tipo de tienda es Office.com.|
| SP_CATALOG | Especifica que el tipo de tienda es el catálogo corporativo de SharePoint.|
| SP_APP | Especifica que el tipo de tienda es una aplicación web de SharePoint.|
| EXCHANGE | Especifica que el tipo de tienda es un servidor de Exchange.|
| FILE_SYSTEM | Especifica que el tipo de almacén es un recurso compartido de sistema de archivos.|
| REGISTRY | Especifica que el tipo de almacén es el registro del sistema.|
| EX_CATALOG | Especifica que el tipo de tienda es Implementación centralizada a través de Exchange.|
