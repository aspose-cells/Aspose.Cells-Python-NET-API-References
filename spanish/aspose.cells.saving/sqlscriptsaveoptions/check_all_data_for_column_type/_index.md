##  check_all_data_for_column_type propiedad

Verifique todos los datos para encontrar el tipo de datos de las columnas.

###  Observaciones

El valor predeterminado es falso, solo verificamos el rendimiento de la primera fila.
Si esta propiedad es verdadera y las columnas contienen un tipo de valor mixto, el tipo de las columnas será texto.
