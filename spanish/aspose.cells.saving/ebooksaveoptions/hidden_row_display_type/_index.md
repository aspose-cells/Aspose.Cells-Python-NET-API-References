##  hidden_row_display_type propiedad

Fila oculta (la altura de esta fila es 0) en Excel, antes de guardarla en formato html,
Si HtmlHiddenRowDisplayType es "Eliminar", la fila oculta no se mostrará.
Si el valor es "Oculto", la fila se habría mostrado, pero estaba oculta; el valor predeterminado es "Oculto".
