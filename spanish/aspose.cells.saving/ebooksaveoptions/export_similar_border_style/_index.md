##  export_similar_border_style propiedad

Indica si se exporta el estilo de borde similar cuando el estilo de borde no es compatible con los navegadores.
Si desea importar el archivo html o mht para Excel, mantenga el valor predeterminado.
El valor predeterminado es falso.
