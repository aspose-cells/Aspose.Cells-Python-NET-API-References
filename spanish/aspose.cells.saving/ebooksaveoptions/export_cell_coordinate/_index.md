##  export_cell_coordinate propiedad

Indica si se exportan las coordenadas de Excel de celdas no vacías al guardar el archivo en HTML. El valor predeterminado es "false".
Si desea importar el HTML de salida a Excel, mantenga el valor predeterminado.
