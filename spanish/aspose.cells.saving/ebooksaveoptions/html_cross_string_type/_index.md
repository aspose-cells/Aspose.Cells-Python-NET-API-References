##  html_cross_string_type propiedad

Indica si una cadena entre celdas se mostrará de la misma manera que MS Excel al guardar un archivo Excel en formato html.
De forma predeterminada, el valor es Predeterminado, por lo que, para cadenas entre celdas, hay poca diferencia entre los archivos html creados por Aspose.Cells y MS Excel.
Pero el rendimiento para crear archivos html grandes, al configurar el valor en Cross, sería varias veces más rápido que al configurarlo en Default o Fit2Cell.
