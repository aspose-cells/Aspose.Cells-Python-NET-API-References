##  width propiedad

Obtiene o establece el width del marco en unidades de 1/4000 del área del gráfico.

###  Observaciones

 ¿Cómo convertir unidades de 1/4000 a píxeles?
Ancho en píxeles = Ancho * Chart.ChartObject.Height / 4000d;
