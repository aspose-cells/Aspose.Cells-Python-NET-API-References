##  object_source_full_name propiedad

Devuelve el nombre completo del archivo de origen del objeto OLE vinculado.

###  Observaciones

Solo se admite la configuración del nombre completo de la fuente cuando el tipo de archivo es OleFileType.Unknown.
Como archivos wav, archivos avi, etc.
