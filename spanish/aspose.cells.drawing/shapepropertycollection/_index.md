##  ShapePropertyCollection clase
Esta clase especifica las propiedades de forma visual para un elemento o forma de gráfico.



El tipo ShapePropertyCollection expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [glow_effect](/cells/python-net/es/aspose.cells.drawing/shapepropertycollection/glow_effect) | Representa un objeto [ShapePropertyCollection.glow_effect](/cells/python-net/es/aspose.cells.drawing/shapepropertycollection#glow_effect) que especifica el efecto de brillo para el elemento o la forma del gráfico.|
| [format_3d](/cells/python-net/es/aspose.cells.drawing/shapepropertycollection/format_3d) | Representa un objeto [ShapePropertyCollection.format_3d](/cells/python-net/es/aspose.cells.drawing/shapepropertycollection#format_3d) que especifica propiedades de forma 3D para la forma o el elemento del gráfico.|
| [soft_edge_radius](/cells/python-net/es/aspose.cells.drawing/shapepropertycollection/soft_edge_radius) | Obtiene y establece el radio de desenfoque que se aplica a los bordes, en unidades de puntos.|
| [shadow_effect](/cells/python-net/es/aspose.cells.drawing/shapepropertycollection/shadow_effect) | Representa un objeto [ShapePropertyCollection.shadow_effect](/cells/python-net/es/aspose.cells.drawing/shapepropertycollection#shadow_effect) que especifica el efecto de sombra para la forma o el elemento del gráfico.|


###  Métodos
| Método| Descripción|
| :- | :- |
