Obtiene el primer párrafo matemático de la propiedad TextBody del objeto TextBox.


###  Devoluciones

Si hay un párrafo matemático, devuelve el primero, de lo contrario, devuelve nulo.

##  get_equation_paragraph(self, index) {#int}
Obtenga el párrafo matemático especificado de la propiedad TextBody del objeto TextBox.
Aviso:
(1) Devuelve NULL cuando el índice está fuera de los límites o no se encuentra.
(2) También devuelve NULL si la posición de índice especificada no es un párrafo matemático.


###  Devoluciones

Devuelve el párrafo matemático especificado por el índice.


```python

def get_equation_paragraph(self, index):
    ...
```


| Parámetro| Tipo| Descripción|
| :- | :- | :- |
| index | int | El índice de posición del párrafo de matemáticas, comenzando desde 0.|
