##  input_range propiedad

Obtiene o establece el rango de la hoja de cálculo utilizado para llenar el cuadro combinado especificado.

###  Ejemplo

```python

if shape.input_range == "$B$6:$B10":
    shape.input_range = "$A$1:$A$5"
```
