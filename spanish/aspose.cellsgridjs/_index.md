 El**Aspose.Cells.GridJs** El espacio de nombres contiene todas las clases de GridJ que proporcionan operaciones básicas para administrar hojas de cálculo sin utilizar Microsoft Excel.

| Clase| Descripción|

| [`GridCacheForStream`](/cells/python-net/es/aspose.cellsgridjs/gridcacheforstream) | Esta clase contiene las operaciones de caché para GridJs. El usuario deberá implementar su propia lógica de negocios para el almacenamiento basada en ella.|
