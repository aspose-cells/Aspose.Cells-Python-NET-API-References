##  calculated_value propiedad


Obtiene/establece el valor calculado para esta función.

###  Observaciones


El usuario debe establecer esta propiedad en su motor de cálculo personalizado para aquellas funciones que admite el motor.
y el valor establecido se devolverá al obtener esta propiedad.
Obtener esta propiedad antes de configurarla hará que la función sea calculada por el motor de cálculo predeterminado de Aspose.Cells y se devolverá el valor calculado.
