##  max_shape_or_image_width_or_height propiedad


Establece/obtiene el ancho o alto máximo para una forma/imagen. GridJs ignorará la forma/imagen con un ancho o alto mayor que esto, afectará cuando IslimitShapes=true.
el valor predeterminado es 10000.
