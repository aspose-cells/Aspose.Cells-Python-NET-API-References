##  Config clase

Representa todas las configuraciones para GridJs.



El tipo Config expone los siguientes miembros:

###  Constructores
| Constructor| Descripción|
| :- | :- |
| [__init__](/cells/python-net/es/aspose.cellsgridjs/config/__init__/#) | Construye una nueva instancia de Config.|


###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [save_html_as_zip](/cells/python-net/es/aspose.cellsgridjs/config/save_html_as_zip) | Establece/obtiene si se debe guardar el archivo html como archivo zip, el valor predeterminado es falso.|
| [same_image_detecting](/cells/python-net/es/aspose.cellsgridjs/config/same_image_detecting) |
<br/>El valor por defecto es verdadero. |
| [auto_optimize_for_large_cells](/cells/python-net/es/aspose.cellsgridjs/config/auto_optimize_for_large_cells) |
<br/>ignorar algunos estilos/bordes para reducir el tiempo de carga
<br/>El valor por defecto es verdadero. |
| [islimit_shape_or_image](/cells/python-net/es/aspose.cellsgridjs/config/islimit_shape_or_image) |
<br/>GridJs limitará el tamaño total de la forma/imagen de visualización dentro de una hoja de trabajo a MaxShapeOrImageCount
<br/>El valor por defecto es verdadero. |
| [max_shape_or_image_count](/cells/python-net/es/aspose.cellsgridjs/config/max_shape_or_image_count) |
<br/>el valor predeterminado es 100. |
| [max_total_shape_or_image_count](/cells/python-net/es/aspose.cellsgridjs/config/max_total_shape_or_image_count) |
<br/>el valor predeterminado es 300. |
| [max_shape_or_image_width_or_height](/cells/python-net/es/aspose.cellsgridjs/config/max_shape_or_image_width_or_height) |
<br/>el valor predeterminado es 10000. |
| [max_pdf_save_seconds](/cells/python-net/es/aspose.cellsgridjs/config/max_pdf_save_seconds) |
<br/>el valor predeterminado es 10. |
| [ignore_empty_content](/cells/python-net/es/aspose.cellsgridjs/config/ignore_empty_content) |
<br/> si la última fila o columna contiene celdas sin valor ni fórmula pero tiene un estilo personalizado
<br/>entonces no mostraremos esta fila/columna cuando este valor sea verdadero.
<br/>El valor por defecto es verdadero . |
| [use_print_area](/cells/python-net/es/aspose.cellsgridjs/config/use_print_area) |
<br/>el valor predeterminado es falso. |
| [load_time_out](/cells/python-net/es/aspose.cellsgridjs/config/load_time_out) |
<br/>el valor predeterminado es -1, lo que significa que no se establece ninguna interrupción de tiempo de espera. |
| [show_chart_sheet](/cells/python-net/es/aspose.cellsgridjs/config/show_chart_sheet) |
<br/>el valor predeterminado es falso. |
| [page_size](/cells/python-net/es/aspose.cellsgridjs/config/page_size) |
<br/>GridJs limitará el tamaño de la fila según el tamaño de la página, si el tamaño de la página es -1, no realizará la paginación
<br/>el valor predeterminado es -1 |
| [empty_sheet_max_row](/cells/python-net/es/aspose.cellsgridjs/config/empty_sheet_max_row) |
<br/>el valor predeterminado es 12. |
| [empty_sheet_max_col](/cells/python-net/es/aspose.cellsgridjs/config/empty_sheet_max_col) |
<br/>el valor predeterminado es 15. |
| [picture_cache_directory](/cells/python-net/es/aspose.cellsgridjs/config/picture_cache_directory) |
<br/>la ruta predeterminada será "_piccache" dentro de FileCacheDirectory. |
| [file_cache_directory](/cells/python-net/es/aspose.cellsgridjs/config/file_cache_directory) |
<br/>Necesitamos configurarlo en una ruta específica antes de usar GridJs. |
