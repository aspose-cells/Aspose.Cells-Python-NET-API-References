##  use_print_area propiedad


Establece/obtiene si se debe usar PageSetup.PrintArea para el rango de visualización de la interfaz de usuario cuando la hoja de trabajo tiene la configuración PageSetup para PrintArea.
el valor predeterminado es falso.
