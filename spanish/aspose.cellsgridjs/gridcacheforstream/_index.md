##  GridCacheForStream clase

Esta clase contiene las operaciones de caché para GridJs. El usuario deberá implementar su propia lógica de negocios para el almacenamiento basada en ella.



El tipo GridCacheForStream expone los siguientes miembros:

| [save_stream](/cells/python-net/es/aspose.cellsgridjs/gridcacheforstream/save_stream/#io.RawIOBase-str) | Implementa este método para guardar el caché y guardar la transmisión en el caché con la clave uid.|
