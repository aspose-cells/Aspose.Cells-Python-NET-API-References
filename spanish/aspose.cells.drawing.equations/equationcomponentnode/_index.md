##  EquationComponentNode clase
 Esta clase especifica los componentes de una ecuación o expresión matemática.
Diferentes tipos de componentes combinados en diferentes ecuaciones.
Por ejemplo, una fracción consta de dos partes, un componente numerador y un componente denominador.
Para conocer más tipos de componentes, consulte 'EquationNodeType'.
