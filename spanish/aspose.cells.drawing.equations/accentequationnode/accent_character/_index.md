##  accent_character propiedad

Este atributo especifica el tipo de marca diacrítica de combinación que se añade a la base de la función de acento. El carácter de acento predeterminado es U+0302.
Se recomienda encarecidamente utilizar el atributo AccentType para establecer el carácter de acento.
Utilice esta configuración de propiedad si no puede encontrar el carácter que necesita en un tipo conocido.

###  Observaciones

Se debe tener en cuenta que esta propiedad solo acepta un carácter y, si se pasan varios caracteres, solo se acepta el primero.
