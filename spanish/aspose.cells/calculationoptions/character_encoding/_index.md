##  character_encoding propiedad

Especifica la codificación utilizada para codificar/decodificar caracteres al calcular fórmulas.
Para funciones como CHAR, CODE, el resultado calculado depende de la configuración de la región y del conjunto de caracteres predeterminado del entorno.
Con esta propiedad, el usuario puede especificar la codificación adecuada utilizada para esas funciones para obtener el resultado esperado.
