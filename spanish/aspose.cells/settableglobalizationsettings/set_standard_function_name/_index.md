##  set_standard_function_name(self, local_name, standard_name, bidirectional) {#str-str-bool}
Establece el nombre de la función dependiente de la configuración regional de acuerdo con el nombre de función estándar dado.



```python

def set_standard_function_name(self, local_name, standard_name, bidirectional):
    ...
```


| Parámetro| Tipo| Descripción|
| :- | :- | :- |
| local_name | str | Nombre de función dependiente de la configuración regional|
| standard_name | str | Nombre de función estándar (configuración regional en-US).|
| bidirectional | bool | Si asigna automáticamente el nombre de la función estándar al nombre de la función local.<br/>Si es verdadero, el nombre estándar se asignará automáticamente al nombre local.<br/>Para que el usuario no tenga que volver a llamar al [`SettableGlobalizationSettings.set_local_function_name`](/cells/python-net/es/aspose.cells/settableglobalizationsettings/set_local_function_name)<br/> para el mismo par de nombres estándar y locales|
