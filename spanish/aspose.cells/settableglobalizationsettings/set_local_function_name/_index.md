##  set_local_function_name(self, standard_name, local_name, bidirectional) {#str-str-bool}
Establece el nombre de la función dependiente de la configuración regional correspondiente al nombre de función estándar dado.



```python

def set_local_function_name(self, standard_name, local_name, bidirectional):
    ...
```


| Parámetro| Tipo| Descripción|
| :- | :- | :- |
| standard_name | str | Nombre de función estándar (configuración regional en-US).|
| local_name | str | Nombre de función dependiente de la configuración regional|
| bidirectional | bool | Si asigna automáticamente el nombre de la función local al nombre de la función estándar.<br/>Si es verdadero, el nombre local se asignará automáticamente al nombre estándar.<br/>Para que el usuario no tenga que volver a llamar al [`SettableGlobalizationSettings.set_standard_function_name`](/cells/python-net/es/aspose.cells/settableglobalizationsettings/set_standard_function_name)<br/> para el mismo par de nombres estándar y locales|
