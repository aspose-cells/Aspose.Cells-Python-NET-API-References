##  set_standard_built_in_name {#str-str-bool}
Establece el nombre de la función dependiente de la configuración regional de acuerdo con el nombre de función estándar dado.



```python
def set_standard_built_in_name(self, local_name, standard_name, bidirectional):
    ...
```


| Parámetro| Tipo| Descripción|
| :- | :- | :- |
| local_name | str | Nombre de la función dependiente de la configuración regional|
| standard_name | str | Nombre de la función estándar (localización en EE. UU.).|
| bidirectional | bool | Si asigna automáticamente el texto del nombre estándar al texto del nombre local.<br/>Si es verdadero, el texto del nombre estándar se asignará automáticamente al texto del nombre local.<br/>para que el usuario no necesite volver a llamar al [`SettableGlobalizationSettings.set_local_built_in_name`](/cells/python-net/es/aspose.cells/settableglobalizationsettings/set_local_built_in_name)<br/> para el mismo par de nombres estándar y local|
