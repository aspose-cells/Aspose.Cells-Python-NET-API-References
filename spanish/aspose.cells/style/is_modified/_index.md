##  is_modified(modify_flag) {#StyleModifyFlag}
Comprueba si se han modificado las propiedades especificadas del estilo.
Se usa para el estilo de ConditionalFormattings para verificar si las propiedades especificadas de este estilo deben usarse al aplicar ConditionalFormattings en una celda.
