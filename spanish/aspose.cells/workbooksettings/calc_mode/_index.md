##  calc_mode propiedad

Especifica si calcular las fórmulas manualmente,
de forma automática o automática, excepto para operaciones con varias tablas.

###  Observaciones

Solo conjuntos para MS Excel.
Llame manualmente al método Workbook.CalculateFormula() para calcular fórmulas con Aspose.Cells.
NOTA: Este miembro ahora está obsoleto. En su lugar, utilice FormulaSettings.CalculationMode.
 Esta propiedad se eliminará 12 meses después desde febrero de 2022.
Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.
