##  is_h_scroll_bar_visible propiedad

Obtiene o establece un valor que indica si la hoja de cálculo generada contendrá una barra de desplazamiento horizontal.

###  Observaciones

El valor por defecto es verdadero.

###  Ejemplo

El siguiente código hace que la barra de desplazamiento horizontal sea invisible para la hoja de cálculo.

```python

#  Hide the horizontal scroll bar of the Excel file.
settings.is_h_scroll_bar_visible = False

```
