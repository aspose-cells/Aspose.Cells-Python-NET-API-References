##  show_tabs propiedad

Obtiene o establece un valor si se muestran las pestañas del libro de trabajo.

###  Observaciones

El valor predeterminado es verdadero.

###  Ejemplo

El siguiente código oculta las pestañas de la hoja y los botones de desplazamiento de pestañas de la hoja de cálculo.

```python

#  Hide the spreadsheet tabs.
workbook.settings.show_tabs = False

```
