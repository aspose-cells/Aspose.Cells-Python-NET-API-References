##  show_tabs propiedad

Obtiene o establece un valor si se muestran las pestañas del Libro de trabajo.

###  Observaciones

El valor por defecto es verdadero.

###  Ejemplo

El siguiente código oculta las pestañas de hoja y los botones de desplazamiento de pestañas para la hoja de cálculo.

```python

#  Hide the spreadsheet tabs.
workbook.settings.show_tabs = False

```
