##  is_v_scroll_bar_visible propiedad

Obtiene o establece un valor que indica si la hoja de cálculo generada contendrá una barra de desplazamiento vertical.

###  Observaciones

El valor predeterminado es verdadero.

###  Ejemplo

El siguiente código hace que la barra de desplazamiento vertical sea invisible para la hoja de cálculo.

```python

#  Hide the vertical scroll bar of the Excel file.
settings.is_v_scroll_bar_visible = False

```
