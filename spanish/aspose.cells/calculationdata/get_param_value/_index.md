##  get_param_value(index) {#int}
Obtiene el objeto de valor representado del parámetro en el índice dado.


###  Devoluciones

Si el parámetro es un valor simple, devuelve el valor simple.
Si el parámetro es una referencia, devuelve el objeto ReferedArea.
Si el parámetro hace referencia a varios conjuntos de datos, devuelve una matriz de objetos.
