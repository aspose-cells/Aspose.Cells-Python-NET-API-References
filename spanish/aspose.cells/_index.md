 El**Aspose.Cells** El espacio de nombres proporciona clases básicas y fundamentales para generar, convertir, modificar, representar e imprimir hojas de cálculo sin utilizar Microsoft Excel.
Por ejemplo, la clase [`Workbook`](/cells/python-net/es/aspose.cells/workbook) es la entrada principal para crear o cargar y luego manipular un archivo de hoja de cálculo.
La clase [`Cells`](/cells/python-net/es/aspose.cells/cells) es la clase principal para manipular todos los datos y propiedades en una hoja, etc.
