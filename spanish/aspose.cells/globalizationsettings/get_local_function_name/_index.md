##  get_local_function_name(standard_name) {#str}
Obtiene el nombre de la función dependiente de la configuración regional de acuerdo con el nombre de la función estándar dado.


###  Devoluciones

Nombre de función dependiente de la configuración regional. La configuración regional fue especificada por el libro de trabajo para el que se utiliza esta configuración.
