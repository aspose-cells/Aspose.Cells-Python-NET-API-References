##  get_named_ranges(self) {#}

###  Devoluciones

Una matriz de objetos Range.
Si la referencia del Nombre definido es externa o tiene múltiples rangos, no se devolverá ningún objeto Rango para este Nombre.



Devuelve nulo si el rango nombrado no existe.
