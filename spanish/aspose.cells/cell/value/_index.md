###  Observaciones

Tipo posible:

booleano,


Fecha y hora,


Doble,


Entero


Cadena.


Para el int value, puede devolverse como un objeto Integer o un objeto Double.
Y no hay garantía de que el value devuelto se mantenga siempre como el mismo tipo de objeto.
