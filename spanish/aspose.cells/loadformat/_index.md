##  LoadFormat enumeración
Representa el formato del archivo de carga.



El tipo LoadFormat expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| AUTO | Representa reconocer el formato automáticamente.|
| CSV | Archivo de texto de valores separados por comas (CSV).<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`LoadFormat.CSV`](/cells/python-net/es/aspose.cells/loadformat#CSV).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| XLSX | Representa una plantilla o un libro de trabajo de hoja de cálculo XML de Office Open XML, con o sin macros.|
| TSV | Archivo de texto de valores separados por tabulaciones (TSV).<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`LoadFormat.TSV`](/cells/python-net/es/aspose.cells/loadformat#TSV).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| TAB_DELIMITED | Representa un archivo de texto delimitado por tabulaciones, lo mismo que [`LoadFormat.TSV`](/cells/python-net/es/aspose.cells/loadformat#TSV).|
| HTML | Representa un archivo html.|
| M_HTML | Representa un archivo mhtml.|
| ODS | Abra el archivo Hoja de documento (ODS).<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`LoadFormat.ODS`](/cells/python-net/es/aspose.cells/loadformat#ODS).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| EXCEL_97_TO_2003 | Representa un archivo xls de Excel97-2003.|
| SPREADSHEET_ML | Representa un archivo xml de Excel 2003.|
| XLSB | Representa un archivo xlsb.|
| OTS | Abra el archivo de hoja de plantilla de documento (OTS).|
| NUMBERS | Representa un archivo de números.|
| FODS | Representa el formato de archivo OpenDocument Flat XML Spreadsheet (.fods).<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`LoadFormat.FODS`](/cells/python-net/es/aspose.cells/loadformat#FODS).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| SXC | Representa el formato de archivo de hoja de cálculo de StarOffice Calc (.sxc).<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`LoadFormat.SXC`](/cells/python-net/es/aspose.cells/loadformat#SXC).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| XML | Representa un archivo xml simple.|
| EPUB | Representa un archivo EPUB.|
| AZW3 |Representa un archivo AZW3.|
| UNKNOWN | Representa un formato no reconocido, no se puede cargar.|
| IMAGE | Imagen|
| JSON | json|
