##  custom_path propiedad

La ruta personalizada (URL) del usuario guardada en el archivo HTML generado para la fuente de referencia. Si el usuario no la define, se utilizará la ruta predeterminada.
Por ejemplo, si el usuario guardará los datos de la hoja en d:/sheet001.htm, la URL utilizada en el archivo html principal debe ser "d:/sheet001.htm" u otra ruta relativa válida a la que se pueda acceder mediante el archivo html principal.
