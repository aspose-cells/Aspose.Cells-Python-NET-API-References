##  ConditionalFormattingValue clase
Describe los valores de los puntos de interpolación en una escala de degradado, dataBar o iconSet.



El tipo ConditionalFormattingValue expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [value](/cells/python-net/es/aspose.cells/conditionalformattingvalue/value) | Obtenga o establezca el valor de este objeto de valor de formato condicional.<br/> Debe utilizarse junto con Tipo.|
| [type](/cells/python-net/es/aspose.cells/conditionalformattingvalue/type) | Obtenga o establezca el tipo de este objeto de valor de formato condicional.<br/> Establecer el tipo en FormatConditionValueType.Min o FormatConditionValueType.Max<br/> Establecerá automáticamente "Valor" en nulo.|
| [is_gte](/cells/python-net/es/aspose.cells/conditionalformattingvalue/is_gte) | Obtenga o establezca la bandera Mayor o igual que.<br/> Usar solo para conjuntos de iconos, determina si se utiliza este valor de umbral<br/> El operador mayor o igual a.<br/>'falso' indica que se utiliza 'mayor que' en lugar de 'mayor o igual que'.<br/> El valor predeterminado es verdadero.|
