##  ConditionalFormattingValue clase
Describe los valores de los puntos de interpolación en una escala de gradiente, barra de datos o conjunto de iconos.



El tipo ConditionalFormattingValue expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [value](/cells/python-net/es/aspose.cells/conditionalformattingvalue/value) |Obtenga o establezca el valor de este objeto de valor de formato condicional.<br/> Debe usarse junto con Type.|
| [type](/cells/python-net/es/aspose.cells/conditionalformattingvalue/type) | Obtenga o establezca el tipo de este objeto de valor de formato condicional.<br/> Establecer el tipo en FormatConditionValueType.Min o FormatConditionValueType.Max<br/> establecerá automáticamente el "Valor" en nulo.|
| [is_gte](/cells/python-net/es/aspose.cells/conditionalformattingvalue/is_gte) | Obtenga o establezca el indicador Mayor que o igual.<br/> Usar solo para conjuntos de íconos, determina si este valor de umbral usa<br/> el operador mayor o igual que.<br/>falso' indica que se utiliza 'mayor que' en lugar de 'mayor o igual que'.<br/> El valor predeterminado es verdadero.|
