##  password propiedad

Representa el password para proteger la hoja de trabajo.

###  Observaciones

Si password se establece como una cadena nula o en blanco,
Puede desproteger la hoja de cálculo o el libro sin utilizar un password.
De lo contrario, debe especificar password para desproteger la hoja de trabajo o el libro.
