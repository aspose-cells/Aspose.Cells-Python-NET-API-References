##  password propiedad

Representa el password para proteger la hoja de trabajo.

###  Observaciones

Si password se establece en una cadena nula o en blanco, puede desproteger la hoja de trabajo o el libro sin usar un password. De lo contrario, debe especificar el password para desproteger la hoja de trabajo o el libro de trabajo.
