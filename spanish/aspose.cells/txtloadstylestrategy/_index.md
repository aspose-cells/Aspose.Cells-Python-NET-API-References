##  TxtLoadStyleStrategy enumeración
Especifica cómo aplicar estilo a los valores analizados al convertir un valor de cadena en número o fecha y hora.



El tipo TxtLoadStyleStrategy expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| NONE |No establece el estilo para el valor analizado.|
| BUILT_IN | Establezca el estilo como número/fecha y hora incorporados cuando el valor analizado sean valores numéricos/fecha y hora simples.|
| EXACT_FORMAT | Establezca el formato personalizado exacto para el valor analizado para que el valor formateado sea el mismo que el de entrada original.|
