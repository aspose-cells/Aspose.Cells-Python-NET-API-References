##  MergeEmptyTdType enumeración
Representa el tipo de combinación para un elemento TD vacío al exportar un archivo a html.



El tipo MergeEmptyTdType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| DEFAULT | Mostrar como MS Excel.|
| NONE | Los elementos TD vacíos no se fusionarán al exportar un archivo a html.<br/> Esto generará un archivo html significativamente más grande.|
| MERGE_FORCELY | Fusionar un elemento TD vacío a la fuerza al exportar un archivo a html.<br/>El tamaño del archivo html se reducirá significativamente después de establecer el valor en verdadero.<br/> Si desea importar el archivo html para Excel o exportar líneas de cuadrícula perfectas al guardar el archivo en html,<br/> mantenga el valor predeterminado.|
