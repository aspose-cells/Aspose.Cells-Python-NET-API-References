##  HtmlExportDataOptions enumeración
Representa las opciones para exportar datos html.



El tipo HtmlExportDataOptions expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| TABLE | Exportar archivo a html que solo contiene la parte de la tabla.|
| ALL | Exportar todos los datos a html.|
