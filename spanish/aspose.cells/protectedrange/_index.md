##  ProtectedRange clase
Un rango específico que se le permitirá editar cuando la protección de la hoja esté activada.



El tipo ProtectedRange expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [name](/cells/python-net/es/aspose.cells/protectedrange/name) | Obtiene el título Range. Esto se usa como un descriptor, no como una definición de rango con nombre.|
| [cell_area](/cells/python-net/es/aspose.cells/protectedrange/cell_area) | Obtiene el objeto [ProtectedRange.cell_area](/cells/python-net/es/aspose.cells/protectedrange#cell_area) que representa el área de la celda a proteger.|
| [is_protected_with_password](/cells/python-net/es/aspose.cells/protectedrange/is_protected_with_password) |Indica si las hojas de trabajo están protegidas con contraseña.|
| [password](/cells/python-net/es/aspose.cells/protectedrange/password) | Representa la contraseña para proteger el rango.|
| [security_descriptor](/cells/python-net/es/aspose.cells/protectedrange/security_descriptor) | El descriptor de seguridad define las cuentas de usuario que pueden editar este rango sin proporcionar una contraseña para acceder al rango.|
