##  FormulaSettings clase
Configuración de fórmulas y cálculo.



El tipo FormulaSettings expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [calculate_on_open](/cells/python-net/es/aspose.cells/formulasettings/calculate_on_open) | Indica si se requiere que la aplicación realice un cálculo completo cuando se abre el libro.|
| [calculate_on_save](/cells/python-net/es/aspose.cells/formulasettings/calculate_on_save) | Indica si se debe volver a calcular el libro antes de guardar el documento, cuando se está en modo de cálculo manual.|
| [force_full_calculation](/cells/python-net/es/aspose.cells/formulasettings/force_full_calculation) | Indica si calcula todas las fórmulas cada vez que se activa un cálculo.|
| [calculation_mode](/cells/python-net/es/aspose.cells/formulasettings/calculation_mode) | Obtiene o establece el modo para el cálculo del libro en MS Excel.|
| [calculation_id](/cells/python-net/es/aspose.cells/formulasettings/calculation_id) | Especifica la versión del motor de cálculo utilizado para calcular los valores en el libro.|
| [enable_iterative_calculation](/cells/python-net/es/aspose.cells/formulasettings/enable_iterative_calculation) |Indica si habilitar el cálculo iterativo para resolver referencias circulares.|
| [max_iteration](/cells/python-net/es/aspose.cells/formulasettings/max_iteration) | Las iteraciones máximas para resolver una referencia circular.|
| [max_change](/cells/python-net/es/aspose.cells/formulasettings/max_change) | El cambio máximo para resolver una referencia circular.|
| [precision_as_displayed](/cells/python-net/es/aspose.cells/formulasettings/precision_as_displayed) | Si se debe establecer la precisión del resultado calculado tal como se muestra al calcular las fórmulas|
| [enable_calculation_chain](/cells/python-net/es/aspose.cells/formulasettings/enable_calculation_chain) | Si habilitar la cadena de cálculo para fórmulas. El valor predeterminado es falso.|
| [preserve_padding_spaces](/cells/python-net/es/aspose.cells/formulasettings/preserve_padding_spaces) | Indica si se conservan los espacios y saltos de línea que se rellenan entre los tokens de fórmula.<br/>mientras obtiene y establece fórmulas.<br/> El valor predeterminado es falso.|
