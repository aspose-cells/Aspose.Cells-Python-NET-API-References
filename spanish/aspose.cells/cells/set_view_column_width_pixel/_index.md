##  set_view_column_width_pixel {#int-int}
Establece el ancho de la columna en una vista diferente.



```python
def set_view_column_width_pixel(self, column, pixels):
    ...
```


| Parámetro| Tipo| Descripción|
| :- | :- | :- |
| column | int | El índice de la columna.|
| pixels | int | El ancho en unidades de píxeles.|
###  Observaciones

Si el tipo de vista actual es [`ViewType.PAGE_LAYOUT_VIEW`](/cells/python-net/es/aspose.cells/viewtype#PAGE_LAYOUT_VIEW), el ancho de la columna es el mismo que el ancho impreso.
