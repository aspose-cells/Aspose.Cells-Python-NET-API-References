##  process_built_in_functions propiedad

Si las funciones integradas que han sido compatibles con el motor integrado
debe ser verificado y procesado por esta implementación.
El valor predeterminado es falso.

###  Observaciones

Si el usuario necesita cambiar la lógica de cálculo de algunas funciones integradas, esta propiedad debe establecerse como verdadera.
De lo contrario, deje esta propiedad como falsa para considerar el rendimiento.
