##  ColorScale clase
 Describe la regla de formato condicional ColorScale.
Esta regla de formato condicional crea una escala de colores graduada en las celdas.



El tipo ColorScale expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [is_3_color_scale](/cells/python-net/es/aspose.cells/colorscale/is_3_color_scale) | Indica si el formato condicional es en escala de 3 colores.|
| [min_cfvo](/cells/python-net/es/aspose.cells/colorscale/min_cfvo) | Obtenga o establezca el objeto de valor mínimo de este ColorScale.<br/> No se puede establecer null o CFValueObject con tipo FormatConditionValueType.Max.|
| [mid_cfvo](/cells/python-net/es/aspose.cells/colorscale/mid_cfvo) | Obtenga o establezca el objeto de valor medio de este ColorScale.<br/>No se puede establecer CFValueObject con el tipo FormatConditionValueType.Max o FormatConditionValueType.Min.|
| [max_cfvo](/cells/python-net/es/aspose.cells/colorscale/max_cfvo) | Obtenga o establezca el objeto de valor máximo de este ColorScale.<br/> No se puede establecer null o CFValueObject con tipo FormatConditionValueType.Min.|
| [min_color](/cells/python-net/es/aspose.cells/colorscale/min_color) | Obtenga o establezca el color del degradado para el valor mínimo en el rango.|
| [mid_color](/cells/python-net/es/aspose.cells/colorscale/mid_color) | Obtenga o establezca el color del degradado para el valor medio del rango.|
| [max_color](/cells/python-net/es/aspose.cells/colorscale/max_color) | Obtenga o establezca el color del degradado para el valor máximo del rango.|
