##  html_cross_string_type propiedad

Indica si una cadena de celdas cruzadas se mostrará de la misma manera que MS Excel al guardar un archivo de Excel en formato html.
De forma predeterminada, el valor es Predeterminado, por lo que, para cadenas de celdas cruzadas, hay poca diferencia entre los archivos html creados por Aspose.Cells y MS Excel.
Pero el rendimiento para crear archivos html grandes, establecer el valor en Cruz sería varias veces más rápido que establecerlo en Predeterminado o Fit2Cell.
