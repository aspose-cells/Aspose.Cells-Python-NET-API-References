##  export_similar_border_style propiedad

Indicar si se exporta el estilo de borde similar cuando los navegadores no admiten el estilo de borde.
Si desea importar el archivo html o mht a Excel, mantenga el valor predeterminado.
El valor predeterminado es falso.
