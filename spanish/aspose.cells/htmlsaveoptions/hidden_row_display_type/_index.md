##  hidden_row_display_type propiedad

Fila oculta (la altura de esta fila es 0) en Excel, antes de guardarla en formato html,
si HtmlHiddenRowDisplayType es "Eliminar", la fila oculta no se generará,
Si el valor está "Oculto", la fila se generará, pero estaba oculta, el valor predeterminado es "Oculto".
