##  enable_css_custom_properties propiedad

Optimice la salida HTML mediante propiedades personalizadas de CSS. Por ejemplo, si una imagen base64 tiene varias instancias, con una propiedad personalizada, los datos de la imagen solo se guardan una vez para mejorar el rendimiento del HTML resultante.
El valor predeterminado es falso.
