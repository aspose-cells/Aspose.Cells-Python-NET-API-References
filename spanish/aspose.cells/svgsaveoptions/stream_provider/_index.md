##  stream_provider propiedad

Obtiene o establece IStreamProvider para exportar objetos.

###  Observaciones

Si se guarda como SVG o Tiff, esta propiedad se ignora.
De lo contrario, si se debe guardar más de una imagen, escribiremos otras imágenes junto a esta.
