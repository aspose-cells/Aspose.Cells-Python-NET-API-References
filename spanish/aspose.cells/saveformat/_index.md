##  SaveFormat enumeración
Representa el formato en el que se guarda el libro.



El tipo SaveFormat expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| CSV | Archivo de texto de valores separados por comas (CSV).<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`SaveFormat.CSV`](/cells/python-net/es/aspose.cells/saveformat#CSV).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| XLSX | Representa un archivo xlsx.|
| XLSM | Representa un archivo xlsm que habilita macros.|
| XLTX | Representa un archivo xltx.|
| XLTM | Representa un archivo xltm que habilita macros.|
| XLAM | Representa un archivo xltm que habilita macros complementarias.|
| TSV | Archivo de texto de valores separados por tabulaciones (TSV).<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`SaveFormat.TSV`](/cells/python-net/es/aspose.cells/saveformat#TSV).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| TAB_DELIMITED | Representa un archivo de texto delimitado por tabulaciones, lo mismo que [`SaveFormat.TSV`](/cells/python-net/es/aspose.cells/saveformat#TSV).|
| HTML | Representa un archivo html.|
| M_HTML | Representa un archivo mhtml.|
| ODS | Abra el archivo Hoja de documento (ODS).<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`SaveFormat.ODS`](/cells/python-net/es/aspose.cells/saveformat#ODS).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| EXCEL_97_TO_2003 | Representa un archivo xls de Excel97-2003.|
| SPREADSHEET_ML | Representa un archivo xml de Excel 2003.|
| XLSB | Representa un archivo xlsb.|
| AUTO | Si guarda el archivo en el disco, el formato del archivo corresponde a la extensión del nombre del archivo.<br/>Si guarda el archivo en la secuencia, el formato del archivo es xlsx.|
| UNKNOWN | Representa un formato no reconocido, no se puede guardar.|
| PDF | Representa un archivo Pdf.|
| XPS | Formato XPS (especificación de papel XML).<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`SaveFormat.XPS`](/cells/python-net/es/aspose.cells/saveformat#XPS).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| TIFF | Representa un archivo TIFF.<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`SaveFormat.TIFF`](/cells/python-net/es/aspose.cells/saveformat#TIFF).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| SVG | Archivo SVG.<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`SaveFormat.SVG`](/cells/python-net/es/aspose.cells/saveformat#SVG).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| DIF | Formato de intercambio de datos.|
| OTS | Abra el archivo de hoja de plantilla de documento (OTS).|
| XLT | Archivo de plantilla Excel 97-2003.|
| XML | Representa un archivo xml simple.|
| NUMBERS | Representa un archivo de números.|
| MARKDOWN | Representa un documento de rebajas.|
| FODS | Representa el formato de archivo OpenDocument Flat XML Spreadsheet (.fods).<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`SaveFormat.FODS`](/cells/python-net/es/aspose.cells/saveformat#FODS).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| SXC | Representa el formato de archivo de hoja de cálculo de StarOffice Calc (.sxc).<br/> NOTA: Este miembro ahora está obsoleto. En cambio,<br/>utilice la propiedad [`SaveFormat.SXC`](/cells/python-net/es/aspose.cells/saveformat#SXC).<br/> Esta propiedad se eliminará 6 meses después desde abril de 2021.<br/> Aspose se disculpa por cualquier inconveniente que pueda haber experimentado.|
| PPTX | Representa el archivo .pptx.|
| DOCX | Representa el archivo .docx.|
| EMF | Windows Metarchivo mejorado.|
| JPG | JPEGJFIF.|
| PNG | Gráficos de red portátiles.|
| BMP | Windows Mapa de bits|
| GIF | GIF|
| JSON | json|
| SQL_SCRIPT | SQL|
| X_HTML | Representa el archivo XHtml.|
| EPUB | Representa un archivo Epub.|
