##  HtmlOfficeMathOutputType enumeración
Representa cómo exportar OfficeMath a HTML.



El tipo HtmlOfficeMathOutputType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| IMAGE | Convierte OfficeMath a HTML como imagen especificada por<img> etiqueta.|
| MATH_ML | Convierte OfficeMath a HTML usando MathML.|
