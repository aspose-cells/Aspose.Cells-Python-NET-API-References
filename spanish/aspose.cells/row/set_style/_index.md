| Parámetro| Tipo| Descripción|
| :- | :- | :- |
| style | [`Style`](/cells/python-net/es/aspose.cells/style) | el estilo que se utilizará como estilo predeterminado para las celdas de esta fila.|
###  Observaciones

Este método solo establece el estilo dado como el estilo predeterminado para esta fila,
sin cambiar la configuración de estilo de las celdas existentes en esta fila.
Para actualizar la configuración de estilo de las celdas existentes al estilo especificado al mismo tiempo,
Por favor utilice [`Row.apply_style`](/cells/python-net/es/aspose.cells/row/apply_style)
