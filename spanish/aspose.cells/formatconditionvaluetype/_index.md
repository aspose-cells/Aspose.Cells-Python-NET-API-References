##  FormatConditionValueType enumeración
Tipo de valor de condición.



El tipo FormatConditionValueType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| FORMULA | El valor mínimo/punto medio/máximo para el<br/> el gradiente está determinado por una fórmula.|
| MAX | Indica que el valor máximo en el rango será<br/> utilizado como el valor máximo para el gradiente.|
| MIN | Indica que el valor mínimo en el rango será<br/> utilizado como el valor mínimo para el gradiente.|
| NUMBER | Indica que el mínimo/punto medio/máximo<br/>el valor del gradiente se especifica mediante una constante<br/> valor numérico.|
| PERCENT | El valor indica un porcentaje entre el mínimo<br/>y los valores máximos en el rango se utilizarán como<br/> valor mínimo/punto medio/máximo para el gradiente.|
| PERCENTILE | El valor indica una clasificación percentil en el rango deberá<br/>ser utilizado como el valor mínimo / punto medio / máximo<br/>para el gradiente.|
| AUTOMATIC_MAX | Indica que el valor máximo automático en el rango será<br/> utilizado como el valor máximo automático para el gradiente.|
| AUTOMATIC_MIN | Indica que el valor mínimo automático en el rango será<br/> utilizado como el valor mínimo automático para el gradiente.|
