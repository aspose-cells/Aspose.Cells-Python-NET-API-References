##  DefaultStyleSettings clase
Configuración de los valores predeterminados de las propiedades de estilo del libro de trabajo.



El tipo DefaultStyleSettings expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [built_in_preference](/cells/python-net/es/aspose.cells/defaultstylesettings/built_in_preference) | Indica si la propiedad para el formato de número es preferible cuando el estilo define tanto un número integrado como un patrón personalizado.<br/> El valor predeterminado es falso, lo que significa que, de forma predeterminada, se utilizará un patrón personalizado para formatear valores siempre que no esté vacío para un estilo.|
| [font_name](/cells/python-net/es/aspose.cells/defaultstylesettings/font_name) | Obtiene/establece el nombre de fuente predeterminado para el libro de trabajo|
| [font_size](/cells/python-net/es/aspose.cells/defaultstylesettings/font_size) | Obtiene o establece el tamaño de fuente estándar predeterminado para el libro de trabajo.|
| [horizontal_alignment](/cells/python-net/es/aspose.cells/defaultstylesettings/horizontal_alignment) | Obtiene/establece el valor predeterminado para la alineación horizontal|
| [vertical_alignment](/cells/python-net/es/aspose.cells/defaultstylesettings/vertical_alignment) | Obtiene/establece el valor predeterminado para la alineación vertical|
