##  HtmlCrossType enumeración
Representa cinco tipos de cadena cruzada html.



El tipo HtmlCrossType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| DEFAULT | Mostrar como MS Excel, depende de la siguiente celda.<br/>Si la siguiente celda es nula, la cadena se cruzará o se truncará|
| MS_EXPORT | Muestre la cadena como MS Excel exportando html.|
| CROSS | Mostrar HTML cadena cruzada, este rendimiento para crear archivos html de gran tamaño será más de diez veces más rápido que establecer el valor en Predeterminado o FitToCell.|
| CROSS_HIDE_RIGHT | Muestre HTML cadena cruzada y oculte la cadena correcta cuando los textos se superponen.|
| FIT_TO_CELL | Solo se muestra la cadena dentro del ancho de la celda.|
