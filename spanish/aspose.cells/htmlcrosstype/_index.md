##  HtmlCrossType enumeración
Representa cinco tipos de cadenas cruzadas HTML.



El tipo HtmlCrossType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| DEFAULT | Se muestra como MS Excel, depende de la siguiente celda.<br/> Si la siguiente celda es nula, la cadena se cruzará o se truncará.|
| MS_EXPORT | Muestra la cadena como MS Excel exportando HTML.|
| CROSS |Mostrar HTML cadena cruzada, este rendimiento para crear archivos html grandes será más de diez veces más rápido que configurar el valor en Predeterminado o Ajustar a celda.|
| CROSS_HIDE_RIGHT | Muestra la cadena cruzada HTML y oculta la cadena derecha cuando los textos se superponen.|
| FIT_TO_CELL | Solo se muestra la cadena dentro del ancho de la celda.|
