##  PageSetup clase
 Encapsula el objeto que representa la descripción de la configuración de la página.
El objeto PageSetup contiene todas las opciones de configuración de página.



El tipo PageSetup expone los siguientes miembros:

###  Ejemplo

```python
from aspose.cells import Workbook

workbook = Workbook()
sheets = workbook.worksheets
# Add a worksheet
sheets.add()
sheet = sheets[1]
pageSetup = sheet.page_setup
pageSetup.print_area = "D1:K13"

```
