##  ScenarioCollection clase
Representa la lista de escenarios.



El tipo ScenarioCollection expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [active_index](/cells/python-net/es/aspose.cells/scenariocollection/active_index) | Obtiene y establece qué escenario se selecciona.|
| [last_selected](/cells/python-net/es/aspose.cells/scenariocollection/last_selected) | Indica qué escenario fue seleccionado por última vez por el usuario para ejecutarse o mostrarse.|
| [capacity](/cells/python-net/es/aspose.cells/scenariocollection/capacity) | Obtiene o establece el número de elementos que puede contener la lista de matriz.|


###  Métodos
| Método| Descripción|
| :- | :- |
| [copy_to](/cells/python-net/es/aspose.cells/scenariocollection/copy_to/#list) | Copia toda la lista de matrices en una lista de matrices unidimensional compatible, comenzando desde el principio de la lista de matrices de destino.|
| [copy_to](/cells/python-net/es/aspose.cells/scenariocollection/copy_to/#int-list-int-int) | Copia un rango de elementos de la lista de matriz a una lista de matriz unidimensional compatible, comenzando en el índice especificado de la lista de matriz de destino.|
| [index_of](/cells/python-net/es/aspose.cells/scenariocollection/index_of/#aspose.cells.Scenario-int) | Busca el objeto especificado y devuelve el índice de base cero de la primera aparición dentro del rango de elementos en la lista de matriz que se extiende desde el índice especificado hasta el último elemento.|
| [index_of](/cells/python-net/es/aspose.cells/scenariocollection/index_of/#aspose.cells.Scenario-int-int) |Busca el objeto especificado y devuelve el índice de base cero de la primera aparición dentro del rango de elementos en la lista de matriz que comienza en el índice especificado y contiene el número especificado de elementos.|
| [last_index_of](/cells/python-net/es/aspose.cells/scenariocollection/last_index_of/#aspose.cells.Scenario) | Busca el objeto especificado y devuelve el índice de base cero de la última aparición dentro de toda la lista de la matriz.|
| [last_index_of](/cells/python-net/es/aspose.cells/scenariocollection/last_index_of/#aspose.cells.Scenario-int) | Busca el objeto especificado y devuelve el índice de base cero de la última aparición dentro del rango de elementos en la lista de matriz que se extiende desde el primer elemento hasta el índice especificado.|
| [last_index_of](/cells/python-net/es/aspose.cells/scenariocollection/last_index_of/#aspose.cells.Scenario-int-int) | Busca el objeto especificado y devuelve el índice de base cero de la última aparición dentro del rango de elementos en la lista de matriz que contiene el número especificado de elementos y termina en el índice especificado.|
| [add](/cells/python-net/es/aspose.cells/scenariocollection/add/#str) | Agrega un escenario.|
| [binary_search](/cells/python-net/es/aspose.cells/scenariocollection/binary_search/#aspose.cells.Scenario) | Busca un elemento en toda la lista de matriz ordenada utilizando el comparador predeterminado y devuelve el índice de base cero del elemento.|
