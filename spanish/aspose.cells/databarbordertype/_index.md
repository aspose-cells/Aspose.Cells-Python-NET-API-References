##  DataBarBorderType enumeración
Especifica el tipo de borde de una barra de datos.



El tipo DataBarBorderType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| NONE | La barra de datos no tiene borde.|
| SOLID | La barra de datos tiene un borde sólido.|
