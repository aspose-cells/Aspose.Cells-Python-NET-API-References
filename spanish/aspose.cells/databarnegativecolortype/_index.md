##  DataBarNegativeColorType enumeración
Especifica si se usa el mismo borde y color de relleno que las barras de datos positivos.



El tipo DataBarNegativeColorType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| COLOR | Utilice el color especificado en el cuadro de diálogo Valor negativo y configuración del eje<br/> o usando las propiedades ColorType y BorderColorType del objeto NegativeBarFormat.|
| SAME_AS_POSITIVE | Utilice el mismo color que las barras de datos positivos.|
