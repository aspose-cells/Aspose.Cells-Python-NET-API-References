##  HtmlVersion enumeración
Indica que se utiliza la versión HTML al guardar en formatos HTML.



El tipo HtmlVersion expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| DEFAULT | Guarde el documento de acuerdo con la exportación de MS Excel HTML.|
| X_HTML |Guarda el documento de acuerdo con el estándar XHTML 1.0 Transitional.|
| HTML5 | Guarda el documento de acuerdo con el estándar HTML 5.|
