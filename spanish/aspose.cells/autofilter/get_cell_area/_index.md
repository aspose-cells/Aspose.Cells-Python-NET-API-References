| Parámetro| Tipo| Descripción|
| :- | :- | :- |
| refresh_applied_range | bool | Si desea actualizar el rango aplicado.<br/>Para el rango aplicado de filtro automático, la última fila puede cambiar cuando cambian los datos de las celdas.<br/> Si esta bandera es verdadera, entonces la última fila del rango será recalculada de acuerdo a los datos de las celdas actuales.|
