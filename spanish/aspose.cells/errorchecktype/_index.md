##  ErrorCheckType enumeración
Representa todos los tipos de comprobación de errores.



El tipo ErrorCheckType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| EVALUATION_ERROR |Ignorar errores cuando las celdas contienen fórmulas que generan un error.|
| CALC |  |
| EMPTY_CELL_REF | Ignorar errores cuando las fórmulas hacen referencia a celdas vacías.|
| NUMBER_STORED_AS_TEXT | Ignorar errores cuando los números están formateados como texto o están precedidos por un apóstrofe|
| TEXT_NUMBER | Ignorar errores cuando los números están formateados como texto o están precedidos por un apóstrofe|
| INCONSIST_RANGE | Ignorar errores cuando las fórmulas omiten ciertas celdas en una región.|
| INCONSIST_FORMULA | Ignore los errores cuando una fórmula en una región de su hoja de cálculo difiere de otras fórmulas en la misma región.|
| TWO_DIGIT_TEXT_YEAR | Ignorar errores cuando las fórmulas contienen celdas con formato de texto con años representados como 2 dígitos.|
| TEXT_DATE | Ignorar errores cuando las fórmulas contienen celdas con formato de texto con años representados como 2 dígitos.|
| UNLOCKED_FORMULA | Ignorar errores cuando las celdas desbloqueadas contienen fórmulas.|
| UNPROCTED_FORMULA | Ignorar errores cuando las celdas desbloqueadas contienen fórmulas.|
| TABLE_DATA_VALIDATION | Ignorar errores cuando el valor de una celda en una tabla no cumple con las reglas de validación de datos especificadas.|
| VALIDATION | Ignorar errores cuando el valor de una celda en una tabla no cumple con las reglas de validación de datos especificadas.|
| CALCULATED_COLUMN | Ignorar errores cuando las celdas contienen un valor diferente de una fórmula de columna calculada.|
