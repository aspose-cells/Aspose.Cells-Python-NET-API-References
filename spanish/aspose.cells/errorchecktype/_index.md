##  ErrorCheckType enumeración
Representa todos los tipos de verificación de errores.



El tipo ErrorCheckType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| EVALUATION_ERROR | Ignore los errores cuando las celdas contengan fórmulas que den como resultado un error.|
| CALC |  |
| EMPTY_CELL_REF | Ignore los errores cuando las fórmulas hagan referencia a celdas vacías.|
| NUMBER_STORED_AS_TEXT | Ignorar errores cuando los números tienen formato de texto o están precedidos por un apóstrofo|
| TEXT_NUMBER | Ignorar errores cuando los números tienen formato de texto o están precedidos por un apóstrofo|
| INCONSIST_RANGE | Ignore los errores cuando las fórmulas omitan determinadas celdas en una región.|
| INCONSIST_FORMULA | Ignore los errores cuando una fórmula en una región de su hoja de cálculo difiere de otras fórmulas en la misma región.|
| TWO_DIGIT_TEXT_YEAR |Ignore los errores cuando las fórmulas contengan celdas con formato de texto con años representados como 2 dígitos.|
| TEXT_DATE |Ignore los errores cuando las fórmulas contengan celdas con formato de texto con años representados como 2 dígitos.|
| UNLOCKED_FORMULA | Ignore los errores cuando las celdas desbloqueadas contengan fórmulas.|
| UNPROCTED_FORMULA | Ignore los errores cuando las celdas desbloqueadas contengan fórmulas.|
| TABLE_DATA_VALIDATION | Ignore los errores cuando el valor de una celda en una tabla no cumple con las reglas de validación de datos especificadas.|
| VALIDATION | Ignore los errores cuando el valor de una celda en una tabla no cumple con las reglas de validación de datos especificadas.|
| CALCULATED_COLUMN | Ignore los errores cuando las celdas contengan un valor diferente de una fórmula de columna calculada.|
