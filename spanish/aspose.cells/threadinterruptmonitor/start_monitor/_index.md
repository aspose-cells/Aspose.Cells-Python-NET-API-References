##  start_monitor(self, ms_limit) {#int}
Inicia el monitor con el límite de tiempo especificado. El tiempo de inicio para calcular el coste del tiempo es justo cuando se llama a este método.
Por lo tanto, el procedimiento a supervisar debe iniciarse justo después de esta llamada.



```python

def start_monitor(self, ms_limit):
    ...
```


| Parámetro| Tipo| Descripción|
| :- | :- | :- |
| ms_limit | int | límite de tiempo (ms) para requerir la interrupción.|
