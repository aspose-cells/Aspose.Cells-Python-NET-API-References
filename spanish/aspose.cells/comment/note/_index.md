##  note propiedad

Representa el contenido del comentario.

###  Observaciones

Si se trata de un comentario en hilo, no se pudo cambiar el note, de lo contrario, MS Excel no pudo procesarlo como un comentario en hilo.

###  Ejemplo

```python

comment1.note = "First note."

```
