##  DataBarBorder clase
Representa el borde de las barras de datos especificadas por una regla de formato condicional.



El tipo DataBarBorder expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [color](/cells/python-net/es/aspose.cells/databarborder/color) | Obtiene o establece el color del borde de las barras de datos especificado por una regla de formato condicional.|
| [type](/cells/python-net/es/aspose.cells/databarborder/type) | Obtiene o establece el tipo de barras de datos del borde especificado por una regla de formato condicional.|
