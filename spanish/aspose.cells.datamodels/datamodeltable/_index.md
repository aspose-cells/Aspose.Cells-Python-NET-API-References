##  DataModelTable clase
Representa las propiedades de una sola tabla en el modelo de datos de una hoja de cálculo.



El tipo DataModelTable expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [id](/cells/python-net/es/aspose.cells.datamodels/datamodeltable/id) | Obtiene el identificador de la tabla del modelo de datos.|
