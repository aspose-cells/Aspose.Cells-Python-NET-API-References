##  CustomXmlPart clase
Representa una parte de almacenamiento de datos XML personalizados (datos XML personalizados dentro de un paquete).



El tipo CustomXmlPart expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [data](/cells/python-net/es/aspose.cells.markup/customxmlpart/data) | Obtiene o establece el contenido XML de esta parte de almacenamiento de datos XML personalizados.|
| [schema_data](/cells/python-net/es/aspose.cells.markup/customxmlpart/schema_data) | Obtiene o establece el contenido XML de esta parte de almacenamiento de datos de esquema XML personalizado.|
| [id](/cells/python-net/es/aspose.cells.markup/customxmlpart/id) |Obtiene y establece la identificación de la parte xml personalizada.|
