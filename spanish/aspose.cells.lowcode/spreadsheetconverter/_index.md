##  SpreadsheetConverter clase
Conversor para conversión entre diferentes formatos de archivos de hojas de cálculo, como xls, xlsx, xlsb, spreadsheet ml...



El tipo SpreadsheetConverter expone los siguientes miembros:

###  Métodos
| Método| Descripción|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/es/aspose.cells.lowcode/spreadsheetconverter/process/#str-str) | Convierte el archivo de plantilla dado entre formatos de archivos de hoja de cálculo.|
| [`process(, load_options, save_options)`](/cells/python-net/es/aspose.cells.lowcode/spreadsheetconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Convierte entre diferentes formatos de archivos de hojas de cálculo.|



###  Ejemplo

```python
from aspose.cells.lowcode import SpreadsheetConverter

SpreadsheetConverter.process("template.xlsx", "template.xlsb")

```
