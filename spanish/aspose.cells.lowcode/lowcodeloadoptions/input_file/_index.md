##  input_file propiedad

Obtiene y establece el archivo (con ruta si es necesario) de la plantilla.

###  Observaciones

Al establecer una ruta no nula y no vacía a esta propiedad,
El valor establecido previamente para [`LowCodeLoadOptions.input_stream`](/cells/python-net/es/aspose.cells.lowcode/lowcodeloadoptions#input_stream) será ignorado.
