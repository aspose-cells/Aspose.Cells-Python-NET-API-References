##  JsonConverter clase
Conversor para conversión entre estructura de datos json y otros formatos de archivos de hojas de cálculo.



El tipo JsonConverter expone los siguientes miembros:

###  Métodos
| Método| Descripción|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/es/aspose.cells.lowcode/jsonconverter/process/#str-str) | Convierte el archivo de plantilla dado entre json y otros formatos.|
| [`process(, load_options, save_options)`](/cells/python-net/es/aspose.cells.lowcode/jsonconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Convierte entre datos json y otros formatos de archivos de hojas de cálculo.|



###  Ejemplo

```python
from aspose.cells.lowcode import JsonConverter

JsonConverter.process("data.json", "res.xlsx")
```
