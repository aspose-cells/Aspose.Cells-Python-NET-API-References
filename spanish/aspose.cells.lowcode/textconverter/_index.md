##  TextConverter clase
Conversor para conversión entre formatos basados en texto (csv, tsv, dif...) y otros formatos de archivos de hojas de cálculo.



El tipo TextConverter expone los siguientes miembros:

###  Métodos
| Método| Descripción|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/es/aspose.cells.lowcode/textconverter/process/#str-str) | Convierte el archivo de plantilla dado entre archivos basados en texto y otros formatos.|
| [`process(, load_options, save_options)`](/cells/python-net/es/aspose.cells.lowcode/textconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Convierte formatos de archivo entre formatos basados en texto y otros formatos de archivos de hojas de cálculo|



###  Ejemplo

```python
from aspose.cells.lowcode import TextConverter

TextConverter.process("template.csv", "res.xlsx")

```
