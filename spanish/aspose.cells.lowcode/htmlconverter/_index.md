##  HtmlConverter clase
Conversor para conversión entre archivos html (html o mht) y otros formatos de archivos de hojas de cálculo.



El tipo HtmlConverter expone los siguientes miembros:

###  Métodos
| Método| Descripción|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/es/aspose.cells.lowcode/htmlconverter/process/#str-str) | Convierte el archivo de plantilla dado entre HTML y otros formatos.|
| [`process(, load_options, save_options)`](/cells/python-net/es/aspose.cells.lowcode/htmlconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Convierte archivos entre HTML y otros formatos de archivos de hojas de cálculo.|



###  Ejemplo

```python
from aspose.cells.lowcode import HtmlConverter

HtmlConverter.process("Template.xlsx", "res.html")

```
