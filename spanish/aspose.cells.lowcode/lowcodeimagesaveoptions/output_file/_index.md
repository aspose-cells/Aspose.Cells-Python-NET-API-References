##  output_file propiedad

Obtiene y establece el archivo (con ruta si es necesario) para guardar los datos generados.
Al configurar esta propiedad con un valor distinto de una cadena nula o vacía, se ignorará [`LowCodeSaveOptions.output_stream`](/cells/python-net/es/aspose.cells.lowcode/lowcodesaveoptions#output_stream).
