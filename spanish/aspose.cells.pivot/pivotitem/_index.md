##  PivotItem clase
Representa un elemento en un informe de PivotField.



El tipo PivotItem expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [is_hidden](/cells/python-net/es/aspose.cells.pivot/pivotitem/is_hidden) | Obtiene y establece si el elemento dinámico está oculto.|
| [is_hide_detail](/cells/python-net/es/aspose.cells.pivot/pivotitem/is_hide_detail) | Obtiene y establece si el elemento dinámico oculta detalles.|
| [value](/cells/python-net/es/aspose.cells.pivot/pivotitem/value) | Obtiene el valor del elemento pivote|
| [name](/cells/python-net/es/aspose.cells.pivot/pivotitem/name) | Obtiene el nombre del elemento dinámico.|
| [index](/cells/python-net/es/aspose.cells.pivot/pivotitem/index) | Obtiene el índice del elemento pivote en el campo pivote|


###  Métodos
| Método| Descripción|
| :- | :- |
| [move(count, is_same_parent)](/cells/python-net/es/aspose.cells.pivot/pivotitem/move/#int-bool) | Mueve el elemento hacia arriba o hacia abajo|
| [get_string_value()](/cells/python-net/es/aspose.cells.pivot/pivotitem/get_string_value/#) | Obtiene el valor de cadena del elemento pivote<br/> Si el valor es nulo, devolverá ""|
| [get_double_value()](/cells/python-net/es/aspose.cells.pivot/pivotitem/get_double_value/#) | Obtiene el valor doble del elemento pivote<br/> Si el valor es nulo o no es un número, devolverá 0|
| [get_date_time_value()](/cells/python-net/es/aspose.cells.pivot/pivotitem/get_date_time_value/#) | Obtiene el valor de fecha y hora del elemento pivote<br/> Si el valor es nulo, devolverá DateTime.MinValue|
