##  PivotConditionFormatScopeType enumeración
Representa el tipo de alcance de formato de condición de tabla dinámica.



El tipo PivotConditionFormatScopeType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| DATA | Indica que se aplica formato condicional a los campos de datos seleccionados.|
| FIELD | Indica que se aplica formato condicional a las intersecciones de campos de tabla dinámica seleccionados.|
| SELECTION |Indica que se aplica formato condicional a las celdas seleccionadas.|
