##  PivotShowValuesSetting clase
Representa la configuración para mostrar valores cuando se utiliza el cálculo ShowDataAs.



El tipo PivotShowValuesSetting expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [calculation_type](/cells/python-net/es/aspose.cells.pivot/pivotshowvaluessetting/calculation_type) | Representa cómo mostrar los valores de un campo de datos en el informe dinámico.|
| [base_field_index](/cells/python-net/es/aspose.cells.pivot/pivotshowvaluessetting/base_field_index) | Representa el campo base para un cálculo ShowDataAs cuando se utiliza dicho cálculo.|
| [base_item_position_type](/cells/python-net/es/aspose.cells.pivot/pivotshowvaluessetting/base_item_position_type) | Representa el tipo de elemento pivote base en el campo base cuando se utiliza el cálculo ShowDataAs.<br/> Válido sólo para campos de datos.<br/>Debido a que PivotItemPosition.Custom es solo para lectura, si necesita configurar PivotItemPosition.Custom,<br/> Por favor, configure el atributo PivotField.BaseItemIndex.|
| [base_item_index](/cells/python-net/es/aspose.cells.pivot/pivotshowvaluessetting/base_item_index) | Representa el índice personalizado del elemento pivote en el campo base cuando se utiliza el cálculo ShowDataAs.<br/> Válido sólo para campos de datos.|
