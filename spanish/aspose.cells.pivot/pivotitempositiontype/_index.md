##  PivotItemPositionType enumeración
Representa el tipo de posición del elemento base pivote en el campo base cuando se utiliza el cálculo ShowDataAs.



El tipo PivotItemPositionType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| PREVIOUS | Representa el elemento pivote anterior en el campo pivote.|
| NEXT | Representa el siguiente elemento pivote en el campo pivote.|
| CUSTOM | Muestra valores como un formato diferente según el índice del elemento dinámico en el campo dinámico.|
