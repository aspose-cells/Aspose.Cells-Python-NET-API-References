##  ReserveMissingPivotItemType enumeración
Representa cómo conservar los elementos pivote faltantes.



El tipo ReserveMissingPivotItemType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| DEFAULT | Elimina los elementos de pivote faltantes antiguos y reserva los elementos visibles que la fuente de datos actual no contiene como elementos faltantes.|
| ALL | Reserva todos los artículos faltantes.|
| NONE | Elimina todos los elementos pivotantes faltantes.|
