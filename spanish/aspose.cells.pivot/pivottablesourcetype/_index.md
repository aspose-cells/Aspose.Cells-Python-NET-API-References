##  PivotTableSourceType enumeración
Representa el tipo de fuente de datos de la tabla dinámica.



El tipo PivotTableSourceType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| SHEET | Especifica que los datos de origen son un rango.|
| EXTERNAL |Especifica que se utilizan datos de fuente externa.|
| CONSOLIDATION | Especifica que se utilizan múltiples rangos de consolidación como datos de origen.|
| SCENARIO | Los datos de origen se completan desde una estructura interna temporal.|
| UNKNOWN | Fuente de datos desconocida.|
