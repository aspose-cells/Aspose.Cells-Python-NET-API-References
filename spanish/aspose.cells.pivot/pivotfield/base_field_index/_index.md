##  base_field_index propiedad

Representa el campo base para un cálculo personalizado cuando se utiliza el cálculo ShowDataAs.

 NOTA: Esta propiedad ya no está disponible. En su lugar,
Utilice la propiedad PivotField.ShowValuesSetting.BaseFieldIndex en su lugar.
 Este método se eliminará 12 meses después, a partir de junio de 2024.
