##  PivotFieldType enumeración
Representa el tipo de campo de tabla dinámica.



El tipo PivotFieldType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| UNDEFINED |Presenta el tipo de campo pivote base.|
| ROW | Presenta el tipo de campo pivote de fila.|
| COLUMN | Presenta el tipo de campo pivote de columna.|
| PAGE | Presenta el tipo de campo pivote de la página.|
| DATA | Presenta el tipo de campo pivote de datos.|
