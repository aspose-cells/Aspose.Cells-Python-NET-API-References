##  ActiveXPersistenceType enumeración
Representa el método de persistencia para persistir un control ActiveX.



El tipo ActiveXPersistenceType expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| PROPERTY_BAG | Los datos se almacenan como datos xml.|
| STORAGE | Los datos se almacenan como datos binarios de almacenamiento.|
| STREAM | Los datos se almacenan como un flujo de datos binarios.|
| STREAM_INIT | Los datos se almacenan como datos binarios streaminit.|
