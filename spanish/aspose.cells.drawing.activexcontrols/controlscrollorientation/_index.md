##  ControlScrollOrientation enumeración
Representa el tipo de orientación del desplazamiento.



El tipo ControlScrollOrientation expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| AUTO | El control se representa horizontalmente cuando el ancho del control es mayor que su altura.<br/> De lo contrario, el control se representa verticalmente.|
| VERTICAL | El control se representa verticalmente.|
| HORIZONTAL | El control se representa horizontalmente.|
