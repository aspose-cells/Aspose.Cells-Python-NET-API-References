##  add_radio_button(self, upper_left_row, top, upper_left_column, left, height, width) {#int-int-int-int-int-int}

###  İadeler

Bir RadioButton nesnesi.


```python

def add_radio_button(self, upper_left_row, top, upper_left_column, left, height, width):
    ...
```


| Parametre| Tip| Tanım|
| :- | :- | :- |
| upper_left_row | int | Sol üst sıra dizini.|
| top | int | RadioButton'ın sol satırından itibaren dikey uzaklığını piksel cinsinden temsil eder.|
| upper_left_column | int | Sol üst sütun dizini.|
| left | int | RadioButton'ın sol sütunundan yatay uzaklığını piksel cinsinden temsil eder.|
| height | int | RadioButton'ın yüksekliğini piksel cinsinden temsil eder.|
| width | int | RadioButton'ın genişliğini piksel cinsinden temsil eder.|

###  Örnek

