##  HtmlVersion numaralandırma
Html formatlarına kaydederken HTML versiyonunun kullanıldığını gösterir.



HtmlVersion türü aşağıdaki üyeleri ortaya çıkarır:

###  Alanlar
| Alan| Tanım|
| :- | :- |
| DEFAULT | Belgeyi MS Excel HTML'e uygun olarak dışa aktararak kaydedin.|
| X_HTML |Belgeyi XHTML 1.0 Transitional standardına uygun olarak kaydeder.|
| HTML5 | HTML 5 standardına uygun olarak belgeyi kaydeder.|
