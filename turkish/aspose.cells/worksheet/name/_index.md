##  name mülk

Çalışma sayfasının name değerini alır veya ayarlar.

###  Notlar

 name numaralı sayfanın maksimum uzunluğu 31'dir. Ayrıca aynı name numarasını (büyük/küçük harf ayrımı gözetmeden) iki çalışma sayfasına atayamazsınız.
Örneğin, "SheetName1"i ilk çalışma sayfasına, "SHEETNAME1"i ise ikinci çalışma sayfasına ayarlayamazsınız.
