##  export_cell_name mülk

 Hücre adının Excel2007 .xlsx (.xlsm, .xltx, .xltm) dosyasına aktarılıp aktarılmayacağını belirtir.
Çıktı dosyasına SQL Server DTS üzerinden erişilebiliyorsa bu değerin true olması gerekir.
Değeri false olarak ayarlamak, büyük dosya oluştururken performansı önemli ölçüde artıracak ve dosya boyutunu azaltacaktır.
Varsayılan değer doğrudur.
