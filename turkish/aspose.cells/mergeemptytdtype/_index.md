##  MergeEmptyTdType numaralandırma
Dosyayı HTML'ye aktarırken boş TD öğesinin birleştirme türünü temsil eder.



MergeEmptyTdType türü aşağıdaki üyeleri ortaya çıkarır:

###  Alanlar
| Alan| Tanım|
| :- | :- |
| DEFAULT | MS Excel gibi görüntülenebilir.|
| NONE | Boş TD elemanları, dosyayı html'e aktarırken birleştirilmeyecektir.<br/> Bu, önemli ölçüde daha büyük bir HTML dosyası oluşturacaktır.|
| MERGE_FORCELY | Dosyayı html'e aktarırken boş TD öğesinin zorla birleştirilmesi.<br/>Değeri true olarak ayarlandıktan sonra html dosyasının boyutu önemli ölçüde azalacaktır.<br/> Html dosyasını Excel'e aktarmak veya dosyayı HTML'e kaydederken mükemmel ızgara çizgileri dışa aktarmak istiyorsanız,<br/> Lütfen varsayılan değeri koruyun.|
