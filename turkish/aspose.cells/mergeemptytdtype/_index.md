##  MergeEmptyTdType numaralandırma
Dosyayı html'ye aktarırken boş TD öğesinin birleştirme türünü temsil eder.



MergeEmptyTdType türü aşağıdaki üyeleri ortaya çıkarır:

###  Alanlar
| Alan| Tanım|
| :- | :- |
| DEFAULT | MS Excel gibi görüntüleyin.|
| NONE | Dosya html'ye aktarılırken boş TD öğeleri birleştirilmeyecektir.<br/> Bu, çok daha büyük bir html dosyası oluşturacaktır.|
| MERGE_FORCELY | Dosyayı html'ye aktarırken boş TD öğesinin zorla birleştirilmesi.<br/>Değeri true olarak ayarladıktan sonra html dosyasının boyutu önemli ölçüde azalacaktır.<br/> Html dosyasını excel'e aktarmak veya dosyayı html'ye kaydederken mükemmel ızgara çizgilerini dışa aktarmak istiyorsanız,<br/> lütfen varsayılan değeri koruyun.|
