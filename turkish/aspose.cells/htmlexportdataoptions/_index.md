##  HtmlExportDataOptions numaralandırma
HTML verilerinin dışa aktarılmasına yönelik seçenekleri temsil eder.



HtmlExportDataOptions türü aşağıdaki üyeleri ortaya çıkarır:

###  Alanlar
| Alan| Tanım|
| :- | :- |
| TABLE | Sadece tablo kısmını içeren dosyayı html'e aktarın.|
| ALL | Tüm verileri html'e aktarın.|
