##  SpreadsheetConverter sınıfı
Xls, xlsx, xlsb, spreadsheet ml gibi farklı elektronik tablo dosya formatları arasında dönüşüm sağlayan dönüştürücü.



SpreadsheetConverter türü aşağıdaki üyeleri ortaya çıkarır:

###  Yöntemler
| Yöntem| Tanım|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/tr/aspose.cells.lowcode/spreadsheetconverter/process/#str-str) | Verilen şablon dosyasını elektronik tablo dosya biçimleri arasında dönüştürür.|
| [`process(, load_options, save_options)`](/cells/python-net/tr/aspose.cells.lowcode/spreadsheetconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Farklı elektronik tablo dosya formatları arasında dönüşüm sağlar.|



###  Örnek

```python
from aspose.cells.lowcode import SpreadsheetConverter

SpreadsheetConverter.process("template.xlsx", "template.xlsb")

```
