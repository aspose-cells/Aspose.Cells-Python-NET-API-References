##  JsonConverter sınıfı
Json veri yapısı ile diğer elektronik tablo dosya formatları arasında dönüşüm sağlayan dönüştürücü.



JsonConverter türü aşağıdaki üyeleri ortaya çıkarır:

###  Yöntemler
| Yöntem| Tanım|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/tr/aspose.cells.lowcode/jsonconverter/process/#str-str) | Verilen şablon dosyasını json ve diğer formatlar arasında dönüştürür.|
| [`process(, load_options, save_options)`](/cells/python-net/tr/aspose.cells.lowcode/jsonconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Json verileri ile diğer elektronik tablo dosya formatları arasında dönüşüm yapar.|



###  Örnek

```python
from aspose.cells.lowcode import JsonConverter

JsonConverter.process("data.json", "res.xlsx")

```
