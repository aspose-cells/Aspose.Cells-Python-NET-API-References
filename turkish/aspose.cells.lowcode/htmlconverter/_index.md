##  HtmlConverter sınıfı
Html dosyaları (html veya mht) ile diğer elektronik tablo dosya formatları arasında dönüşüm sağlayan dönüştürücü.



HtmlConverter türü aşağıdaki üyeleri ortaya çıkarır:

###  Yöntemler
| Yöntem| Tanım|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/tr/aspose.cells.lowcode/htmlconverter/process/#str-str) | Verilen şablon dosyasını html ve diğer formatlar arasında dönüştürür.|
| [`process(, load_options, save_options)`](/cells/python-net/tr/aspose.cells.lowcode/htmlconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) | Dosyayı HTML ve diğer elektronik tablo dosya biçimleri arasında dönüştürür.|



###  Örnek

```python
from aspose.cells.lowcode import HtmlConverter

HtmlConverter.process("Template.xlsx", "res.html")

```
