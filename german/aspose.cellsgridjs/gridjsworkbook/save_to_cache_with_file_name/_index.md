##  save_to_cache_with_file_name {#str-str-str}

Speichert die Speicherdaten in der Cache-Datei mit dem angegebenen Dateinamen und legt außerdem das Öffnungskennwort fest. Das Speicherformat basiert auf der Dateierweiterung des Dateinamens.



```python
def save_to_cache_with_file_name(self, uid, filename, password):
    ...
```


| Parameter| Typ| Beschreibung|
| :- | :- | :- |
| uid | str | Die eindeutige ID für den Dateicache.|
| filename | str | Der zu speichernde Dateiname.|
| password | str | Das Öffnungskennwort der Excel-Datei. Der Wert kann null sein, wenn kein Passwort festgelegt ist.|
