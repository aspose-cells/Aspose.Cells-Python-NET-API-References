##  page_size Eigentum


Legt fest/ruft ab, ob eine Paginierung durchgeführt werden soll
GridJs begrenzt die Zeilengröße basierend auf PageSize. Wenn PageSize -1 ist, wird keine Paginierung durchgeführt
Der Standardwert ist -1
