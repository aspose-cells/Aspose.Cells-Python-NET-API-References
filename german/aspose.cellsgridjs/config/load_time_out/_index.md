##  load_time_out Eigentum


Setzt/ruft einen Timeout-Interrupt in Millisekunden beim Laden der Datei ab. Wenn die Ladezeit der Datei länger als erwartet ist, wird eine Ausnahme ausgelöst.
Der Standardwert ist -1, was bedeutet, dass kein Timeout-Interrupt festgelegt ist.
