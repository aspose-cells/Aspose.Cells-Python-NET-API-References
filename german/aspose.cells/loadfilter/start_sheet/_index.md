##  start_sheet(sheet) {#Worksheet}
Bereitet Filteroptionen vor dem Laden eines bestimmten Arbeitsblatts vor.
Die Implementierung von LoadFilter durch den Benutzer kann die LoadDataFilterOptions hier ändern
um anzugeben, wie Daten für dieses Arbeitsblatt geladen werden.



```python
def start_sheet(self, sheet):
    ...
```


| Parameter| Typ| Beschreibung|
| :- | :- | :- |
| sheet | [Worksheet](/cells/python-net/de/aspose.cells/worksheet) | Das zu ladende Arbeitsblatt.<br/>Hier können nur wenige Eigenschaften für das angegebene Arbeitsblattobjekt verwendet werden<br/>da die meisten Daten und Eigenschaften nicht geladen wurden. Die verfügbaren Eigenschaften sind:<br/> Name, Index, Sichtbarkeitstyp|
