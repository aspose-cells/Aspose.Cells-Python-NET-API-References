##  is_h_scroll_bar_visible Eigentum

Ruft einen Wert ab, der angibt, ob das generierte Arbeitsblatt eine horizontale Bildlaufleiste enthält, oder legt diesen fest.

###  Bemerkungen

Der Standardwert ist wahr.

###  Beispiel

Der folgende Code macht die horizontale Bildlaufleiste für die Tabelle unsichtbar.

```python

#  Hide the horizontal scroll bar of the Excel file.
settings.is_h_scroll_bar_visible = False
```
