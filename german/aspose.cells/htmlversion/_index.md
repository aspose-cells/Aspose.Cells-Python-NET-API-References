##  HtmlVersion Aufzählung
Gibt an, dass beim Speichern im HTML-Format die Version HTML verwendet wird.



Der Typ HtmlVersion macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| DEFAULT | Speichern Sie das Dokument gemäß dem MS Excel-Export HTML.|
| X_HTML |Speichert das Dokument in Übereinstimmung mit dem XHTML 1.0 Transitional-Standard.|
| HTML5 | Speichert das Dokument gemäß dem Standard HTML 5.|
