##  ColorScale Klasse
 Beschreiben Sie die bedingte Formatierungsregel ColorScale.
Diese bedingte Formatierungsregel erstellt eine abgestufte Farbskala für die Zellen.



Der Typ ColorScale macht die folgenden Member verfügbar:

###  Eigenschaften
| Eigentum| Beschreibung|
| :- | :- |
| [is_3_color_scale](/cells/python-net/de/aspose.cells/colorscale/is_3_color_scale) | Gibt an, ob die bedingte Formatierung eine 3-Farben-Skala ist.|
| [min_cfvo](/cells/python-net/de/aspose.cells/colorscale/min_cfvo) | Rufen Sie das Minimalwertobjekt dieser ColorScale ab oder legen Sie es fest.<br/> Null oder CFValueObject mit dem Typ FormatConditionValueType.Max können nicht darauf festgelegt werden.|
| [mid_cfvo](/cells/python-net/de/aspose.cells/colorscale/mid_cfvo) | Rufen Sie das Mittelwertobjekt dieser ColorScale ab oder legen Sie es fest.<br/> CFValueObject mit dem Typ FormatConditionValueType.Max oder FormatConditionValueType.Min kann nicht darauf festgelegt werden.|
| [max_cfvo](/cells/python-net/de/aspose.cells/colorscale/max_cfvo) | Rufen Sie das Maximalwertobjekt dieser ColorScale ab oder legen Sie es fest.<br/> Null oder CFValueObject mit dem Typ FormatConditionValueType.Min können nicht darauf festgelegt werden.|
| [min_color](/cells/python-net/de/aspose.cells/colorscale/min_color) | Rufen Sie die Verlaufsfarbe für den Mindestwert im Bereich ab oder legen Sie sie fest.|
| [mid_color](/cells/python-net/de/aspose.cells/colorscale/mid_color) |Rufen Sie die Verlaufsfarbe für den mittleren Wert im Bereich ab oder legen Sie sie fest.|
| [max_color](/cells/python-net/de/aspose.cells/colorscale/max_color) | Rufen Sie die Verlaufsfarbe für den Maximalwert im Bereich ab oder legen Sie sie fest.|
