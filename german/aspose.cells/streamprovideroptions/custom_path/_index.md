##  custom_path Eigentum

Der benutzerdefinierte Pfad (URL) des Benutzers, der in der generierten HTML-Datei für die angegebene Quelle gespeichert ist. Wenn nicht vom Benutzer definiert, wird DefaultPath verwendet.
Beispielsweise werden die Blattdaten vom Benutzer in d:/sheet001.htm gespeichert, die in der Haupt-HTML-Datei verwendete URL sollte „d:/sheet001.htm“ oder ein anderer gültiger relativer Pfad sein, auf den über das Haupt-HTML zugegriffen werden kann Datei.
