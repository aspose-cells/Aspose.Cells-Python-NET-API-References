##  custom_path Eigentum

Der benutzerdefinierte Pfad (URL), der in der generierten HTML-Datei für die referenzierte Quelle gespeichert ist. Falls nicht vom Benutzer definiert, wird DefaultPath verwendet.
Beispielsweise werden die Blattdaten vom Benutzer unter d:/sheet001.htm gespeichert. Die in der Haupt-HTML-Datei verwendete URL sollte „d:/sheet001.htm“ oder ein anderer gültiger relativer Pfad sein, auf den von der Haupt-HTML-Datei aus zugegriffen werden kann.
