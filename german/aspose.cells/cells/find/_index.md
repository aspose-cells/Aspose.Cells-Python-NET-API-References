###  Kehrt zurück

```python

    ...
```


| Parameter| Typ| Beschreibung|
| :- | :- | :- |
| what | any | Das Objekt, nach dem gesucht werden soll.<br/> Der Typ sollte int,double,DateTime,string,bool sein.|
| previous_cell | [Cell](/cells/python-net/de/aspose.cells/cell) | Vorherige Zelle mit demselben Objekt.<br/> Dieser Parameter kann auf null gesetzt werden, wenn von Anfang an gesucht wird.|
