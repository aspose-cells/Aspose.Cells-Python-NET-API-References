##  ConditionalFormattingValue Klasse
Beschreibt die Werte der Interpolationspunkte in einer Verlaufsskala, dataBar oder iconSet.



Der Typ ConditionalFormattingValue macht die folgenden Member verfügbar:

###  Eigenschaften
| Eigentum| Beschreibung|
| :- | :- |
| [value](/cells/python-net/de/aspose.cells/conditionalformattingvalue/value) |Rufen Sie den Wert dieses Wertobjekts für bedingte Formatierung ab oder legen Sie ihn fest.<br/> Es sollte in Verbindung mit Type verwendet werden.|
| [type](/cells/python-net/de/aspose.cells/conditionalformattingvalue/type) | Rufen Sie den Typ dieses Wertobjekts für bedingte Formatierung ab oder legen Sie ihn fest.<br/> Festlegen des Typs auf FormatConditionValueType.Min oder FormatConditionValueType.Max<br/> setzt „Wert“ automatisch auf Null.|
| [is_gte](/cells/python-net/de/aspose.cells/conditionalformattingvalue/is_gte) | Rufen Sie das Flag „Größer als oder gleich“ ab oder setzen Sie es.<br/> Nur für Symbolsätze verwenden, bestimmt, ob dieser Schwellenwert verwendet wird<br/> der Größer-gleich-Operator.<br/>„false“ gibt an, dass „größer als“ anstelle von „größer als oder gleich“ verwendet wird.<br/> Der Standardwert ist wahr.|
