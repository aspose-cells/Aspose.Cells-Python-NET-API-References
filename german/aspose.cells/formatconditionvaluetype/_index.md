##  FormatConditionValueType Aufzählung
Bedingungswerttyp.



Der Typ FormatConditionValueType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| FORMULA | Der minimale/mittlere/maximale Wert für<br/> Der Gradient wird durch eine Formel bestimmt.|
| MAX | Gibt an, dass der Maximalwert im Bereich liegen soll<br/> als Maximalwert für den Gradienten verwendet.|
| MIN | Gibt an, dass der Mindestwert im Bereich liegen soll<br/> als Minimalwert für den Gradienten verwendet.|
| NUMBER | Zeigt das Minimum / den Mittelpunkt / das Maximum an<br/>Der Wert für den Gradienten wird durch eine Konstante angegeben<br/> Zahlenwert.|
| PERCENT | Der Wert gibt einen Prozentsatz zwischen dem Minimum an<br/>Als Werte sind die Maximalwerte im Bereich zu verwenden<br/> Minimal-/Mittelpunkt-/Maximalwert für den Gradienten.|
| PERCENTILE |Der Wert gibt eine prozentuale Rangfolge im Bereich an<br/>als Minimal-/Mittelwert/Maximalwert verwendet werden<br/> für den Farbverlauf.|
| AUTOMATIC_MAX | Gibt an, dass der automatische Maximalwert im Bereich liegen soll<br/> wird als automatischer Maximalwert für den Gradienten verwendet.|
| AUTOMATIC_MIN | Gibt an, dass der automatische Mindestwert im Bereich liegen soll<br/> wird als automatischer Minimalwert für den Gradienten verwendet.|
