##  FormatConditionValueType Aufzählung
Werttyp der Bedingung.



Der Typ FormatConditionValueType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| FORMULA | Der minimale/mittlere/maximale Wert für die<br/> Steigung wird durch eine Formel bestimmt.|
| MAX | Gibt an, dass der Höchstwert im Bereich sein soll<br/> als Maximalwert für den Gradienten verwendet.|
| MIN | Gibt an, dass der Mindestwert im Bereich sein soll<br/> als Minimalwert für den Gradienten verwendet.|
| NUMBER | Gibt an, dass das Minimum / Mittelpunkt / Maximum<br/>Wert für die Steigung wird durch eine Konstante angegeben<br/> Zahlenwert.|
| PERCENT | Der Wert gibt einen Prozentsatz zwischen dem Minimum an<br/>und Maximalwerte im Bereich sind als zu verwenden<br/> Minimal- / Mittelpunkt- / Maximalwert für die Steigung.|
| PERCENTILE | Der Wert gibt eine prozentuale Rangfolge innerhalb des Bereichs an<br/>als Minimal-/Mittelpunkt-/Maximalwert verwendet werden<br/>für den Gradienten.|
| AUTOMATIC_MAX | Gibt an, dass der automatische Maximalwert im Bereich liegen soll<br/> als automatischer Maximalwert für den Gradienten verwendet.|
| AUTOMATIC_MIN | Gibt an, dass der automatische Mindestwert im Bereich sein soll<br/> als automatischer Minimalwert für den Gradienten verwendet.|
