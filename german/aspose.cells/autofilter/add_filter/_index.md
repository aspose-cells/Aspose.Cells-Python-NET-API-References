##  add_filter(self, field_index, criteria) {#int-str}
Fügt einen Filter für eine Filterspalte hinzu.



```python

def add_filter(self, field_index, criteria):
    ...
```


| Parameter| Typ| Beschreibung|
| :- | :- | :- |
| field_index | int | Der ganzzahlige Offset des Feldes, auf dem der Filter basieren soll<br/> (von links in der Liste; das äußerste linke Feld ist Feld 0).|
| criteria | str | Die angegebenen Kriterien (eine Zeichenfolge, z. B. „101“).<br/>Es kann nur Null sein oder einer der Zellenwerte in dieser Spalte.|
###  Bemerkungen

MS Excel 2007 unterstützt Mehrfachauswahl in einer Filterspalte.
