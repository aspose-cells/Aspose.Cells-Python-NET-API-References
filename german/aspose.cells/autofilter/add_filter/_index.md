##  add_filter(field_index, criteria) {#int-str}
Fügt einen Filter für eine Filterspalte hinzu.



```python
def add_filter(self, field_index, criteria):
    ...
```


| Parameter| Typ| Beschreibung|
| :- | :- | :- |
| field_index | int | Der ganzzahlige Offset des Felds, auf dem der Filter basieren soll<br/> (von links in der Liste; das Feld ganz links ist Feld 0).|
| criteria | str | Die angegebenen Kriterien (eine Zeichenfolge, z. B. „101“).<br/> Es kann nur null oder einer der Werte der Zellen in dieser Spalte sein.|
###  Bemerkungen

MS Excel 2007 unterstützt die Mehrfachauswahl in einer Filterspalte.
