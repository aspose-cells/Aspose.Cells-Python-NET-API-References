##  LoadFormat Aufzählung
Stellt das Ladedateiformat dar.



Der Typ LoadFormat macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| AUTO | Stellt die automatische Erkennung des Formats dar.|
| CSV | Textdatei mit durch Kommas getrennten Werten (CSV).<br/> HINWEIS: Dieses Mitglied ist mittlerweile veraltet. Stattdessen<br/>Bitte verwenden Sie die [`LoadFormat.CSV`](/cells/python-net/de/aspose.cells/loadformat#CSV)-Eigenschaft.<br/> Diese Eigenschaft wird 6 Monate später (ab April 2021) entfernt.<br/> Aspose entschuldigt sich für etwaige Unannehmlichkeiten, die Ihnen möglicherweise entstanden sind.|
| XLSX | Stellt eine Office Open XML-Tabellenkalkulationsarbeitsmappe oder -Vorlage mit oder ohne Makros dar.|
| TSV | Textdatei mit tabulatorgetrennten Werten (TSV).<br/> HINWEIS: Dieses Mitglied ist mittlerweile veraltet. Stattdessen<br/>Bitte verwenden Sie die [`LoadFormat.TSV`](/cells/python-net/de/aspose.cells/loadformat#TSV)-Eigenschaft.<br/> Diese Eigenschaft wird 6 Monate später (ab April 2021) entfernt.<br/> Aspose entschuldigt sich für etwaige Unannehmlichkeiten, die Ihnen möglicherweise entstanden sind.|
| TAB_DELIMITED | Stellt eine durch Tabulatoren getrennte Textdatei dar, dasselbe mit [`LoadFormat.TSV`](/cells/python-net/de/aspose.cells/loadformat#TSV).|
| HTML | Stellt eine HTML-Datei dar.|
| M_HTML |Stellt eine MHTML-Datei dar.|
| ODS | Öffnen Sie die Datei „Dokumentblatt (ODS)“.<br/> HINWEIS: Dieses Mitglied ist mittlerweile veraltet. Stattdessen<br/>Bitte verwenden Sie die [`LoadFormat.ODS`](/cells/python-net/de/aspose.cells/loadformat#ODS)-Eigenschaft.<br/> Diese Eigenschaft wird 6 Monate später (ab April 2021) entfernt.<br/> Aspose entschuldigt sich für etwaige Unannehmlichkeiten, die Ihnen möglicherweise entstanden sind.|
| EXCEL_97_TO_2003 | Stellt eine Excel97-2003-XLS-Datei dar.|
| SPREADSHEET_ML | Stellt eine Excel 2003-XML-Datei dar.|
| XLSB | Stellt eine XLSB-Datei dar.|
| OTS | Öffnen Sie die OTS-Datei (Document Template Sheet).|
| NUMBERS | Stellt eine Zahlendatei dar.|
| FODS | Stellt das OpenDocument Flat XML Spreadsheet (.fods)-Dateiformat dar.<br/> HINWEIS: Dieses Mitglied ist mittlerweile veraltet. Stattdessen<br/>Bitte verwenden Sie die [`LoadFormat.FODS`](/cells/python-net/de/aspose.cells/loadformat#FODS)-Eigenschaft.<br/> Diese Eigenschaft wird 6 Monate später (ab April 2021) entfernt.<br/> Aspose entschuldigt sich für etwaige Unannehmlichkeiten, die Ihnen möglicherweise entstanden sind.|
| SXC | Repräsentiert das StarOffice Calc Spreadsheet-Dateiformat (.sxc).<br/> HINWEIS: Dieses Mitglied ist mittlerweile veraltet. Stattdessen<br/>Bitte verwenden Sie die [`LoadFormat.SXC`](/cells/python-net/de/aspose.cells/loadformat#SXC)-Eigenschaft.<br/> Diese Eigenschaft wird 6 Monate später (ab April 2021) entfernt.<br/> Aspose entschuldigt sich für etwaige Unannehmlichkeiten, die Ihnen möglicherweise entstanden sind.|
| XML | Stellt eine einfache XML-Datei dar.|
| EPUB | Stellt eine EPUB-Datei dar.|
| AZW3 | Stellt eine AZW3-Datei dar.|
| CHM | Stellt eine CHM-Datei dar.|
| MARKDOWN | Stellt eine Markdown-Datei dar.|
| UNKNOWN |Stellt ein unbekanntes Format dar und kann nicht geladen werden.|
| IMAGE | Bild|
| JSON | Json|
| DIF | Datenaustauschformat.|
| DBF | Xbase-Datendatei|
