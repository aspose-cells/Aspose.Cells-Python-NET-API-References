##  add_printer_font(self, font_full_name, font_pcl_name) {#str-str}
Fügt Informationen zu Schriftarten hinzu, die vom Hersteller bereits zum Drucker hinzugefügt wurden.



```python

def add_printer_font(self, font_full_name, font_pcl_name):
    ...
```


| Parameter| Typ| Beschreibung|
| :- | :- | :- |
| font_full_name | str | Vollständiger Name der Schriftart (z. B. „Times New Roman Bold Italic“), die in der Quelldatei verwendet wird.|
| font_pcl_name | str | Name der Schriftart, die im PCL-Ausgabedokument verwendet wird.|
###  Bemerkungen

Es gibt 52 Schriftarten, die gemäß PCL-Spezifikation in jeden Drucker integriert werden können.
Hersteller können ihren Geräten jedoch einige andere Schriftarten hinzufügen.
