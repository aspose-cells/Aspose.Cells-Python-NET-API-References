##  name Eigentum

Ruft die name des Arbeitsblatts ab oder legt sie fest.

###  Bemerkungen

 Die maximale Länge des Blatts name beträgt 31. Und Sie können nicht dasselbe name (Groß-/Kleinschreibung beachten) zwei Arbeitsblättern zuweisen.
Beispielsweise können Sie „SheetName1“ nicht auf das erste Arbeitsblatt und „SHEETNAME1“ auf das zweite Arbeitsblatt festlegen.
