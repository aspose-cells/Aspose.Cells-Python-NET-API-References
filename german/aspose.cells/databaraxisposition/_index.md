##  DataBarAxisPosition Aufzählung
Gibt die Achsenposition für einen Zellbereich mit bedingter Formatierung als Datenbalken an.



Der Typ DataBarAxisPosition macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| AUTOMATIC | Zeigen Sie die Achse an einer variablen Position an, basierend auf dem Verhältnis des minimalen negativen Werts zum maximalen positiven Wert im Bereich.<br/> Positive Werte werden von links nach rechts angezeigt.<br/> Negative Werte werden von rechts nach links angezeigt.<br/>Wenn alle Werte positiv oder alle Werte negativ sind, wird keine Achse angezeigt.|
| MIDPOINT | Zeigen Sie die Achse unabhängig von den Werten im Bereich in der Mitte der Zelle an.<br/> Positive Werte werden von links nach rechts angezeigt.<br/> Negative Werte werden von rechts nach links angezeigt.|
| NONE | Es wird keine Achse angezeigt und sowohl positive als auch negative Werte werden in der Richtung von links nach rechts angezeigt.|
