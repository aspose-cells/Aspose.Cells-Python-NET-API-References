##  ErrorCellValueType Aufzählung
Stellt einen Zellenwert dar, der einen Fehler enthält.



Der Typ ErrorCellValueType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| BLOCKED | Stellt den Wert einer Zelle dar, die einen #BLOCKED!-Fehler enthält.|
| BUSY |Stellt den Wert einer Zelle dar, die einen #BUSY!-Fehler enthält.|
| CALC | Stellt den Wert einer Zelle dar, die einen #CALC!-Fehler enthält.|
| CONNECT | Stellt den Wert einer Zelle dar, die einen #CONNECT!-Fehler enthält.|
| NAME | Stellt den Wert einer Zelle dar, die einen #NAME?-Fehler enthält.|
| FIELD | Stellt den Wert einer Zelle dar, die einen #FIELD!-Fehler enthält.|
| SPILL | Stellt den Wert einer Zelle dar, die einen #SPILL!-Fehler enthält.|
| UNKNOWN | Stellt den Wert einer Zelle dar, die einen #UNKNOWN!-Fehler enthält.|
| TIME_OUT | Stellt den Wert einer Zelle dar, die einen #TIMEOUT!-Fehler enthält.|
| EXTERNAL | Stellt den Wert einer Zelle dar, die einen #EXTERNAL!-Fehler enthält.|
