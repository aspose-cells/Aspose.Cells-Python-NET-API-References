##  ErrorCheckType Aufzählung
Stellt alle Fehlerprüfungstypen dar.



Der Typ ErrorCheckType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| EVALUATION_ERROR | Fehler ignorieren, wenn Zellen Formeln enthalten, die zu einem Fehler führen.|
| CALC |  |
| EMPTY_CELL_REF | Fehler ignorieren, wenn Formeln auf leere Zellen verweisen.|
| NUMBER_STORED_AS_TEXT | Fehler ignorieren, wenn Zahlen als Text formatiert sind oder ihnen ein Apostroph vorangestellt ist|
| TEXT_NUMBER | Fehler ignorieren, wenn Zahlen als Text formatiert sind oder ihnen ein Apostroph vorangestellt ist|
| INCONSIST_RANGE | Fehler ignorieren, wenn Formeln bestimmte Zellen in einem Bereich auslassen.|
| INCONSIST_FORMULA | Ignorieren Sie Fehler, wenn sich eine Formel in einem Bereich Ihres Arbeitsblatts von anderen Formeln im selben Bereich unterscheidet.|
| TWO_DIGIT_TEXT_YEAR |Ignorieren Sie Fehler, wenn Formeln textformatierte Zellen enthalten, in denen das Jahr als zwei Ziffern dargestellt wird.|
| TEXT_DATE |Ignorieren Sie Fehler, wenn Formeln textformatierte Zellen enthalten, in denen das Jahr als zwei Ziffern dargestellt wird.|
| UNLOCKED_FORMULA | Fehler ignorieren, wenn entsperrte Zellen Formeln enthalten.|
| UNPROCTED_FORMULA | Fehler ignorieren, wenn entsperrte Zellen Formeln enthalten.|
| TABLE_DATA_VALIDATION | Fehler ignorieren, wenn der Wert einer Zelle in einer Tabelle nicht den angegebenen Datenvalidierungsregeln entspricht.|
| VALIDATION | Fehler ignorieren, wenn der Wert einer Zelle in einer Tabelle nicht den angegebenen Datenvalidierungsregeln entspricht.|
| CALCULATED_COLUMN | Fehler ignorieren, wenn Zellen einen anderen Wert als eine berechnete Spaltenformel enthalten.|
