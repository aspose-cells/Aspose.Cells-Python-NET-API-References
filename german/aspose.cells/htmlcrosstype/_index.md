##  HtmlCrossType Aufzählung
Stellt fünf Typen von HTML-Crossstrings dar.



Der Typ HtmlCrossType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| DEFAULT | Anzeige wie MS Excel, abhängig von der nächsten Zelle.<br/> Wenn die nächste Zelle null ist, wird die Zeichenfolge gekreuzt oder abgeschnitten.|
| MS_EXPORT | Zeigen Sie die Zeichenfolge wie beim HTML-Export von MS Excel an.|
| CROSS |Zeigt die Kreuzzeichenfolge HTML an. Diese Leistung beim Erstellen großer HTML-Dateien ist mehr als zehnmal schneller als das Festlegen des Werts auf „Default“ oder „FitToCell“.|
| CROSS_HIDE_RIGHT | Zeigen Sie die Querzeichenfolge HTML an und verbergen Sie die rechte Zeichenfolge, wenn sich die Texte überlappen.|
| FIT_TO_CELL | Es wird nur die Zeichenfolge innerhalb der Zellenbreite angezeigt.|
