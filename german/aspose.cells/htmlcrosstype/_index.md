##  HtmlCrossType Aufzählung
Stellt fünf Arten von HTML-Kreuzzeichenfolgen dar.



Der Typ HtmlCrossType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| DEFAULT | Anzeige wie MS Excel, abhängig von der nächsten Zelle.<br/> Wenn die nächste Zelle null ist, kreuzt sich die Zeichenfolge oder sie wird abgeschnitten|
| MS_EXPORT | Zeigen Sie die Zeichenfolge an, als würde MS Excel HTML exportieren.|
| CROSS | Wenn Sie die Kreuzzeichenfolge HTML anzeigen, ist diese Leistung beim Erstellen großer HTML-Dateien mehr als zehnmal schneller als das Festlegen des Werts auf „Default“ oder „FitToCell“.|
| CROSS_HIDE_RIGHT | Zeigen Sie die Querzeichenfolge HTML an und blenden Sie die rechte Zeichenfolge aus, wenn sich die Texte überlappen.|
| FIT_TO_CELL | Die Zeichenfolge wird nur innerhalb der Breite der Zelle angezeigt.|
