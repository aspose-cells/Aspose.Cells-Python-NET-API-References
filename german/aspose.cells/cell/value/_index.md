Ruft die in dieser Zelle enthaltene value ab/legt sie fest.

Boolesch,


Datum/Uhrzeit,


Doppelt,


Ganze Zahl


Zeichenfolge.

Und es gibt keine Garantie, dass die zurückgegebene Nummer value immer als Objekt vom gleichen Typ beibehalten wird.
