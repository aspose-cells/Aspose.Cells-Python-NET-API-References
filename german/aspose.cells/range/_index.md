###  Bemerkungen

Die Klasse Range bezeichnet einen Bereich der Excel-Tabelle.
Damit können Sie den Bereich formatieren und festlegen.
Und Sie können auch einfach eine Reihe von Excel kopieren.

###  Beispiel

Das folgende Beispiel zeigt, wie Sie einen Bereich erstellen und den Bereichswert in Excel festlegen.

```python
from aspose.cells import Workbook

# Instantiating a Workbook object
workbook = Workbook()
#  Get the first Worksheet Cells.
cells = workbook.worksheets[0].cells
#  Create a range (A1:D3).
range = cells.create_range("A1", "D3")
#  Set value to the range.
range.value = "Hello"
# Save the Excel file
workbook.save("book1.xlsm")

```
