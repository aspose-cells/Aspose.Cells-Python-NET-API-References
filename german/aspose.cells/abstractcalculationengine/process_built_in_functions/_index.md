##  process_built_in_functions Eigentum

Ob integrierte Funktionen, die von der integrierten Engine unterstützt wurden
sollten von dieser Implementierung überprüft und verarbeitet werden.
Der Standardwert ist falsch.

###  Bemerkungen

Wenn der Benutzer die Berechnungslogik einiger integrierter Funktionen ändern muss, sollte diese Eigenschaft auf „true“ festgelegt werden.
Andernfalls belassen Sie diese Eigenschaft aus Leistungsgründen bitte auf „false“.
