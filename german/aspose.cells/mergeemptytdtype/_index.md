##  MergeEmptyTdType Aufzählung
Stellt den Zusammenführungstyp für leere TD-Elemente beim Exportieren einer Datei in HTML dar.



Der Typ MergeEmptyTdType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| DEFAULT | Anzeige wie MS Excel.|
| NONE | Leere TD-Elemente werden beim Exportieren der Datei in HTML nicht zusammengeführt.<br/> Dadurch wird eine erheblich größere HTML-Datei generiert.|
| MERGE_FORCELY | Beim Exportieren einer Datei in HTML wird das Zusammenführen eines leeren TD-Elements erzwungen.<br/>Die Größe der HTML-Datei wird erheblich reduziert, wenn der Wert auf „true“ gesetzt wird.<br/> Wenn Sie die HTML-Datei in Excel importieren oder beim Speichern der Datei im HTML-Format perfekte Rasterlinien exportieren möchten,<br/> Bitte behalten Sie den Standardwert bei.|
