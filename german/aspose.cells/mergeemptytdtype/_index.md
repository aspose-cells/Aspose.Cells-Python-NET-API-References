##  MergeEmptyTdType Aufzählung
Stellt den Zusammenführungstyp für leere TD-Elemente beim Exportieren einer Datei in HTML dar.



Der Typ MergeEmptyTdType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| DEFAULT | Anzeige wie MS Excel.|
| NONE | Leere TD-Elemente werden beim Exportieren der Datei in HTML nicht zusammengeführt.<br/> Dadurch wird eine deutlich größere HTML-Datei generiert.|
| MERGE_FORCELY | Beim Exportieren der Datei ins HTML werden leere TD-Elemente zwangsweise zusammengeführt.<br/>Die Größe der HTML-Datei wird erheblich reduziert, nachdem der Wert auf „true“ gesetzt wurde.<br/> Wenn Sie die HTML-Datei in Excel importieren oder perfekte Gitterlinien exportieren möchten, wenn Sie die Datei im HTML-Format speichern,<br/> Bitte behalten Sie den Standardwert bei.|
