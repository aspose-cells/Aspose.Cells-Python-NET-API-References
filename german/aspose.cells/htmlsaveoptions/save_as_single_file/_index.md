##  save_as_single_file Eigentum

Gibt an, ob der HTML-Code als einzelne Datei gespeichert wird.
Der Standardwert ist false.

###  Bemerkungen

Wenn die Arbeitsmappe mehrere Arbeitsblätter oder andere erforderliche Ressourcen wie Bilder enthält,
 Normalerweise müssen diese Arbeitsblätter und andere Ressourcen in separaten Dateien gespeichert werden.
In einigen Szenarien muss der Benutzer möglicherweise nur eine resultierende Datei abrufen, beispielsweise um die Übertragung zu vereinfachen.
Wenn dies der Fall ist, kann der Benutzer diese Eigenschaft auf „true“ setzen.
