##  default_font_name Eigentum

Geben Sie den Namen der Standardschriftart für den HTML-Export an. Die Standardschriftart wird verwendet, wenn die Schriftart des Stils nicht vorhanden ist.
Wenn diese Eigenschaft null ist, verwendet Aspose.Cells eine universelle Schriftart, die dieselbe Familie wie die ursprüngliche Schriftart hat.
der Standardwert ist null.
