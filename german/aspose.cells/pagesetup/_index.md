##  PageSetup Klasse
 Kapselt das Objekt, das die Beschreibung der Seiteneinrichtung darstellt.
Das Objekt PageSetup enthält alle Seiteneinrichtungsoptionen.



Der Typ PageSetup macht die folgenden Member verfügbar:

###  Beispiel

```python
from aspose.cells import Workbook

workbook = Workbook()
sheets = workbook.worksheets
# Add a worksheet
sheets.add()
sheet = sheets[1]
pageSetup = sheet.page_setup
pageSetup.print_area = "D1:K13"

```
