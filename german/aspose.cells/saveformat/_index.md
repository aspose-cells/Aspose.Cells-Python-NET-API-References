##  SaveFormat Aufzählung
Stellt das Format dar, in dem die Arbeitsmappe gespeichert wird.



Der Typ SaveFormat macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| CSV |Kommagetrennte Werte (CSV) Textdatei.<br/> HINWEIS: Dieses Mitglied ist jetzt veraltet. Stattdessen,<br/>Bitte verwenden Sie die Eigenschaft [SaveFormat.CSV](/cells/python-net/de/aspose.cells/saveformat#CSV).<br/> Diese Property wird 6 Monate später seit April 2021 entfernt.<br/> Aspose entschuldigt sich für etwaige Unannehmlichkeiten.|
| XLSX | Stellt eine xlsx-Datei dar.|
| XLSM | Stellt eine xlsm-Datei dar, die Makros aktiviert.|
| XLTX | Stellt eine xltx-Datei dar.|
| XLTM | Stellt eine xltm-Datei dar, die Makros aktiviert.|
| XLAM | Stellt eine XLTM-Datei dar, die Add-in-Makros aktiviert.|
| TSV | Tabulatorgetrennte Werte (TSV) Textdatei.<br/> HINWEIS: Dieses Mitglied ist jetzt veraltet. Stattdessen,<br/>Bitte verwenden Sie die Eigenschaft [SaveFormat.TSV](/cells/python-net/de/aspose.cells/saveformat#TSV).<br/> Diese Property wird 6 Monate später seit April 2021 entfernt.<br/> Aspose entschuldigt sich für etwaige Unannehmlichkeiten.|
| TAB_DELIMITED | Stellt eine tabulatorgetrennte Textdatei dar, wie [SaveFormat.TSV](/cells/python-net/de/aspose.cells/saveformat#TSV).|
| HTML | Stellt eine HTML-Datei dar.|
| M_HTML | Stellt eine mhtml-Datei dar.|
| ODS | Öffnen Sie die Dokumentblattdatei (ODS).<br/> HINWEIS: Dieses Mitglied ist jetzt veraltet.
