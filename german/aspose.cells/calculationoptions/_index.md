###  Eigenschaften
| Eigentum| Beschreibung|
| :- | :- |
| [ignore_error](/cells/python-net/de/aspose.cells/calculationoptions/ignore_error) | Gibt an, ob beim Berechnen von Formeln aufgetretene Fehler ignoriert werden sollen.<br/>Der Fehler kann auf eine nicht unterstützte Funktion, externe Links usw. zurückzuführen sein.<br/> Der Standardwert ist „true“.|
| [recursive](/cells/python-net/de/aspose.cells/calculationoptions/recursive) | Gibt an, ob die abhängigen Zellen rekursiv berechnet werden, wenn eine Zelle berechnet wird und diese von anderen Zellen abhängt.<br/> Der Standardwert ist „true“.|
| [calc_stack_size](/cells/python-net/de/aspose.cells/calculationoptions/calc_stack_size) | Die Stapelgröße für die rekursive Berechnung von Zellen. Der Standardwert ist 200.|
| [precision_strategy](/cells/python-net/de/aspose.cells/calculationoptions/precision_strategy) | Gibt die Strategie zur Verarbeitung der Berechnungsgenauigkeit an.|
| [linked_data_sources](/cells/python-net/de/aspose.cells/calculationoptions/linked_data_sources) | Gibt die Datenquellen für in Formeln verwendete externe Links an.|
| [character_encoding](/cells/python-net/de/aspose.cells/calculationoptions/character_encoding) | Gibt die Kodierung an, die zum Kodieren/Dekodieren von Zeichen beim Berechnen von Formeln verwendet wird.<br/>Bei Funktionen wie CHAR und CODE hängt das berechnete Ergebnis von den Regionseinstellungen und dem Standardzeichensatz der Umgebung ab.<br/>Mit dieser Eigenschaft kann der Benutzer die richtige Kodierung für diese Funktion angeben, um das erwartete Ergebnis zu erhalten.|
