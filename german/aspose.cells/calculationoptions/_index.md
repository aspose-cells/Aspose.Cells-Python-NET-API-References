###  Eigenschaften
| Eigentum| Beschreibung|
| :- | :- |
| [ignore_error](/cells/python-net/de/aspose.cells/calculationoptions/ignore_error) | Gibt an, ob beim Berechnen von Formeln aufgetretene Fehler ignoriert werden sollen.<br/>Der Fehler kann auf nicht unterstützte Funktionen, externe Links usw. zurückzuführen sein.<br/> Der Standardwert ist wahr.|
| [recursive](/cells/python-net/de/aspose.cells/calculationoptions/recursive) | Gibt an, ob die abhängigen Zellen rekursiv berechnet werden, wenn eine Zelle berechnet wird und sie von anderen Zellen abhängt.<br/> Der Standardwert ist wahr.|
| [custom_engine](/cells/python-net/de/aspose.cells/calculationoptions/custom_engine) | Die benutzerdefinierte Formelberechnungs-Engine zur Erweiterung der Standardberechnungs-Engine von Aspose.Cells.|
| [calculation_monitor](/cells/python-net/de/aspose.cells/calculationoptions/calculation_monitor) | Der Monitor, mit dem der Benutzer den Fortschritt der Formelberechnung verfolgen kann.|
| [calc_stack_size](/cells/python-net/de/aspose.cells/calculationoptions/calc_stack_size) | Die Stapelgröße für die rekursive Berechnung von Zellen. Der Standardwert ist 200.|
| [precision_strategy](/cells/python-net/de/aspose.cells/calculationoptions/precision_strategy) | Gibt die Strategie zur Verarbeitung der Berechnungsgenauigkeit an.|
| [linked_data_sources](/cells/python-net/de/aspose.cells/calculationoptions/linked_data_sources) | Gibt die Datenquellen für externe Links an, die in Formeln verwendet werden.|
| [character_encoding](/cells/python-net/de/aspose.cells/calculationoptions/character_encoding) | Gibt die Kodierung an, die zum Kodieren/Dekodieren von Zeichen beim Berechnen von Formeln verwendet wird.<br/>Bei Funktionen wie CHAR, CODE hängt das berechnete Ergebnis von den Regionseinstellungen und dem Standardzeichensatz der Umgebung ab.<br/> Mit dieser Eigenschaft kann der Benutzer die richtige Kodierung angeben, die für diese Funktion verwendet wird, um das erwartete Ergebnis zu erhalten.|
