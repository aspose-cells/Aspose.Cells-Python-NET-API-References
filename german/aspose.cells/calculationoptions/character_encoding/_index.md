##  character_encoding Eigentum

Gibt die Kodierung an, die zum Kodieren/Dekodieren von Zeichen beim Berechnen von Formeln verwendet wird.
Bei Funktionen wie CHAR und CODE hängt das berechnete Ergebnis von den Regionseinstellungen und dem Standardzeichensatz der Umgebung ab.
Mit dieser Eigenschaft kann der Benutzer die richtige Kodierung für diese Funktion angeben, um das erwartete Ergebnis zu erhalten.
