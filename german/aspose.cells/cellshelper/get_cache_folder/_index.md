##  get_cache_folder() {#}
Ruft den Ordner für temporäre Dateien ab, die als Datencache verwendet werden können.


###  Kehrt zurück

Angegebener Ordner für Cache-Dateien.
Wenn es nicht angegeben wurde, wird null zurückgegeben
und der temporäre Pfad des Systems wird bei Bedarf verwendet.


```python

@staticmethod
def get_cache_folder():
    ...
```


###  Bemerkungen

Cache-Dateien werden im Allgemeinen für einige Funktionen zur Berücksichtigung der Speicherleistung verwendet.
wie das Speichern großer Datensätze in einer XLS-Datei,
oder Verwendung des Speichermodus mit Dateicache für das Zellenmodell.
