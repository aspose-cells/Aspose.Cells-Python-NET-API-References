##  TxtLoadStyleStrategy Aufzählung
Gibt an, wie der Stil auf analysierte Werte angewendet wird, wenn ein Zeichenfolgenwert in eine Zahl oder eine Datums-/Uhrzeitkonvertierung umgewandelt wird.



Der Typ TxtLoadStyleStrategy macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| NONE |Legt keinen Stil für den analysierten Wert fest.|
| BUILT_IN | Legen Sie den Stil als integrierte Zahl/Datum/Uhrzeit fest, wenn es sich bei den geparsten Werten um einfache numerische/Datum/Uhrzeit-Werte handelt.|
| EXACT_FORMAT | Legen Sie das genaue benutzerdefinierte Format für den analysierten Wert fest, damit der formatierte Wert mit dem ursprünglichen Eingabewert übereinstimmt.|
