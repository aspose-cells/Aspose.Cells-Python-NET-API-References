##  TextCrossType Aufzählung
Listet den anzuzeigenden Texttyp auf, wenn die Textbreite größer als die Zellenbreite ist.



Der Typ TextCrossType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| DEFAULT | Text wie in Microsoft Excel anzeigen.|
| CROSS_KEEP | Zeigen Sie den gesamten Text an, indem Sie andere Zellen durchstreichen, und behalten Sie den Text der durchgestrichenen Zellen bei.|
| CROSS_OVERRIDE | Zeigen Sie den gesamten Text an, indem Sie andere Zellen durchstreichen, und überschreiben Sie den Text der durchgestrichenen Zellen.|
| STRICT_IN_CELL | Zeigen Sie nur den Text innerhalb der Zellenbreite an.|
