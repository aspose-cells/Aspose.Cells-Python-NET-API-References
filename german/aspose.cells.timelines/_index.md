 Der**Aspose.Cells.Timelines** Der Namespace enthält Klassen, die den Zugriff auf und das Zeichnen von Zeitleisten in Excel-Tabellendateien ermöglichen.

###  Klassen
| Klasse| Beschreibung|
| :- | :- |
| [`Timeline`](/cells/python-net/de/aspose.cells.timelines/timeline) | Zusammenfassende Beschreibung der Zeitleistenansicht<br/> Aufgrund von MS Excel unterstützt Excel 2003 keine Timeline|
| [`TimelineCollection`](/cells/python-net/de/aspose.cells.timelines/timelinecollection) | Gibt die Auflistung aller Timeline-Objekte im angegebenen Arbeitsblatt an.<br/> Aufgrund von MS Excel unterstützt Excel 2003 die Zeitleiste nicht.|
