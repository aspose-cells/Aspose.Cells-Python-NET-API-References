##  upper_delta_y Eigentum

Ruft den vertikalen Versatz des Shapes von seiner oberen linken Eckzeile ab oder legt diesen fest.

###  Bemerkungen

Der Wertebereich ist 0 bis 256.

###  Beispiel

```python

if shape.upper_delta_y == 3:
    shape.upper_delta_y = 1
```
