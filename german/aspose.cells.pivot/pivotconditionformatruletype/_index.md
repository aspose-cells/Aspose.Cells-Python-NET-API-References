##  PivotConditionFormatRuleType Aufzählung
Stellt den Formatierungsregeltyp der PivotTable-Bedingung dar.



Der Typ PivotConditionFormatRuleType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| NONE | Gibt an, dass die Top N-bedingte Formatierung nicht ausgewertet wird|
| ALL | Zeigt an, dass die Top N-bedingte Formatierung<br/> über den gesamten Umfangsbereich ausgewertet.|
| ROW | Gibt an, dass die Top N-bedingte Formatierung für jede Zeile ausgewertet wird.|
| COLUMN | Zeigt an, dass die Top N-bedingte Formatierung<br/> für jede Spalte ausgewertet.|
