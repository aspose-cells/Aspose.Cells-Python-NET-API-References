##  PivotConditionFormatRuleType Aufzählung
Stellt den Typ der PivotTable-Bedingungsformatierungsregel dar.



Der Typ PivotConditionFormatRuleType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| NONE | Zeigt an, dass die bedingte Formatierung der Top N nicht ausgewertet wird|
| ALL | Gibt an, dass Top N bedingte Formatierung ist<br/> über den gesamten Umfangsbereich ausgewertet.|
| ROW | Gibt an, dass die bedingte Formatierung der Top N für jede Zeile ausgewertet wird.|
| COLUMN | Gibt an, dass Top N bedingte Formatierung ist<br/> für jede Spalte ausgewertet.|
