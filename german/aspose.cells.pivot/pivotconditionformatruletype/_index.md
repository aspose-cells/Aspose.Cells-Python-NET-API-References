##  PivotConditionFormatRuleType Aufzählung
Stellt den Formatierungsregeltyp für PivotTable-Bedingungen dar.



Der Typ PivotConditionFormatRuleType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| NONE | Gibt an, dass die bedingte Top-N-Formatierung nicht ausgewertet wird|
| ALL | Gibt an, dass Top N bedingte Formatierung ist<br/> über den gesamten Anwendungsbereich ausgewertet.|
| ROW | Gibt an, dass die bedingte Top-N-Formatierung für jede Zeile ausgewertet wird.|
| COLUMN | Gibt an, dass Top N bedingte Formatierung ist<br/> für jede Spalte ausgewertet.|
