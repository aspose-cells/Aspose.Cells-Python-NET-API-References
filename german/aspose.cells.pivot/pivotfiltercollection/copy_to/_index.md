```python

```


| Parameter| Typ| Beschreibung|
| :- | :- | :- |
| index | int | Der nullbasierte Index in der Quellarrayliste, bei dem das Kopieren beginnt.|

| count | int | Die Anzahl der zu kopierenden Elemente.|
