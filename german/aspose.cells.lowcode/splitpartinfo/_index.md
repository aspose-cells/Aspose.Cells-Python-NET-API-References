##  SplitPartInfo Klasse
Stellt die Informationen eines Ein-/Ausgangs für mehrere Ein-/Ausgänge dar,
beispielsweise die aktuelle Seite, die beim Konvertieren einer Tabelle in ein Bild gerendert werden soll.



Der Typ SplitPartInfo macht die folgenden Member verfügbar:

###  Eigenschaften
| Eigentum| Beschreibung|
| :- | :- |
| [part_index](/cells/python-net/de/aspose.cells.lowcode/splitpartinfo/part_index) | Index des aktuellen Teils in der Sequenz (basierend auf 0).<br/> -1 bedeutet, dass es keine Mehrfachteile gibt und das Ergebnis daher einzeln ist.|
| [sheet_index](/cells/python-net/de/aspose.cells.lowcode/splitpartinfo/sheet_index) | Index des Blatts, in dem sich das aktuelle Teil befindet. -1 bedeutet, dass nur ein Blatt vorhanden ist.|
| [sheet_name](/cells/python-net/de/aspose.cells.lowcode/splitpartinfo/sheet_name) | Name des Blatts, in dem sich das aktuelle Teil befindet.|
