 Der**Aspose.Cells.QueryTables** Der Namespace stellt Klassen für den Zugriff auf und die Aktualisierung von Arbeitsblatttabellen bereit, die aus Daten erstellt wurden, die von einer externen Datenquelle zurückgegeben wurden, z. B. einem SQL-Server oder einer Microsoft Access-Datenbank.
