##  enable_css_custom_properties Eigentum

Optimieren Sie die HTML-Ausgabe mithilfe benutzerdefinierter CSS-Eigenschaften. Beispielsweise müssen die Bilddaten bei mehreren Vorkommen eines Base64-Bildes mithilfe benutzerdefinierter Eigenschaften nur einmal gespeichert werden, wodurch die Leistung des resultierenden HTML verbessert wird.
Der Standardwert ist „false“.
