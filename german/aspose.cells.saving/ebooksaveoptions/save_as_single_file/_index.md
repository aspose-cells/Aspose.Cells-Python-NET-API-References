##  save_as_single_file Eigentum

Gibt an, ob das HTML als einzelne Datei gespeichert werden soll.
Der Standardwert ist „false“.

###  Bemerkungen

Wenn die Arbeitsmappe mehrere Arbeitsblätter oder andere erforderliche Ressourcen wie Bilder enthält,
 Normalerweise müssen diese Arbeitsblätter und anderen Ressourcen in separaten Dateien gespeichert werden.
In manchen Fällen benötigt der Benutzer möglicherweise nur eine Ergebnisdatei, beispielsweise aus Gründen der Übertragungsfreundlichkeit.
Wenn ja, kann der Benutzer diese Eigenschaft auf „true“ setzen.
