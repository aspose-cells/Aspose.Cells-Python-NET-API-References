##  encode_entity_as_code Eigentum

Gibt an, ob die HTML-Zeichenentitäten durch Dezimalcode ersetzt werden.
(zB wird „&nbsp;“ durch „&#160;“ ersetzt).
Der Standardwert ist „false“.
