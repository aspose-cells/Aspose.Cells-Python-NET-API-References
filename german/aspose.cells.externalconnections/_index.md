 Der**Aspose.Cells.ExternalConnections** namespace stellt Klassen für externe Verbindungen bereit, um Daten aus externen Quellen wie Datenbanken und dem Web abzurufen.

###  Klassen
| Klasse| Beschreibung|
| :- | :- |
| [ConnectionParameter](/cells/python-net/de/aspose.cells.externalconnections/connectionparameter) | Gibt Eigenschaften zu allen Parametern an, die mit externen Datenverbindungen verwendet werden<br/> Parameter gelten für ODBC- und Webabfragen.|
| [ConnectionParameterCollection](/cells/python-net/de/aspose.cells.externalconnections/connectionparametercollection) | Gibt die Sammlung [ConnectionParameter](/cells/python-net/de/aspose.cells.externalconnections/connectionparameter) an|
| [DBConnection](/cells/python-net/de/aspose.cells.externalconnections/dbconnection) | Gibt alle Eigenschaften an, die einer externen ODBC- oder OLE DB-Datenverbindung zugeordnet sind.|
| [DataModelConnection](/cells/python-net/de/aspose.cells.externalconnections/datamodelconnection) | Gibt eine Datenmodellverbindung an|
| [ExternalConnection](/cells/python-net/de/aspose.cells.externalconnections/externalconnection) | Gibt eine externe Datenverbindung an|
| [ExternalConnectionCollection](/cells/python-net/de/aspose.cells.externalconnections/externalconnectioncollection) | Gibt die Sammlung [ExternalConnection](/cells/python-net/de/aspose.cells.externalconnections/externalconnection) an|
| [WebQueryConnection](/cells/python-net/de/aspose.cells.externalconnections/webqueryconnection) | Gibt die Eigenschaften für eine Webabfragequelle an. Eine Webabfrage ruft Daten aus HTML-Tabellen ab,<br/> und kann auch HTTP-"Get"-Parameter liefern, die vom Webserver beim Generieren der HTML verarbeitet werden sollen<br/> einschließlich der Parameter und Parameterelemente.|
