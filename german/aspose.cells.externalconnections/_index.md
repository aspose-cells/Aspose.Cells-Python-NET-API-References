 Der**Aspose.Cells.ExternalConnections**Der Namespace bietet Klassen für externe Verbindungen, um Daten aus externen Quellen wie Datenbanken und dem Internet abzurufen.

###  Klassen
| Klasse| Beschreibung|
| :- | :- |
| [`ConnectionParameter`](/cells/python-net/de/aspose.cells.externalconnections/connectionparameter) | Gibt Eigenschaften zu allen Parametern an, die mit externen Datenverbindungen verwendet werden<br/> Parameter sind für ODBC- und Webabfragen gültig.|
| [`ConnectionParameterCollection`](/cells/python-net/de/aspose.cells.externalconnections/connectionparametercollection) | Gibt die Sammlung [`ConnectionParameter`](/cells/python-net/de/aspose.cells.externalconnections/connectionparameter) an|
| [`DBConnection`](/cells/python-net/de/aspose.cells.externalconnections/dbconnection) | Gibt alle Eigenschaften an, die einer externen ODBC- oder OLE DB-Datenverbindung zugeordnet sind.|
| [`DataModelConnection`](/cells/python-net/de/aspose.cells.externalconnections/datamodelconnection) | Gibt eine Datenmodellverbindung an|
| [`ExternalConnectionCollection`](/cells/python-net/de/aspose.cells.externalconnections/externalconnectioncollection) | Gibt die ExternalConnection-Sammlung an|
| [`WebQueryConnection`](/cells/python-net/de/aspose.cells.externalconnections/webqueryconnection) | Gibt die Eigenschaften für eine Webabfragequelle an. Eine Webabfrage ruft Daten aus HTML Tabellen ab.<br/> und kann auch HTTP "Get"-Parameter liefern, die vom Webserver bei der Generierung der HTML verarbeitet werden sollen.<br/> einschließlich der Parameter und Parameterelemente.|
