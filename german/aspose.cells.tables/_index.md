 Der**Aspose.Cells.Tables** Namespace enthält Klassen, die Tabellen, Zeilen, Zellen und deren Formatierung für Excel-Tabellendateien darstellen.

###  Klassen
| Klasse| Beschreibung|
| :- | :- |
| [ListColumn](/cells/python-net/de/aspose.cells.tables/listcolumn) | Stellt eine Spalte in einer Liste dar.|
| [ListColumnCollection](/cells/python-net/de/aspose.cells.tables/listcolumncollection) | Stellt eine Auflistung aller [ListColumn](/cells/python-net/de/aspose.cells.tables/listcolumn)-Objekte im angegebenen ListObject-Objekt dar.|
| [ListObject](/cells/python-net/de/aspose.cells.tables/listobject) | Stellt ein Listenobjekt auf einem Arbeitsblatt dar.<br/> Das ListObject-Objekt ist ein Mitglied der ListObjects-Auflistung.<br/> Die ListObjects-Auflistung enthält alle Listenobjekte auf einem Arbeitsblatt.|
| [ListObjectCollection](/cells/python-net/de/aspose.cells.tables/listobjectcollection) | Stellt eine Sammlung von [ListObject](/cells/python-net/de/aspose.cells.tables/listobject)-Objekten im Arbeitsblatt dar.|
| [TableStyle](/cells/python-net/de/aspose.cells.tables/tablestyle) | Stellt den Tabellenstil dar.|
| [TableStyleCollection](/cells/python-net/de/aspose.cells.tables/tablestylecollection) | Stellt alle benutzerdefinierten Tabellenstile dar.|
| [TableStyleElement](/cells/python-net/de/aspose.cells.tables/tablestyleelement) | Repräsentiert das Element des Tabellenstils.|
| [TableStyleElementCollection](/cells/python-net/de/aspose.cells.tables/tablestyleelementcollection) |Stellt alle Elemente des Tabellenstils dar.|
| [TableToRangeOptions](/cells/python-net/de/aspose.cells.tables/tabletorangeoptions) | Repräsentiert die Optionen beim Konvertieren einer Tabelle in einen Bereich.|
