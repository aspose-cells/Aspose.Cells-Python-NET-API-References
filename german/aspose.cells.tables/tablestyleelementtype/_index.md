##  TableStyleElementType Aufzählung
Stellt den Stilelementtyp Table oder PivotTable dar.



Der Typ TableStyleElementType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| BLANK_ROW | Tabellenformatelement, das für die leeren Zeilen von PivotTable gilt.|
| FIRST_COLUMN | Tabellenstilelement, das für die erste Spalte der Tabelle gilt.|
| FIRST_COLUMN_STRIPE | Tabellenstilelement, das auf die ersten Spaltenstreifen der Tabelle angewendet wird.|
| FIRST_COLUMN_SUBHEADING | Tabellenformatelement, das für die Unterüberschrift der ersten Spalte von PivotTable gilt.|
| FIRST_HEADER_CELL | Tabellenstilelement, das auf die erste Kopfzeilenzelle der Tabelle angewendet wird.|
| FIRST_ROW_STRIPE | Tabellenstilelement, das für die Streifen in der ersten Zeile der Tabelle gilt.|
| FIRST_ROW_SUBHEADING | Tabellenformatelement, das für die Unterüberschrift der ersten Zeile von PivotTable gilt.|
| FIRST_SUBTOTAL_COLUMN | Tabellenformatelement, das für die erste Zwischensummenspalte von PivotTable gilt.|
| FIRST_SUBTOTAL_ROW | Tabellenstilelement, das auf die erste Zwischensummenzeile der Pivot-Tabelle angewendet wird.|
| GRAND_TOTAL_COLUMN | Tabellenstilelement, das auf die Gesamtsummenspalte der Pivot-Tabelle angewendet wird.|
| GRAND_TOTAL_ROW |Tabellenstilelement, das auf die Gesamtsummenzeile der Pivot-Tabelle angewendet wird.|
| FIRST_TOTAL_CELL | Tabellenstilelement, das auf die erste Summenzeilenzelle der Tabelle angewendet wird.|
| HEADER_ROW | Tabellenstilelement, das für die Kopfzeile der Tabelle gilt.|
| LAST_COLUMN | Tabellenstilelement, das für die letzte Spalte der Tabelle gilt.|
| LAST_HEADER_CELL | Tabellenstilelement, das auf die letzte Kopfzeilenzelle der Tabelle angewendet wird.|
| LAST_TOTAL_CELL | Tabellenstilelement, das auf die letzte Summenzeilenzelle der Tabelle angewendet wird.|
| PAGE_FIELD_LABELS | Tabellenstilelement, das für die Seitenfeldbeschriftungen der Pivot-Tabelle gilt.|
| PAGE_FIELD_VALUES | Tabellenstilelement, das auf die Seitenfeldwerte der Pivot-Tabelle angewendet wird.|
| SECOND_COLUMN_STRIPE | Tabellenstilelement, das auf die Streifen der zweiten Spalte der Tabelle angewendet wird.|
| SECOND_COLUMN_SUBHEADING | Tabellenstilelement, das auf die Unterüberschrift der zweiten Spalte der Pivot-Tabelle angewendet wird.|
| SECOND_ROW_STRIPE | Tabellenstilelement, das für die Streifen in der zweiten Reihe der Tabelle gilt.|
| SECOND_ROW_SUBHEADING | Tabellenstilelement, das auf die Unterüberschrift der zweiten Zeile der Pivot-Tabelle angewendet wird.|
| SECOND_SUBTOTAL_COLUMN | Tabellenformatelement, das für die zweite Zwischensummenspalte von PivotTable gilt.|
| SECOND_SUBTOTAL_ROW |Tabellenformatelement, das für die zweite Zwischensummenzeile von PivotTable gilt.|
| THIRD_COLUMN_SUBHEADING | Tabellenformatelement, das für die Unterüberschrift der dritten Spalte von PivotTable gilt.|
| THIRD_ROW_SUBHEADING | Tabellenformatelement, das für die Unterüberschrift der dritten Zeile von PivotTable gilt.|
| THIRD_SUBTOTAL_COLUMN | Tabellenstilelement, das auf die dritte Zwischensummenspalte der Pivot-Tabelle angewendet wird.|
| THIRD_SUBTOTAL_ROW | Tabellenformatelement, das für die dritte Zwischensummenzeile von PivotTable gilt.|
| TOTAL_ROW | Tabellenstilelement, das für die Gesamtzeile der Tabelle gilt.|
| WHOLE_TABLE | Tabellenstilelement, das für den gesamten Inhalt der Tabelle gilt.|
