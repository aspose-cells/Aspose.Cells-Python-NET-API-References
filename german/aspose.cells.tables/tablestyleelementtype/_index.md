##  TableStyleElementType Aufzählung
Stellt den Stilelementtyp „Tabelle“ oder „PivotTable“ dar.



Der Typ TableStyleElementType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| BLANK_ROW | Tabellenstilelement, das auf die leeren Zeilen der PivotTable angewendet wird.|
| FIRST_COLUMN | Tabellenstilelement, das für die erste Spalte der Tabelle gilt.|
| FIRST_COLUMN_STRIPE | Tabellenstilelement, das auf die Streifen der ersten Spalte der Tabelle angewendet wird.|
| FIRST_COLUMN_SUBHEADING | Tabellenstilelement, das auf die Unterüberschrift der ersten Spalte der PivotTable angewendet wird.|
| FIRST_HEADER_CELL | Tabellenstilelement, das auf die erste Kopfzeilenzelle der Tabelle angewendet wird.|
| FIRST_ROW_STRIPE | Tabellenstilelement, das auf die Streifen der ersten Zeile der Tabelle angewendet wird.|
| FIRST_ROW_SUBHEADING | Tabellenstilelement, das auf die Unterüberschrift der ersten Zeile der PivotTable angewendet wird.|
| FIRST_SUBTOTAL_COLUMN |Tabellenstilelement, das auf die erste Zwischensummenspalte der PivotTable angewendet wird.|
| FIRST_SUBTOTAL_ROW | Tabellenstilelement, das auf die erste Zwischensummenzeile der Pivot-Tabelle angewendet wird.|
| GRAND_TOTAL_COLUMN | Tabellenstilelement, das auf die Gesamtsummenspalte der Pivot-Tabelle angewendet wird.|
| GRAND_TOTAL_ROW | Tabellenstilelement, das auf die Gesamtsummenzeile der Pivot-Tabelle angewendet wird.|
| FIRST_TOTAL_CELL | Tabellenstilelement, das auf die erste Gesamtzeilenzelle der Tabelle angewendet wird.|
| HEADER_ROW | Tabellenstilelement, das auf die Kopfzeile der Tabelle angewendet wird.|
| LAST_COLUMN | Tabellenstilelement, das auf die letzte Spalte der Tabelle angewendet wird.|
| LAST_HEADER_CELL | Tabellenstilelement, das auf die letzte Kopfzeilenzelle der Tabelle angewendet wird.|
| LAST_TOTAL_CELL | Tabellenstilelement, das auf die letzte Gesamtzeilenzelle der Tabelle angewendet wird.|
| PAGE_FIELD_LABELS | Tabellenstilelement, das auf die Seitenfeldbeschriftungen der Pivot-Tabelle angewendet wird.|
| PAGE_FIELD_VALUES | Tabellenstilelement, das auf die Seitenfeldwerte der Pivot-Tabelle angewendet wird.|
| SECOND_COLUMN_STRIPE | Tabellenstilelement, das auf die Streifen der zweiten Spalte der Tabelle angewendet wird.|
| SECOND_COLUMN_SUBHEADING | Tabellenstilelement, das auf die Unterüberschrift der zweiten Spalte der Pivot-Tabelle angewendet wird.|
| SECOND_ROW_STRIPE |Tabellenstilelement, das auf die Streifen der zweiten Tabellenzeile angewendet wird.|
| SECOND_ROW_SUBHEADING | Tabellenstilelement, das auf die Unterüberschrift der zweiten Zeile der Pivot-Tabelle angewendet wird.|
| SECOND_SUBTOTAL_COLUMN | Tabellenstilelement, das auf die zweite Zwischensummenspalte der PivotTable angewendet wird.|
| SECOND_SUBTOTAL_ROW | Tabellenstilelement, das auf die zweite Zwischensummenzeile der PivotTable angewendet wird.|
| THIRD_COLUMN_SUBHEADING | Tabellenstilelement, das auf die Unterüberschrift der dritten Spalte der PivotTable angewendet wird.|
| THIRD_ROW_SUBHEADING | Tabellenstilelement, das auf die Unterüberschrift der dritten Zeile der PivotTable angewendet wird.|
| THIRD_SUBTOTAL_COLUMN | Tabellenstilelement, das auf die dritte Zwischensummenspalte der Pivot-Tabelle angewendet wird.|
| THIRD_SUBTOTAL_ROW | Tabellenstilelement, das auf die dritte Zwischensummenzeile der PivotTable angewendet wird.|
| TOTAL_ROW | Tabellenstilelement, das auf die Gesamtzeile der Tabelle angewendet wird.|
| WHOLE_TABLE | Tabellenstilelement, das für den gesamten Inhalt der Tabelle gilt.|
