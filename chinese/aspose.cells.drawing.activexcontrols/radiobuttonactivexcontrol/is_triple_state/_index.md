## is_triple_state 属性

指示指定的控件将如何显示 Null 值。

### 评论

|环境|描述|
| :- | :- |
|真的|控件将循环显示 Yes、No 和 Null 值的状态。当控件的 Value 属性设置为 Null 时，该控件显示为灰色（灰色）。|
|错误的|（默认）控件将循环显示 Yes 和 No 值的状态。 Null 值显示为 No 值。|
