## JsonExportHyperlinkType枚举
表示将超链接导出到 json 的类型。



JsonExportHyperlinkType 类型公开了以下成员：

### 领域
|场地|描述|
| :- | :- |
| DISPLAY_STRING |导出显示字符串|
| ADDRESS |导出地址|
| HTML_STRING |导出为 html 字符串。|
