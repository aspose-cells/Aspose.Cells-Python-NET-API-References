## CustomXmlPart类
表示自定义 XML 数据存储部件（包内的自定义 XML 数据）。



CustomXmlPart 类型公开了以下成员：

### 特性
|属性|描述|
| :- | :- |
| [data](/cells/python-net/zh/aspose.cells.markup/customxmlpart/data) |获取或设置此自定义 XML 数据存储部件的 XML 内容。|
| [schema_data](/cells/python-net/zh/aspose.cells.markup/customxmlpart/schema_data) |获取或设置此自定义 XML 架构数据存储部件的 XML 内容。|
| [id](/cells/python-net/zh/aspose.cells.markup/customxmlpart/id) |获取和设置自定义 xml 部件的 ID。|
