## PropertyType枚举
指定文档属性的数据类型。



PropertyType 类型公开以下成员：

### 字段
|字段|描述|
| :- | :- |
| BOOLEAN |该属性是一个布尔值。|
| DATE_TIME |该属性是一个日期时间值。|
| DOUBLE |该属性是一个浮点数。|
| NUMBER |该属性是一个整数。|
| STRING |该属性是一个字符串值。|
| BLOB |该属性是一个字节数组。|
