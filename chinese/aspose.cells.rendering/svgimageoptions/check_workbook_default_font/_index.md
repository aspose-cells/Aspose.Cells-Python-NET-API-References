## check_workbook_default_font属性

当 Excel 中的字符是 Unicode 且未在单元格样式中设置正确的字体时，
它们可能在 pdf、图像中以块的形式出现。
将其设置为 true 以尝试首先使用工作簿的默认字体显示这些字符。

### 注意事项

默认为真。
