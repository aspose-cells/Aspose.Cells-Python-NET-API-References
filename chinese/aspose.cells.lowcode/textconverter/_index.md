## TextConverter类
用于在基于文本的格式（csv、tsv、dif...）和其他电子表格文件格式之间进行转换的转换器。



TextConverter 类型公开以下成员：

### 方法
|方法|描述|
| :- | :- |
| [`process(, template_file, result_file)`](/cells/python-net/zh/aspose.cells.lowcode/textconverter/process/#str-str) |在基于文本的文件和其他格式之间转换给定的模板文件。|
| [`process(, load_options, save_options)`](/cells/python-net/zh/aspose.cells.lowcode/textconverter/process/#aspose.cells.lowcode.lowcodeloadoptions-aspose.cells.lowcode.lowcodesaveoptions) |在基于文本的格式和其他电子表格文件格式之间转换文件格式|



### 例子

```python
from aspose.cells.lowcode import TextConverter

TextConverter.process("template.csv", "res.xlsx")

```
