## ReserveMissingPivotItemType枚举
表示如何保留缺失的枢轴项。



ReserveMissingPivotItemType 类型公开以下成员：

### 字段
|字段|描述|
| :- | :- |
| DEFAULT |删除旧的 missint 数据透视表项，并保留当前数据源不包含的可见项作为缺失项。|
| ALL |保留所有缺失的物品。|
| NONE |删除所有缺失的枢轴项目。|
