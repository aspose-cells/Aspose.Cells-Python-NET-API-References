## base_item_position_type属性

表示使用 ShowDataAs 计算时基本字段中基本数据透视表项的类型。
仅对数据字段有效。
因为 PivotItemPosition.Custom 仅供读取，如果需要设置 PivotItemPosition.Custom，
请设置 PivotField.BaseItemIndex 属性。
