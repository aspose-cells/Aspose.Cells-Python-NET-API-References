## width_in_shape 属性

表示形状的宽度，以父形状的 1/4000 为单位。

### 评论

仅当此形状位于组或图表中时适用。

### 例子

```python

if shape.upper_delta_y == 2000:
    shape.upper_delta_y = 4000

```
