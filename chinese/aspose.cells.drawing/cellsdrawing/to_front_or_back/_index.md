##  to_front_or_back(self, orders) {#int}
将形状置于前面或将形状置于后面。



```python

def to_front_or_back(self, orders):
    ...
```


|范围|类型|描述|
| :- | :- | :- |
| orders | int |如果小于零，则将形状设置为背面。<br/>如果大于零，则将形状置于前面。|

### 例子

```python

shape.to_front_or_back(2)
```
