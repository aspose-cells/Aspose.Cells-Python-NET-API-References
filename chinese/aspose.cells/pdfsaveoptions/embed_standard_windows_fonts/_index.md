## embed_standard_windows_fonts 属性

True 以嵌入 True Type 字体。
仅影响 ASCII 字符 32-127。
字符代码大于 127 的字体总是被嵌入。
始终嵌入 PDF/A-1a、PDF/A-1b 标准的字体。
默认为真。
