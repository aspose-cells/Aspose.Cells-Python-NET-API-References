##  get_param_value(index) {#int}
获取给定索引处参数的表示值对象。


### 返回

如果参数是普通值，则返回普通值。
如果参数是引用，则返回 ReferredArea 对象。
如果参数引用多个数据集，则返回对象数组。


```python
def get_param_value(self, index):
    ...
```


|范围|类型|描述|
| :- | :- | :- |
| index | int |参数索引（基于0）|
