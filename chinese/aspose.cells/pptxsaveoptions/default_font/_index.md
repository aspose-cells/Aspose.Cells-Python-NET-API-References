## default_font 属性

当Excel中的字符是Unicode且未在单元格样式中设置正确的字体时，
它们可能在 pdf、图像中显示为块。
设置 DefaultFont，例如 MingLiu 或 MS Gothic 以显示这些字符。
如果不设置该属性，Aspose.Cells 将使用系统默认字体来显示这些 unicode 字符。
