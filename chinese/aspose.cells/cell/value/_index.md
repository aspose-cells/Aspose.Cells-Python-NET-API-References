## value属性

获取/设置此单元格包含的 value。

### 注意事项

可能的类型：

无效的，


布尔值，


日期时间，


双倍的，


整数


细绳。


对于 int value，它可能作为 Integer 对象或 Double 对象返回。
并且无法保证返回的 value 将始终保持为同一类型的对象。
### 定义：
```python
@property
def value(self):
    ...
@value.setter
def value(self, value):
    ...
```
