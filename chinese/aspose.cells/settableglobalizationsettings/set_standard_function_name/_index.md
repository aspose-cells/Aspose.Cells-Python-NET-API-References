##  set_standard_function_name {#str-str-bool}
根据给定的标准函数名称设置与区域设置相关的函数名称。



```python
def set_standard_function_name(self, local_name, standard_name, bidirectional):
    ...
```


|范围|类型|描述|
| :- | :- | :- |
| local_name | str |与区域设置相关的函数名称|
| standard_name | str |标准（en-US 区域设置）函数名称。|
| bidirectional | bool |是否自动将标准函数名映射到本地函数名。<br/>如果为 true，标准名称将自动映射到本地名称<br/>所以用户不需要再次拨打[`SettableGlobalizationSettings.set_local_function_name`](/cells/python-net/zh/aspose.cells/settableglobalizationsettings/set_local_function_name)<br/>对于相同的标准和本地名称对|
