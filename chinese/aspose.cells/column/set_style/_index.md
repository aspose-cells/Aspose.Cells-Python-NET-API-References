##  set_style(self, style) {#aspose.cells.Style}
设置此列的样式。



```python

def set_style(self, style):
    ...
```


|范围|类型|描述|
| :- | :- | :- |
| style | [`Style`](/cells/python-net/zh/aspose.cells/style) |该样式将用作此列中单元格的默认样式。|
### 注意事项

此方法仅将给定的样式设置为此列的默认样式，
而不改变此列中现有单元格的样式设置。
要同时将现有单元格的样式设置更新为指定的样式，
请使用 [`Column.apply_style`](/cells/python-net/zh/aspose.cells/column/apply_style)
