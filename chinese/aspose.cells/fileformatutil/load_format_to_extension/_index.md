##  load_format_to_extension（，加载格式）{#aspose.cells.LoadFormat}

### 返回

返回的扩展名是一个以点开头的小写字符串。


```python

@staticmethod
def load_format_to_extension(load_format):
    ...
```


|范围|类型|描述|
| :- | :- | :- |
| load_format | [`LoadFormat`](/cells/python-net/zh/aspose.cells/loadformat) |已加载的文件格式。|
### 注意事项
