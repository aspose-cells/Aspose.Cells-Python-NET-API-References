## show_tabs属性

获取或设置是否显示工作簿选项卡的值。

### 注意事项

默认值为 true。

### 例子

以下代码隐藏了电子表格的工作表选项卡和选项卡滚动按钮。

```python

#  Hide the spreadsheet tabs.
workbook.settings.show_tabs = False

```
