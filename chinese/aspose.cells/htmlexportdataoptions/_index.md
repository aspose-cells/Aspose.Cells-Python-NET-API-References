## HtmlExportDataOptions枚举
代表导出 html 数据的选项。



HtmlExportDataOptions 类型公开以下成员：

### 领域
|场地|描述|
| :- | :- |
| TABLE |将文件导出为仅包含表格部分的 html。|
| ALL |将所有数据导出为 html。|
