## format_data_ignore_column_width属性

指示溢出列时是否显示单元格的全部格式化数据。
如果为 true，则忽略列宽，并且将导出单元格的整个数据。
如果为 false，则数据将以与 Excel 相同的方式导出。
默认值为 false。
