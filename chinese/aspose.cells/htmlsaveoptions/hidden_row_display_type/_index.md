## hidden_row_display_type属性

excel 中的隐藏行（此行的高度为 0），在将其保存为 html 格式之前，
如果HtmlHiddenRowDisplayType为“Remove”，则隐藏行将不会输出，
如果值为“Hidden”，则该行会被输出，但被隐藏，默认值为“Hidden”
