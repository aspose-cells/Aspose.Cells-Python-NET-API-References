## std_dev 属性

获取或设置标准差的数量，以包括高于或低于平均值的标准差
条件格式规则。
输入值必须在 0 到 3 之间（包括 0 和 3）。
将此值设置为 0 意味着未设置 stdDev。
默认值为 0。
