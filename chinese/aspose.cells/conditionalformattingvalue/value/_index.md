## value属性

获取或设置此条件格式 value 对象的 value。
它应该与 Type 一起使用。

### 注意事项

如果 value 是字符串并以“=”开头，它将被处理为公式，
否则我们将把它处理为简单的 value。
### 定义：
```python
@property
def value(self):
    ...
@value.setter
def value(self, value):
    ...
```
