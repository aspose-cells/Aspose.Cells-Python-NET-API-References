## value 属性

获取或设置此条件格式 value 对象的 value。
它应该与类型一起使用。

### 评论

如果value为字符串，以“=”开头，则按公式处理，
否则我们会将其作为简单的 value 处理。

```python

def value(self):
    ...
@value.setter
def value(self, value):
    ...
```
