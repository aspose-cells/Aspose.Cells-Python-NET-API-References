## check_mixed_value_type 属性

False，Aspose.Cells 将根据第一行的值类型设置 DataColumn 的类型以提高性能。
 True, Aspose.Cells 会在设置DataColumn的类型前检查列中的值类型是否混合
和值类型混合，DataColumn 的类型将是字符串。
