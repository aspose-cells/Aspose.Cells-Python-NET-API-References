## HtmlFormatHandlingType枚举
指定如何处理来自 HTML 源的格式



HtmlFormatHandlingType 类型公开了以下成员：

### 领域
|场地|描述|
| :- | :- |
| ALL |将所有 HTML 格式连同数据一起传输到工作表中。|
| NONE |将数据作为无格式文本引入（设置数据类型仍然发生）。|
| RTF |将带入工作表的数据的 HTML 格式转换为富文本格式。|
