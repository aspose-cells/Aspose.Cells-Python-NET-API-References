## HtmlFormatHandlingType枚举
指定如何处理来自 HTML 源的格式



HtmlFormatHandlingType 类型公开以下成员：

### 字段
|字段|描述|
| :- | :- |
| ALL |将所有 HTML 格式与数据一起传输到工作表中。|
| NONE |将数据作为未格式化的文本引入（仍然会设置数据类型）。|
| RTF |将 HTML 格式转换为工作表中数据的富文本格式。|
