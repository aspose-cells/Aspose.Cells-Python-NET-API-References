## only_use_connection_file 属性

指示电子表格应用程序是否应始终且仅使用
odcFile 属性指示的外部连接文件中的连接信息
刷新连接时。如果为假，则电子表格应用程序
应遵循 reconnectionMethod 属性指示的过程
