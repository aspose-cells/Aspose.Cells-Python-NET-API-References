## OLEDBCommandType枚举
指定 OLE DB 命令类型。



OLEDBCommandType 类型公开以下成员：

### 字段
|字段|描述|
| :- | :- |
| NONE |未指定命令类型。|
| CUBE_NAME |指定多维数据集名称|
| SQL_STATEMENT |指定 SQL 语句|
| TABLE_NAME |指定表名|
| DEFAULT_INFORMATION |指定已给出默认信息，由提供商决定如何解释。|
| WEB_BASED_LIST |指定针对基于 Web 的列表数据提供程序的查询。|
| TABLE_COLLECTION |指定表列表。|
