## command_type 属性

指定 OLE DB 命令类型。
1. 查询指定一个立方体名称
2.Query指定一条SQL语句
3.查询指定表名
4. Query指定已经给出了默认信息，由provider如何解读。
5. 查询针对基于 Web 的列表数据提供程序。
