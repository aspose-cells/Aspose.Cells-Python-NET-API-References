## odc_file 属性

指定此连接来自的外部连接文件的完整路径
创建。如果在尝试刷新数据时连接失败，并且 reconnectionMethod=1，
然后电子表格应用程序将使用来自外部连接文件的信息重试
而不是工作簿中嵌入的连接对象。
