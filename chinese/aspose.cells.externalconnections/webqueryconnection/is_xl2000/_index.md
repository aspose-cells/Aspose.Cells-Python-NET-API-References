## is_xl2000 属性

此标志的存在是为了向后兼容旧的现有电子表格文件，并已设置
如果此 Web 查询在更新或等于的电子表格应用程序中刷新，则为真
至 Microsoft Excel 2000。
这是一个可以忽略的可选属性。
