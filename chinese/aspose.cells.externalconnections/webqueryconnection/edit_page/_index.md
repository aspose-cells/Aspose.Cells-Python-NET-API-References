## edit_page属性

显示 Web 查询数据的面向用户的网页的 URL。此 URL 会持久保存
在 sourceData="true" 和 url 已被重定向以引用 XML 文件的情况下。
然后，可以在 UI 中显示面向用户的页面，并且可以检索 XML 数据
幕后。

### 注意事项

注意：此属性现已过时。取而代之的是
请使用 WebQueryConnection.EditWebPage 属性。
该房产将于2017年10月起12个月后被拆除。
Aspose 对于您所遇到的不便深表歉意。
