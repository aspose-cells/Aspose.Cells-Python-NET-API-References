## edit_page 属性

显示网络查询数据的面向用户的网页的 URL。此网址已保存
在 sourceData="true" 和 url 已被重定向以引用 XML 文件的情况下。
然后可以在 UI 中显示面向用户的页面，并且可以检索 XML 数据
在幕后。

### 评论

注意：此属性现在已过时。反而，
请使用 WebQueryConnection.EditWebPage 属性。
自 2017 年 10 月起，此属性将在 12 个月后移除。
Aspose 对您可能遇到的任何不便深表歉意。
