## merge_empty_td_forcely属性

指定导出文件为html时是否强制合并空的TD元素。
设置为 true 后，html 文件的大小会明显减小，默认值为 false。
如果您想将 html 文件导入到 excel 或在将文件保存为 html 时导出完美的网格线，
请保留默认值。
