## html_cross_string_type属性

指示在以 html 格式保存 Excel 文件时跨单元格字符串是否以与 MS Excel 相同的方式显示。
默认情况下该值为默认值，因此，对于跨单元格字符串，Aspose.Cells 和 MS Excel 创建的 html 文件之间几乎没有区别。
但是对于创建大型 html 文件的性能，将值设置为 Cross 会比将其设置为 Default 或 Fit2Cell 快几倍。
