## Config类

代表GridJs的所有设置



Config 类型公开以下成员：

### 构造函数
|构造函数|描述|
| :- | :- |
| [__init__](/cells/python-net/zh/aspose.cellsgridjs/config/__init__/#) |构造一个新的 Config 实例|


### 属性
|属性|描述|
| :- | :- |
| [save_html_as_zip](/cells/python-net/zh/aspose.cellsgridjs/config/save_html_as_zip) |设置/获取是否将 html 文件保存为 zip 存档，默认为 false。|
| [same_image_detecting](/cells/python-net/zh/aspose.cellsgridjs/config/same_image_detecting) |
<br/>默认值是true。 |
| [auto_optimize_for_large_cells](/cells/python-net/zh/aspose.cellsgridjs/config/auto_optimize_for_large_cells) |
<br/>忽略一些样式/边框以减少加载时间
<br/>默认值是true。 |
| [islimit_shape_or_image](/cells/python-net/zh/aspose.cellsgridjs/config/islimit_shape_or_image) |
<br/>GridJs 会将一个工作表内的总显示形状/图像大小限制为 MaxShapeOrImageCount
<br/>默认值是true。 |
| [max_shape_or_image_count](/cells/python-net/zh/aspose.cellsgridjs/config/max_shape_or_image_count) |
<br/>默认值为 100。
| [max_total_shape_or_image_count](/cells/python-net/zh/aspose.cellsgridjs/config/max_total_shape_or_image_count) |
<br/>默认值为 300。
| [max_shape_or_image_width_or_height](/cells/python-net/zh/aspose.cellsgridjs/config/max_shape_or_image_width_or_height) |
<br/>默认值为 10000。
| [max_pdf_save_seconds](/cells/python-net/zh/aspose.cellsgridjs/config/max_pdf_save_seconds) |
<br/>默认值为 10。
| [ignore_empty_content](/cells/python-net/zh/aspose.cellsgridjs/config/ignore_empty_content) |
<br/>如果最后一行或最后一列包含没有值和公式但具有自定义样式的单元格
<br/>那么当这个vlaue为true时我们将不会显示该行/列。
<br/>默认值是true 。 |
| [use_print_area](/cells/python-net/zh/aspose.cellsgridjs/config/use_print_area) |
<br/>默认值为 false 。 |
| [load_time_out](/cells/python-net/zh/aspose.cellsgridjs/config/load_time_out) |
<br/>默认值为-1，即不设置超时中断。 |
| [show_chart_sheet](/cells/python-net/zh/aspose.cellsgridjs/config/show_chart_sheet) |
<br/>默认值为 false 。 |
| [page_size](/cells/python-net/zh/aspose.cellsgridjs/config/page_size) |
<br/>GridJs会根据PageSize限制行大小，如果PageSize为-1，则不会进行分页
<br/>默认值为-1 |
| [empty_sheet_max_row](/cells/python-net/zh/aspose.cellsgridjs/config/empty_sheet_max_row) |
<br/>默认值为 12。
| [empty_sheet_max_col](/cells/python-net/zh/aspose.cellsgridjs/config/empty_sheet_max_col) |
<br/>默认值为 15。
| [picture_cache_directory](/cells/python-net/zh/aspose.cellsgridjs/config/picture_cache_directory) |
<br/>默认路径是 FileCacheDirectory 内的“_piccache”。 |
| [file_cache_directory](/cells/python-net/zh/aspose.cellsgridjs/config/file_cache_directory) |
<br/>在使用GridJs之前，我们需要将其设置为特定路径。 |
