##  get_value {#int-int}

获取距该区域左上角给定偏移量的单元格值。


### 返回


“#REF！”如果该区域无效；
如果给定的偏移量超出该区域，则为“#N/A”；
否则返回给定位置的单元格值。


```python
def get_value(self, row_offset, col_offset):
    ...
```


|范围|类型|描述|
| :- | :- | :- |
| row_offset | int |距该区域起始行的行偏移量|
| col_offset | int |列距该区域起始行的偏移量|
